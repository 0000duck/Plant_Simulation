Maximum number of samples
Enter the Maximum number of samples, which Plant Simulation draws for probability distributions, to prevent it from spending an excessive amount of time for throwing the dice.
You can enter a number between 1 and 32000. Plant Simulation keeps track of values outside of the upper and lower bounds which you enter to prevent values outside this range. For a normal distribution it may happen that Plant Simulation draws large values. When the bounds are close together, this may result in many values located outside of the range. This way Plant Simulation has to draw a large number of samples before a valid value is available, which may take up a lot of time.
