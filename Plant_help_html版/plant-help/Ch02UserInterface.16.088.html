Preferences
The command Preferences opens the Preferences in which you can select global settings for Plant Simulation.
•
On the tabs Modeling, User Interface, Editor, 3D, and License you can select general settings, which are independent of the simulation model.
•
On the tabs Simulation and Units you can select model-specific settings, which apply to new models. The tab General shows both types of settings.
Click the tabs to access the respective set of features.
•
Click OK to apply your entries and save the new configuration to the registry under HKEY_CURRENT_USER for each user separately.
•
Click Cancel to close the window without saving any changes.
The Model Settings only apply to the current simulation model!
