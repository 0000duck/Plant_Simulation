OpenGL Version
Select the OpenGL version you would like to use for rendering the 3D scenes in your simulation models. When the 3D Viewer is running while the dialog Preferences is opened, the drop-down list only shows the OpenGL versions that the graphics card driver of the graphics card of your computer supports.
You can check out the features of the individual OpenGL versions on the Internet.
