Open Class
The command Open Class opens the window of the class of the selected object. The class object is located in the Class Library.
Suppose that we derived Frame B from Frame A, which we derived from the root Frame Frame in the Class Library. Then the Frame is the class of Frame A and of Frame B.
When you select the command Open Class for an instance of an object, Plant Simulation moves up in the inheritance structure until it reaches an object in the Class Library. It then opens the window of this object.
•
When you are working with a Frame, the command opens the window of the Frame class in the Class Library.
•
When you are working with a Method, the command opens the window of the Method class in the Class Library.
•
When you are working with a List, the command opens the window of the respective List class in the Class Library.
Instead, you can also click the object with the right mouse button and click the button on the mini toolbar above the context menu.
When you hold down the Shift key while clicking the button, Plant Simulation shows the location of the class of the object.
