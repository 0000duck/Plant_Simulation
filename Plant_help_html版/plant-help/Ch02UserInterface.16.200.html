Open Frame in 2D/3D
The command Open Frame in 3D opens the window of the counterpart of the current object in the 3D Viewer or in Plant Simulation 2D respectively.
•
When you are working with a Frame, the command opens the window of that Frame in the 3D Viewer or vice versa.
•
When you are working with an object, the command opens the dialog window of that object in the 3D Viewer or vice versa.
