Auto graphics active
To activate auto graphics mode, select this, and click OK or Apply. Plant Simulation then uses the color of the 2D vector graphic, which you can activate with the check box Vector graphics active and set on the Tab Graphics. In this case the 3D Viewer uses the color of the vector graphic as the Diffuse color.
To define a color of your own with the settings of the tab, select Material active and then select the settings of your choice.
