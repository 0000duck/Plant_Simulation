Connector positioning
Select on which side of the object the program places the starting point or the end point of Connectors:
•
Fixed side places the starting point of the Connector at the exit of the object and the end point at its entrance.
•
Angle-dependent takes the angle between the objects into account when determining the starting point or the end point of the Connector.
