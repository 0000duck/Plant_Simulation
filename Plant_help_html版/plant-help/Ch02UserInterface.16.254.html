Configuration
The toggle button activates or deactivates inheritance of the configuration settings of the conveyor. To deactivate inheritance, click the button so that it looks like this .
•
A green button means that inheritance is active. Then the object inherits, i.e., uses, the polycurve settings of the parent object from which it was derived. When you modify the polycurve settings of the parent object, the program also modifies the polycurve settings of the child object.
•
An orange button with a minus means that inheritance is not active.
You can:
•
Select the Type of conveyor and the settings that pertain to that conveyor.
•
Select settings for the Legs of the conveyor.
•
Select settings for the Lanes of the conveyor.
•
Select settings for the Frame of the conveyor.
To edit extrusion profiles of the selected length-oriented object, click Advanced.
