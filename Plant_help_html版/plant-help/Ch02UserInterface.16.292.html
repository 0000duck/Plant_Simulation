Create Rotation Path
To create a new rotation path for the selected object, click this.
The 3D Viewer opens the dialog Create Rotation Path. Here you can define how the 3D Viewer rotates the mobile object that the selected object processes on the path, which you create.
•
Select the Start angle for the rotation in degrees, the End angle in degrees, and the Radius of the entire rotation under Span.
•
Select the Axis X/Y/Z around which the animated object rotates.
•
Select the Center around which the animated object rotates.
•
Click OK to accept or Cancel to discard these settings.
