License
On the tab License you can select the license type you purchased and you can select how Plant Simulation acquires this license. Plant Simulation supports a license file or three redundant license servers for retrieving your license. Redundant license servers significantly increase the stability and the availability of the licenses.
When you are using a license server triad, two out of the three redundant license servers always have to be reachable, meaning that one server can fail.
Activating Licenses describes how to go about in setting up and activating the licenses.
