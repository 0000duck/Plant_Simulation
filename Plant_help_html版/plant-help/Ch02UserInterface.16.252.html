Base height
Enter the distance of the selected object from the floor.
The base height can be inherited, meaning that you can define it once in the class instead of having to define it for each instance individually. The z-coordinate defines the floor of the installation and the base height defines the distance of the legs from the floor. By changing the setting ΔZ in the Segments table, you can add an additional offset. The setting ΔZ also affects the simulation time in Plant Simulation 2D.
