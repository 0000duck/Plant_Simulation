Rename
The command Rename opens the dialog Rename, where you change the name and the label of the selected model or of the selected object.
•
Enter a new Name for the simulation model or object.
•
Enter a new Label.
•
To accept the new name and close the dialog window, click OK.
•
To accept the changes, and keep the dialog window open, click Apply.
•
To discard the change, click Cancel.
Instead, you can also right-click the object in the Frame and select Rename on the context menu.
