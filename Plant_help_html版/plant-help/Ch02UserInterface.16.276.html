Edit
To edit the anchor points of the path you selected, click this.
You can:
•
Add a new anchor point.
•
Click Insert Before to insert a new anchor point before the selected anchor point.
•
Delete the selected anchor point.
•
Click Move Up to move the selected anchor point up one position in the list up. Click Move Down to move it one position down in the list.
•
Duplicate the selected anchor point.
•
Click Edit Values to edit the values of the selected anchor point or double-click the row of the anchor point, whose values you want to change, in the list.
•
Add Common Offset to add a common offset to the selected anchor point.
•
Hide or visualize the selected path by clicking the cell under animation paths.
