Test path
The group box Test path provides settings and actions for testing the animation path of the selected object. When you start testing the 3D Viewer creates a temporary test object which moves on the selected animation path.
You can:
•
Enter the Factor with which the test object moves on the animation path.
•
Select if the test object moves in the forward direction or if it moves Backwards.
•
Play the animation with the settings you selected.
•
Pause the animation.
•
Stop the animation.
The 3D Viewer does not save these settings to the model file.
