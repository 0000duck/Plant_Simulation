Event Controller
The command Event Controller provides these commands:
•
Reset Simulation
•
Start/Stop Simulation
•
Start Fast Forward Simulation
•
Single Step Simulation
•
Debug Next Simulation Step
•
Real-Time
•
Simulation Speed
Instead of clicking the buttons described below, you can also click the corresponding buttons in the dialog of the EventController.
