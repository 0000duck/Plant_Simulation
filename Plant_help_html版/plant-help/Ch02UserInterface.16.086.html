Show states as colors if no state graphics are present
To show the external representation of simulation objects in different colors when no state group with individual state graphics for the different states exists, select the check box. Then the 3D Viewer automatically colors the visible graphics of the object red for failed, blue for paused or yellow for waiting, when the object is in the respective state during the simulation.
When you clear the check box, the 3D Viewer immediately removes all state colors.
