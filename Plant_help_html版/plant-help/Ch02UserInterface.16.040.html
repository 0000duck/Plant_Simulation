Model language
Select the language of the currently open simulation model. Plant Simulation saves this setting to the model file.
The language of the operating system of your computer determines the language of the Plant Simulation user interface.
When you create a new simulation model, the model language determines:
•
The names of folders and objects, such as MaterialFlow or Materialfluss, SingleProc or Einzelstation, etc.
•
The items of the drop-down lists, such as the names of the distributions (Uniform or Gleich).
•
The return value of the attributes that correspond to the entries of the drop-down lists. If you query the attribute ResourceType in SimTalk, the program will return the value in the model language "Storage" in English, "Lagerung" in German.
•
The contents of the window Show Attributes and Methods.
•
The names, which the feature Auto Complete in the method editor suggests.
