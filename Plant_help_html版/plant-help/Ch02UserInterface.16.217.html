Settings for the Position
Enter a value into the text box, or click the spin buttons to move the position of the selected object on the x-axis, the y-axis and the z-axis. The settings for the three axes are independent of each other.
For MU instances the settings for the position show the local positions calculated by the animation.
The toggle button activates or deactivates inheritance of the position .
•
A green button means that inheritance is active. Then the object inherits, i.e., uses, the position of the parent object from which it was derived. When you modify the position of the parent object, the program also modifies the position of the child object.
•
An orange button with a minus means that inheritance is not active.
