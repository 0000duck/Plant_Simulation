Advanced
To edit extrusion profiles of the selected length-oriented object, click Advanced.
To create a new extrusion profile, click New. Then enter a name of the path. The 3D Viewer adds this path to the bottom of the list next to profile name.
To rename the selected extrusion profile, click Rename. Enter the new name into the dialog Rename.
To delete the selected extrusion profile, click Delete.
To edit the selected extrusion profile, click Edit. Then, edit the settings in the dialog that opens. The settings you can select here are described under Editing an animation path.
