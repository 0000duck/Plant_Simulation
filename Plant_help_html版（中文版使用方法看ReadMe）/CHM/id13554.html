Exit a Loop with exitLoop
SimTalk Reference > General SimTalk Functions > Control Flow Statements > Loops > Exit a Loop with exitLoop

You can use the keyword exitLoop to exit a loop. Optionally, you can set the number of loops to be exited by entering an integer number: exitloop 2-- two loops.

Example:

 
for var y := 1 to MyTableFile.yDim 
   for var x := 1 to MyTableFile.xDim 
      if MyTableFile[x,y] ~= "looking for this" 
         exitloop  -- exits the inner loop
      end
   next
next
print x, " ", y

Example:

 
var i: integer
while i  100 
   for i := i to i+10 
      repeat
         print i
         if i = 20 
            exitloop 2 -- exits the repeat-loop and the for-loop
         end
      until i > 5
    next
end
