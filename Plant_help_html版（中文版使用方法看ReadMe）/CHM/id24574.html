_3D.ExtConfiguration

Syntax:

 
Path._3D.ExtConfiguration

Parameters of the Extrusion Configuration

The parameters of the extrusion configuration designate the following settings:

  • Index Name: Is the name of the extrusion group, it is only entered once for each group in the first column of the group, i.e., in column 0.

  • Extrusion Profile: The sub-table [2 x number of points], which you can open by double-clicking in the cell Extrusion Profile , specifies the profile that will be extruded according to the settings below.

  • Configuration Type: You can set the following values:

    Column/Data Type of the Value You Can Assign in

    Meaning

    Name

    Is the name of the profile. For new profiles this is the name you entered as the path name.

    Type

    Select the profile type of the selected profile.

    The profile types are described in the table below.

    Profile

    Click Edit to edit the selected profile.

    Material

    In the sub-table Material, which you can open by double-clicking in the cell Material, you can set the following:

    The Diffuse color, compare makeRGBValue.

    The Ambient color.

    The Specular color.

    The Emissive color.

    The Transparency.

    The Shininess.

    Start Offset

    Is the offset of the start reference in the extrusion direction.

    End Offset

    Is the offset of the end reference in the extrusion direction.

    Repetition Interval

    For the configuration types 14, 15, 16, 17, 18, 19, 22, and 23 the value 0 means that the extrusion is being generated once at the start point of the object. A value greater than 0 means that the extrusion is generated every number of meters that you enter. If you enter 0.5, the extrusion will be generated every 50 cm along the path. For the other values, this values is meaningless and the program only accepts 0.

    Graphic Group ID

    By default this value is 0. A value between 0 and 10 generates additional graphic groups into which the program places these thus marked extrusions. The Fluid object Pipe for example uses this feature.

    Name/Value for Assignment

    Description

    Path, z max, Move abs/0

    Extrude once along the path with z as defined in the anchors, the distance from the edge remains fixed when changing the width of the conveyor.

    Path, z max, Move rel/1

    Extrude once along the path with z as defined in the anchors and pin the profile base point to its relative position on width change.

    Path, z max, Scale x/2

    Extrude once along the path with z as defined in the anchors and scale in the x-direction when changing the width.

    Path, z max, Scale xy/3

    Extrude once along the path with z as defined in the anchors and scale in the x-direction and the y-direction when changing the width.

    Path, Floor, Move abs/4

    Extrude once along the path with z as defined in the anchors, the distance from the edge remains fixed when changing the width of the conveyor.

    Path, Floor, Move rel5

    Extrude once along the path with z = 0.0 and pin to the borders on width change.

    Path, Floor, Scale x/6

    Extrude once along the path with z = 0.0 and scale in the x-direction when changing the width.

    Path, Floor, Scale xy/7

    Extrude once along the path with z = 0.0 and scale in the x-direction and the y-direction when changing the width.

    Horizontal, Anchors, z max, Move/8

    Extrude horizontally with z as defined in the anchors for every anchor point and pin to the borders on width change. The graphic tilts according to the incline.

    Horizontal, Anchors, z max, Scale/9

    Extrude horizontally with z as defined in the anchors for every anchor point and pin to the center on width change. The graphic tilts according to the incline.

    Horizontal, Anchors, z 1/5, Move/10

    Extrude horizontally with z = 1/5 as defined in the anchors for every anchor point and pin to the borders on width change.

    Horizontal, Anchors, z 1/5, Scale/11

    Extrude horizontally with z = 0.0 for every anchor point and pin to the borders on width change.

    Horizontal, Anchors, Floor, Move/12

    Extrude horizontally with z = 0.0 for every anchor point and pin to the center on width change.

    Horizontal, Anchors, Floor, Scale/13

    Extrude horizontally with z as defined in the anchors with a fixed interval and pin to the borders on width change.

    Horizontal, Interval, z max, Move/14

    Extrude horizontally with z as defined in the anchors with a fixed interval and pin to the center on width change. The graphic tilts according to the incline.

    Horizontal, Interval, z max, Scale/15

    Extrude horizontally with z as defined in the anchors with a fixed interval and pin to the center on width change. The graphic tilts according to the incline.

    Extrude horizontally with z = 1/5 as defined in the anchors with a fixed interval and pin to the center on width change.

    Horizontal, Interval, z 1/5, Scale/17

    Extrude horizontally with z = 1/5 as defined in the anchors with a fixed interval and pin to the center on width change.

    Horizontal, Interval, Floor, Move/18

    Extrude horizontally with z = 0.0 with a fixed interval and pin to the borders on width change.

    Horizontal, Interval, Floor, Scale/19

    Extrude horizontally with z = 0.0 with a fixed interval and pin to the center on width change.

    Vertical, Floor, Anchors, Move/20

    Extrude vertically from the path anchor to the floor for every anchor point and pin to the borders on width change.

    Vertical, Floor, Anchors, Fix/21

    Extrude vertically from the path anchor to the floor for every anchor point and pin to the center on width change.

    Vertical, Interval, Move/22

    Extrude vertically from the path anchor to the floor with a fixed interval and pin to the borders on width change.

    Vertical, Interval, Fix/23

    Extrude vertically from the path anchor to the floor with a fixed interval and pin to the center on width change.

    Forward, z max, Anchors, Move abs/24

    Repeatedly extrude a small segment in the path direction for each anchor point, the distance from the borders remains fixed when changing the width of the conveyor.

    Forward, z max, Anchors, Move rel/25

    Repeatedly extrude a small segment in the path direction for each anchor point, and pin the base point of the profile to its relative position when changing the width.

    Forward, z max, Anchors, Fix/26

    Repeatedly extrude a small segment in the path direction for each anchor point, the distance from the path remains fixed when changing the width of the conveyor.

    Forward, z max, Interval, Move abs/27

    Repeatedly extrude a small segment in the path direction with a fixed distance, the distance from the borders remains fixed when changing the width of the conveyor.

    Forward, z max, Interval, Move rel/28

    Repeatedly extrude a small segment in the path direction with a fixed distance, and pin the base point of the profile to its relative position when changing the width.

    Forward, z max, Interval, Fix/29

    Repeatedly extrude a small segment in the path direction with a fixed distance, the distance from the path remains fixed when changing the width of the conveyor.

    Vertical, z max, Anchors, Move/30

    Extrude vertically at the height of the path anchor point for each anchor point and pin to the borders when changing the width of the conveyor.

    Vertical, z max, Anchors, Fix/31

    Extrude vertically at the height of the path anchor point with a fixed distance and pin to the center when changing the width of the conveyor.

    Vertical, z max, Interval, Move/32

    Extrude vertically at the height of the path anchor point for each anchor point and pin to the borders when changing the width of the conveyor.

    Vertical, z max, Interval, Fix/33

    Extrude vertically at the height of the path anchor point with a fixed distance and pin to the center when changing the width of the conveyor.

    Forward, Floor, Anchors, Move abs/34

    Repeatedly extrude a small segment in the path direction with z = 0.0 for each anchor point, the distance from the borders remains fixed when changing the width of the conveyor.

    Forward, Floor, Anchors, Move rel/35

    Repeatedly extrude a small segment in the path direction with z = 0.0 for each anchor point, and pin the base point of the profile to its relative position when changing the width.

    Forward, Floor, Anchors, Fix/36

    Repeatedly extrude a small segment in the path direction with z = 0.0 for each anchor point, the distance from the path remains fixed when changing the width of the conveyor.

    Forward, Floor, Interval, Move abs/37

    Repeatedly extrude a small segment in the path direction with z = 0.0 with a fixed distance, the distance from the borders remains fixed when changing the width of the conveyor.

    Forward, Floor, Interval, Move rel/38

    Repeatedly extrude a small segment in the path direction with z = 0.0 with a fixed distance, and pin the base point of the profile to its relative position when changing the width.

    Forward, Floor, Interval, Fix/39

    Repeatedly extrude a small segment in the path direction with z = 0.0 with a fixed distance, the distance from the path remains fixed when changing the width of the conveyor.

    Vertical, Ceiling, Anchors, Move/40

    Extrude vertically from the path anchor point to the ceiling for each anchor point and pin to the borders when changing the width of the conveyor. The end offset denotes the ceiling height.

    Vertical, Ceiling, Anchors, Fix/41

    Extrude vertically from the path anchor point to the ceiling with a fixed distance and pin to the center when changing the width of the conveyor. The end offset denotes the ceiling height.

    Vertical, Ceiling, Interval, Move/42

    Extrude vertically from the path anchor point to the ceiling for each anchor point and pin to the borders when changing the width of the conveyor. The end offset denotes the ceiling height.

    Vertical, Ceiling, Interval, Fix/43

    Extrude vertically from the path anchor point to the ceiling with a fixed distance and pin to the center when changing the width of the conveyor. The end offset denotes the ceiling height.

    Extrude horizontally with z as designed in the anchor points for each anchor point and pin to the borders when changing the width of the conveyor.

    Parameter

    The parameter of data type table contains the parameters defining the extrusion configuration.

The 3D attributes of the data types table and array return a copy of the values.

Compare these examples:

_3D.ExtConfiguration := myTable -- assigns a new value
myTable :=_3D.ExtConfiguration -- reads a value

This example seems to have the correct syntax, but does not have any effects:

_3D.ExtConfiguration[1,1] := 7 -- does not assign the sub-table to the first cell of the attribute

Example:

 
Conveyor._3D.ExtConfiguration := MyExtrusionConfigurationTable
MyExtrusionConfigurationTable := Conveyor._3D.ExtConfiguration

If you would like to copy the attribute _3D.ExtConfiguration to a TableFile in a Frame, this does not work via a simple assignment with := but by copying format and contents of the TableFile. Compare the example below:

Example:

 
var MyExtrusionConfigurationTable : table := Conveyor._3D.ExtConfiguration
Table1.delete
MyExtrusionConfigurationTable.copyFormatTo(Table1)
MyExtrusionConfigurationTable.copyRangeTo({0,0}..{*,*}, Table1, 0, 0)
