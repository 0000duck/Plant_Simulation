Insert Date or Time
2D Reference Help > Display and User Interface Objects > Comment > Dialog Window of the Comment > Tab Comment > Context Menu for Formatting the Comment > Insert Date or Time

The menu command Insert Date or Time opens the dialog Insert Date and Time.

  • Select one of the available formats from the list box.

  • Click OK to enter this time or date at the position of the cursor into the text box.

Instead, you can also click on the Formatting Toolbar .
