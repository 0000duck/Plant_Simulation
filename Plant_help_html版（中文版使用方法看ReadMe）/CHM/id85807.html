2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Preferences > 3D > 3D

On the tab 3D you can select general settings for 3D as such and settings for the 3D model. You can also select some of these settings for each scene in the individual scene windows. These settings will then override the general preferences.

When you click OK or Apply, 3D saves the 3D toolbars of the 3D scene windows. Thus it also shows these toolbars in any new 3D scene windows, which you open afterwards. This only applies for the setting User Interface > User interface concept > Menus and Toolbars.

You can:

  • Select the Visualization settings Screen coverage threshold, Show states as colors if no state graphics are present, and Grid Settings.

  • Enter a value for the Modeling setting Curve precision.

  • Select the View setting Show grid.

  • Select the Mouse wheel behavior.

  • Select the OpenGL Version.
