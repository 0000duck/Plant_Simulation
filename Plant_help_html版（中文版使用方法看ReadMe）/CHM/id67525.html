StatEmptyMu
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Read-Only Attributes for Statistics of the Material Flow Objects > StatEmptyMu

Syntax:

 
Path.StatEmptyMu → real

The read-only attribute StatEmptyMu returns the mean duration of the time span during which the object designated by Path was Empty, i.e., not Failed, not Paused, or not Stopped.

Return Value

The return value has the data type real.

Example:

 
print ParallelStation.StatEmptyMu
