MaterialsTable
2D Reference Help > Fluid Objects > FluidSource > Attributes of the FluidSource > MaterialsTable

Syntax:

 
Path.MaterialsTable

The attribute MaterialsTable sets the MaterialsTable which contains the data of the different materials which the FluidSource designated by Path can produce.

Data Type of the Value You Can Assign

You can assign a value of data type path.

Example:

 
MyFluidSource.MaterialsTable := MyMaterialsTable
