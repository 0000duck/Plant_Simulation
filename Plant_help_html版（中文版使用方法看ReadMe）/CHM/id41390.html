Resources
2D Reference Help > Resource Objects > ShiftCalendar > Attributes of the ShiftCalendar > Resources

Syntax:

 
Path.Resources

The attribute Resources sets the resource objects that the ShiftCalendar controls. Plant Simulation copies the contents of the table object, which you enter, to the internal resources/Objects table on the Tab Resources.

Select the data type object for the first column of the table file and enter all objects to be controlled by the shift calendar.

Note that assigning the table overwrites the contents of any existing resource table. You can use a column index and a row index. You can also use the rest of the table for any other purpose.

Data Type of the Value You Can Assign

You can assign a value of data type any.

Example:

 
shifts.Resources := ResourcesTable
