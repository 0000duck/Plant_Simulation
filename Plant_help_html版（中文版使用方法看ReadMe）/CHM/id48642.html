Transparent
2D Reference Help > Information Flow Objects > Variable > Attributes of the Variable > Transparent

Syntax:

 
&Variable.Transparent

The attribute Transparent sets if the background of the Variable, which you inserted into a Frame, is transparent (true) or not (false).

Then, the Variable shows the space around the text in the color of the background image or it shows the grid points of the Frame. If you clear Transparent, Plant Simulation shows the space around the text in white.

Not transparent

Transparent

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
&MyVariable.Transparent := false
