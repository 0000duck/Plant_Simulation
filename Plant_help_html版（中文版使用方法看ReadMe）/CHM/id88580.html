Create a Bulleted List
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Create a Bulleted List

Enter an asterisk followed by an empty space (* ) at the start of a line to create a bulleted list. Each asterisk followed by an empty space at the start of a line creates a new bullet. Each bullet can span several lines. Terminate the bulleted list with an empty line.

Instead, you can also click on the toolbar .

When the asterisk is not located directly at the start of the line, but after a tab stop or after four empty spaces, Plant Simulation creates a sub-list.

Each additional four empty spaces or each additional tab stop create sub-sub-lists which are indented further to the right.

* List item 1
* List item 2
    * Sub-list item 2.1
    * Sub-list item 2.2
* List item 3
