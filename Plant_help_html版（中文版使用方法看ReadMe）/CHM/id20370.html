DragDropCtrl
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of All Objects > Attributes for Controls of All Objects > DragDropCtrl

Syntax:

 
Path.DragDropCtrl

The attribute DragDropCtrl designates a Method object. Plant Simulation calls it for the object, which you drag onto another object and drop it there. The dropped object can be an object or text.

Parameter

Plant Simulation passes the parameter PathOfDroppedObject of data type string to the called method, which contains the path of the dropped object.

A drag-and-drop control might look like this:

Example:

 
param str: string
print "Dragged object ",str," onto ",?," and dropped it there."
-- ?,@ object with control method that the other object is dropped onto.

Example:

 
param draggedObjects : object[]) -- one-dimensional array with n objects
                            -- the size of which can change
var obj: object
-- @,?: target object onto which the dragged object was dropped

for var i := 1 to draggedObjects.dim 
   print i,": ",draggedObjects[i]
   -- prints the dragged objects to the Console
next

Data Type of the Value You Can Assign

You can assign a value of data type method.

Example:

 
MyStation.DragDropCtrl := &myDragDropCtrl
