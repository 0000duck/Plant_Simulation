yDim
SimTalk Reference > General SimTalk Functions > Data Types > Array > Methods and Attributes of Arrays > yDim

Syntax:

 
yDim

The method yDim returns the y-dimension of a two-dimensional array.

For a one-dimensional array it returns 0. This way you can find out if a local variable of data type any contains a one-dimensional array or a two-dimensional array and if it was assigned to this station, not if it actually was there yet.

Example:

 
var matrix:real[2, 3]
print matrix.yDim -- prints 3
