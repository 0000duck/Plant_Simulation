isAlignmentColumn
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Columns > isAlignmentColumn

Syntax:

 
Path.isAlignmentColumn(Column:any, Alignment:string) → boolean

The method isAlignmentColumn returns if the alignment of a column of the TimeSequence designated by Path matches the designated value (true) or not (false).

Parameters

  • The parameter Column of data type any designates the column.

  • The parameter Alignment of data type string designates the value: Right, Left or Center.

Return Value

The return value has the data type boolean.

Example:

 
print timeSequence1.isAlignmentColumn("column4","right")
