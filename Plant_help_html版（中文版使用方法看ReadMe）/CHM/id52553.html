Y
2D Reference Help > The Plant Simulation Program Window > Display Panel > Attributes of the Display Panel > Y

Syntax:

 
Path.DisplayPanel.Y

A 3D Only model does not provide the attribute DisplayPanel.Y.

The attribute Y sets the y-coordinate of the left top corner of the display panel of the object designated by Path. This position is relative to the coordinates of the reference point of the object.

You can enter a number between -5000 and +5000.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
Station.DisplayPanel.X := 200
Station.DisplayPanel.Y := 200
