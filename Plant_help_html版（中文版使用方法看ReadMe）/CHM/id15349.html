resetValues
2D Reference Help > Display and User Interface Objects > Chart > Methods of the Chart > resetValues

Syntax:

 
Path.resetValues

The method resetValues resets the Category > Histogram and the Category > Plotter of the Chart. This means it deletes the existing data and starts collecting data anew. You can, for example, use it to record a weekly histogram or to suppress data collected during the ramp-up phase.

Example:

 
MyChart.resetValues
