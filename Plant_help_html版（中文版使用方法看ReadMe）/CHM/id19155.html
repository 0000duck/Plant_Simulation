Discrete Empirical (dEmp)
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Empirical Distributions > Discrete Empirical (dEmp)

If the random numbers are real or integer numbers, you need to use a table with two columns. Column 1 contains the Time, column 2 contains the respective Frequency. You can enter values of data type time and of data type real.

The relative frequency will be used as an empirical estimate for the probability of the respective value to appear.

Clicking Apply in the dialog of the object automatically formats the table so that the data types of the columns match the parameters.

You can also create the discrete empirical distribution with this SimTalk function: 

z_dEmp(Stream:integer,Table:table)

The parameter Stream (integer) designates the random number stream.

Parameter

The parameter Table designates the frequency table.
