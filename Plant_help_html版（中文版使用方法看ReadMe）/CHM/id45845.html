_3D.TypeOfAnimations.AnimationPathName.getTable
SimTalk Reference > 3D-related SimTalk Functions > Built-in Functions for Saved Animations > _3D.TypeOfAnimations.AnimationPathName.getTable

Syntax:

 
Path._3D.MUAnimations.AnimationPathName.getTable(Target:table)
Path._3D.SelfAnimations.AnimationPathName.getTable(Target:table)
Path._3D.CameraAnimations.AnimationPathName.getTable(Target:table)

The method _3D.TypeOfAnimations.AnimationPathName.getTable formats the passed table and then writes the animation data of all saved animations of the designated animation type into this table.

Parameter

The parameter Target of data type table contains the data of each saved animation.

Example:

 
var mua:= Buffer._3D.MUAnimations
var created : boolean := mua.createAnimationLine("Animation2")
if created -- The animation line was created. 
   var t: table -- Copy the ‘Default’ line values to the table.
   mua.Default.getTable(t)
   mua.Animation2.setTable(t)
end
