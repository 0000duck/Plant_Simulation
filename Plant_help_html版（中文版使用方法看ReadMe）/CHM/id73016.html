Exiting Methods Using Return
SimTalk Reference > General SimTalk Functions > Control Flow Statements > Exiting Methods Using Return

Syntax:

 
return

The keyword return terminates the currently executed method and returns to the calling method.

Example:

 
if ParallelStation.failed   -- cannot process?
   return                -- return immediately
else
   ...
end

You can terminate methods that return a result with return expression. This has the same result as result := expression return.

Example:

 
param n: integer -> integer -- computes the factorial of n 
if n = 0 
   return 1
end
return n * self.execute(n-1)
