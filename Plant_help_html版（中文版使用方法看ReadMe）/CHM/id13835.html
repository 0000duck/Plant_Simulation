Comment
2D Reference Help > Display and User Interface Objects > Comment > Comment

The object Comment shows explanatory notes to your simulation model which you added. This helps you and your colleagues to better understand the intentions behind your model and how it is supposed to work.

You can access it like you do any other object, be aware though that the Comment usually is not involved in the simulation itself. Select Options > Show Comments on the General ribbon tab of the Frame or click to display all Comment objects, which you inserted into that Frame. Plant Simulation displays the text in the Frame window, which you entered under Display > Text.

To open a window that only shows the comment you entered, without formatting and editing options, click the Comment object with the right mouse button and select Open Comment Window on the context menu.

To add the Comment to your simulation model, click Manage Class Library > Basic Objects > UserInterface > Comment on the Home ribbon tab.
