createNestedList
2D Reference Help > Information Flow Objects > QueueFile and StackFile > Methods of the QueueFile and the StackFile > createNestedList

Syntax:

 
Path.createNestedList([Name:string])

The method createNestedList creates a nested list in the StackFile and QueueFile designated by Path. The StackFile or QueueFile has to be of data type list, stack, queue, or table.

Plant Simulation moves existing entries down one position. When you enter an index, which is larger than the largest valid index, Plant Simulation places the entry at the first available position in the list, regardless of the actual index, preventing gaps between entries.

Parameter

The parameter Name of data type string designates the name of the list to be created.

Example:

 
MyStackFile.createNestedList("My Sublist")
