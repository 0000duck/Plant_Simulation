Locked
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Show 3D Graphic Structure > Context Menu of a Graphic Group > Locked

The context menu command Locked locks the graphic group. This prevents you from selecting this graphic group in the 3D windows which are opened for this object. This makes, for example, sense for the graphic of a factory building which might otherwise be selected inadvertently.
