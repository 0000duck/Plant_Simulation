Remove Parts from Stock
Step-by-Step Help > Modeling in 2D > Modeling Transport Systems > Modeling a Transport System with Passive Objects of Type Track > Model a Facility Using a Storage Crane > Temporarily Store Parts, Put together the Order, Remove the Parts from Stock > Remove Parts from Stock

Once the number of parts in store is the same as the number of ordered parts, the crane removes these parts from stock with the method releaseFromStock, which we programmed.

The source code looks like this:

param row: integer 
crane.getPart(InventoryList["OrderNo", row ],InventoryList["Number", row ], 10)
InventoryList.cutRow(row)

This Method retrieves all parts of a certain type, identified by its name, employing the built-in method getPart. As we chose the name of the part to be identical with the order number, we can directly retrieve the order number and the order quantity. After this, the crane deletes the order number from the InventoryList.
