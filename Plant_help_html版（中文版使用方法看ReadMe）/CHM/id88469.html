Internal
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Show 3D Graphic Structure > Context Menu of a Graphic Group > Internal

The context menu command Internal determines if the graphic group is only visible within its object and will only be used for its internal decoration or not. Internal graphic groups are also visible on the outside of the object if Show content is selected. The external graphics on the other hand represent its object towards the outside.

The dialog 3D Graphic Structure shows internal graphic groups with two yellow cubes, it shows external graphic groups with two red cubes.
