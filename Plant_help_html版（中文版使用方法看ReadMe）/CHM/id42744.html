2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Curve > Transparent

To make the inside of the curved object transparent, so that the background color of the Frame shines through, select this. The border of the object remains as it is, i.e. non-transparent. To completely hide the curved object, enter a value less than 0 for the Pen width.
