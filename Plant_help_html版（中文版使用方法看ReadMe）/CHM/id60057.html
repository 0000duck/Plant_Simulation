Edit Dialog
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Edit Dialog

To edit a dialog item which you created in the Dialog, click this. The Dialog then selects the dialog item in the display window and you can move it by dragging it to another to another position. To edit its properties, click it with the right mouse button and select Open on the context menu.

Instead, you can also right-click the Dialog object and select Edit Dialog on the context menu in the Frame.

The dialog window of the object Dialog uses the standard Siemens PLM theme.
