Animate the Camera
Step-by-Step Help > Modeling in 3D > Controlling Your View > Animate the Camera

You can make the camera move on any animation path of the selected object or of the root object of the scene. Here, you fly on this path and view the scene from it.

In addition, 3D:

  • Visualizes the animated camera, i.e., inserts an animatable object that represents the camera.

  • Allows you to change between animation view and normal view during the animation.

  • Allows you to manually transform the animated camera: By changing the view while in animation view, or by manipulating the object, while in normal view.

  • Allows you to define the speed of the animation and the direction of the animation.

  • Allows you to pause, continue and stop the camera animation, independent of other animations and simulations.
