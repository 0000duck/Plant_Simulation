User-defined Attributes
2D Reference Help > Information Flow Objects > Method > The Method Window > Method Debugger > Menu Bar of the Method Debugger > Tools Menu > User-defined Attributes

The menu command User-defined Attributes opens a dialog in which you can add new user-defined attributes to the Method or in which you can edit and delete existing user-defined attributes.

Click New to open the dialog User-defined Attributes, which is described under the Tab User-defined.
