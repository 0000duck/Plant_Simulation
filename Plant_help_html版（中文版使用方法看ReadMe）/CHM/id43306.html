getLaneNo
2D Reference Help > Mobile Objects > Transporter > Methods of the Transporter > getLaneNo

Syntax:

 
MU-path.getLaneNo([LaneNumber:length]) → integer

The method getLaneNo returns the number of the lane on which the Transporter is located on a TwoLaneTrack: 1 for lane A, 2 for lane B. When the Transporter is located on a single-lane Track, the method always returns 1.

Parameter

The optional parameter LaneNumber of data type length designates the lane number.

Return Value

The return value has the data type integer.

Example:

 
print.MUs.Transporter:1.getLaneNo
