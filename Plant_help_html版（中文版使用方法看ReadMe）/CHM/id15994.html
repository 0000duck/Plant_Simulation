Export to File
2D Reference Help > Information Flow Objects > Method > The Method Window > Edit Ribbon Tab > Export to File

The command Export to File opens the dialog Save As. Enter the file name and select the folder to which you would like to save the source code of the active Method as a text file and click OK.

Plant Simulation adds a comment before the source code proper with the path of Method. This way you can always tell which text file belongs to which source code.
