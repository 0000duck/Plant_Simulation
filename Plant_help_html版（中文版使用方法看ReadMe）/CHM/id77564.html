Width
2D Reference Help > Material Flow Objects > TwoLaneTrack > Dialog Window of the TwoLaneTrack > Tab Attributes > Width

Enter the width of both lanes of the TwoLaneTrack in meters. The width only applies when the setting Transfer length is activated.

You can only enter the Curve width when you deactivate Transfer length. When you activate it, Plant Simulation computes the width in meters.
