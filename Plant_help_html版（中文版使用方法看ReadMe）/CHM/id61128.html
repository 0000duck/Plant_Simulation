In our sample model the Sources create parts. The object Source moves its part directly on to the pick-and-place robot, which picks it up, rotates, and delivers it to the Drain. The object Source1 moves its part to the pick-and-place robot, which picks it up, rotates, and delivers it to the processing station Station. After the processing station has processed the part, it feeds it back to the pick-and-place robot, which picks it up and delivers it to Drain1.
