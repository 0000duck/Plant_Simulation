findCeil
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for the Order of Cells within Lists and Tables > findCeil

Syntax:

 
Path.findCeil([Range:listrange, ...,] Value:any)

The method findCeil searches in the list designated by Path for the value greater than or equal to the value you specify.

The method only applies to lists and tables of data types integer, real and string, and to data types, which can be converted into the data type real, such as dateTime, weight, etc.

Parameters

  • The parameter Value of data type any designates the value you want to find.

  • The parameter Range of data type listrange designates the range between which Plant Simulation searches to the position of the cursor, for tables it designates the position of both cursors.

Return Value

The return value has the data type boolean.

When Plant Simulation finds a value, the method returns true and sets the cursor in that cell. It returns false when it did not find the value. When several values exist, Plant Simulation sets the cursor to the cell that is closest to the value of data type any. The search starts at the current cursor position. This holds only true when the cursor is located within one of the ranges specified. When that is not the case, the search begins in the first range specified and, if finished without success, is continued in the next range, etc.

When the cursor is located in one of the ranges specified, Plant Simulation does not search the preceding ranges, not even after the search was negative for the other ranges. When the cursor is located in the middle of the range, Plant Simulation only searches the range after the cursor, but not the range before. When you do not enter a range, Plant Simulation searches the list starting at the position of the cursor to the end of the list.

For the following example we suppose that a QueueFile contains the values 30, 50, 20, 0, 40, 10. We then apply the method findCeil to it:

Example:

 
MyQueueFile.Cursor := 1 -- prepares the search
MyQueueFile.findCeil(25)

As the result Plant Simulation sets the cursor to 30 (the Cursor is located in cell 1). 30 is the value greater than or equal to the search value 25 and is closest to 25.
