Information Flow Objects
2D Reference Help > Information Flow Objects > Information Flow Objects

Tecnomatix Plant Simulation provides these objects for providing the simulation with information during the simulation runs:

  • The Method for programming controls and the Variable for storing data over an extended period of time.

  • The lists and tables CardFile , QueueFile and StackFile , TableFile , and TimeSequence .

  • The Trigger and the Generator for controlling when and how mobile objects (MUs) are created.

  • The AttributeExplorer for managing the attributes defining the settings of the individual stations in your simulation model.

  • The FileInterface for exchanging data with other applications.

  • The FileLink for exchanging data with other applications.

  • The XMLInterface for exchanging data with other applications.

Note that the Inter-Process Communication Interfaces and the Database Interfaces are also part of the information flow objects.
