PermitDeleteCtrl
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of All Objects > Attributes for Controls of All Objects > PermitDeleteCtrl

Syntax:

 
Path.PermitDeleteCtrl

The attribute PermitDeleteCtrl designates a Method object. The Frame or the folder calls it, when you attempt to delete the object for which you programmed the control. When the method returns false, the program does not delete the object. When the method returns true, the program deletes the object and calls a Destructor control method, provided you entered one.

The permit delete control only applies to the Frame and to the Folder.

Data Type of the Value You Can Assign

You can assign a value of data type method.

Example:

 
Frame.PermitDeleteCtrl := &myPermitDeleteCtrl
