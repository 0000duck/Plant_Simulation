createNestedList
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Instantiating the TableFile > createNestedList

Syntax:

 
Path.createNestedList(Column:any,Row:any[, Name:string]) → list

The method createNestedList creates a nested list in the designated cell in the TableFile designated by Path. The data type of the columns of this TableFile have to be list, stack, queue, or table.

Parameters

  • The parameter Column of data type any designates the column of the cell.

  • The parameter Row of data type any designates the row of the cell.

  • The optional parameter Name of data type string designates the name of the sub-table to be created.

Return Value

The return value has the data type list which designates the nested list (CardFile, QueueFile, or StackFile) that was created.

Example:

 
MyTableFile.createNestedList(1,2,"My subtable")
