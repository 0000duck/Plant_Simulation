Export Object File
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Export Object File

The command Export Object File saves the list as an object file ( .obj). An Plant Simulation object file contains the contents of the list and, in addition information about Plant Simulation-specific formatting, such as data types and other formatting of columns and rows, dimension, column and row index etc. Thus the list will open with the same properties, when you open it in another Plant Simulation simulation models. Other programs cannot use this information. Click Save to save the file.
