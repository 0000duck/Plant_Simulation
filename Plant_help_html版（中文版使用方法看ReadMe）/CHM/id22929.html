Full
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Read-Only Attributes for the State of the Material Flow Objects > Full

Syntax:

 
Path.Full → boolean
Path[X:integer, Y:integer].Full → boolean

The read-only attribute Full returns if the capacity of the object designated by Path is exhausted (true) or not (false).

For point-oriented objects Full is true, when all processing stations are occupied. If more than one processing station exists, each one of them might be asked if it is ready to receive MUs. As a station may process a single MU, Full is identical with occupied.

Parameters

  • The parameter X of data type integer designates the x-coordinate of the processing station.

  • The parameter Y of data type integer designates the y-coordinate.

For length-oriented objects Full returns true, when the number of MUs on the object corresponds to its capacity or the length occupied takes the value of its length.

Return Value

The return value has the data type boolean.

Example:

 
-- queries the entire object
if NOT storeNorth.Full 
   @.move(storeNorth)
elseif NOT storeEast.Full 
   @.move(storeEast)
else -- if all are all Full, use spare 
   @.move(spareStore)
end -- queries each single station
if ParallelStation[3,4].Full 
   ParallelStation[3,4].Cont.delete
end
