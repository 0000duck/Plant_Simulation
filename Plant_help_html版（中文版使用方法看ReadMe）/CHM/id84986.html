_3D.AniRotationAxis
SimTalk Reference > 3D-related SimTalk Functions > Built-in Functions for Saved Animations > _3D.AniRotationAxis

Syntax:

 
Path._3D.AniRotationAxis

The attribute _3D.AniRotationAxis sets the rotation axis for the function _3D.TypeOfAnimations.scheduleRotation of the object designated by Path.

Parameter

The parameter of data type array of three values of data type real sets the rotation axis for the animation.

Example:

 
MyStation._3D.AniRotationAxis := [0,1,0]
-- rotates the object around its y-axis
