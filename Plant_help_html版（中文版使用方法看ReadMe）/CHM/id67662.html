Operators and Expressions
SimTalk Reference > General SimTalk Functions > Operators and Expressions > Operators and Expressions

The most basic and most simple form of an expression is a constant value or a variable value. By combining constants or variables and operators, you can create increasingly complex expressions.

The order in which Plant Simulation analyzes complex expressions depends on the priority of the operators involved.

Plant Simulation provides arithmetic operators, the logical operators AND and OR, the logical operator NOT, and relational operators. Note that some operators are restricted to certain data types. Boolean operators in Plant Simulation combine expressions and relational operators compare expressions.
