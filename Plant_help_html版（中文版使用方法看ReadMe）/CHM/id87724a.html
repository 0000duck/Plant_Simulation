increment
2D Reference Help > Information Flow Objects > Variable > Methods of the Variable > increment

Syntax:

 
&Variable.increment
&Variable.increment(Value:integer)

The method increment adds 1 to the Variable.

Parameter

The method increment(Value:integer) adds the Value designated by the parameter of data type integer to the Variable.

The method only applies to numerical data types. It applies to the object Variable and to user-defined attributes.

Example:

 
x := &MyVariable.increment(-3) 
-- accomplishes the same as 
-- Variable := Variable - 3 x := Variable
