Plant Simulation provides these attributes and methods for MU animations in 3D:

_3D.AnimationObject, _3D.MUAnimations.AnimationPathName.cancelRedirection, _3D.MUAnimations.AnimationPathName.isRedirected, _3D.MUSideToAttach, _3D.MUAnimations.AnimationPathName.redirectTo, and _3D.resetAnimationTime.
