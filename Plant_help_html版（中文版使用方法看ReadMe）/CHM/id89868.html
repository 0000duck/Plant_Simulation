PowerInputOff
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for the Energy Simulation of the Material Flow Objects > PowerInputOff

Syntax:

 
Path.PowerInputOff

The attribute PowerInputOff sets the power input of the state off of the object designated by Path.

Data Type of the Value You Can Assign

You can assign a value of data type real or a formula.

Example:

 
Station.PowerInputOff := 1
