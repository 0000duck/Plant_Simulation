Create an Access Key Using the & Sign
2D Reference Help > Material Flow Objects > Frame > The Frame Window > General Ribbon Tab > Configure User-defined Ribbon Tab > Text to be displayed > Create an Access Key Using the & Sign

Enter an ampersand character in front of a letter in a command to make this letter the access key in the user-defined ribbon. Pressing the Alt key and that letter activate this command.

If you do not enter the ampersand, Plant Simulation assigns an automatically numbered access key.

The built-in access keys take precedence over any access keys you define! This means that your access key does not work if it is already defined on one of the pre-defined ribbon tabs.

In our example we made the S in the command My Simple Dialog the access key. Holding down the Alt key, pressing the predefined keys U and T, and pressing the S key opens our simple dialog.
