Distribution Functions
SimTalk Reference > General SimTalk Functions > Predefined Functions > Mathematical Functions > Distribution Functions > Distribution Functions

You can create random numbers for times in the dialogs of the objects on the Tab Times and on the Tab Failures, with a Variable of data type randtime, and in SimTalk with the functions described below.

Parameter

The parameter Stream designates the random number stream (compare Random Number Seed Values) and has the data type integer. All other parameters are the parameters of the corresponding distribution function as described under Probability Distributions. They all either are of data type real or integer.

All distribution functions, except for z_emp, return a value of data type real.
