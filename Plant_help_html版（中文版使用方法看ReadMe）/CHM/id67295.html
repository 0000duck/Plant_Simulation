Show default buttons
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Show default buttons

To show the standard buttons in the dialog you are creating, select this. To hide these buttons, clear the check box.

If you do not show this button combination, you have to define your own buttons to provide the user with a way to apply or to discard settings he chose.
