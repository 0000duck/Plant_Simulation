StatServicesTransportingPortion
2D Reference Help > Resource Objects > Worker > Read-Only Attributes of the Worker > StatServicesTransportingPortion

Syntax:

 
Path.StatServicesTransportingPortion

The method StatServicesTransportingPortion returns the portion of the overall statistics time during which the Worker was carrying parts from Workplace to Workplace. The overall statistics time is the statistics collection period without the unplanned time and the paused time.

Return Value

The return value has the data type real.

Example:

 
print .Resources.MyWorker:1.StatServicesTransportingPortion
