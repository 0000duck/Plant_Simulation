toLower
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Strings > toLower

Syntax:

 
toLower(Text:string) → string

The function toLower changes upper-case letters within a string to lower-case letters.

Parameter

The parameter of data type string can consist of any combination of upper-case letter, lower-case letters, and special characters. The result string then only contains lower-case letters and special characters.

Return Value

The return value has the data type string.

Example:

 
print toLower("SHANIA") -- returns "shania"

Example:

 
print toLower("Hello World!") -- returns "hello world!"
