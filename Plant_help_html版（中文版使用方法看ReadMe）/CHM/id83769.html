Search in the Help
Using HTML Help > Working with Full Text Search > Search in the Help

These topics describe the features on the Search tab of the Help Viewer. Here you can search through every word in a help file to find a match. If you run a full-text search for the word "index," for example, the tab lists every topic that contains the word "index."

Advanced full-text search allows you to search using boolean, wildcard, and nested expressions.

To use full-text search:

  • Click the Search tab, and enter the word or phrase you want to find into the text box Type in the word(s) to search for.

  • Click List Topics, select the topic you want, and then click Display.
