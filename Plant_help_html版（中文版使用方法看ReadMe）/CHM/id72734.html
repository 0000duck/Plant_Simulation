New List View
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New List View

A list view displays a table in the dialog.

When displaying strings, Plant Simulation only shows the first 260 characters.

  • Enter the Name of the list view.

  • Enter the Callback argument, which will be passed to the Callback method as a parameter. It will be executed, when the user double-clicks a row in the list view.

    You can only select rows, but no columns or individual cells.

  • Enter the name of a table or click the button and select a table in the dialog Select Object.

    Enter the items the list view shows into the cells of the columns.

  • Enter its X and Y position in the Dialog.

  • Enter its Width and Height in the Dialog.

  • Select if you want to Enable it or not.

  • To insert the dialog item into your dialog, click OK.
