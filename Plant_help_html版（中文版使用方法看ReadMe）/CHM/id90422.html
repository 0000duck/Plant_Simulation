Is Tractor
2D Reference Help > Mobile Objects > Transporter > Dialog Window of the Transporter > Tab Attributes > Is Tractor

To designate the active Transporter as the tractor of the tugger train, which pulls or pushes the trailers, select this.

The Transporter then shows the icon tractor instead of the icon Operational. As a vector graphic the icon of the tractorlooks like this .

To prevent the tractor from loading parts, enter 0 as the X-dimension and the Y-dimension for the loading space type > Store. For the loading space types Track and Line, enter 0 as the Capacity.
