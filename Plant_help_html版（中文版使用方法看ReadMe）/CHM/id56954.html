Underrun
2D Reference Help > Fluid Objects > Tank > Attributes of the Tank > Attributes of the Sensors of the Tank > Underrun

Syntax:

 
Path.Sensor.Underrun

The attribute Underrun sets if the sensor is to be triggered when the amount of material in the Tank designated by Path has underrun, i.e., is located below the sensor position (true) or not (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTank.sensorID(1).Underrun := true
