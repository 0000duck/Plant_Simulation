Display an Object of Type Display
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Select an Object or an Object Attribute to be Displayed > Display an Object of Type Display

To display the numeric value of an object of type Display on the HTML page, click on the toolbar .

Select the Display whose numeric value you would like to show.

To use the font size and the font color, which you selected for the object Display, on the HTML page, select this .

Instead, you can also type in the instructions that correspond to the settings you can enter into the dialog.

You can enter true or false as the optional parameter. When you enter true, Plant Simulation displays the value using the color and the font size which you selected on the tab Display of the Display. 

[Display]
[Display, true]
[Display, false]
