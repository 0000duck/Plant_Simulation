Attributes of the Method
2D Reference Help > Information Flow Objects > Method > Attributes of the Method > Attributes of the Method

The object Method provides:

  • The attributes Program and UsingNewSyntax.

  • The Attributes of All Objects.

To view all of the methods and attributes of the Method, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.

To access the Method, use the reference operator or the anonymous identifier self.

Example:

 
&MyMethod.Program := "->real return 0.0"

self.Program := "->real return 0.0"
