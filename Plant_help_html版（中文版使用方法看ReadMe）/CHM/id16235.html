PenWidth
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Curved Objects > PenWidth

Syntax:

 
Path.PenWidth

The attribute PenWidth sets the line width of the outline around the border of the length-oriented curved object designated by Path. This can be a value between -1 and 100. The value -1 makes the outline invisible. The value 0 stands for a pen width of 1 pixel. The value 1 stands for a stroke weight of 1. This affects zooming in the Frame.

3D does not use this setting.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyTrack.PenWidth := 4
