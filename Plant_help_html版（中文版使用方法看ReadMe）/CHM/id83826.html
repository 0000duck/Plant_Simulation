Length
2D Reference Help > Tools > CrossSlidingCar > Dialog Window of the CrossSlidingCar > Tab Track Attributes > Length

Type in the length of the track of the cross-sliding car. This length is the length of the underlying Track. The Track can consist of straight and/or curved segments. When calculating the loading and unloading positions, Plant Simulation considers the Track to be a straight segment.
