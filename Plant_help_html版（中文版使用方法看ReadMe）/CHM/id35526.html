Capacity
2D Reference Help > Material Flow Objects > ParallelStation > Read-only attributes of the ParallelStation > Capacity

Syntax:

 
Path.Capacity

The read-only attribute Capacity returns the capacity of the ParallelStation designated by Path. It is the product of XDim times YDim.

Return Value

The return value has the data type integer.

Example:

 
if ParallelStation.Capacity >= lotsize 
   @.move(ParallelStation)
end
