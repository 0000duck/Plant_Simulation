setXYWH
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > General Methods of All Objects > setXYWH

Syntax:

 
Path.setXYWH(X:integer, Y:integer, Width:integer, Height:integer)

A 3D Only model does not provide the method setXYWH.

The method setXYWH sets the position on the screen of the dialog of the object designated by Path.

Parameters

  • The parameter X of data type integer designates the x-coordinate of the dialog on screen.

  • The parameter Y of data type integer designates the y-coordinate.

  • The parameter Width of data type integer designates the width of the dialog.

  • The parameter Height of data type integer designates the height of the dialog.

    The values width and height only apply for windows that you can resize, i.e., for Object windows of the Frame, the Method, the Method Debugger, the QueueFile and StackFile, the CardFile, the TableFile, and the Icon Editor.

Example:

 
MyStation.setxywh(10,100,10,20)
frame1.setxywh(500,100,15,25)
