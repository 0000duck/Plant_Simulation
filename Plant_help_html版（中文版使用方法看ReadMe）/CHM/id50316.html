Methods for Animating All Objects
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Animating All Objects > Methods for Animating All Objects

The objects provide these methods to animate them:

closeImg, createIcon, delAniPoint, delAniPoints, deleteIcon, existsIcon, getAniPoints, getBoundingBox, getIconSize, getPixel, getRefPoint, inheritPosition, inheritSizeAndOrientation, linkAniPoint, mirrorX, mirrorY, moveAniPoint, openImg, putIconToClipboard, resetIcon, saveIconToFile, setAniPoint, setAniConveyor, setCurrIconFromClipboard, setIconFromFile, setIconSize, setPixel, setRefPoint, turnIcon, and unlinkAniPoint.

To view all of the methods and attributes of the material flow object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.
