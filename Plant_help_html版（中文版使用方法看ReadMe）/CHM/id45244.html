setAlignmentColumn
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > setAlignmentColumn

Syntax:

 
Path.setAlignmentColumn(Column:any [,Column:any,...], Aligmnent:string)

The method setAlignmentColumn sets the alignment of column(s) of the TableFile designated by Path.

Parameters

  • The parameter Column of data type any designates the column whose alignment you want to set.

  • The optional parameters Column of data type any designate additional columns whose alignment you want to set.

  • The parameter Alignment of data type string designates the alignment you want to assign: Left, Right or Centered.

Example:

 
MyTableFile.setAlignmentColumn(1,"right")
MyTableFile.setAlignmentColumn(1,2,"right")
