_3D.MUAnimations.Animation.redirectTo
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for MU Animations > _3D.MUAnimations.Animation.redirectTo

Syntax:

 
Path._3D.MUAnimations.Animation.redirectTo(TargetPath:string) → boolean

The method _3D.MUAnimations.AnimationPathName.redirectTo redirects the animation to another designated MU animation . The animation has to be located within the same object.

Parameter

The parameter TargetPath of data type string designates the name of the target animation.

Return Value

The return value has the data type boolean. It tells if the TargetPath was found (true) or not found (false).

Example:

 
self.~._3D.MUAnimations.getAnimation("Default").redirectTo("backwards alternative")
