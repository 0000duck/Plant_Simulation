2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab User-defined > Edit

To change the name and data type of the selected user-defined attribute and to activate or deactivate inheritance, click Edit or double-click the name of the user-defined attribute in the list field and modify the values in the dialog.

To show the statistics values, which the user-defined attribute collected, click Edit. Then, click the tab Statistics. Here you can set statistics to Active, show the Statistics Table, Charts, and show a Frequency Histogram.
