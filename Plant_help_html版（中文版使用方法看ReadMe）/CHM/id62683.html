Test Our Station in the Simulation Model
Step-by-Step Help > Modeling in 3D > Creating an Animatable Object > Test Our Station in the Simulation Model

Now we are ready to test our screwdriver station in the simulation model.

  • Insert a Source, which produces the pallets which we want to screw together. We named it SourcePallets.

    Insert a second Source, which feeds the screws required for screwing together the pallets. We named it SourceScrews. We received the screw, including the JT file, from a colleague.

  • Insert a Drain, which removes the finished pallets from the plant.

  • Insert our Screwdriver station.

  • Insert three Conveyors between the Sources, the Drain, and the Screwdriver station.

  • Connect all objects with Connectors. Connect the SourcePallets, that produces the MU > PartsCarrier, with the ConveyorPallets, and with the InterfaceInPalette of the Screwdriver. Connect the SourceScrews that produces the MU > Screw, with the ConveyorScrews, and with the InterfaceInScrew. Connect the ConveyorOut with the InterfaceOut and then with the Drain. The simulation model should now look like this:

    You might be wondering how we inserted the Conveyors with an inclination to smoothly connect to the Screwdriver station. We proceeded as follows:

    • For the ConveyorPallets we entered an anchor point height of 1 meter for the first anchor point and an anchor point height of 0.15 meters for the second anchor point. This is the height of the object SourcePallets to which we connect it on the left-hand side.

    • For the ConveyorOut we edited values in the dialog Edit 3D Properties > Appearance. We entered a base height of 0.15 meters, which means that the left-hand side of the ConveyorOut is placed 15 centimeters above the floor. We then clicked Segments and entered 1 meter as the Z timension so that the right-hand side connects smoothly to the Drain.

    • When you run the simulation, you might want to run it with a low real-time factor to be able to watch how the screwdriver actually screws the screws in.
