setTableRow
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > setTableRow

Syntax:

 
Path.setTableRow(Table:string, Row:integer) → boolean

The method setTableRow sets the number of the row of the Items table, which the user selected in a New List View.

Parameters

  • The parameter Table of data type string designates the table of the items table.

  • The parameter Row of data type integer designates the row which the user selected.

Return Value

The return value has the data type boolean.

Example:

 
MyDialog.setTableRow("MyListViewTable",4)
