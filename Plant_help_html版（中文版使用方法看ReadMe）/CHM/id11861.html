write
2D Reference Help > Information Flow Objects > FileInterface > Methods of the FileInterface > write

Syntax:

 
Path.write(Data:any) → boolean

The method write adds the data designated by the parameter of data type data to the end of the file, without starting a new line. The FileInterface places the cursor at the end of the entry. When the file does not exist, the FileInterface creates it. When the file is not open for write, the FileInterface opens it, writes the data, and closes it afterwards.

When the file cannot be opened, for example because the path to the file is invalid, the function fails with an error. To avoid this error, you can explicitly call the method open, which returns false if the file cannot be opened.

Parameter

The parameter Data of data type any designates the data that is appended to the end of the file.

Return Value

The return value has the data type boolean.

Example:

 
var FIF : object
FIF := FileInterface
FIF.write("ProcTime: ")
FIF.write(.machine.ProcTime,"[s]")
