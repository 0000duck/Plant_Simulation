Model Settings
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > Model Settings

The command Model Settings opens the dialog Model Settings in which you can select the settings for the currently open simulation model. Plant Simulation saves theses settings in the mode file. Click the tabs to access the respective set of features.

The Preferences set general settings, which apply to new simulation models!
