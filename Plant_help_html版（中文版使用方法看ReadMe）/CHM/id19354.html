p
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Attributes of the Distributions > p

Syntax:

 
Attribute_path.p

The attribute p sets the probability of the event of interest to occur.

Data Type of the Value You Can Assign

You can assign a value of data type real.

You can assign the following attributes/parameters to these distributions:

Distribution

Set of parameters

Range of values for the parameters

Binomial

n, p

integer n > 0, 0 p 1

Geometric

p, LowerBound, UpperBound

0 p 1, Upper Bound > Lower Bound

Hypergeometric

m, n, p

1 > n > m, 0 p 1
