setAntitheticRandomNumbers
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > setAntitheticRandomNumbers

Syntax:

 
setAntitheticRandomNumbers(AntitheticRandomNumbers:boolean) → boolean

The function setAntitheticRandomNumbers sets if Plant Simulation uses Antithetic Random Numbers (true) or normal random numbers (false).

Parameter

The parameter AntitheticRandomNumbers of data type boolean designates the action to be taken.

Return Value

The return value has the data type boolean and shows the previous setting.

Example:

 
setAntitheticRandomNumbers(true)
