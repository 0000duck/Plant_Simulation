DecimalSeparator
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Attributes for the Text Format of Lists and Tables > DecimalSeparator

Syntax:

 
Path.DecimalSeparator

The attribute DecimalSeparator sets the decimal separator, when saving the list designated by Path in text format.

The attribute applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Data Type of the Value You Can Assign

You can assign a value of data typestring: . (period) or , (comma).

Example:

 
MyStackFile.DecimalSeparator := "," -- comma
