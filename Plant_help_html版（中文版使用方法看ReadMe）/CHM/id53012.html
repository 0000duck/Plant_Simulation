BackgroundColor
2D Reference Help > Material Flow Objects > Frame > Attributes of the Frame > BackgroundColor

Syntax:

 
Path.BackgroundColor

The attribute BackgroundColor sets the background color of the Frame.

You can set the RGB values of the color with the method makeRGBValue.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
.Models.Frame.BackgroundColor := makeRGBValue(100,100,100)
.Models.Frame.BackgroundColor := 6579300 -- is the same as the color above
