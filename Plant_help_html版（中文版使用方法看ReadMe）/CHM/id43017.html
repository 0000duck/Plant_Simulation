Material Flow Properties
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Statistics of the Material Flow Objects > Statistics Report > Resource Statistics of Stationary Resources > Material Flow Properties

The topic material flow properties shows the following values, which are of interest:

The column

Shows the

Read-only attribute

Number of Entries

number of MUs that entered the object. When an MU enters the object, the object only counts the MU itself, not its contents: When a Transporter/Container transporting several parts enters the object, the number of entries increases by 1, not by the number of parts located on the Transporter/Container!

StatNumIn

Number of Exits

number of MUs that exited the object. The restrictions described under Number of Entries apply.

StatNumOut

Minimum Contents

minimum number of MUs that was located on the object The restrictions described under Number of Entries apply.

StatMinNumMU

Maximum Contents

maximum number of MUs that was located on the object. The restrictions described under Number of Entries apply.

StatMaxNumMUs

Relatively Empty

portion of the statistics collection period during which the object was Empty in relation to the time during which the object was available.

StatRelativeEmptyPortion

Relatively Full

portion of the statistics collection period during which all temporary storage places of the Buffer were occupied in relation to the time during which the object was available.

StatRelativeFullPortion

Relative Occupation without Interruptions

the sum of all dwelling times of all MUs during the statistics collection period during which the object was not paused and not failed. This sum is then divided by the capacity and the entire statistics collection period without pauses and failures.

StatRelativeOccupation

Relative Occupation with Interruptions

time during which the object was occupied and paused and failed divided by the capacity and the statistics collection period.
