startPauseIn
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods for the State of the Material Flow Objects > startPauseIn

Syntax:

 
Path.startPauseIn(StartPauseIn:time)

The method startPauseIn pauses the object designated by Path.

Parameter

The parameter StartPauseIn of data type time designates the time after which the pause starts.

The method sets the attribute Pause to true. The time interval has to be equal to or greater than zero.

Example:

 
ParallelStation.startPauseIn(10) -- in 10 seconds
