Add User-defined Entries to the Report
2D Reference Help > Tools > ExperimentManager > Viewing Evaluations > View Output Values of Interest > Add User-defined Entries to the Report

You can also add your own items to the report using the HtmlReport named UserReport which is inserted into the ExperimentManager. 

ExperimentManager.UserReport.appendConveyor(to_str("Observed waiting time ", Machine.statWaitingTime," in experiment ", ExperimentManager.currExpNo))
