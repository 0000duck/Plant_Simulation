Sigma
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Attributes of the Distributions > Sigma

Syntax:

 
Attribute_path.Sigma

The attribute Sigma sets the standard deviation of the distribution.

Data Type of the Value You Can Assign

You can assign a value of data type time.

You can assign the following attributes/parameters to these distributions:

Distribution

Set of parameters

Range of values for the parameters

Erlang

Mu, Sigma, LowerBound, UpperBound

Mu > 0, Sigma > 0, Upper Bound > Lower Bound

Laplace Distribution

Stream, Mu, Sigma, LowerBound, UpperBound

Mu can be any value, Sigma > 0, Upper Bound > Lower Bound

Logistic Distribution

Stream, Mu, Sigma, LowerBound, UpperBound

Mu can be any value, Sigma > 0, Upper Bound > Lower Bound

Lognormal

Mu, Sigma, LowerBound, UpperBound

Mu > 0, Sigma > 0, Upper Bound > Lower Bound

Normal

Mu, Sigma, LowerBound, UpperBound

Mu can be any value, Sigma > 0, Upper Bound > Lower Bound
