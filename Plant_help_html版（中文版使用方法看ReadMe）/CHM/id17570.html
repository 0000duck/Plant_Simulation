Default angle
2D Reference Help > Material Flow Objects > Turntable > Dialog Window of the Turntable > Tab Attributes > Default angle

Enter the angle in degrees to which the Turntable rotates, when you selected Go to default position. The default position is located at this angle. If you would like the Turntable to rotate to the side on which the start point is located, add 180 degrees to this angle.
