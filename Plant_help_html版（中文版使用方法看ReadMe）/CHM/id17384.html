VectorgraphicsBorderColor
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes for Vector Graphics of the Mobile Objects > VectorgraphicsBorderColor

Syntax:

 
MU-path.VectorgraphicsBorderColor

The attribute VectorgraphicsBorderColor sets the border color of the vector graphic of the MU designated by MU-path. Plant Simulation surrounds the fill color with the border color in the Border width that you select. The border color also sets the color of the direction arrow.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
.MUs.MyPart.VectorgraphicsBorderColor := 15780518 -- light blue
.MUs.MyPart.VectorgraphicsBorderColor := makeRGBValue(100,200,200)
