External Connections
2D Reference Help > Material Flow Objects > Interface > Dialog Window of the Interface > The View Menu > External Connections

The menu command External Connections opens a table displaying the paths of the external connections in the Frame in which the Interface object is located.

Column 1 contains the Connector at the exit of the Frame, column 2 the name of the succeeding object.
