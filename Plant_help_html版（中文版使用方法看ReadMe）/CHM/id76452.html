Data Source
Add-Ins Reference Help > Database Interfaces > ODBC > Dialog Window of the ODBC Interface > Data Source

To open the table Info Data Source, click this. It shows information about the used data source.

Entry

Description

DBMS name

Name of the DBMS (database management system)

DBMS version

Version of the DBMS

Server name

Name of the server belonging to the data source

Database name

Name of the current database. This can also be a folder, for example for dBase, or a file, for example for Access.

Qualifier location

Shows if a qualifier (database name) stands at the start or at the end of a table name identifier. Possible values are Qualifier at the start or Qualifier at the end.

Qualifier separator

Shows the qualifier separator.

Qualifier name

Qualifier name of the DBMS developer, for example DATABASE for Access or Workbook for Excel.

Qualifier usage

Lists the instruction groups in which a qualifier can be specified. Possible values are DML_STATEMENTS (all data manipulation statements), PROCEDURE_INVOC (procedure invocation statement), TABLE_DEF (all instructions for defining tables), INDEX_DEF (all instructions for defining indexes), or PRIVILEGE_DEF (all instructions for defining privileges).
