querySyncs
Add-Ins Reference Help > Inter-Process Communication Interfaces > Teamcenter > Methods of the Teamcenter Interface > querySyncs

Syntax:

 
Path.querySyncs(AINo:integer, ResultsTable:table)

The method querySyncs returns the Teamcenter syncs for the designated ApplicationInterface. You first have to query the application interface with the method queryApplicationInterfaces.

Parameters

  • The parameter AINo of data type integer designates the Teamcenter application interface for which you want to query syncs. The integer value is the position of the application interface on the table into which the method queryApplicationInterfaces wrote all available application interfaces.

  • The parameter ResultsTable of data type table designates the table into which Plant Simulation writes the names of the available syncs.

Example:

 
Teamcenter.login
Teamcenter.queryApplicationInterfaces(MyTable)
MyTable.CursorX := 1
MyTable.CursorY := 1
if MyTable.find("My application interface") 
   Teamcenter.querySyncs(MyTable.CursorY, MyTable)
   MyTable.CursorX := 1
   MyTable.CursorY := 1
   if MyTable.find("My sync") 
      Teamcenter.retrieveData(MyTable.CursorY)
end
Teamcenter.logout
