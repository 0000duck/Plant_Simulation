StatStoppedCount
2D Reference Help > Resource Objects > LockoutZone > Read-Only Attributes of the LockoutZone > StatStoppedCount

Syntax:

 
Path.StatStoppedCount

The read-only attribute StatStoppedCount returns how often the LockoutZone designated by Path stopped the stations.

Return Value

The return value has the data type integer.

Example:

 
print MyLockoutZone.StatStoppedCount
