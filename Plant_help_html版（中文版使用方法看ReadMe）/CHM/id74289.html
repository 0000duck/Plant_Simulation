bool_to_num
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Numerical Values > bool_to_num

Syntax:

 
bool_to_num(boolean) → integer

The function bool_to_num converts the number designated by the parameter of data type boolean into a number of data type integer.

Parameter

The parameter has the data type boolean.

Return Value

The return value has the data type integer. true returns 1, false returns 0.

Example:

 
var gross: real
var inclVAT: boolean
print bool_to_num(true)
print "Price:"
print gross + 0.16*gross*bool_to_num(inclVAT)
