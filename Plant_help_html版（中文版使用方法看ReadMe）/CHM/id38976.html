Time
2D Reference Help > Material Flow Objects > EventController > Dialog Window of the EventController > Time

The program shows the current simulation time of the simulation run in the box next to Time.

Click Time to change the display of the time between the

  • Relative time: The program resets the relative time to zero, when it starts the simulation run. This is the default.

  • Current time plus simulation time: The program adds the simulation time to the time and date at which it started the simulation run.

Let us assume that today is the second of January 2017 midnight and that the simulation is to run for two days.

  • For the setting relative time the program shows 2:00:00:0000, i.e., two days, after the simulation run is finished.

  • For the setting current time plus simulation time the program shows We, 2017/01/04 00:00:00.0000, i.e., Wednesday, January 04, 2017, midnight.

When you are running the simulation with the EventController buttons on the Home ribbon tab, Plant Simulation shows the time as it progresses in the status bar . When the simulation is finished, the status bar shows the end time .
