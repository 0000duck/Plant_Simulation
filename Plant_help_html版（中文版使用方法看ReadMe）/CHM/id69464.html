getIcon
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > getIcon

Syntax:

 
Path.getIcon(DialogItem:string) → string

The method getIcon returns the Image ID or the image name of the image designated by the parameter of data type string.

Parameter

The parameter DialogItem of data type string designates the dialog item.

Return Value

The return value has the data type string.

Example:

 
print MyDialog.getIcon("My_Image")
