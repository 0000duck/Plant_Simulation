  • Which distributions to select for the Processing time, the Set-up time, the Recovery time, the Cycle time, and the Failure times for the Plant Simulation objects.

For this, we provide Statistics Wizards.

You can:

  • Employ Distribution-Fitting with DataFit.

  • Use Distributions with Bounds.
