Modeling Hierarchically
Step-by-Step Help > Modeling in 3D > Modeling Hierarchically

Using hierarchical modeling techniques you can add any level of detail to the areas and machines you insert into your simulation model. Hierarchical modeling means that you create a machine, a production area, etc. in a Frame using the built-in object classes or classes you yourself designed. You can then insert this Frame, Machine for example, into another Frame in which you modeled a production area, and then insert the production area Frame into the Frame that contains your entire factory.
