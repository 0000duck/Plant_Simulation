standardDeviation
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Querying Statistics Values of List and Tables > standardDeviation

Syntax:

 
Path.standardDeviation([Range:listrange, ...])

The method standardDeviation returns the standard deviation from the mean of all values in a range of cells.

The method applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

The method only applies to entries of data types real or integer. Plant Simulation ignores entries of other data types in the ranges to be searched or considers their value to be 0.

Parameter

The parameters Range of data type listrange designate the range of cells.

Return Value

The return value has the data type real.

Example:

 
print MyStackFile.standardDeviation({*})
print MyQueueFile.standardDeviation({1}..{*})
print MyCardFile.standardDeviation({*})
print MyTableFile.standardDeviation({3,3}..{3,*})
