Using Redundant License Servers
Setting-Up and Starting > Licensing Plant Simulation > Using Redundant License Servers

When you use a floating license or a node-locked license, you can also employ three redundant license servers to significantly increase the availability of licenses within your computer network. The redundant servers are three separate computers within the network using the same license file which is required for the redundant license server mechanism.

As long as at least two of the three computers are available within the computer network, all licenses are available.

Enter the names and the ports of the license servers on the tab License. Then, select Use a license server triad.
