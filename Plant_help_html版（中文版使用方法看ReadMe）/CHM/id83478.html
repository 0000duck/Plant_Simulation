UniqueKeyColumnIndex
2D Reference Help > Information Flow Objects > TableFile > Attributes of the TableFile > UniqueKeyColumnIndex

Syntax:

 
Path.UniqueKeyColumnIndex

The attribute UniqueKeyColumnIndex sets if Plant Simulation applies a unique key to the user-defined column index (true) or not (false).

The attribute only works while you manually edit a TableFile. It does not work when writing to an index using a Method. Plant Simulation does not detect multiple entries via information flow.

You can assign a value of data type boolean.

Example:

 
Tab.UniqueKeyColumnIndex := true
