NumSensors
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Read-Only Attributes of Sensors of the Material Flow Objects > NumSensors

Syntax:

 
Path.NumSensors → integer

The read-only attribute NumSensors returns the number of sensors defined for the object (Track, TwoLaneTrack, Conveyor) designated by Path.

Creating and manipulating sensors via SimTalk for the Transporter only works if you select the loading space Types Track or Line!

Return Value

The return value has the data type integer.

Example:

 
print MyTrack.NumSensors

Example:

 
print .MUs.Transporter:1.NumSensors
