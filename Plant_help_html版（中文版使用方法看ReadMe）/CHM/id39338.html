setCursor
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Miscellaneous Methods of the TableFile > setCursor

Syntax:

 
Path.setCursor(Column:any, Row:any) → boolean

The method setCursor sets the internal cursors of the TableFile designated by Path to the designated cell. The method find starts searching from the current cursor position and sets the cursor to the next found item. The methods maxand min set the cursor to the maximum or minimum value of the given range.

Parameters

  • The parameter Column of data type any designates the column, i.e., the internal cursor CursorX.

  • The parameter Row of data type any designates the row, i.e., the internal cursor CursorY.

Return Value

The return value has the data type boolean. It tells if the cursors were set to the desired position.

Example:

 
table.setCursor(1,1)

Example:

 
MyTableFile.setCursor(1, "Test") -- returns false if the index Test does not exist
