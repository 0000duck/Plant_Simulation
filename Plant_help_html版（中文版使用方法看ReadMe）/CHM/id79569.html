Append Segment
3D Reference Help > 3D Reference > Context Menus in 3D > Context Menu of the Selected Animatable Object in the 3D Scene > Segments > Append Segment

The context menu command Append Segment opens the dialog Edit Parameters of Curve. The settings of this dialog are the same as those of the corresponding dialog in Plant Simulation 2D. You can then add additional curved or straight segments to the selected length-oriented object.

Only length-oriented objects provide this command.

You can then extend the polycurve of the selected length-oriented object by appending points to the extrusion polycurve path that defines it.

Just like in 2D:

  • 3D shows a movable line, which connects a path tool at the starting point with another path tool attached to the cursor.

  • You can insert straight path segments by clicking the left mouse button. You can insert curved path segments by holding down the Shift key and by clicking the left mouse button.

  • You can terminate curve mode by clicking the right mouse button or by pressing the Esc key.
