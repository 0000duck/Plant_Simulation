Most recently used
2D Reference Help > Material Flow Objects > FlowControl > Dialog Window of the FlowControl > Tab Entry Strategy > Strategy > Most recently used

To receive the MUs from the predecessor, from which the FlowControl received an MU the last time, select Most recently used from the drop-down list. As long as this predecessor can move MUs on, the FlowControl receives them from this predecessor.
