File path
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Point Cloud > File path

Click the button and select the point cloud file that you want to import. To remove the point cloud from the Frame, delete its name from the text box.

Provided the path is relative, it refers to the folder in which your simulation model is located.

The abbreviation .pod stands for point database.
