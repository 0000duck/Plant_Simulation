SimTalk Reference > General SimTalk Functions > Names > Anonymous Identifiers > ?

The anonymous identifier ? (question mark) designates the material flow object or the control (Method) that called the Method. Employing the anonymous identifier ? allows a control to be used without modifications by several objects.

Example:

 
?.Cont.move(E2) -- moves the contents of the object that 
                 -- called the method to the object E2
