Activate 3D
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Window Ribbon Tab > Activate 3D

The command Activate 3D activates or deactivates 3D (compare the 3D Reference Help).

After you started 3D, the 2D Frame window shows the x-axis and the y-axis in red. The point where these two red lines intersect is the point, which will be mapped to the zero point, i.e., the Position of the grid, in 3D.

You can set the origin of the grid in the Frame in 2D with the attribute AxesOrigin.
