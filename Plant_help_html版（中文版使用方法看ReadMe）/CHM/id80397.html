getCommandLineArg
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > getCommandLineArg

Syntax:

 
getCommandLineArg(CommandLineArgument:string, byref Value:string) → boolean

The function getCommandLineArg queries if a command-line argument has been passed to Plant Simulation when starting it.

Parameters

  • The parameter CommandLineArgument designates the command-line argument that has been passed to Plant Simulation when starting it.

  • The parameter Value is assigned the value of the command-line argument above. Note that some command-line arguments do not have a value. In this case Plant Simulation assigns an empty string.

Return Value

The return value has the data type boolean.

It is true when Plant Simulation was started with the specified command-line argument. Otherwise it is false.

Suppose we started Plant Simulation with these command-line arguments: 

PlantSimulation14.exe /L Runtime /abc 42 /xyz /myfile "C:\Program Files\a.txt"

Example:

 
var arg: string
if getCommandLineArg("abc", arg) 
   print arg -- outputs 42
end
if getCommandLineArg("myfile", arg) 
   print arg -- outputs C:\Program Files\a.txt
end
if getCommandLineArg("xyz", arg) 
   print arg -- outputs an empty string
end
