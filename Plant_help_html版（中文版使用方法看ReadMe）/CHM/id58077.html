Align to Grid
2D Reference Help > Material Flow Objects > Frame > The Frame Window > Icons Ribbon Tab > Align to Grid

The command Align to Grid places the reference points of the icons of the selected objects onto grid points, when Show Grid on the General ribbon tab of the Frame was deactivated while you placed the objects. It thus aligns objects to the grid that weren’t aligned while you inserted them, for example when the grid was deactivated.

This only works, when you activated the grid on the General ribbon tab.
