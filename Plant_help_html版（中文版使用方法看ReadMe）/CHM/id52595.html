exitFree
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods for the State of the Material Flow Objects > exitFree

Syntax:

 
Path.exitFree → boolean

The method exitFree returns if the exit of the object designated by Path is not Blocked by an MU/Transporter (true), or blocked (false) or if the maximum number of MUs is located on the object, i.e., its capacity is exhausted or not.

Return Value

The return value has the data type boolean.

Example:

 
print MyConveyor.exitFree
print MyTrack.exitFree

Only the Track and the Conveyor provide this method.
