3D-related SimTalk Functions
SimTalk Reference > 3D-related SimTalk Functions > 3D-related SimTalk Functions

In addition to the general SimTalk functions Plant Simulation provides a number of functions for accessing the objects in 3D.

You can view and access the attributes and methods of the 3D objects by clicking Auto Complete on the Edit ribbon tab of the method editor.

To access the attributes and methods of the 3D objects, type in _3D, followed by a period.

Most of the 3D attributes and 3D methods listed below use the array data types for setting and getting values.
