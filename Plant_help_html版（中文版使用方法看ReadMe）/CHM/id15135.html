_3D.createSphere
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating 3D Shapes > _3D.createSphere

Syntax:

 
_3D.createSphere(BasePoint:length[3], Radius:length [,GraphicGroupName:string]) → integer

The method _3D.createSphere creates a sphere in the scene window.

Parameters

  • The parameter BasePoint is an array of data type length which sets the base point of the sphere.

  • The parameter Radius of data type length sets its radius.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described underThe Parameters GraphicPath and GraphicGroupName.

Return Value

The return value has the data type integer and specifies the numerical index of the sphere, compare _3D.getGraphicIndex.

Example:

 
_3D.createSphere([2.0,2.0,1],2)

Example:

 
_3D.createSphere([2.0,2.0,1],2,"deco")

Example:

 
var index := _3D.createSphere([2.0,2.0,1],2,"deco")
_3D.setGraphicMaterial([index], makeRGBValue(255,0,0), 0, 0, 0, 0, 0, "deco")
