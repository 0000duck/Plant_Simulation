Order next part only when correct part type was processed
Add-Ins Reference Help > Object Libraries > Kanban Objects > KanbanSingleProc > Dialog Window of the KanbanSingleProc > Tab Advanced2 > Order next part only when correct part type was processed

To wait until a certain ordered part has been processed, select the check box.

To create an order each time after a part has been processed on the station, clear the check box. This ensures that between the supplier and the KanbanStation one part is on its way.
