add
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > add

Syntax:

 
Path.add(ValueToBeAdded:any)

The method add either adds a numerical value to all entries of the value column of a TimeSequence, or combines the time and value columns of two TimeSequence objects.

Parameter

If the parameter ValueToBeAdded of data type any which is to be added is a numerical value, Plant Simulation adds it to each entry of the value column. The time column remains unchanged.

In case the parameter is another TimeSequence, Plant Simulation adds time value pairs in sections. The values in the time columns determine the extent of the individual sections. If for a certain time value there are multiple entries in one time sequence, and only one entry in the other time sequence, the single entry will be added to all other entries. If both time sequences have the same number of multiple entries with a particular time value, the entries will be combined in pairs and added afterwards. If the number of entries with the same time value differs, the last entry will be duplicated, before the addition is performed, until the number of entries with the same time matches.

Example:

 
MyTimeSequence.add(5)
shiftA.add(shiftB)
