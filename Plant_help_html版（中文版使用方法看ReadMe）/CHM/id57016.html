ShiftCalendarObject
2D Reference Help > Material Flow Objects > Frame > Attributes of the Frame > ShiftCalendarObject

Syntax:

 
Path.ShiftCalendarObject

The attribute ShiftCalendarObject sets the ShiftCalendar, which controls the Frame designated by Path.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
EngineAssembly.ShiftCalendarObject := MyShiftCalendar
