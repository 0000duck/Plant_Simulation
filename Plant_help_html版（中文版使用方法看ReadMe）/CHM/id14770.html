remove
2D Reference Help > Information Flow Objects > FileInterface > Methods of the FileInterface > remove

Syntax:

 
Path.remove → boolean

The method remove closes the file if it was open and deletes it for good.

Return Value

The return value has the data type boolean. Plant Simulation returns true if it succeeded in deleting the file. If the method fails, check if the file exists and make sure that it is not read-only.

Example:

 
MyFileInterface.remove
