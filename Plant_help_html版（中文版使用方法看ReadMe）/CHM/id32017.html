BrokerPath
2D Reference Help > Resource Objects > WorkerPool > Attributes of the WorkerPool > BrokerPath

Syntax:

 
Path.BrokerPath

The attribute BrokerPath sets the path to the Broker, which the WorkerPool uses to provide the services.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
print MyWorkerPool.BrokerPath
MyWorkerPool.BrokerPath := myBroker
MyWorkerPool.BrokerPath := Broker3
