setSensitive
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > setSensitive

Syntax:

 
Path.setSensitive(DialogItem:string, Activated:boolean) → boolean

The method setSensitive sets if the Dialog shows and activates a dialog item in the dialog window of the dialog you defined or shows it and gray it out.

Parameters

  • The parameter DialogItem of data type string designates the dialog item.

  • The parameter Activated of data boolean sets if the dialog item will be shown (true) or be shown and grayed out (false).

The method applies to all dialog items, except for the Tab Control, the Tab Page and the Image.

Return Value

The return value has the data type boolean. It shows if the state of the dialog item, activated or deactivated, has changed or not.

Example:

 
if MyDialog.setSensitive("MyButton",false) 
   print "The button 'MyButton' was deactivated."
else
   print "The button 'MyButton' has already been deactivated."
end
