Assign a background color of its own
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Background > Assign a background color of its own

To assign a background color to the selected Frame, select this. 3D activates the other settings on the tab.

If you do not want to assign a background color to the selected Frame/scene, deactivate the check box. 3D then uses the background color of the parent Frame of the selected Frame.
