Activate Profiler
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Debugger Ribbon Tab > Activate Profiler > Activate Profiler

The command Activate Profiler activates or deactivates the Profiler. We recommend to only activate the Profiler when you actually intend to collect profiler data, as activating it all the time may have negative effects on the run time of your simulation model.
