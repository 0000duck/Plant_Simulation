StatTspReadyTime
2D Reference Help > Mobile Objects > Transporter > Read-Only Attributes of the Transporter > StatTspReadyTime

Syntay:

 
MU-path.StatTspReadyTime

The read-only attribute StatTspReadyTime returns the sum of all ready-times of the Transporter designated by MU-path.

Return Value

The return value has the data type time.

Example:

 
print .MUs.Transporter:1.StatTspReadyTime
