formFeed
2D Reference Help > Information Flow Objects > FileInterface > Methods of the FileInterface > formFeed

Syntax:

 
Path.formFeed → boolean

The method formFeed adds a FormFeedcharacter to the end of the file and places the cursor at the end of the entry. When printing, the printer starts a new page after this character.

If the file is not open, the FileInterface opens it and leaves it open after accessing it

Return Value

The return value has the data type boolean.

The method returns true if it succeeded in writing data to file, false if it did not succeed.

Example:

 
MyFileInterface.formFeed
