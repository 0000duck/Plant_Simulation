Show Origin
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Show Origin

The context menu command Show Origin shows the origin of the selected object, i.e., the highlight jumps from the selected object to the object in the Class Library from which the selected object was derived. If the highlight does not change, then the object was not derived, but duplicated!
