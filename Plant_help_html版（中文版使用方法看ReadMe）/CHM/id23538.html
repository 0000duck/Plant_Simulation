Dialog Items of the Fluid Objects
2D Reference Help > Fluid Objects > Shared Properties of the Fluid Objects > Dialog Items of the Fluid Objects > Dialog Items of the Fluid Objects

The fluid objects share a number of dialog items and menus. To open the dialog window of an object, double-click its icon.

After opening the dialog, Plant Simulation shows the current values. Enter new properties into the text boxes or select them from the drop-down lists. Click OK or Apply to accept changes you made and to update the dialog. Instead, you can also press the F5 key to show the most current results of a simulation run on some of the tabs.
