portNumber
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > portNumber

Syntax:

 
portNumber → integer

The function portNumber returns the TCP/IP port number, which Plant Simulation uses for incoming connections, such as Browser and Explorer connections.

For security reasons Plant Simulation only allows access from the local computer. To access Plant Simulation from another computer, create the following registry key of data type string: HKEY_CURRENT_USER/SOFTWARE/Tecnomatix/Tecnomatix Plant Simulation X.X/AllowHTTPAccessFrom. If you want to set the key for all users, you can create the string for HKEY_LOCAL_MACHINE/SOFTWARE/Tecnomatix/Tecnomatix Plant Simulation X.X/AllowHTTPAccessFrom. Replace X.X in the key above with the major and the minor version of your version of the program. The contents of the key sets the IP address of the computer or of the Frame from which access is allowed.

To allow access from anywhere, enter an asterisk * instead of the IP address.

Return Value

The return value has the data type integer. It is 0 if the web server has not been started.

Example:

 
print portNumber
