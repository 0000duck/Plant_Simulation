_3D.Width
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Defining Length-oriented Objects > _3D.Width

Syntax:

 
Path._3D.Width

The attribute _3D.Width sets the width of the length-oriented object. The width only applies for the simulation when the setting Transfer length in 2D is activated.

Parameter

The parameter of data type real sets the width in meters.

Example:

 
Conveyor._3D.Width := 2
