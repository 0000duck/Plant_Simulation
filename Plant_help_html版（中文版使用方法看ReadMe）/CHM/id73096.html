getValue
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > getValue

Syntax:

 
Path.getValue(DialogItem:string) → string

The method getValue returns the changed value of the dialog item designated by the parameter of data type string.

This method applies to the dialog items of type New Drop-Down List Box, New Check Box, New Radio Button, New List Box, New Edit Text Box, and New Tab Control. When you apply the method to the Tab Control, Plant Simulation returns the name of the active tab.

Return Value

The return value has the data type string.

Example:

 
var value: string
var number : integer
value := MyDialog.getValue("unit")
number := str_to_num(value)
