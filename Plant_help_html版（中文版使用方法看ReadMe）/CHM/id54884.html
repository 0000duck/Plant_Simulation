Interval
Add-Ins Reference Help > Genetic Algorithms > GARangeAllocation > Dialog Window of the GARangeAllocation > Tab Contents > Interval

Enter the interval between the items into the cells in the column Maximum. You might, for example enter values of 0, 20, 40, 60, 80, 100 for the solution items for the settings minimum 0, maximum 100 and interval 20.
