List (Type)
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > User-defined Distributions > List (Type)

You can use the List(Type) distribution to determine the times depending on the type of MU.

Type in the name of the table into which you want to enter the required data. Clicking Apply in the dialog of the object automatically formats the table so that the data types of the columns match the parameters.

Enter all possible MU types in column 1 and the respective times in seconds in column 2.

During the simulation run Plant Simulation reads the time at which an MU arrives from that list. If you did not enter a list, Plant Simulation displays an error message.
