XMLInterface
2D Reference Help > Information Flow Objects > XMLInterface > XMLInterface

The XMLInterface is an easy to use interface to read and extract data stored in XML files. Basically an XML document contains data in ASCII format, i.e., text that is structured in a certain way.

You can, for example, use the XMLInterface to read data, which you exported from Process Designer or an XML database.

You can select to:

  • Read and Write Data Sequentially

  • Read and Access Data Randomly

  • Access and Traverse Data Randomly

You can then use the data, which the XMLInterface read, and which you manipulated with a Method, to run a simulation in Plant Simulation. You might afterwards use the methods write and writeElement to write the simulation results back to an XML file and continue working with these simulation results in Process Designer or another program.

Working with the XMLInterface requires that you are familiar with XPath (XML Path Language) source code.

To add the XMLInterface to your simulation model, click Manage Class Library > Basic Objects > InformationFlow > XMLInterface on the Home ribbon tab.
