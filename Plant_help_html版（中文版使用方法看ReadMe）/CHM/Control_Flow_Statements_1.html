Control Flow Statements
SimTalk Reference > General SimTalk Functions > Control Flow Statements > Control Flow Statements

A control is not always executed in a linear fashion. Different events influence the course of execution a control takes and require appropriate reactions.

Plant Simulation provides these language constructs for programming control flow statements:

  • Branching with if-else-end

  • Multiple Branching with if-elseif-end

  • Conditional Expressions with When-Then-Else,

  • The control flow statement Switch / inspect

  • Loops
