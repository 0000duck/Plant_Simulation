pushList
2D Reference Help > Information Flow Objects > QueueFile and StackFile > Methods of the QueueFile and the StackFile > pushList

Syntax:

 
Path.pushList(List:any)

The method pushList places the list designated by the parameter of data type any into the StackFile or the QueueFile.

Parameter

The parameter List of data type any has to reference a list of data type stack, queue, a list, or a single column of data type table. You can use the method copy if you only want to place a range of a list into a StackFile or a QueueFile. The data types of the data you want to insert have to match the data types of the StackFile or QueueFile into which you insert it.

Example:

 
-- pastes the contents of the CardFile into the StackFile
MyStackFile.pushList(MyCardFile.copy)
-- pastes the specified range from the CardFile, starting in cell 4,
-- into the QueueFile
MyQueueFile.pushList(MyCardFile.copy({4}..{*}))
