Cont
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Miscellaneous read-only attributes of the Material Flow Objects > Cont

Syntax:

 
Path.Cont

The read-only attribute Cont returns the MU that is located on the object designated by Path.

  • When the object has a capacity of 1, Cont returns the MU that is located on the object.

  • When the object is an AssemblyStation or a DismantleStation, Cont returns the main part.

  • When the object has capacity greater than 1, Cont returns the MU that can exit next.

  • When no MU is located on the object, Cont returns VOID.

When the Worker, who is carrying a part, unloads that part, the method Worker.Cont returns the unloaded part.

Return Value

The return value has the data type object.

Example:

 
if MyStation.Cont /= VOID
  print MyStation.Cont.name
end
ParallelStation.Cont.CurrIconNo := 1
