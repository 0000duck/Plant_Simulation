Minimum Contents
Step-by-Step Help > Modeling in 2D > Modeling the Flow of Materials, Basics > Transferring Parts from Station to Station > Select an Exit Strategy > Minimum Contents

To move the part to the successor that contains the least number of MUs, select Min. contents from the drop-down list.

This only works correctly, when resource statistics of the successor or of the predecessor is active.

  • When you select Blocking , the object moves the MU to the next successor.

  • When you clear Blocking , the object moves the MU the next successor, which can receive the MU.
