updateNodes
2D Reference Help > Information Flow Objects > XMLInterface > Methods of the XML Interface > updateNodes

Syntax:

 
Path.updateNodes(NodesTable:list) → boolean

The method updateNodesupdates the nodes you selected with the method getNodes, and writes them into a table.

Parameter

The parameter NodesTable of data type table designates the name of the table.

You can then manipulate the contents of the resulting table, and finally update the selected nodes. It applies when you randomly access data.

Return Value

The return value has the data type boolean.

Example:

 
-- update the selected nodes of the document
var tbl: table
MyXMLInterface.filename := "D:\MSXML 4.0\books.xml"
MyXMLInterface.openDocument
-- select the nodes to be changed
tbl := MyXMLInterface.getNodes("/catalog/book[title='Midnight Rain']", 1)
-- update the value
tbl[5,1][3,3] := "TEST"
-- write the changed data
MyXMLInterface.updateNodes(tbl)
MyXMLInterface.filename := "D:\MSXML 4.0\tmp.xml"
MyXMLInterface.write
MyXMLInterface.close
