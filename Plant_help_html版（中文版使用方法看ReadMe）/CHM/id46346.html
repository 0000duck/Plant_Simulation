Tab Labels
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Labels > Tab Labels

Enter a Title and Subtitle, and labels for the X-axis, the Y-axis and the Z-axis of the Chart. Select if and where Plant Simulation shows the Legend and open a table into which you enter Annotations on the tab Labels.

When you enter neither Title nor Subtitle, the Chart might, under certain circumstances, cut off the upper part of they Range. Enter a blank space into the text box Subtitle to prevent this.
