Shift calendar
2D Reference Help > Fluid Objects > Tank > Dialog Window of the Tank > Tab Attributes > Shift calendar

Click this and select the ShiftCalendar in the dialog Select Object. This ShiftCalendar object contains the data of the shifts in your installation and controls during which shifts the object works.

Or you can select the ShiftCalendar in a Frame, drag it to the text box and drop it there.

This automatically enters the object into the list of Objects on the tab Resources of the ShiftCalendar.
