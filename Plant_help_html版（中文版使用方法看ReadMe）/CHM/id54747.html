Active
Add-Ins Reference Help > Inter-Process Communication Interfaces > OPCClassic > Attributes of the OPC Interface > Active

Syntax:

 
Path.Active

The attribute Active activates the OPC Classic Interface (true) or deactivates it (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
OPCClassicInterface.Active := true
