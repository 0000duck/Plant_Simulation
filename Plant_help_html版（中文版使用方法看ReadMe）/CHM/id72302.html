_3D.getGraphicIndex
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.getGraphicIndex

Syntax:

 
Path._3D.getGraphicIndex(GraphicPath:any[] [,GraphicGroupName:string]) → integer

The method _3D.getGraphicIndex returns the numeric index of the designated graphic of the object designated by Path.

Parameter

  • The parameter GraphicPath is an array of data type any into which the method writes the numeric index. The parameter is described underThe Parameters GraphicPath and GraphicGroupName.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Return Value

The return value has the data type integer.

Example:

 
var index : integer := MyStation._3D.getGraphicIndex(["Wheels"])
 -- the variable contains the numerical index of the graphic
