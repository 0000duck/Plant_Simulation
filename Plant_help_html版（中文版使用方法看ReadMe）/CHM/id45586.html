ResSetUp
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods for the State of the Material Flow Objects > ResSetUp

Syntax:

 
Path.ResSetUp → boolean

The read-only attribute resSetUp returns if the object designated by Path is Setting-Up and not failed, not paused, or not stopped (true) or not (false).

Return Value

The return value has the data type boolean.

Example:

 
if machA.ResSetUp
@.move(machineB)
end
