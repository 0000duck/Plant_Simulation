Strategy
2D Reference Help > Material Flow Objects > Converter > Attributes of the Converter > Strategy

Syntax:

 
Path.Strategy

The attribute Strategy sets the Strategy of the Converter.

  • Default exit makes all parts exit through the Default exit which you select to the successor attached to that exit.

  • Straight conveys the part straight through to the next station.

  • Feed in allows parts from a branch line to only enter the main line when no part is located on the main line within the free space which you enter. The free space is the distance between two successive parts on the main line.

  • MU Attribute moves the part on according to a built-in or a user-defined attribute of the part. You can set the attributes with the method setAttributeList. You can set the default exit with the attribute DefaultExit.

  • MU Name moves the part on according to its name. You can set the default exit with the attribute DefaultExit. You can set the attributes with the method setAttributeList.

    The numbers designate the side of the Converter at which the part exits. The figures below illustrate the sides depending on the direction in which you inserted the Converter into your simulation model.

  • Method conveys the part according to the source code you enter into the strategy method in which you determine the side of the Converter at which the part will exit the Converter. You can set the control with the attribute StrategyCtrl.

  • Method at Converting Point only determines the target at the converting point, not already before the part enters the Converter. This strategy is only available for a capacity of 1.

    The part always stops at the converting point. Then, Plant Simulation calls the method for determining the target. In the method you can, for example use ?.ExitForMU := 1. You can set the control with the attribute StrategyCtrl.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyConverter.Strategy := "Straight"
