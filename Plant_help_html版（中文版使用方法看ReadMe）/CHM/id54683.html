Dialog Window of the Container
2D Reference Help > Mobile Objects > Container > Dialog Window of the Container > Dialog Window of the Container

Double-click the icon of a Container, which is located on one of the material flow objects in the Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Mobile Units in the Toolbox. Enter settings of your choice to produce the desired results. The dialog items are described under Dialog Items of the Objects.
