Dialog Window of the Chart
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Dialog Window of the Chart

Double-click the icon of the object Chart in the Frame or double-click the Chart window to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab User Interface in the Toolbox. The dialog items are described under Dialog Items of the Objects.

When you click Apply, Plant Simulation applies the settings you selected in the dialog and shows the records. When you resize the Chart window, Plant Simulation also resizes the graph. Double-click anywhere in the display window to open the dialog of the Chart.
