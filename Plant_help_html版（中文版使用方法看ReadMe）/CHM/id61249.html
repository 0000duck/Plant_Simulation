read
2D Reference Help > Information Flow Objects > CardFile > Methods of the CardFile > read

Syntax:

 
Path.read(Row:integer)

The method read returns the contents of a cell.

The method read returns the contents of the cell without removing it. If you want to read and remove the contents, use the method [ ].

Parameter

The parameter of data type integer designates the position of the cell.

The data type of the return value is the same as the data type of the CardFile.

Example:

 
print MyCardFile.read(2)
