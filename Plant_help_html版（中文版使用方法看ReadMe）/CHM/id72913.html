Embed
2D Reference Help > Information Flow Objects > FileLink > Attributes of the FileLink > Embed

Syntax:

 
Path.Embed

The attribute Embed sets if the FileLink embeds the file into the Plant Simulation Frame (true) or not (false). Note that this attribute is not inherited and that it is set to false by default.

To embed the file into the Plant Simulation model, assign true. Then that version of the file will become part of the simulation model, meaning the program does not mind, when you move the original file to another location. Changes to file after embedding it will naturally not be reflected if you embed it. Embedding may significantly increase the file size of your .spp model file.

When you assign false, the program creates a link to the file in the file system of your computer. If you move or delete the file, the link will become invalid and Plant Simulation will not be able to find the file and open it.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
Frame1.FileLink2.Embed := false
