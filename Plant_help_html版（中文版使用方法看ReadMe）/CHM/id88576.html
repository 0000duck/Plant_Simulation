Visualization
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > General > Visualization

Select how you want to create your simulation model:

  • Only in 2D. This creates the model consisting only of the 2D part.

  • In 2D and in 3D. This creates the model consisting of a 2D part and a 3D part.

  • Only in 3D. This creates the model consisting only of the 3D part.

We recommend to either create in 2D only or in 3D only. The 2D and 3D mode provides fewer modeling features as the exclusive 2D only or 3D only mode.

When you created the model in 2D only and need to create its 3D part at a later point in time, you can also click on the Home ribbon tab.

Depending on the setting you selected, the simulation model looks slightly different and provides different tabs on the ribbon bar. Likewise SimTalk provides different attributes and methods.
