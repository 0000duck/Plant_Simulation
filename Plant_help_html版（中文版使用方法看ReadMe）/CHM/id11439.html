insertTriggeredMeth
2D Reference Help > Information Flow Objects > Trigger > Methods of the Trigger > insertTriggeredMeth

Syntax:

 
Path.insertTriggeredMeth(Method:object)

The method insertTriggeredMeth inserts the Method designated by the parameter of data type object into the list Triggered Methods of the Trigger designated by Path.

Parameter

The parameter Method of data type object designates the Method that is to be inserted.

Example:

 
MyTrigger.insertTriggeredMeth(&Shift3)
