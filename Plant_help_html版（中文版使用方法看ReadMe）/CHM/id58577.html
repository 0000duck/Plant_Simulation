Declare a Local Variable at the Beginning of the Source Code
SimTalk Reference > General SimTalk Functions > Constants and Variables > Local Variables > Declare a Local Variable at the Beginning of the Source Code

Enter the name of your local variable to be declared after the keyword var, followed by one or several identifiers for the local variables you are defining. After that you can enter a colon and define the data type of the local variable.

If you would like to declare several variables of the same type, separate their individual names with commas, followed by a colon, the data type and a semicolon.

Example:

 
var index: integer    -- single declaration
var VAT, loctax: real -- two variables with the same data type
var customers: list   -- customer list
var orders: table[integer,string,real] -- table with three columns
var complaints: queue[time]            -- queue of data type time
