2D Reference Help > Material Flow Objects > DismantleStation > Dialog Window of the DismantleStation > The View Menu > Contents

The menu command Contents opens a list of all MUs that were dismantled on the DismantleStation. Note that the main MU is displayed in blue, while the mounting parts are displayed in black.
