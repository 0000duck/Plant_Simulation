getEntryList
2D Reference Help > Material Flow Objects > FlowControl > Methods of the FlowControl > getEntryList

Syntax:

 
Path.getEntryList(EntryList:any)

The method getEntryList returns the contents of the internal entry list of the FlowControl designated by Path for the Entry strategies Percentage or Cyclic sequence, and writes it into a list.

Parameter

The parameter EntryList of data type any designates the name of the list.

Example:

 
var tab: list
MyFlowControl.getEntryList(tab)
MyFlowControl1.setEntryList(tab)

Example:

 
MyFlowControl.getEntryList(CardFile)
