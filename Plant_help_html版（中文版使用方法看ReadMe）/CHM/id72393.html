SampleInterval
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > SampleInterval

Syntax:

 
Path.SampleInterval

The attribute SampleInterval sets the period of time between two successive sampling actions, when you selected Sample mode.

Data Type of the Value You Can Assign

You can assign a value of data type time.

Example:

 
MyChart.SampleInterval := 3600
myMyChart.sampleInterval := 0 -- Prevents the Chart from being updated.
                              -- To update values, use the Method ‘update’.
