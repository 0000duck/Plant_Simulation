Methods of the Importer
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods of the Importer > Methods of the Importer

The importer provides these methods:

getAlternativeServices, getExporters, getImportedServices, getServices, import, importExporter, releaseExporters, setAlternativeServices, setServices, startProcessing, stopProcessing, and startSetup.

Only the Station, the ParallelStation, the AssemblyStation, the DismantleStation, the Mixer, the Portioner, the DePortioner, and the Tank can import services, which you define on the Tab Importer.

In general 0 designates the failure-importer (failImp), 1 the set-up-importer (setupImp), 2 the processing-importer (imp), and 3 the transport-importer (transportImp).

  • Plant Simulation addresses processing-importer methods as sub-methods of imp.

    Example:

    		 
    ParallelStation.imp.releaseExporters

  • Plant Simulation addresses setup-importer methods as sub-methods of setupImp. The setup-importer shares all attributes and methods with the processing-importer. The only exception are the set-up services.

    Example:

    		 
    MyAssembly.setupImp.getServices

  • Plant Simulation addresses failure-importer methods as sub-methods of failImp.

    Example:

    		 
    MyStation.failImp.releaseExporters

  • Plant Simulation addresses transport-importer methods as sub-methods of transportImp.

    Example:

    		 
    MyRemovalStation.transportImp.setServices(TransportServices)
