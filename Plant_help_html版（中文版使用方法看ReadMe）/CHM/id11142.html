Changed
2D Reference Help > Information Flow Objects > TableFile > Attributes of the TableFile > Changed

Syntax:

 
Path.Changed

Plant Simulation sets the attribute Changed to true when the contents of the table changed. You can reset the attribute to false at any time to detect a change at a later point in time.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
while true 
   MyTableFile.Changed := false
   stopuntil MyTableFile.Changed
   print "Contents of the table has changed."
   print "New sum of all values: ", MyTableFile.sum"
end
