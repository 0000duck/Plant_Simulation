Insert Rack
3D Reference Help > 3D Reference > Edit Ribbon Tab > Insert Rack

The command Insert Rack creates and inserts a storage rack into the selected Frame.

  • Select the Graphic group to which you want to add the new graphic. The list shows the graphic groups which are defined for the object.

    • Default (external): Designates the standard graphic group of the 3D objects. It initially is external and represents the object towards the outside. When you deactivate Show External Graphics, 3D hides the graphic groups designated as Default.

    • One of the user-defined graphic groups you added on the Tab Graphics. Newly created 3D Frames by default provide the graphic group Deco (internal), which are intended for graphics for decorating the interior view of the Frame.

  • Enter the bay width, the bay height, and the board depth of the rack that you want to create into the dialog Create Rack.

  • Enter the number of storage bays per board of the rack.

  • Enter the number of boards (board count) that form the rack. Note that the topmost shelf is now included in the amount of shelves (board count). In the example below the rack has four shelves.

  • Click the color box next to Board material to assign a material to the material of the storage boards.

    • Click to copy the current material settings you selected. You can then paste these settings into the graphic of another object.

      The command also copies the corresponding SimTalk code to the clipboard. You can then paste this code into a Method object to set the material of another graphic. The syntax is described under Generate SimTalk Code and Copy to Clipboard.

    • Click to paste the material settings that you copied to the graphic of the object.

  • Click the color box next to Post material to assign a material to the posts of the rack.

  • Select how you would like to optimize the graphic of the rack:

    • For Post-processing Capability: This setting creates a highly granular structure of the graphic.

    • For Performance: This setting creates a particularly flat structure of the graphic.

  • To create the rack, click Create. 3D attaches the shape to the mouse pointer. Drag the mouse to the position at which you would like to insert the rack and click the left mouse button. To cancel inserting, you can click the right mouse button or press the Esc key.

  • To move the rack to the location of your choice, click it and drag the mouse.

  • To change the size of the rack, select it and select 3D Properties > > Tab Transformation. Then change the size under Scale.

  • To change the rack settings of an existing rack, click it with the right mouse button and select Edit 3D Properties. Here you can also select if the rack is an obstacle for the worker.

  • To assign a material or a color to the rack, select it and then click 3D Properties > Tab Material on the ribbon tab Start.

You can also edit the dimensions of the rack by clicking Show Manipulators on the Edit ribbon tab.
