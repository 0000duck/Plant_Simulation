Accumulated
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > Accumulated

Syntax:

 
Path.Accumulated

For the setting Category > Histogram the attribute Accumulated sets if the Chart creates an accumulated histogram (true) or not (false). The accumulated histogram adds the sum of all previous values to each value. The last value thus is 100 percent. Compare this example

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyChart.Accumulated := true
