str_to_length
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Physical Data Types > str_to_length

Syntax:

 
str_to_length(string) → length

The function str_to_length converts the data designated by the parameter of data type string into a number of data type length. You can enter one of these unit into which the function converts the string: mm, m, cm, km, in, ft, yd, or mi.

If you do not enter a unit, Plant Simulation outputs the acceleration values according to the settings you selected under File > Model Settings/Preferences > Units > Length.

Suppose Plant Simulation outputs the length in a dialog window. Plant Simulation outputs the length according to the settings you selected under File > Model Settings/Preferences > Units > Length. When evaluating the value, Plant Simulation has to reconvert it to m (meters) again. The function by str_to_length does this in a single step.

Parameter

The parameter Data has the data type string.

Return Value

The return value has the data type length.

Example:

 
-- reads the values in the dialog window 
var entry: string
entry := dialog.getValue("length")
print str_to_length(entry)
entry := dialog.getValue("width")
print str_to_length(entry)

Example:

 
str_to_length("1.23")
-- interprets 1.23 according to the length setting you selected under Preferences

Example:

 
str_to_length("1.23ft")
-- interprets 1.23 as foot
