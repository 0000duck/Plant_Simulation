Shift calendar
2D Reference Help > Resource Objects > WorkerPool > Dialog Window of the WorkerPool > Tab Attributes > Shift calendar

Click and select the ShiftCalendar in the dialog Select Object. This ShiftCalendar object contains the data of the shifts in your installation and controls during which shifts the Worker works.

Or you can select the ShiftCalendar in a Frame, drag it to the text box and drop it there.

When the Worker works in shifts, you also have to select the Parts buffer. He deposits the parts, which he could not deliver, onto this object, when he walks back to the WorkerPool at the end of his shift.
