Homogeneous loading
2D Reference Help > Tools > TransferStation > Dialog Window of the TransferStation > Tab Advanced Attributes > Homogeneous loading

To only load parts of a single type, select this check box. The first loaded part determines, which part type will be loaded onto the means of transport. If you use type-specific loading, this check box only has an affect, when several part types will be loaded onto a single type of means of transport.
