List (Place)
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > User-defined Distributions > List (Place)

The List(Place) distribution only applies when modeling a ParallelStation with several processing stations that have different but constant processing times.

Type in the name of the table into which you want to enter the required data. Clicking Apply in the dialog of the object automatically formats the table so that the data types of the columns match the parameters.

Within the table, the entry [1,2] matches the processing location at position [1,2], for example.

When an MU arrives at a processing station on a ParallelStation during a simulation run, Plant Simulation takes the appropriate time for that station from the table.
