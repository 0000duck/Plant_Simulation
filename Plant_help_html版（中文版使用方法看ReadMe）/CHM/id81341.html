CurrIconTransparent
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of All Objects > Attributes for Animating All Objects > CurrIconTransparent

Syntax:

 
Path.CurrIconTransparent

A 3D Only model does not provide the attribute CurrIconTransparent.

The attribute CurrIconTransparent makes the background of the Frame shine through the transparent area of the icon of the object designated by Path, which we defined, around the icon proper of the current icon of the object.

This method applies to the icon of the addressed instance, not to the icon of the object class.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
if MyStation.name = "press" 
   MyStation.CurrIcon := "type_12green"
   MyStation.CurrIconTransparent := false
   MyStation.selected := true
else
   MyStation.CurrIcon := "defaultmodel"
end
