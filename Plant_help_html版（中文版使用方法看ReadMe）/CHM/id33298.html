AmountPerMU
2D Reference Help > Fluid Objects > Portioner > Attributes of the Portioner > AmountPerMU

Syntax:

 
Path.AmountPerMU

The attribute AmountPerMU sets the amount of material which the Portioner designated by Path is to transmute into individual MUs.

The amount of material is measured in liters.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
MyPortioner.AmountPerMU := 10
