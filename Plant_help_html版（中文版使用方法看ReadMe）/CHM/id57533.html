Dialog Window of the Interface
2D Reference Help > Material Flow Objects > Interface > Dialog Window of the Interface > Dialog Window of the Interface

Double-click the icon of the Interface, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Material Flow in the Toolbox. The dialog elements and menus, which all material flow objects share, are described under Dialog Items of the Objects.
