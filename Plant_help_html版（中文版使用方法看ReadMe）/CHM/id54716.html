FrontLocation
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for the Location of the Mobile Objects > FrontLocation

Syntax:

 
MU-path.FrontLocation

The read-only attribute FrontLocation returns the object on which the front of the MU designated by MU-path is located.

Return Value

The return value has the data type object.

Example:

 
print @.FrontLocation.NumMu
