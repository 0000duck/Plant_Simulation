IsResetting
2D Reference Help > Material Flow Objects > EventController > Read-Only Attributes of the EventController > IsResetting

Syntax:

 
Path.IsResetting

The read-only attribute IsResetting is set to true when you reset the simulation model. Plant Simulation sets it to false again once all reset methods have been processed.

Return Value

The return value has the data type boolean.

Example:

 
print EventController.IsResetting
