Accessing the Name of a Sub-List with a Method
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Creating Lists within Lists and Tables > Accessing the Name of a Sub-List with a Method

To open the sub-list, which is contained in a cell of type Stack, Queue, List, and Table, hold down the Shift key and double-click into the cell. Instead, you can also right-click the cell and select Open Object or click in the text box, and press the F2 key.

You can use the attribute Name to access the name of the sub-list via information flow.

Example:

 
var str: string
str := .Models.MyPlantAnytown.SteeringTypes
["Jacks","9149"].Name 
-- returns the value of the cell that contains the sub-table, 4 in the example
