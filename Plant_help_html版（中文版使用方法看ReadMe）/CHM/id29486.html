StatServicesEnRouteToJobCount
2D Reference Help > Resource Objects > Worker > Read-Only Attributes of the Worker > StatServicesEnRouteToJobCount

Syntax:

 
Path.StatServicesEnRouteToJobCount

The read-only attribute StatServicesEnRouteToJobCount returns how often the Worker was en-route to a Workplace, to which he was mediated for do a job at the associated station. When you check Get job orders in the pool only, Plant Simulation also counts how often the Worker walked back to the WorkerPool to get the next order.

Return Value

The return value has the data type integer.

Example:

 
print .Resources.MyWorker:1.StatServicesEnRouteToJobCount
