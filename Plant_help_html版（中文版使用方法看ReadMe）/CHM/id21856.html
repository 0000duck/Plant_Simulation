List Favorite Help Topics
Using HTML Help > General Topics > List Favorite Help Topics

To create a list of Favorite help topics:

  • Locate the help topic you want to make a favorite topic.

  • Click the Favorites tab, and then click Add.

    Click the Favorites tab, select the topic, and then click Display to return to a favorite topic. If you want to rename a topic, select it, type a new name in the Current topic text box and click Add. Select the topic and then click the Remove button to remove a favorite topic.
