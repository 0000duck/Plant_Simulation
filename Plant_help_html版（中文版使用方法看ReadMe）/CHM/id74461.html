Name
2D Reference Help > Information Flow Objects > Variable > Dialog Window of the Variable > Name

Enter a name of your choice for the Variable, or accept the name, which the program suggests.

You can enter a combination of letters, digits, and underscore (_) characters, for example MyVariable, MyVariable1, My_Variable_1. The name of an object cannot start with a digit, i.e., you cannot enter 1Variable.
