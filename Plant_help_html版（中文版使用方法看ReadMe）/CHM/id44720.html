CurrentAcceleration
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for the State of the Mobile Objects > CurrentAcceleration

Syntax:

 
MU-path.CurrentAcceleration

The read-only attribute CurrentAcceleration returns the actual acceleration or deceleration of the MU designated by MU-path.

The method CurrentAcceleration is observable, but mostly makes sense, when you deactivated Acceleration, as the actual acceleration or deceleration continuously changes, when Acceleration is active.

Return Value

The return value has the data type acceleration.

Example:

 
print .MUs.Transporter:5.CurrentAcceleration
