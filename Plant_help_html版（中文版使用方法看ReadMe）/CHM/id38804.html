BwDestList
2D Reference Help > Material Flow Objects > Track > Attributes of the Track > BwDestList

Syntax:

 
Path.BwDestList

The attribute BwDestList sets the path to the destination list for a Transporter driving in reverse on the Track. The destination list contains all destinations that the Transporter can reach while driving in reverse on this Track.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
MyTrack.BwDestList := myCardFile
