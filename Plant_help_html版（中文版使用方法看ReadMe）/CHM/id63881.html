DDE
Add-Ins Reference Help > Inter-Process Communication Interfaces > DDE > DDE

The DDE interface, which is integrated in Plant Simulation, allows accessing the DDE (Dynamic Data Exchange) communication system offered by MS Windows. Plant Simulation can serve as a DDE server or as a DDE client.
