Server name
Add-Ins Reference Help > Inter-Process Communication Interfaces > OPCClassic > Dialog Window of the OPC Classic Interface > Tab Attributes > Server name

Enter the name of your OPC server. You might, for example, enter OPC.Simatic.Net or Matricon.OPCServer, etc.

Click this to open a list of all servers which are available via the host you entered. Selecting a server in the list inserts it into the text box server name.

The connection string that you enter here depends on your database provider. Consult the documentation of your database provider for more information.
