Use icon
2D Reference Help > Display and User Interface Objects > Drop-Down List > Dialog Window of the Drop-Down List > Tab Attributes > Use icon

To show the Drop-Down List in the Frame with its Current icon which you draw, select this. Clear the check box to show the drop-down list with its default look. Then, draw your icon in the icon editor. When you select use icon, the drop-down list is shown with the width of the icon of the Drop-Down List, not with the Width you enter into the dialog. To change the width, hold down Ctrl+Shift and drag the left or right side of the icon.
