Insert Box
3D Reference Help > 3D Reference > Edit Ribbon Tab > Insert Box

The command Insert Box creates and inserts a box into the selected Frame.

  • Select the Graphic group to which you want to add the new graphic. The list shows the graphic groups which are defined for the object.

    • Default (external): Designates the standard graphic group of the 3D objects. It initially is external and represents the object towards the outside. When you deactivate Show External Graphics, 3D hides the graphic groups designated as Default.

    • One of the user-defined graphic groups you added on the Tab Graphics. Newly created 3D Frames by default provide the graphic group Deco (internal), which are intended for graphics for decorating the interior view of the Frame.

  • Enter the Width, the Depth and the Height of the box you want to create into the dialog Create Box.

  • Enter the Wall thickness of the box.

  • To create the box, click Create. 3D attaches the shape to the mouse pointer. Drag the mouse to the position at which you would like to insert the shape and click the left mouse button. To cancel inserting, you can click the right mouse button or press the Esc key.

  • To move the box to the location of your choice, click it and drag the mouse.

  • To change the box settings, click the box with the right mouse button and select 3D Properties > Box Settings. Here you can also select if the fence is an obstacle for the worker.

  • To assign a material and a color, select the cylinder and click 3D Properties > Tab Material on the Home ribbon tab.

You can also edit the dimensions of the box by clicking Show Manipulators on the Edit ribbon tab.
