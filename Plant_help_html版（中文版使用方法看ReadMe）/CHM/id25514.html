Top 50 Most Relevant Methods
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Debugger Ribbon Tab > Show Profile > Top 50 Most Relevant Methods

The time consumption of the 50 most important methods groups all instances of methods that inherit the same source code Each row of a group contains information for a single instance. At the end of each group--separated by a blank line-- Plant Simulation shows the total time consumed by these instances.

% global

shows the relation of the time consumed by all instances of the group, to the total time consumed. A large value shows that this method takes up a large part of the simulation time. Check the programming of the method and to modify the source code.

% local

shows the relation of the time consumed by a single instance to the relation to the sum of the times of all instances of the group.

seconds

shows the time consumed by an instance is displayed in milliseconds.

calls

shows the number of calls.

msec/call

shows the mean execution time, which is formed by dividing the values in column 3 (seconds) by the values in column 4 (calls), i.e., the absolute time consumption is divided by the number of calls.

name

shows the name of the instance or class of the method or the material flow object. To open the window of the respective Method object or of the material flow object, click the link.

At first the Profiler only shows the classes. When you click the percentage of the class, it also shows the instances.
