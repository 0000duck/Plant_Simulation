MuSelection
2D Reference Help > Material Flow Objects > Source > Attributes of the Source > MuSelection

Syntax:

 
Path.MuSelection

The attribute MuSelection sets how the Source selects the MUs to be produced.

Depending on the setting of you enter, the attribute Path either sets the path to the delivery list, the sequence table, the frequency table, or the MU class.

Data Type of the Value You Can Assign

You can assign a value of data type string: Constant, Sequence cyclical, Sequence, Random, or Percentage.

Example:

 
MySource.MuSelection := "Sequence"
