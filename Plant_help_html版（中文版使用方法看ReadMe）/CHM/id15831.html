Edit Display Panel
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Edit Display Panel

The context menu command Edit Display Panel opens the Display Panel. Here you can add a new display panel to the selected object or edit, or delete an existing display panel.

3D models do not provide this command as they do not show display panel.
