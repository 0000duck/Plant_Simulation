push
2D Reference Help > Information Flow Objects > QueueFile and StackFile > Methods of the QueueFile and the StackFile > push

Syntax:

 
Path.push(Cell:any)

The method push adds the cell designated by the parameter of data type any to a list with one column. StackFile and QueueFile add the entries according to their built-in properties.

Parameter

The data type of the return value matches the data type of the StackFile or the QueueFile.

Example:

 
MyStackFile.push(true)
MyQueueFile.push("cans")
