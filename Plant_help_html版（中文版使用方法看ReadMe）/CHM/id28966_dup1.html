Visible
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Graphics > Object graphics > Graphic Groups > Add > Visible

To use the new graphic group for visually representing the object in the 3D windows immediately, select this. Clear this to provide the new graphic group as an alternative or additional visual representation for later use and hide it in all 3D windows for now.

If a graphic group marked as Visible is actually shown in a certain 3D window or not depends on additional object settings and on the scene root of this 3D window.

Instead, you can also select the check box on the table on the tab itself.
