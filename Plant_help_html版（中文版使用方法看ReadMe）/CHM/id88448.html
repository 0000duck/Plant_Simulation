Print Setup
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Print Setup

The command Print Setup opens the dialog Print Setup in which you can select, which items Plant Simulation prints, when you print a list and a table. You can also select the print format, Portrait or Landscape.

This Check Box

Prints

Attribute

Row number

Prints the row number in front of each cell.

PrintRowNumber

Column number

Prints the column number above each column.

PrintColumnNumber

Data type

Prints the data type of each column.

PrintDataType

Repeat row index

Prints the user-defined row index on each page.

RepeatRowIndex

Repeat column index

Prints the user-defined column index on each page.

RepeatColumnIndex

Internal lists

Prints nested tables the list or table contains also.

PrintInternalLists

Column width

Prints columns using the column width of the widest column. When you do not check this, the program uses the value you set under Format > List > Column width and entries may overprint other entries.

GenerateColumnWidth
