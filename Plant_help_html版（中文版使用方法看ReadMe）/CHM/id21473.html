WorkerChart
2D Reference Help > Tools > WorkerChart > WorkerChart

To add the WorkerChart to the Class Library, click Manage Class Library > Libraries > Tools > Worker Chart on the Home ribbon tab.

The WorkerChart shows the statistics of all the workers who are located in a single WorkerPool. For this reason it registers the workers at the beginning of a simulation. To start the WorkerChart, drag the WorkerPool onto the icon of the WorkerChart, drop it there and then start the simulation.

We modeled the WorkerChart as an application object in a Frame. For this reason opening help with F1 and What’s This Help in the dialogs of the objects do not work. To open help for the object, select Help > Help on WorkerChart in its dialog.

To assign the object another name or a label, click it with the right mouse button and select Rename.
