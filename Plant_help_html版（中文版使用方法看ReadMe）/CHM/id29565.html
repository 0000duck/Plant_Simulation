_3D.createCylinder
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating 3D Shapes > _3D.createCylinder

Syntax:

 
_3D.createCylinder(BasePoint:length[3], Radius:length, Height:length [,GraphicGroupName:string]) → integer

The method _3D.createCylinder creates a cylinder in the scene window.

Parameters

  • The parameter BasePoint is an array of data type length which sets the base point of the cylinder. The base point designates the center of the base area.

  • The parameter Radius of data type length sets its radius.

  • The parameter Height of data type length sets its height.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Return Value

The return value has the data type integer and specifies the numerical index of the cylinder, compare _3D.getGraphicIndex.

Example:

 
_3D.createCylinder([3.0,2.5,1],3,3)

Example:

 
_3D.createCylinder([3.0,2.5,1],3,3,"deco")

Example:

 
var index := _3D.createCylinder([3.0,2.5,1],3,3,"deco")
_3D.setGraphicMaterial([index], makeRGBValue(255,0,0), 0, 0, 0, 0, 0, "deco")
