2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > Modeling > 2D

On the tab 2D you can select the default settings that determine how a Frame in the current model shows the objects which you insert into it.

You can override these settings for each individual Frame on the General Ribbon Tab > More View Options. The Modeling settings you select here are saved to your model file.
