Insert an Object from the Class Library
Step-by-Step Help > Modeling in 2D > Creating a Simulation Model > Working with the Frame > Model with Objects from the Class Library > Insert an Object from the Class Library

In the structure, navigate to the folder and then to the object you would like to insert. Note that the objects are grouped according to their functions and to the frequency you use them.

To insert an instance of the class object into the Frame:

  • Click the object of your choice with the left mouse button to select it. The insert cursor looks like this .

  • While holding the mouse button down, drag the object to the target position in the Frame window and drop it.
