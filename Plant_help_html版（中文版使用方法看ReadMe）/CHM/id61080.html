MutationPlan
Add-Ins Reference Help > Genetic Algorithms > GASequence > Attributes of the GASequence > MutationPlan

Syntax:

 
Path.MutationPlan

For the setting Mutation > Fixed number of mutations the attribute MutationPlan sets the table in which you define the number of mutations per chromosome and the position offset for each generation.

Data Type of the Value You Can Assign

You can assign a value of data type table.

Example:

 
GASequence.MutationPlan.delete
GAsequence.MutationPlan  := MutationSteps.copy
