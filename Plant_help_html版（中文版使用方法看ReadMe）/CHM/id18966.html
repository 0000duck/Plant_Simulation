MinVal
2D Reference Help > Display and User Interface Objects > Display > Attributes of the Display > MinVal

Syntax:

 
Path.MinVal

The attribute MinVal sets the lower bound of the displayed range in bar and pie modes. The Display shows values below or equal to MinVal as an empty bar/pie.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
MyDisplay.MinVal := 5.35
