Host
Add-Ins Reference Help > Inter-Process Communication Interfaces > Socket > Attributes of the Socket Interface > Host

Syntax:

 
Path.Host

The attribute Host sets the IP address of the host computer with which you want to establish the connection. The number 127.0.0.1 designates the local host.

This setting applies when you are running the socket connection in Server socket

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
Socket.Host := "127.0.0.1"
