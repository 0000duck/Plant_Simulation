Color
2D Reference Help > Information Flow Objects > Variable > Attributes of the Variable > Color

Syntax:

 
&Variable.Color

The attribute Color sets the color with which Plant Simulation displays the object Variable in the Frame. You can set the RGB values of the color with the method makeRGBValue.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
&MyVariable.Color := makeRGBValue(110,0,200)
