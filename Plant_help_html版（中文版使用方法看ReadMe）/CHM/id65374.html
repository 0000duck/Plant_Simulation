Plant Simulation Viewer License
Setting-Up and Starting > Plant Simulation License Types > Plant Simulation Viewer License

With the Plant Simulation Viewer license you can run and visualize simulation models, but you cannot change the model or its parameters. The programming language SimTalk is disabled in the Plant Simulation Viewer license.

The Plant Simulation Viewer license:

  • Visualizes built-in and user-defined objects with integrated parameters.

  • Visualizes values in lists and tables.

  • Executes simulation runs with 2D and 3D graphics, animates simulation models, records data and evaluates simulation runs.

  • Allows using Genetic Algorithms for the automated optimization of your simulation models.

  • Allows using an unlimited number of objects.

  • Provides an integrated presentation of the results in Gantt charts depending on resources and orders.
