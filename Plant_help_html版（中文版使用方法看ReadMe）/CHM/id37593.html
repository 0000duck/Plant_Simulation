Transparent
2D Reference Help > Display and User Interface Objects > Display > Attributes of the Display > Transparent

Syntax:

 
Path.Transparent

The attribute Transparent sets if the Display you inserted in a Frame shows its background transparent (true) or not (false).

Then, the Display shows the space around the text in the color of the background image or it shows the grid points of the Frame. If you clear Transparent, Plant Simulation shows the space around the text white.

Not transparent

Transparent

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyDisplay.Transparent := false
