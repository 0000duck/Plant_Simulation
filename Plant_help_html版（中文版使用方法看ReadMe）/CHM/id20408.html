_3D.getGraphicPosition
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.getGraphicPosition

Syntax:

 
Path._3D.getGraphicPosition(GraphicPath:any [,GraphicGroupName:string]) → real[3]

The method _3D.getGraphicPosition returns the position of the designated graphic of the object designated by Path.

Parameters

  • The parameter GraphicPath of data type any designates the path of the graphic. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Return Value

The return value is an array of data type real.

Example:

 
var a : real := Station._3D.getGraphicPosition([1, 1])
var a : real[3] := ParallelStation._3D.getGraphicPosition(["Graphic1"])

Example:

 
var a : real := Station._3D.getGraphicPosition([1, 1], "deco" )
var a : real[3] := ParallelStation._3D.getGraphicPosition(["Graphic1"], "deco")
