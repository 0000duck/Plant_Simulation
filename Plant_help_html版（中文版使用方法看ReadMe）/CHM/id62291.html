insert
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > insert

Syntax:

 
Path.insert(GivenTime:any, Value:any)

The method insert inserts a time/value pair into the TimeSequence designated by Path. The type of the value, i.e., the second parameter any, must match or be compatible with the data type of the TimeSequence. For the time reference Absolute, the first parameter is expected to be of data type dateTime. Otherwise, the data type time is required.

Parameters

  • The parameter GivenTime of data type any designates the time.

  • The parameter Value of data type any designates the value.

Example:

 
MyTimeSequence.insert(eventcontroller.simtime,3)
