NumMU
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for Accessing the Cargo of Container and Transporter > NumMU

Syntax:

 
MU-path.NumMU

The read-only attribute NumMU returns the number of MUs located on the loading space of the MU designated by MU-path.

Return Value

The return value has the data type integer.

Example:

 
print .MUs.Transporter:1.NumMU
