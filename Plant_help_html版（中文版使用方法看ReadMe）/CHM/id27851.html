_3D.BaseHeight
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Defining Length-oriented Objects > _3D.BaseHeight

Syntax:

 
Path._3D.BaseHeight

The attribute _3D.BaseHeight sets the base height. The base height defines the distance of the legs of a conveyor from the floor.

The base height can be inherited, meaning that you can define it once in the class instead of having to define it for each instance individually. The z-coordinate defines the floor of the installation and the base height defines the distance of the legs from the floor. By changing the setting ΔZ in the Segments table, you can add an additional offset. The setting ΔZ also affects the simulation time in Plant Simulation 2D.

Normally, the base height in 2D and in 3D are identical. They might differ if you clear Connect 2D and 3D positions under Edit 3D Properties > Tab Transformation > Position.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
MyConveyor._3D.BaseHeight := 3
print MyConveyor._3D.BaseHeight
