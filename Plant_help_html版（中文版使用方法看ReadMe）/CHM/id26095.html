Label
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Label

In addition to the Name, you can enter a label for the dialog you create. As opposed to the name, you can enter any term of your choice as the label. This includes special characters and blank spaces.

  • If you enter a label, Plant Simulation shows that label in the title bar of the dialog you created.

    Label

    No label

  • If you do not enter text as the label, Plant Simulation shows the Name of the Dialog object in the title bar.
