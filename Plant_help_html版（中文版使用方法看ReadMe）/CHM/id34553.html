node
2D Reference Help > The Plant Simulation Program Window > Toolbox > Attributes and Methods of the Folder in the Class Library > node

Syntax:

 
Path.node(ObjectNumber:integer) → object

The method node calls an object within a folder. Plant Simulation numbers the objects within folders in your model in the order in which you insert them.

Parameter

The parameter ObjectNumber of data type integer designates the number of the object.

Return Value

The return value has the data type object.

Example:

 
print .Models.MnEnginePlant.node(5)
-- returns the path and the name of the object designated by node,
-- for example .Models.EnginePlant.Track
