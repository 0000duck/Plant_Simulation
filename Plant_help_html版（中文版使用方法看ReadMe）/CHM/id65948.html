Use icon
2D Reference Help > Display and User Interface Objects > Button > Dialog Window of the Button > Tab Attributes > Use icon

To show the object in the Frame with its Current icon, select this. Clear the check box to show the object with its default look.

The default look also displays the label which you entered as button text. Using an icon is especially handy if you want to show a picture on the button instead of text. Then, draw your icon in the icon editor. When you select use icon, the button is shown with the width of the icon of the Button, not with the Width you enter into the dialog. To change the width, hold down Ctrl+Shift and drag the left or right side of the icon.

You can use this feature in several ways:

  • You can create a new icon in the icon editor and draw or paste a picture on that icon. For the example above we drew a green rectangle inside the button . Then, we entered the label .

    To be able to tell that you clicked the button, we recommend to draw two pictures for the button, one for the unclicked state and one for the clicked state . For our example above the name of the unclicked icon is icon3, the name of the clicked icon consequently has to be icon3_down. Note that the button does not stay pressed, but returns to its unclicked state when you release the mouse button.
