createNestedList
SimTalk Reference > General SimTalk Functions > Data Types > Instantiating Local Lists and Tables > createNestedList

Syntax:

 
Local-table.createNestedList(Column:integer, Row:integer)

The method createNestedList creates a new sub-list or sub-table in the local variable designated by local_list or by local-table. The local variable has to be of data type list, stack, queue or table. The column type of this list or table has to be of data type list, stack, queue or table as well.

Parameters

  • The parameter Column of data type integer designates the column of the cell.

  • The parameter Row of data type integer designates the row of the cell.

Example:

 
var t: table[table]
t.create                -- creates a table
t.createNestedList(1,1) -- creates a sub-table
t[1,1][2,3] := "Hello world"

This method also applies to sub-lists and sub-tables!

Example:

 
var subtable: table
MyTableFile.createNestedList(2,3) -- creates a sub-table
subtable := MyTableFile[2,3]
subtable.createNestedList(1,1)    -- creates a sub-table within a sub-table
subtable[1,1][4,4] := "Hello world"
