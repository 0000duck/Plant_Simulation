Graphic complexity
3D Reference Help > 3D Reference > Edit Ribbon Tab > Optimize Selected Graphic > Graphic complexity

The group box Graphic complexity shows the following values for the complexity of the selected graphic.

Compare the number of nodes and the number of polygons before and after optimizing in our example below.

Nodes (CPU)

Shows the number of nodes of the selected graphic. The number of nodes affects the CPU (Central Processing Unit) of your computer. When you optimize your graphics, the values for nodes and for polygons should be as small as possible.

Polygons (GPU)

Shows the number of polygons of the selected graphic. The number of polygons affects the graphics card of your computer. When you optimize your graphics, the values for nodes and for polygons should be as small as possible.
