Importers
2D Reference Help > Resource Objects > Exporter > Dialog Window of the Exporter > The View Menu > Importers

The menu command Importers opens a table showing all importers for which the Exporter provides services.

The table shows the Importers in column 1, and their Type in column 2: The number 0 designates the failure-importer, 1 the set-up-importer, 2 the processing-importer, and 3 the transport-importer.
