time_to_num
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Physical Data Types with Units into Data Types without Units > time_to_num

Syntax:

 
time_to_num(time) → real

The function time_to_num converts the data designated by the parameter of data type time into a value without a unit.

Parameter

The parameter Data has the data type time.

Return Value

The return value has the data type real.

Example:

 
var w: weight, t: time
w := time_to_num(t)
