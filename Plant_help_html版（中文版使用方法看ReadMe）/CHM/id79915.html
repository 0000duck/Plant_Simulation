Locked
2D Reference Help > Display and User Interface Objects > Dialog > Attributes of the Dialog > Locked

Syntax:

 
Path.Locked

The attribute Locked sets what Plant Simulation opens, when the user double-clicks the icon of the Dialog:

  • When you enter true, Plant Simulation locks the dialog window of the object Dialog, preventing him from making changes to the layout of the dialog, and opens the Dialog you designed.

  • When you enter false, Plant Simulation unlocks the dialog window of the object Dialog, and opens it, when the user double-clicks the icon, so he can modify the layout of the dialog.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyDialog.Locked := true
MyDialog.openDialog
