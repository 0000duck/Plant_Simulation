2D Reference Help > The Plant Simulation Program Window > Display Panel > Dialog Window of the Display Panel > New > Type

Select how the display panel shows the Value.

  • As static text, i.e., as an arbitrary sequence of characters that does not change as opposed to the value of an attribute/method.

    You can select the Alignment and the Color of the displayed text.

  • As text.

    You can select the Alignment and the Color and if you want to display the values as a percentage and you can specify the number of Decimal places.

  • As a bar if you want to display values of numeric data types.

    You can select the Color, the Length, the Width, the Direction, and the Maximum value of the bars.

  • As an LED if you want to display boolean values.

    You can select the Width of the LED area.

    Showing Statistics with Display Panels in the Model illustrates how to create items.
