Scrollbar
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Axes > Scrollbar

To add a scrollbar to the Chart window for the setting Category > Plotter, select this. Clear the check box to not add a scrollbar.

When the scrollbar is active, you can enter a Feed rate in grid units, which sets the number of grid units by which the Chart scrolls, when it reaches the right border of the display.
