Define a Heading
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Define a Heading

Enter a number sign (# ) at the start of a line to create a heading of level 1.

Enter two number signs (##) to create a heading of level 2.

You can specify a maximum of six heading levels, i.e., you can enter ######.

The HtmlReport also shows the headings of level 1 and 2 on the table of contents in the structure pane on the left-hand side of the HTML page. 

# Heading 1
## Heading 1.1
Text below Heading 1.1.
## Heading 1.2
Text below Heading 1.2.
