Shift calendar
2D Reference Help > Tools > CrossSlidingCar > Dialog Window of the CrossSlidingCar > Tab Controls > Shift calendar

Click and select the ShiftCalendar in the dialog Select Object. This ShiftCalendar object contains the data of the shifts in your installation and controls during which shifts the cross-sliding car works.

This automatically enters the object into the list of Objects on the tab Resources of the ShiftCalendar .
