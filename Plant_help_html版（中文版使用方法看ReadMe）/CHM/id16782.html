Step-by-Step Help > Animating the Simulation Model and Viewing the Results > Working with Evaluation Tools > SankeyDiagram > Color

Select one of the predefined colors for the Connectors in the Sankey diagram: Red, Cyan, Blue, Magenta, Green, Yellow, Black or White. Clicking Select opens the dialog Colors in which you can select another color of your liking.
