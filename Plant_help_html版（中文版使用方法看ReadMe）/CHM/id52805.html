MUs to be watched
Step-by-Step Help > Animating the Simulation Model and Viewing the Results > Working with Evaluation Tools > SankeyDiagram > MUs to be watched

To designate the classes of MUs, which the SankeyDiagram watches, click Open and enter the MU classes, whose instances Plant Simulation watches, into the table that opens.

You might, for example, enter .MUs.part, .MUs.Part, .MUs.Transporter, etc.
