Methods for Managing User-defined Attributes
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Managing User-defined Attributes > Methods for Managing User-defined Attributes

The objects provide these methods for working with their user-defined attributes of the objects, which you define on the Tab User-defined:

createAttr, deleteAttr, getAttrName, getAttrNo, getAttrType, getAttrValue, getStatisticsTable, increment, NumAttr, setAttrType, setAttrValue, and unshare.

To view all of the methods and attributes of the material flow object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.
