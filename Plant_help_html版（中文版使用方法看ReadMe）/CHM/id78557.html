Detach the Camera from an Object
Step-by-Step Help > Modeling in 3D > Modeling a Fly Through > Detach the Camera from an Object

To detach the camera, which you attached to an object, from it, click in the View ribbon tab of the 3D window. 3D then returns to normal view.

3D also detaches an object camera:

  • When you delete the object to which the camera is attached.

  • When you change the scene.
