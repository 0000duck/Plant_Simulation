MovingDirection
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Attributes of the MultiPortalCrane > Attributes of the Portal > MovingDirection

Syntax:

 
Portal-path.MovingDirection

The attribute MovingDirection queries the direction in which the portal designated by Portal-path is currently moving.

Return Value

The return values of this attribute are "left" or "right". The empty string "" means that portal is not moving at the moment. The moving direction is set to "right" if the portal moves in the direction in which you inserted the runway when you created your simulation model.

Example:

 
currentDirection := Portal.MovingDirection
