Capacity
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > Capacity

Syntax:

 
MU-path.Capacity

The attribute Capacity sets the Capacity, i.e., the number of MUs or MU parts that may be located on the loading space of type Track or Line of the Transporter as a whole or in part at any one time. When this number is reached, the Conveyor does not accept any additional MUs, even if its full length is not exhausted. Enter -1 for an infinite capacity.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
.MUs.Transporter:3.LoadBayType := "Line"
.MUs.Transporter:3.Capacity := 15
