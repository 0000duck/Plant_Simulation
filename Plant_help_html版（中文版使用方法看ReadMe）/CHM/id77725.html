creationTime
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Methods of All Mobile Objects > Methods for the State of the Mobile Object > creationTime

Syntax:

 
MU-path.creationTime → time

The method creationTime returns the time at which the MU designated by MU-path was created during the simulation run.

Return Value

The return value has the data type time.

Example:

 
print Part:1.creationTime
