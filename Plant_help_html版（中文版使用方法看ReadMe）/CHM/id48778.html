endPauseIn
2D Reference Help > Mobile Objects > Transporter > Methods of the Transporter > endPauseIn

Syntax:

 
MU-path.endPauseIn(EndPauseIn:time)

The method endPauseIn terminates the pause of the Transporter after the time you enter. It also sets the attribute Paused to false.

Parameter

The parameter EndPauseIn of data type time designates the time span after which the pause ends. The time you enter has to be equal to or greater than 0.

Example:

 
MyTrack.MU(1).endPauseIn(str_to_time("10:0.0"))
