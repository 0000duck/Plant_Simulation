num_to_str
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Numerical Values > num_to_str

Syntax:

 
num_to_str(real [,integer] → string

The functionnum_to_str converts the number designated by the parameter of data type real into a string.

Parameter

  • The parameter Number of data type real designates the number that is going to be converted.

  • For the numerical data types the optional parameter of data type integer designates the precision, i.e., the number of significant digits, which the resulting string has at the most. When the data type of the first parameter is integer or time, the second parameter has no function.

Return Value

The return value has the data type string.

Example:

 
print num_to_str(pi*100, 5) -- outputs "314.16"
