Look at Object
3D Reference Help > 3D Reference > Context Menus in 3D > Context Menu of Anchor Points of a Path > Look at Object

The context menu command Look at Object rotates the selected anchor point of the path so that it points toward another object.

The command is only active when the anchor point belongs to a camera path of type Conveyors.

When you select the command, the mouse pointer changes into an eye. Then, you can select the object at which the anchor point is to look by clicking the object with the left mouse.

Click the right mouse button or press Esc to cancel the command.
