Trigger type
2D Reference Help > Information Flow Objects > Trigger > Dialog Window of the Trigger > Tab Values > Trigger type

Select the Trigger type. The trigger types differ in the way how they generate events.

  • The Input Trigger takes the values from a single TimeSequence. Click Values, and enter the values in the Value Table of the TimeSequence.

  • The Combination Trigger generates a new Trigger by combining existing Triggers. Click Combination Table and enter the names of the Triggers that are to be combined. The program saves the result of the combination in the combination list that clicking Values opens.
