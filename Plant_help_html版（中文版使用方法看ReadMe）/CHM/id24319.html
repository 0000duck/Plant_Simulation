ReferenceDate
2D Reference Help > Information Flow Objects > Trigger > Attributes of the Trigger > ReferenceDate

Syntax:

 
Path.ReferenceDate

The attribute ReferenceDate sets the Start date of the Trigger.

Data Type of the Value You Can Assign

You can assign a value of data type dateTime.

Example:

 
MyTrigger.ReferenceDate := sysDate
