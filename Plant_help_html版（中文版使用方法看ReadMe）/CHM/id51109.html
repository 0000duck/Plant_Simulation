id
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods of Sensors of the Length-Oriented Objects > id

Syntax:

 
Path.Sensor.id → integer

The method id returns the unique identifier of a sensor of the object designated by Path, enabling you to access it. Our example deletes all front-sensitive sensors of the object Track.

Creating and manipulating sensors via SimTalk for the Transporter only works if you select the loading space Types Track or Line!

Return Value

The return value has the data type integer.

Example:

 
for i := MyTrack.NumSensors downto 1 
   if MyTrack.sensorNo(i).front = true 
       sensorID := MyTrack.sensorNo(i).id
       MyTrack.deleteSensor(sensorID)
   end
next

Example:

 
print .MUs.Transporter:1.id
