_3D.InternalGraphics
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > _3D.InternalGraphics

Syntax:

 
Path._3D.InternalGraphics

The attribute _3D.InternalGraphics represents the list of all internal graphic groups. A visible internal graphic group is visible inside of its object and together with its content (compare _3D.ShowContent) from the outside.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
.Models.Frame.Station1._3D.InternalGraphics := ["deco1", deco2"]
var a : string[] := .Models.Frame.Station1._3D.InternalGraphics
