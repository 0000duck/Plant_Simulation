PositionType
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Sensors of the Length-Oriented Objects > PositionType

Syntax:

 
Path.Sensor.PositionType

The attribute PositionType sets the position type of the sensor of the object designated by Path: relative or length.

Creating and manipulating sensors via SimTalk for the Transporter only works if you select the loading space Types Track or Line!

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyTrack.sensorNo(2).PositionType := "Length"

Example:

 
.MUs.Transporter:1.sensorID(3).PositionType := "Relative"
