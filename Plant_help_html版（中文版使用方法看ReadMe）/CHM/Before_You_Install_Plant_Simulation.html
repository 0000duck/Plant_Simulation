Be aware though that the following restrictions apply:

  • When you employ the C interface to link a DLL, which you yourself programmed, this DLL has to be compiled as a 64-bit DLL.

  • For the object ActiveX you need 64-bit ActiveX controls.

  • For the ODBC interface you also need 64-bit drivers for ODBC. Starting with Microsoft Office 2010 Microsoft provides 64-bit ODBC drivers for Access, Excel, etc.

  • The OPC interface can only address a 64-bit version of an OPC server.
