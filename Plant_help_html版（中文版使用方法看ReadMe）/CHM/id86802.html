Lanes
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Sensors of the Length-Oriented Objects > Lanes

Syntax:

 
Path.Sensor.Lanes

The attribute Lanes sets to which lane of the TwoLaneTrack designated by Path the sensor is going to be assigned: To "Lane A", to "Lane B" or to "Both Lanes".

For a sensor on a Track, the attribute always has the value "Lane A".

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
TwoLaneTrack.A.SensorID(1).Lanes := "Lane B"
TwoLaneTrack.B.SensorID(1).Lanes := "Both Lanes"
