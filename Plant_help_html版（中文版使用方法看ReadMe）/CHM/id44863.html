startPauseIn
2D Reference Help > Mobile Objects > Transporter > Methods of the Transporter > startPauseIn

Syntax:

 
MU-path.startPauseIn(StartPauseIn:time)

The method startPauseIn pauses the Transporter designated by MU-path after the time span you enter.

Parameter

The parameter StartPauseIn of data type time designates the time after which the objects set the attribute Pause to true. The time you enter has to be equal to or greater than 0.

Example:

 
MyTrack.Transporter:1.startPauseIn(10) -- in 10 seconds
