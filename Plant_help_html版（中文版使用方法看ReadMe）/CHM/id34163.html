EntranceOpen
2D Reference Help > Fluid Objects > Mixer > Read-Only-Attributes of the Mixer > EntranceOpen

Syntax:

 
Path.EntranceOpen

The read-only attribute EntranceOpen returns if material can enter the Mixer designated by Path because of current recovery times (true) or not (false).

Return Value

The return value has the data type boolean.

Example:

 
print MyMixer.EntranceOpen
