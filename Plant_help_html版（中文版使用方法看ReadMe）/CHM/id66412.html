Controls
2D Reference Help > Material Flow Objects > Frame > The Frame Window > Context Menu of the Selected Object in the Frame > Controls

To open one of the controls, which you entered on the Tab Controls, by clicking Edit Controls on the Home ribbon tab, select the name of the control on the submenu. The command also shows the controls you entered by selecting Tools >Edit Controls in the dialog of an object. The context menu only shows the menu item Controls if you actually did enter one or several controls.
