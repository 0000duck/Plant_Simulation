Paste
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Show 3D Graphic Structure > State Graphics > Context Menu of a State Graphic > Paste

The context menu command Paste inserts the graphic from the graphics clipboard into the selected state group.

When you paste a graphic into a state group of a 3D object which inherits its graphics, 3D asks if you want to cut graphic inheritance for this object or not. 3D only pastes the graphic if you cut graphic inheritance by clicking Yes.
