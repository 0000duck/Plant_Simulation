Built-in Attributes and Methods for Length-oriented Objects
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Defining Length-oriented Objects > Built-in Attributes and Methods for Length-oriented Objects

Plant Simulation provides these attributes and methods for defining length-oriented objects in 3D:

_3D.BaseHeight, _3D.ExtConfiguration, _3D.getExtSegments, _3D.setExtSegments _3D.ShowSensors, and _3D.Width.

You can view and access the attributes for defining length-oriented objects by clicking Auto Complete on the Edit ribbon tab of the method editor.
