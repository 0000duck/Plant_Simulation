BwExitCtrlFront
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Controls of the Material Flow Objects > BwExitCtrlFront

Syntax:

 
Path.BwExitCtrlFront

The attribute BwExitCtrlFront sets if the length-oriented objects Conveyor, Track, or TwoLaneTrack designated by Path activate the Backward exit control, when the front of the MU exits the objects via their entrance and/or when it moves towards their entrance when exiting (true) or not (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTrack.BwExitCtrlFront := false
MyTwoLaneTrack.A.BwExitCtrlFront := true
MyConveyor.BwExitCtrlFront := true

Example:

 
.MUs.Transporter:4.LoadBayType := "Line"
.MUs.Transporter:4.BwExitCtrlFront := false
