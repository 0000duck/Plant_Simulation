setMaxNumberOfCallChains
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Debugging the Model > setMaxNumberOfCallChains

Syntax:

 
setMaxNumberOfCallChains(NumberOfCallChains:integer) → integer

The function setMaxNumberOfCallChains limits the maximum number of call chains Plant Simulation calls. When Plant Simulation reaches the number you set, it stops the simulation and shows an error message. You can then increase the Maximum number of call chains and continue the simulation.

Parameter

The parameter NumberOfCallChains of data type integer designates the maximum number of call chains.

Return Value

The function returns the previous value of the maximum number of call chains and does not set the new value if 0 is passed. The return value has the data type integer.

Example:

 
setMaxNumberOfCallChains(1500)
