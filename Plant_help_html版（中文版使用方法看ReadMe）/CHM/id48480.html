Password
Add-Ins Reference Help > Inter-Process Communication Interfaces > Teamcenter > Dialog Window of the Teamcenter Interface > Tab Connection > Password

Enter the password with which you want to log on to Teamcenter. Plant Simulation encrypts the password and saves it in the model file.

If you are using the Teamcenter interface with Active Workspace, you have to enter the password once even if it is already saved in the object Teamcenter.

If you do not enter the password here, you have to enter it in the Teamcenter login dialog when you log on to the Teamcenter database for the first time.

SimTalk:

setPassword
