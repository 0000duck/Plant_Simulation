ExitCtrlRear
2D Reference Help > Material Flow Objects > TwoLaneTrack > Attributes of the TwoLaneTrack > ExitCtrlRear

Syntax:

 
Path.A.ExitCtrlRear
Path.B.ExitCtrlRear

The attribute ExitCtrlRear sets if the TwoLaneTrack designated by Path triggers the exit control as soon as the back of the Transporter exits on the specified lane (true) or not (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTwoLaneTrack.A.ExitCtrlRear := true
