New Menu/New Menu Command
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New Menu/New Menu Command

A menu is a list of textual choices from which the user can choose. The Dialog inserts the menus along the top of the dialog you define.

  • Enter the Name of the menu.

  • Enter the Caption of the menu. Enter a hyphen (-) as the caption of the menu command to insert a menu separator between groups of menu commands.

  • Enter the Callback argument, which will be passed to the Callback method as a parameter.

    It will be executed, when the user selects a menu or a menu command.

  • Select if you want to Enable it or not.

  • To insert the menu into your dialog, click OK.

  • To add one menu command at a time to the selected menu, right-click it and select New Menu/New Menu Command again.

    • Enter the Name of the menu command.

    • Enter the Caption of the menu command.

    • Enter the parameter, which will be passed to the Callback method as a parameter (Callback argument).

    • Select if you want to Enable it or not.

    • To insert the menu into your dialog, click OK.

  • To add a submenu to the selected menu command, right click that menu command and select New Menu/New Menu Command again. Repeat the steps described above. A submenu looks like this.
