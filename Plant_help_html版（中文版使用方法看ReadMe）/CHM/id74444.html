Speed
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > Units > Speed

Select the unit for the speed. When you assign a speed in a method, Plant Simulation uses m/s as unit. When you change the unit while modeling, it converts all values, which you already entered into objects, to the new unit.
