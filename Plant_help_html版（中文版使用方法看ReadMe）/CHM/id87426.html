Exit locked
2D Reference Help > Material Flow Objects > TwoLaneTrack > Dialog Window of the TwoLaneTrack > Tab Attributes > Lane B > Exit locked

To close the exit of lane B, select this. The TwoLaneTrack prevents Transporters from being moved on to the successors in the material flow and enters them into the Exit Blocking List of lane B of the TwoLaneTrack.

Once the exit is unlocked again, the TwoLaneTrack moves the first Transporter in the Exit Blocking List of lane B on to its successor.

The Transporter does not trigger the exit control, when you selected the check box Exit locked.
