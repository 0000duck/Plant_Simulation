Tab Statistics
2D Reference Help > Tools > TransferStation > Dialog Window of the TransferStation > Tab Statistics > Tab Statistics

The tab Statistics provides the dialog item Statistics Table. It also shows these statistics values on the tab.

Item

Shows the

Working

portion of the statistics collection period during which the TransferStation was working.

Waiting

portion of the statistics collection period during which the TransferStation was waiting.

Waiting for parts

portion of the statistics collection period during which the TransferStation was waiting for parts. The TransferStation spent time waiting for parts, which it will load onto a Container or a Transporter.

Waiting for target

portion of the statistics collection period during which the TransferStation was waiting for a target, i.e., a Container or a Transporter. The TransferStation spent time waiting for a target (Container or Transporter), i.e., the parts are present, but the Container or Transporter is not present yet.

Blocked

portion of the statistics collection period during which the TransferStation was Blocked.

Failed

portion of the statistics collection period during which the TransferStation was Failed.

Paused

portion of the statistics collection period during which the TransferStation was Paused.

Unplanned

portion of the statistics collection period during which the TransferStation was Unplanned.

Entries

number of parts, which entered the TransferStation.
