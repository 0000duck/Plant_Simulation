ScalingFactor
2D Reference Help > Material Flow Objects > Frame > Attributes of the Frame > ScalingFactor

Syntax:

 
Path.ScalingFactor

A 3D Only model does not provide the attribute ScalingFactor.

The attribute ScalingFactor sets the Scaling Factor for displaying the objects in the Frame. Plant Simulation uses the scaling factor for converting the pixel coordinates to length coordinates.

When you change the scaling factor, Plant Simulation keeps the lengths of the length-oriented objects and the 3D coordinates and changes the dimension or the representation of the pixels respectively. Plant Simulation moves the axes origin if your change would create negative pixel coordinates.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
MyFrame.ScalingFactor := 0.5
