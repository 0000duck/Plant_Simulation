Chute conveyor
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Appearance of the Length-oriented Objects > Configuration > Conveyor type > Chute conveyor

For a length-oriented object of type chute conveyor, you can enter its base height, its width. In addition, you can select settings for its Legs and for the chute.

For the chute conveyor 3D shows the group box chute instead of Lanes.

You can:

  • Select the type of the chute.

  • Enter the thickness of the chute proper.

  • For the chute type rails, you can enter how many rails support the chute, i.e., the count.

  • Click material and select the material of the chute.
