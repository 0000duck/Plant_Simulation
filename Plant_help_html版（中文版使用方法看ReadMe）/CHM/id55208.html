testImportFor
2D Reference Help > Resource Objects > Broker > Methods of the Broker > testImportFor

Syntax:

 
Path.testImportFor(Importer:object,Type:integer,Exporter:table) → boolean

The method testImportFor executes a test request for the Importer. The return value shows if the Broker designated by Path is able to satisfy the request (true) or not (false).

Parameters

  • The parameter Importer of data type object designates the importer proper.

  • The parameter Type of data type integer designates its type: 0 designates the failure-importer, 1 the set-up-importer, 2 the processing-importer, and 3 the transport-importer.

  • The parameter Exporter of data type table passes a table containing all Exporters that may be assigned after the call. You can use this method to find out how the Broker will treat the request of the importer.

Data Type of the Value You Can Assign

The return value has the data type boolean.

Example:

 
MyBroker.testImportFor(sp1,1,Tab)
