Pause control
2D Reference Help > Mobile Objects > Transporter > Dialog Window of the Transporter > The Tools Menu > Pause control

Click and select a Method object in the dialog Select Object. Enter the source code of the pause control. The Transporter calls it, whenever its Paused state changes. To accomplish this, either select the check box Paused or assign true to the attribute Pause. Within the control, you can access the object starting the control using the anonymous identifier ?. You can also reference the control with the attribute PauseCtrl within methods.
