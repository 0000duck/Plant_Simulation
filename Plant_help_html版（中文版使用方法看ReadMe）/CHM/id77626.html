Attributes of the Variable
2D Reference Help > Information Flow Objects > Variable > Attributes of the Variable > Attributes of the Variable

The object Variable provides:

  • The attributes Color, Comment, DataType, DecimalPlaces, Font, InheritValue, InitValue, Name, ShowDataType, ShowUnit, StatisticsActive, SupportDDEHotlinks, Transparent, and Value.

  • The Attributes of All Objects.

To view all of the methods and attributes of the Variable, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Classs Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methodsof the selected Instance.

You can only access the attributes of the object Variable, referring to the object itself, with the reference operator.
