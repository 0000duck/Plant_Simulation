asString
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Indirectly Accessing Lists and Tables > asString

Syntax:

 
Path.asString(Column:any, Row:any) → string

The method asString returns the designated cell of the TableFile designated by Path as a string.

This is especially handy if the cell of data type object contains a path for which Plant Simulation would return the contents either as an object or as VOID.

The method applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Parameters

  • The parameter Column of data type any designates the column of the table.

  • The parameter Row of data type any designates its row.

Return Value

The return value has the data type string.

Example:

 
MyTableFile[1,2] := "Station" -- writes an absolute path
MyTableFile.asString(1,2)  -- reads a relative path
