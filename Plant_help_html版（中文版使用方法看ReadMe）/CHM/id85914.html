Method to execute
2D Reference Help > Material Flow Objects > Frame > The Frame Window > General Ribbon Tab > Configure User-defined Context Menu > Method to execute

Enter the name of the Method , which the Text to be displayed calls and executes, in the respective text box below that executes method.

Parameter

If the method expects a parameter, Plant Simulation automatically passes it. The selected objects, which are located within the same Frame as the Frame whose context menu you open, are passed to this parameter, which has to be of data type list.
