deleteIcon
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Animating All Objects > deleteIcon

Syntax:

 
Path.deleteIcon(IconName:string)
Path.deleteIcon(IconNumber:integer)

A 3D Only model does not provide the method deleteIcon.

The method deleteIcon deletes the icon of the object designated by Path.

Parameter

You can enter the number of the icon as a parameter of data type integer or the name of the icon as a parameter of data type string.

This method applies to the icon of the addressed instance not to an icon of the object class.

Example:

 
MyParallelStation.deleteIcon("MyIcon")
MyParallelStation.deleteIcon(1)
