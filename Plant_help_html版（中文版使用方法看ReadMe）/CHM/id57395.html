Context Menu of the Selected Animatable Object in the 3D Scene
3D Reference Help > 3D Reference > Context Menus in 3D > Context Menu of the Selected Animatable Object in the 3D Scene > Context Menu of the Selected Animatable Object in the 3D Scene

Click an animatable object with the right mouse button in the 3D scene to select it. 3D then shows a context menu with the commands listed below.

You can also close the context menu and leave the mini toolbar open by clicking on the border of the mini toolbar with the left mouse button.

Normal animatable objects and animatable length-oriented objects provide different context menu commands.

Don’t confuse the selected object in 3D with the selected graphic of the object!

Dark green denotes that an object is selected. A yellowish green denotes that a graphic is selected, meaning that you can use the + key to contract the selection along the graphic structure.
