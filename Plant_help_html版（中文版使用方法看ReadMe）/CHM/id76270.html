_3D.setExtSegments
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Defining Length-oriented Objects > _3D.setExtSegments

Syntax:

 
Path.setExtSegments(Table:table)

The method setExtSegments sets the segments table of the length-oriented object designated by Path.

Parameter

The parameter Table of data type table designates the table from which Plant Simulation imports the settings of the segment table.

Example:

 
MyConveyor._3D.setExtSegments(extSegmentsTable)
