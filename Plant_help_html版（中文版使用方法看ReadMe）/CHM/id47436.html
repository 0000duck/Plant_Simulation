openHTMLBrowser
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > openHTMLBrowser

Syntax:

 
openHTMLBrowser(WWWAddress:string) → string

The function openHTMLBrowser starts the HTML Browser and opens the address designated by the parameter of data type string.

To make the HTML Browser open a connection to the local, calling Plant Simulation process, enter an L or a forward slash / as the first character of the address.

URLs that begin with L/ only work when the web server has been started.

The safety setting File > Model Settings > General > Access to the computer is permitted affects the function openHTMLBrowser as follows:

  • It permits only URLs with the protocols http, https, file, UNC paths and normal paths into the file system.

  • It does not permit references to executable files for paths into the file system.

    For Plant Simulation executable files are files whose file extension is listed in the environment variable PATH_EXT.

  • For paths to files it only permits those paths that point to the model folder or its sub-folders.

Parameter

The parameter WWWAddress of data type string designates the address to be opened.

Return Value

The return value has the data type string.

Example:

 
openHTMLBrowser("www.siemens.com")
openHTMLBrowser("L/default/start.htm")
