Plate
3D Reference Help > 3D Reference > Edit Ribbon Tab > Insert Textured Plate > Plate

Define the settings for the plate here.

  • Select the orientation of the plate in the 3D window. You can place it flat on the floor of the window, as a front wall, or as a side wall.

  • Enter the width of the plate.

  • Enter the height of the plate.

    If you leave one of the settings for width or height empty, the respective other value is calculated from the width to height ratio.

  • Enter the thickness of the plate.
