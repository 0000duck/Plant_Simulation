Blocked
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Statistics of the Material Flow Objects > Resource Statistics > Blocked

A material flow resource is blocked, when:

  • It is fully occupied.

  • It is not Paused.

  • It is not Failed.

  • It is not Stopped.

  • All of its stations are not Working.

  • All of its stations are not Setting-Up.

    The read-only attribute ResBlocked returns the state blocked of the resource.

    The LED of a blocked station is yellow .

    The Statistics Report shows the Blocked Time below Resource Statistics of Stationary Resources
