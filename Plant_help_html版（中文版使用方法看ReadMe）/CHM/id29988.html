Color
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Curved Objects > Color

Syntax:

 
Path.Color

The attribute Color sets the color of the curved object designated by Path. You can set the RGB values of the color with the method makeRGBValue. When you assign an invalid value to the attribute Color, Plant Simulation retains the current color.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyConveyor.Color := makeRGBValue(10,100,100)
