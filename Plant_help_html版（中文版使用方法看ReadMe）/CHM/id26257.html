In this chapter we introduce you to some of the more advanced features of modeling the flow of materials in your simulation model. You will learn:

  • How to program entrance controls and exit controls that override the built-in, default transfer behavior of the material flow objects.

  • How to define sensors.

  • How to define observers.

  • How to create and work with a distance control.

  • How to customize the behavior of the objects with controls and user-defined attributes.
