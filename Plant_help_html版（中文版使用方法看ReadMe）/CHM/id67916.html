Enter the time at which the simulation will be finished. Enter a relative time, i.e., the period of time during which the simulation runs.

The program compares this period with the simulation time and stops the simulation run, when both are identical. Let us assume that today is July 04, 12 o’clock noon and that the simulation is to run for two days.

To make the simulation run for two days, type in 2:00:00:00. If you do not want to write this out in full, you can also just type 2::: and click Apply to make the program translate this to the full format 2:00:00:00.
