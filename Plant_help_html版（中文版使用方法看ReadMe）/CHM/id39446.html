NumLeavingMU
2D Reference Help > Material Flow Objects > DismantleStation > Read-Only Attributes of the DismantleStation > NumLeavingMU

Syntax:

 
Path.NumLeavingMU

The read-only attribute NumLeavingMU returns the number of MUs from the table Exiting MUs of the DismantleStation designated by Path.

Return Value

The return value has the data type integer.

Example:

 
for i := 1 to MyDismantleStation.NumLeavingMU 
   print MyDismantleStation.leavingMU(i)
next
