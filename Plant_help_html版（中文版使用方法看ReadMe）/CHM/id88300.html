cutRow
2D Reference Help > Information Flow Objects > CardFile > Methods of the CardFile > cutRow

Syntax:

 
Path.cutRow(Row:integer)

The method cutRow removes the contents of a cell from the CardFile and returns it. The CardFile moves all cells with an index greater than integer up by one position.

Parameter

The parameter Row of data type integer designates the position of the cell.

The data type of the return value matches the data type of the CardFile.

Example:

 
value := MyCardFile.cutRow(2)
