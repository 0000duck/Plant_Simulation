The Plant Simulation program window shows the 3D window, the Edit Ribbon Tab, the View Ribbon Tab, the Video Ribbon Tab, the Toolbox, the Console, and a Frame for building your simulation models. The Frame window, the dialogs of the objects and the 3D windows windows, run as child windows within the common parent window, instead of loosely on the desktop.

You can access the 3D features on the Edit Ribbon Tab, the View Ribbon Tab, and on the Video Ribbon Tab. 3D also provides context menus, which you activate by clicking an object or into the background with the right mouse button. The context menus provide access to features depending on your present location and modeling situation. The title bar of the 3D model window shows the level in the hierarchy, i.e., the path of the object you opened.
