Open modal
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Open modal

To open the dialog window you are creating modal, select this. Then, the user cannot open any other Plant Simulation dialog windows until he closes the window of the user-defined dialog.

To allow the user to open other dialog windows in addition to the dialog you are creating, clear the check box.
