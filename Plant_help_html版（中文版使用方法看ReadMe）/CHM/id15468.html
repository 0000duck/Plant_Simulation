StatTranspStoppedTime
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for Statistics of the Mobile Objects > StatTranspStoppedTime

Syntax:

 
MU-path.StatTranspStoppedTime

The read-only attribute StatTranspStoppedTime returns the time during which the MU designated by MU-path was located on a material flow object of type Transport that was stopped by a LockoutZone.

Return Value

The return value has the data type real.

Example:

 
print .MUs.Transporter.StatTranspStoppedTime
