setAlignmentCells
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for the Format of Lists and Tables > setAlignmentCells

Syntax:

 
Path.setAlignmentCells(Range:listrange [,Range:listrange, ...], Alignment:string)

The method setAlignmentCells sets the alignment of the cells of the StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Parameters

  • The parameters Range of data type listrange designate the range of cells, whose alignment you want to set.

  • The parameter Alignment of data type string designate the value: Left, Right, or Centered.

Example:

 
MyTableFile.setAlignmentCells({1,2}..{1,6},"left")
timeseq.setAlignmentCells({1,5}..{1,112},"right")
