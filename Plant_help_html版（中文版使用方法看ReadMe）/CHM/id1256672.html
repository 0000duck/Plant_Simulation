Switch3Way
Add-Ins Reference Help > Object Libraries > Electrical Overhead Monorail > Switch3Way > Switch3Way

You can use the object Switch3Way to model a switch which distributes the Transporters in three directions. The directions are straight, right, or left.

The object Switch3Way provides the method switchTo to change the position of the switch.
