Variable
2D Reference Help > Information Flow Objects > Variable > Variable

The Variable is a global variable that other objects and methods can access during a simulation run. A variable can represent an item which stores a quantity. The opposite of a variable is a constant, whose value is known and does not change. You might, for example, use it to store data over an extended period of time during your simulation run, to count values up and down, to assign values, etc.

You can also show the contents of a Variable in an HtmlReport, compare Display a Variable.

To add the Variable to your simulation model, click Manage Class Library > Basic Objects > InformationFlow > Variable on the Home ribbon tab.
