setEditorRightsRow
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Rows of the TableFile > setEditorRightsRow

Syntax:

 
Path.setEditorRightsRow(Row:any [,Row:any, ...], ReadOnly:boolean)

The method setEditorRightsRow sets the editing permissions of row(s) of the TableFile designated by Path to the designated value.

Parameters

  • The parameters Row of data type any designate the row(s) editing permissions you want to set.

  • The parameter ReadOnly of data type boolean designates read-only access (true) or read and write access (false).

Example:

 
-- sets the editor rights of row 1
Tab1.setEditorRightsRow(1,true)
-- sets the editor rights of row 3 and 5
Tab1.setEditorRightsRow(3,5,false)
