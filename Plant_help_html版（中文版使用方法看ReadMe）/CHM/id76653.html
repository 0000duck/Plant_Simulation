Tab Statistics
Add-Ins Reference Help > Object Libraries > Value Stream Mapping > Value Stream Mapping Objects > Process_assembly > Dialog Window of the Object Process_assembly > Tab Statistics > Tab Statistics

The tab Statistics shows the statistics values which the built-in material flow objects also show. In addition it shows the number of reworked products and the number of scrapped products.

You can view the Current length of the waiting queue, the Mean value, and click Show Waiting Queue.
