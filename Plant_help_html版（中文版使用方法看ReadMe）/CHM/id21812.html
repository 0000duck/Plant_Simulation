UseMetricPrefix
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > UseMetricPrefix

Syntax:

 
Path.UseMetricPrefix

The attribute UseMetricPrefix sets if the Chart shows a unit prefix on the y-axis for very great or very small values, for example 100K instead of 100000, (true) or not (false).

The prefix

stands for

p

pico

n

nano

u

micro

m

milli

K

kilo

M

million

B

billion

T

trillion

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyChart.UseInputChannels := true
