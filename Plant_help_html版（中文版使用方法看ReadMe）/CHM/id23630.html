Insert Column
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Insert Column

The command Insert Column inserts a new Empty column. All existing columns move one position to the right.

In the example below we clicked in Column 2 and selected the command Insert Column. Plant Simulationthen inserted the new Empty column in Column 2 and moved the previous Column 2, and the columns to the right of it, one position to the right.
