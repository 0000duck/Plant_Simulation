SetupAttrName
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Setting-Up the Material Flow Objects > SetupAttrName

Syntax:

 
Path.SetupAttrName

The attribute SetupAttrName sets if the object designated by Path sets up depending on a MU name or depending on a user-defined attribute. For the former, you have to enter an empty string (""), for the latter the name of the attribute.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
ParallelStation.SetupAttrName := ""
MyStation.SetupAttrName := "MySetupAttribute"
