Inserting a Dialog Item into Your Dialog
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > Inserting a Dialog Item into Your Dialog

Before you start creating your dialog, take a few moments to decide about its structure, i.e., which dialog items do you need, can you show them all in a flat hierarchy in the dialog or do you need to insert tabs.

The structure you decide upon determines how and in which order you create the dialog items. Let us say, you want to show the dialog items on tabs. Then, you first have to create the Tab Control, which is the container that accommodates the individual tabs. After this you have to create the individual Tabs, and finally you can insert the dialog items on each one of these tabs.

  • To create a new dialog item, select one of these items on the context menu:

    New Button

    New List View

    New Check Box

    New Menu/New Menu Command

    New Drop-Down List Box

    New Radio Button

    New Edit Text Box

    New Static Text Box

    New Group Box

    New Tab Control

    New Image

    New Tab Page

    New List Box

  • To add one tab at a time to a New Tab Control, click it with the right mouse button on the tab Elements, select New Tab Page on the context menu, and enter a name and caption into the text box Dialog Item Tab Page.

  • To add a dialog item to a tab on a New Tab Control you created, click the name of the tab with the right mouse button in the structure on the tab Elements and select the item you want to insert on the context menu.

You can enter the same Name for dialog items, as long as those dialog items are not located in the same group of items. Groups of items are: Group Box, Menu, Tab Control and Tab.

To rearrange the order of items of type Tab within a Tab Control and Menu in the structure on the tab Elements within these groups, use Shift+Up arrow or Shift+Down arrow. The structure shows all other dialog items according to their position in the user-defined dialog itself.

To move items out of one of the groups or place them there, click the item with the left mouse button in the structure, drag it to the new location and drop it there.
