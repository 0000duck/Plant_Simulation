_3D.isGraphicGroupInternal
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Graphic Groups > _3D.isGraphicGroupInternal

Syntax:

 
Path._3D.isGraphicGroupInternal(Name:string) → boolean

The method _3D.isGraphicGroupInternal returns if the specified graphic group of the object designated by Path is internal (true) or not (false).

Parameter

The parameter Name of data type string designates the name of the graphic group.

Return Value

The return value has the data type boolean.

Example:

 
print Station._3D.isGraphicGroupInternal("MyGraphicGroup")
