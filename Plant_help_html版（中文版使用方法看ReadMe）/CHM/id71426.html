Edit 3D Properties
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Navigate Menu > Edit 3D Properties

The menu command Edit 3D Properties opens the Edit 3D Properties window of the respective object in 3D. You can then change the 3D properties of the object.

Plant Simulation only shows this command when you created or opened the model in 3D Only mode or in 2D/3D Mode.
