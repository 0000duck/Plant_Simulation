TypeTable
2D Reference Help > Tools > TransferStation > Attributes of the TransferStation > TypeTable

Syntax:

 
Path.TypeTable

The attribute TypeTable sets the table in which you define, which parts you want to load or unload. Enter the part type in column 1 of the table, enter the container type in column 2. The TransferStation only takes parts from the column part type from the parts station when loading and moving it takes them directly from the parts station, when unloading and reloading it takes them from the means of transport. When loading and reloading parts, you have to enter the type of means of transport onto which you want to load the part.

Data Type of the Value You Can Assign

You can assign a value of data type table.

Example:

 
MyTransferStation.TypeTable := MyUnloadTable
