Installing the Siemens PLM License Server
Setting-Up and Starting > Licensing Plant Simulation > Installing the Siemens PLM License Server > Installing the Siemens PLM License Server

If you install Tecnomatix Plant Simulation for the first time and if you are Using a Floating License or if you are Using a Named-User License, you will have to install the Siemens PLM License Server.

  • To install the Siemens PLM License Server, log on as Administrator on the computer on which you would like to run the license server.

  • Double-click the program setup.exe on the DVD. Or double-click the MSI file if you downloaded the program.

  • Activate Tecnomatix License Server and click Next.

  • Select the installation language, English in our case, and click OK.

  • Follow the instructions in the dialog.

    For detailed installation instructions, consult the file SPLM_Licensing_Install.pdf within the folder above.

    If you would like to use a WIBU dongle on your computer, you have to Install the WIBU Drivers first.
