_3D.generateHollowEllipse
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating Extrusion Profiles > _3D.generateHollowEllipse

Syntax:

 
basis.~._3D.generateHollowEllipse(BasePoint:length[2], Width:length, Height:length, BarThickness:length, CurvePrecision:real)

The method _3D.generateHollowEllipse creates a two-dimensional hollow ellipse profile which you can use in your user-defined extrusion configuration.

  • The parameter BasePoint is an array of data type length which sets the base point of the hollow ellipse profile.

  • The parameter Width of data type length sets its width.

  • The parameter Height of data type length sets its height.

  • The parameter BarThickness of data type length sets the thickness of the bar.

  • The parameter CurvePrecision of data type real sets the sampling accuracy for the generate ellipse.

Example:

 
var profile : any := basis.~._3D.generateHollowEllipse([2.6,4.5], 4, 2, 3, 0.1]
 --  returns a two-dimensional array containing the values defining a profile; to be used for an extrusion profile
