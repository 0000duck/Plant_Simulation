Watch Window
2D Reference Help > Information Flow Objects > Method > The Method Window > Method Debugger > Watch Window > Watch Window

To open the Watch window, you can either:

  • Click on the Toolbar of the Method Debugger .

  • Press the F12 key.

  • Select the menu command Debug > Watch Window. By default, the Watch Window is part of the Debugger window.

You can access the features of the Watch Window on the Tab Variables, the Tab Anonymous Identifiers, the Tab Call Stack, the Tab Call Chains, the Tab Suspended, and the Tab Expressions.
