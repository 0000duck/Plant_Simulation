SpeedCtrl
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > SpeedCtrl

Syntax:

 
MU-path.SpeedCtrl

The attribute SpeedCtrl designates a Method object, the Speed control, which the Transporter designated by MU-path activates as soon as it reaches its final speed after speeding up or slowing down or when it stops after decelerating. When you did not enter the name of a Method, the attribute is VOID.

This only applies, when you selected the check box Acceleration.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
.MUs.Transporter.SpeedCtrl := &mySpeedCtrl
