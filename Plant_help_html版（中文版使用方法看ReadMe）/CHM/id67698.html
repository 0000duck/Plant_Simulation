Edit Formula
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Formula

The command Formula activates or deactivates formula mode. A formula can be any expression accessing and linking other cells of the list or accessing any of the attributes of the objects. When defining a formula, you will use the same notation you use for defining a Method. You can also insert and read formulas with methods.

  1. To activate Formula mode, click the button so that it is selected.

  2. To enter a formula into a cell, click in it and type the expression in. Instead, you can also type it into the text box above the list.

  3. To show the result of the calculation of the formula, i.e., its value, in the cell, press Enter.

  4. To show the formula itself in the text box, click in the cell containing the formula.

    The data type of the result of a formula has to have the same data type as the cell or the column in which the cell containing the formula is located.
