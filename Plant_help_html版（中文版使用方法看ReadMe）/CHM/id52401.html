StatWaitingResPortion
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Read-Only Attributes for Statistics of the Material Flow Objects > StatWaitingResPortion

Syntax:

 
Path.StatWaitingResPortion → real

The read-only attribute StatWaitingResPortion returns the portion of the statistics collection period the object designated by Path was Waiting for Exporters.

Return Value

The return value has the data type real.

Example:

 
print ParallelStation.StatWaitingResPortion
