bell
SimTalk Reference > General SimTalk Functions > Predefined Functions > Output Functions > bell

Syntax:

 
bell(Frequency:integer, Duration:integer)

The function bell outputs an acoustic signal on the computer speaker.

Parameters

  • The parameter Frequency of data type integer designates the frequency of the bell sound.

  • The parameter Duration of data type integer designates its duration.

Example:

 
bell(440,1000)
