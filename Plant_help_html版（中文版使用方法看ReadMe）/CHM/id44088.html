insert
SimTalk Reference > General SimTalk Functions > Data Types > Array > Methods and Attributes of Arrays > insert

Syntax:

 
insert(Index:integer, Value:any)

The method insert inserts a value into a one-dimensional array which does not have a fixed size.

Parameters

  • The parameter Index of data type integer designates the index position at which Plant Simulation inserts the value.

  • The parameter Value of data type any designates the value which you want to insert.

Example:

 
var a:real[3]
print a.dim -- prints 3
