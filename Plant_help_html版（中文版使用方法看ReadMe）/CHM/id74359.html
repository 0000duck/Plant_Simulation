multiply
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > multiply

Syntax:

 
Path.multiply(Factor:any)

The method multiply multiplies all items of the value column with a numerical value. In addition, a time sequence may be multiplied with another time sequence. Then, the time column of the result is the union of the time column of both time sequences. The value column of the result contains the product of the values of both time sequences at the respective times.

Parameter

The parameter Factor of data type object designates the multiplication factor.

Example:

 
MyTimeSequence.multiply(5)
dimx.multiply(dimy)
