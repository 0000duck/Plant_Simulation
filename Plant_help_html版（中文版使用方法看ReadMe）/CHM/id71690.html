RearMU
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for the Location of the Mobile Objects > RearMU

Syntax:

 
MU-path.RearMU

The read-only attribute RearMU returns the MU, which is located directly behind the MU designated by MU-path on the same material flow object.

Return Value

The return value has the data type object.

Example:

 
if @.RearMU /= VOID 
   @.RearMU.Stopped := true
end
