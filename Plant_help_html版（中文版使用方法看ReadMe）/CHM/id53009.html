Step-by-Step Help > Animating the Simulation Model and Viewing the Results > Viewing and Visualizing Statistics > Viewing Statistics in the Dialogs of the Objects > Check the Contents List of the Stations > Print the Contents List to the Console as an Array

To print the contents lists of the objects Buffer and Conveyorfrom our example to the Console as an array each, type the following source code into a Method and run the method. 

print Buffer.contentsList
print Conveyor.contentsList
