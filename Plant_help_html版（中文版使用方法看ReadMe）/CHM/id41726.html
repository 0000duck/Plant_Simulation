Using a Node-Locked License
Setting-Up and Starting > Licensing Plant Simulation > Using a Node-Locked License

A node-locked license is only available for the user on whose computer the Plant Simulation license file is installed.

The SPLM License Service is not required for this type of license.

When using a WIBU dongle, you can also move the licenses and the dongle to a different computer.
