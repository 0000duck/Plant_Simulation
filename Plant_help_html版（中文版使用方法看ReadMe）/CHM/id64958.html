_3D.InheritGraphics
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > _3D.InheritGraphics

Syntax:

 
Path._3D.InheritGraphics

The attribute _3D.InheritGraphics activates (true) or deactivates (false) graphic inheritance of the object designated by Path.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
.Models.Frame.Station1._3D.InheritGraphics := false
var a: boolean := .Models.Frame.Station1._3D.InheritGraphics 
-- returns whether the graphic is inherited or not
