addObject
2D Reference Help > The Plant Simulation Program Window > Toolbox > Attributes and Methods of the Toolbox > addObject

Syntax:

 
Path.addObject(Icon:object [,Position:integer]) → boolean

The method addObject adds the icon of an object to the toolbar designated by Path.

Parameters

  • The parameter Icon of data type object designates the entire path to the object.

  • The optional parameter Position of data type integer designates the position of the icon/button on the toolbar. The position 01 is to the right of the Select button .

Return Value

The return value has the data type boolean.

Example:

 
-- adds the icon of the object MyStation to the toolbar Material Flow 
-- and positions it to the right of the object Station
.MaterialFlow.Toolbar.addObject(.Materialflow.MyStation,08)
