Content
2D Reference Help > Display and User Interface Objects > HtmlReport > Attributes of the HtmlReport > Content

Syntax:

 
Path.Content

The attribute Content sets the content which the HtmlReport designated by Path displays.

Assignment value

You can assign a value of data type string.

Example:

 
HtmlReport.Content := "[!self, Header, *]\
# General Information\
[Receiving, \"Statistics of the Source named 'Receiving' and the station named 'MyStation'\"]\
[MyStation]\
Created on [=day(sysdate)].[=month(sysdate)].[=year(sysdate)+1900]"
