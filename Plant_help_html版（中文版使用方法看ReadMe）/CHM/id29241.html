BackgroundColor
2D Reference Help > Display and User Interface Objects > Comment > Attributes of the Comment > BackgroundColor

Syntax:

 
Path.BackgroundColor

The attribute BackgroundColor sets the background color of the Comment, which you inserted into a Frame. You can set the RGB values of the color with the method makeRGBValue.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyComment.BackgroundColor := makeRGBValue(100,100,100)
MyComment.BackgroundColor := 6579300 -- is the same as the color above
