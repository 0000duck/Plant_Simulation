BitShift
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Individual Bits > BitShift

Syntax:

 
BitShift(Value:integer, NumberOfDigits:integer) → integer

The function BitShift shifts the bits of the value designated by the parameter integer using the number of digits designated by integer and returns the result.

Parameters

  • The parameter Value of data type integer designates the value.

  • The parameter NumberOfDigits of data type integer designates number of digits. It moves them to the left, if you specified digits greater than 0. It moves them to the right, if you specified digits greater less than 0.

Return Value

The return value has the data type integer.
