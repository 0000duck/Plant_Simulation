Format string
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Format > Tab Data Type > Format string

For the data types Integer, Real, Length, Weight, Speed, Acceleration, Money, and String you can restrict the data the user can enter in the window of the list object to certain values by entering a format string. Note that the Format String does not restrict assignments via Methods.

  • Data type String

    For the data type string Plant Simulation differentiates for each digit:

    Format

    Enter

    A

    letters only

    U

    upper case letters only

    X

    all characters

    C

    upper case letters and numbers

    N

    numbers only

    L

    letters and numbers

    I

    all characters permitted for object names: letters, including umlaut, numbers and underscore

    So, if you enter AANXU, the first two digits have to be letters, such as a lower case a or an upper case A. The third digit has to be a number, such as 1, the fourth digit may be any character, such as e, E, 5or a special character, such as the question mark ?. The fifth digit has to be an upper case letter, such as B. This way you set the length of an expression and the format of each single digit.

    When you enter several terms of your choice, separated by a semicolon, and not followed by a blank, Plant Simulation creates a drop-down list in the cell of the formatted range into which the user clicks. You can then select one of the terms you entered from the drop-down list.

    When you enter stack; list; random, Plant Simulation creates a drop-down list with the commands stack, list and random and enters it into the cell you click in the range you formatted.

  • Data type Integer

    Enter a number into the text box to limit the number of digits Plant Simulation shows. When you enter 3, for example, Plant Simulation only shows the first three digits, when you enter data, such as 123. Entering a minus character, such as -3, allows you to enter negative values also, such as -123.

    For the data type integer you can also enter the value color as the format string. When you then double-click cell in this column, Plant Simulation opens the Windows dialog Color. You can then select a color for the background of this cell here. Plant Simulation calculates the RGB value and enters it as an integer value into the cell, compare makeRGBValue.

  • Data type Real

    Enter two numbers, separated by a point, into the text box to restrict the number of digits before and after the decimal point. When you enter 5.3, for example, the number the user enters may have 5 digits altogether; three of these digits may be after the decimal point 1.234 for example. Entering a minus character, such as -6.3, allow you to enter negative values also, such as -123.456.
