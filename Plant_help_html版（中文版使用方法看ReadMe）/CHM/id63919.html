Export Text File
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Export Text File

The command Export Text File saves the contents of the list as text file (.txt). A text file only saves the contents of the list. You can also create a text file in a text editor or a spreadsheet application and then open it as a *.txt file in an Plant Simulation list. Click Save to save the file.

Enter the name with which you want to save the file and select the file type with the encoding you want to use: ANSI, UTF-8, or Unicode.

When saving or loading a list file as a text file, the settings you selected with the command Text File Format determine how Plant Simulation saves data and how it interprets that data while opening this ASCII file.
