Column width
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Format > Tab Dimension > Column width

Enter the column width in character widths of a non-proportional font. The default setting is 20, the maximum column width is 180.

To simultaneously change the width of several columns, mark the range of columns before entering a number here.
