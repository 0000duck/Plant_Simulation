Full
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Read-Only Attributes of Lists and Tables > Full

Syntax:

 
Path.Full

The read-only attribute Full returns if the list designated by Path has reached the number of cells you set in the window or by with the attributes MaxDim (lists) or MaxXDim and MaxYDim (true) or not (false).

Normally the number of cells in lists/tables is not limited, i.e., lists/tables are never full.

The read-only attribute applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Return Value

The return value has the data type boolean.

Example:

 
print MyStackFile.Full
print MyCardFile.Full
print MyTableFile.Full
