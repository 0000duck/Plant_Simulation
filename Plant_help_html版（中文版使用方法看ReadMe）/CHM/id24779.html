Open Class
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Navigate Menu > Open Class

The menu command Open Class opens the dialog of the class of the object in the Class Library.

When you select the command Open Class for an instance of an object, Plant Simulation moves up in the inheritance structure until it reaches an object in the Class Library. It then opens the dialog of this object.
