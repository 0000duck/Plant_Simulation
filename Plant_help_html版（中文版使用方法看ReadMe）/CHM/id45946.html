Day Start
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > Units > Daylight saving time > Day Start

Enter the day when daylight saving time starts. For the U.S. and the EU, enter 0for Sunday. 0 designates Sunday, 1 Monday, 2Tuesday, 3 Wednesday, 4 Thursday, 5 Friday, and 6 Saturday.
