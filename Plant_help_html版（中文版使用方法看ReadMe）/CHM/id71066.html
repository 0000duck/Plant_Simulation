Output file
3D Reference Help > 3D Reference > Video Ribbon Tab > Video Settings > Output file

Accept the default name--default.avi--for the finished AVI file or enter a different name into the text box. When you also enter the file extension avi, the Windows Explorer shows the icon for Video Clip files in the list window in the browser window. To select another folder, than the default folder where you installed Plant Simulation, to save the AVI file to, click Browse.

Once you click Finish Recording on the View ribbon tab, 3D writes the video it recorded to the file and folder you set above.

  • To apply the settings you selected, click Apply.

  • To close the dialog without applying them, click Cancel.
