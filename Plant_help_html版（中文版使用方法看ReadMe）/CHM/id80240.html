Product-Oriented Statistics of all Existing and Deleted MUs by Classes

This topic provides details about the product-oriented statistics of all existing MUs and those, which were removed from the plant.

The column

Shows

Read-only attribute

Count

how many MU instances of this class existed during the statistics collection period. Plant Simulation only counts those MUs whose product statistics was active during the simulation run. The MUs may be located in different Frame hierarchies.

StatCount

Deleted

how many MU instances of this class were removed from the plant during the statistics collection period. Plant Simulation only counts those MUs whose product statistics was active during the simulation run.

StatDeleted

Mean Life Time

the average life-span of the MUs, which were created and removed from the plant during the statistics collection period. Plant Simulation only counts those MUs whose product statistics was active during the simulation run.

StatAvgLifeSpan
