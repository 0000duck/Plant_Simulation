getRegistry
SimTalk Reference > General SimTalk Functions > Predefined Functions > Operating System Functions > getRegistry

Syntax:

 
getRegistry(Key:string, Value:string)

The function getRegistry returns a key from the folders HKEY_CLASSES_ROOT, HKEY_CURRENT_USER and HKEY_LOCAL_MACHINE in the MS Windows Registry Editor.

Parameters

  • The parameter Key of data type string designates the key in a folder.

  • The parameter Value of data type string designates the value of the key.

The function returns values of these data types:

  • void, when the value does not exist

  • integer for a DWORD value (REG_DWORD)

  • string in all other cases (REG_SZ, REG_EXPAND_SZ, REG_MULTZI_SZ, REG_BINARY)

Example:

 
print getRegistry("HKEY_LOCAL_MACHINE\SOFTWARE\Tecnomatix\eM-Plant\14", "eMPlantHome")
-- returns the folder in which the program is installed
print getRegistry("HKEY_CURRENT_USER\Software\Tecnomatix\eM-Plant\14", "LangSession")
