length_to_num
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Physical Data Types with Units into Data Types without Units > length_to_num

Syntax:

 
length_to_num(length) → real

The function length_to_num converts the data designated by the parameter of data type length into a value without a unit.

Parameter

The parameter Data has the data type length.

Return Value

The return value has the data type real.

Example:

 
var t: time, le: length
t := length_to_num(le)
