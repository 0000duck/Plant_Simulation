UseIcon
2D Reference Help > Display and User Interface Objects > Drop-Down List > Attributes of the Drop-Down List > UseIcon

Syntax:

 
Path.UseIcon

The attribute UseIcon sets if the Drop-Down List designated by Path is displayed in the Frame with its current icon (true) or with the default look which we defined (false).

Use icon off

Use icon on

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyDropdownList.UseIcon := false
