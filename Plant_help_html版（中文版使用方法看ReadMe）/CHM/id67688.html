Ready
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Read-Only Attributes for the State of the Material Flow Objects > Ready

Syntax:

 
Path.Ready → boolean

The read-only attribute Ready returns if the object designated by Path is occupied and an MU is ready to exit (true) or not (false).

For the ParallelStation the read-only attribute Ready returns if it is occupied and at least one MU is ready to exit (true) or not (false).

Return Value

The return value has the data type boolean.

Example:

 
print MyStation.Ready
