increment
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Miscellaneous Methods of the TableFile > increment

Syntax:

 
Path.getFormatData(TargetTable:table) → any

The method increment increments the value in the designated cell of the TableFile designated by Path by the value you pass.

Parameters

  • The parameter Column of data type any designates the column.

  • The parameter Row of data type any designates the row.

  • The optional parameter Increment of data type real or integer designates the value by which the existing value is incremented.

If you do not enter a value, the method increments the value by 1.

Return Value

The return value has the data type any. It designates the new value in the cell.

Example:

 
MyTableFile.increment(2,3) -- increments the value in the cell at the position (2,3) by 1
