Debug Method
2D Reference Help > Information Flow Objects > Method > The Method Window > Tools Ribbon Tab > Debug Method

The command Debug Method, or the F11 key, runs the Method, i.e., executes the source code one step at a time. Plant Simulation opens the Method Debugger, where you can then step through the source code using the Step Into or Step Over commands/buttons. Note that the command Debug has the same function as running the Method while holding down the Ctrl+Alt+Shift keys.

  • When Plant Simulation is stuck in an endless loop and when Plant Simulation is running in the foreground, hold down the Ctrl+Alt+Shift left keys for 5 seconds to open the debuggerwhile a Method is being executed.

  • When Plant Simulation is stuck in an endless loop and when Plant Simulation is not running in the foreground, hold down the Ctrl+Alt+Shift right keys for 5 seconds to open the debuggerwhile a Method is being executed.
