BackgroundColor
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Attributes for the Format of Lists and Tables > BackgroundColor

Syntax:

 
Path.BackgroundColor

The attribute BackgroundColor sets the Background color for all cells of the list or table. You can set the RGB values of the color with the method makeRGBValue.

The attribute applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Data Type of the Value You Can Assign

You can assign a value of data type integer: 1 designates Black, 2 Red, 3 Green, 4 Blue, 5 Magenta, 6 Yellow, and 7 Cyan.

Example:

 
var rgb: integer
MyTableFile.backgroundcolor := makeRGBValue(120,120,120)
rgb := MyTableFile.backgroundcolor
print rgb
