requestParts
Add-Ins Reference Help > Object Libraries > Kanban Objects > KanbanSource > Methods of the KanbanSource > requestParts

Syntax:

 
Path.requestParts(Part:string, Amount:integer)

The method requestParts insert a new row of orders on the table Open requests, and starts a production order if necessary.

Parameters

  • The parameter Part of data type string designates the name of the part (PartType).

  • The parameter Amount of data type integer designates the amount of requested parts.

Example:

 
var requiredWheelType: string
requiredWheelType := BOM[1,@.PartType]
KanbanBuffer.requestParts(requiredWheelType,4)
