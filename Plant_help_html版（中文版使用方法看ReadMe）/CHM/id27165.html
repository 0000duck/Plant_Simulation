Using a Floating License
Setting-Up and Starting > Licensing Plant Simulation > Using a Floating License

A floating license is available for any user within your computer network. If you purchased several floating licenses, several users can work with the number of Plant Simulation licenses installed on the license server computer. This license server can, but does not have to, be the computer on which the applications or application servers run.

The SPLM License Service has to run on a server to assign and manage the licenses. You do not have to install it on the individual client computers running Plant Simulation.
