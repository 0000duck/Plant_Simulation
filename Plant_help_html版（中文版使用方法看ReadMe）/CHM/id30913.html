Capacity
2D Reference Help > Material Flow Objects > PickAndPlace Robot > Dialog Window of the Pick-and-Place Robot > Tab Attributes > Capacity

Enter the Capacity, i.e., the number of MUs that the pick-and-place robot can transport at any one time. Enter a number greater than 1.

It enables the pick-and-place robot to transport more than one part at the same time. It might, for example, pick up a number of bottles and place same into the appropriate slots of a bottle crate one at a time (FIFO).

If you want to unload the parts according to another rule, for example LIFO, you can accomplish this in an exit control.
