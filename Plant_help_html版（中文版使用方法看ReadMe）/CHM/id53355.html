Conditional Expressions with when-then-else
SimTalk Reference > General SimTalk Functions > Control Flow Statements > Conditional Expressions with when-then-else

The when-then-else-statement selects an expression based on the value of a boolean expression.

Syntax:

 
when condition then expression1 else expression2

  • The statement condition is a boolean expression.

  • The statement expression1 is the expression to be selected if the condition is true.

  • The statement expression2 is the expression to be selected if the condition is false.

Example:

 
@.color := when ?.entranceLocked then "red" else "green"

Example:

 
Method(x, y + (when dy > 0 then y+dy else 100))
