FeedRate
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > FeedRate

Syntax:

 
Path.FeedRate

For the setting Category > Plotter the attribute FeedRate sets the feed rate by which the Chart scrolls, when it reached the right hand side of the display.

Data Type of the Value You Can Assign

You can assign a value of data type real. Enter 0.1 for smooth scrolling, enter 12 to scroll the entire page.

Example:

 
MyChart.Scrollbar := true
MyChart.Feedrate := 4
