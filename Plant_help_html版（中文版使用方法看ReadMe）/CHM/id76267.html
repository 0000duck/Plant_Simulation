Take preferred position
2D Reference Help > Tools > CrossSlidingCar > Dialog Window of the CrossSlidingCar > Tab Vehicle Attributes > Take preferred position

Select the check box Take transferred position to make the cross-sliding car move to the feeding line whose name you enter into the text box Preferred position when no transport order exists.

This way you can send the cross-sliding car to a certain feeding line, for example to the Conveyor which provides the most parts to be transported.
