_3D.createCone
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating 3D Shapes > _3D.createCone

Syntax:

 
_3D.createCone(BasePoint:length[3], Radius:length, Height:length [,GraphicGroupName:string]) → integer

The method _3D.createCone creates a cone in the scene window.

Parameters

  • The parameter BasePoint is an array of data type length which sets the base point of the cone. The base point designates the center of the base area.

  • The parameter Radius of data type length sets its radius.

  • The parameter Height of data type length sets its height.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Return Value

The return value has the data type integer and specifies the numerical index of the cone, compare _3D.getGraphicIndex.

Example:

 
_3D.createCone([3.0,2.5,1],3,2)

Example:

 
_3D.createCone([3.0,2.5,1],3,2,"deco")

Example:

 
var index := _3D.createCone([3.0,2.5,1],3,2,"deco")
_3D.setGraphicMaterial([index], makeRGBValue(255,0,0), 0, 0, 0, 0, 0, "deco")
