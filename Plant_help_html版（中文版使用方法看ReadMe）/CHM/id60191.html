num_to_acceleration
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Values without Units into Physical Data Types with Units > num_to_acceleration

Syntax:

 
num_to_acceleration(real) → acceleration

The function num_to_acceleration converts a number without a unit into a value of data type acceleration.

Parameter

The parameter Number of data type real designates the number.

Return Value

The return value has the data type acceleration.

Example:

 
var s: speed
s := num_to_acceleration(3.7) * Conveyor3.Time
