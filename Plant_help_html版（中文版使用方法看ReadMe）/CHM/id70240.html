Active
2D Reference Help > Information Flow Objects > TimeSequence > Attributes of the TimeSequence > Active

Syntax:

 
Path.Active

The attribute Active activates the TimeSequence (true), i.e., makes it record data, or deactivates it (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTimeSequence.Active := true
