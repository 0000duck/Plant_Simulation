getFontSizeRow
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Rows > getFontSizeRow

Syntax:

 
Path.getFontSizeRow(Row:any) → integer

The method getFontSizeRow returns the font size of a column of the TimeSequence designated by Path.

Parameter

The parameter Row of data type any designates the row whose font size you want to get.

Return Value

The return value has the data type integer. 1 stands for Small, 2 for Medium, 3 for Large and 4 for Extra Large.

Example:

 
print timeSequence1.getFontSizeRow(34)
