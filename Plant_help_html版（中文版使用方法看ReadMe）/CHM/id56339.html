Name
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Defining Properties of a Dialog Item > Name

Enter the name of the dialog item. The Dialog shows this Name in the text box on the tab Elements. You can enter the same Name for dialog items, as long as those dialog items are not located in the same group of items. Groups of items are: Group Box, Menu, Tab Control and Tab. We recommend to enter a unique identifier, containing the type of dialog item, for example xyz_button, xyz_list, etc.

You cannot use a period (.) in the name of a dialog item. As SimTalk treats this name as a string, you can also enter special characters.

A Method object can call this dialog item using the Name you entered. For some dialog items you can enter a Caption in addition.
