getAttrValue
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Managing User-defined Attributes > getAttrValue

Syntax:

 
Path.getAttrValue(AttributeNumber:integer) → any

The method getAttrValue returns the value of the user-defined attribute of the object designated by Path.

Parameter

The parameter AttributeNumber of data type integer designates the user-defined attribute. The data type of the value depends on the type of the attribute.

Return Value

The return value has the data type any.

Example:

 
print MyStation.getAttrValue(1)
