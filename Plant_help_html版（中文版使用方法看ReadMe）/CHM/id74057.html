CommonFormatColumnIndex
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Attributes for the Format of Lists and Tables > CommonFormatColumnIndex

Syntax:

 
Path.CommonFormatColumnIndex

The attribute CommonFormatColumnIndex activates the Common format (true ) or deactivates it (false ), if the column index is of data type table, list, stack or queue.

The attribute applies to the object TableFile.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTableFile.CommonFormatColumnIndex := true
