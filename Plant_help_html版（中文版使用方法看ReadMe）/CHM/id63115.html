Exiting MU
2D Reference Help > Material Flow Objects > AssemblyStation > Dialog Window of the AssemblyStation > Tab Attributes > Exiting MU

Select how the AssemblyStation handles the MU, which is leaving the station:

  • Main MU: It moves the main MU to the succeeding object.

  • New MU: It moves the assembled/new MU to the succeeding object.

When you select New MU, the AssemblyStation shows the button and the text box MU. Click this to open the dialog Select Object and select the new MU in the dialog.
