New Tab Page
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New Tab Page

To add one tab at the time to the tab control, click it with the right mouse button on the tab Elements and select New Tab Page on the context menu.

  • Enter the Name of the tab into the dialog Dialog Item Tab Page.

  • Enter the Caption of the tab into the dialog Dialog Item Tab Page. The tab on the tab control shows this caption as the title on the tab itself.

    To change the order of the tabs in the dialog, select a tab on the tab Elements, hold down the Shift key and press the up arrow or the down arrow to move this tab to the left or to the right. The dialog shows the first tab in the structure as the leftmost tab.
