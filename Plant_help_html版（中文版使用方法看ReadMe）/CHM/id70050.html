str_to_bool
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Numerical Values > str_to_bool

Syntax:

 
 str_to_bool(string) → boolean

The function str_to_bool converts the parameter designated by the parameter of data type string into a value of data type boolean. Note that the spelling of the string is not case-sensitive.

Parameter

The parameter has the data type string.

Return Value

The return value has the data type boolean. true returns true, any other value returns false.

Example:

 
table[1,1] := str_to_bool("true")
