dayOfYear
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Managing Date and Time > dayOfYear

Syntax:

 
dayOfYear(Date:date) → integer
dayOfYear(DateTime:dateTime) → integer

The function dayOfYear returns the number of days that has elapsed since January first.

Parameter

The parameter Date or DateTime of data type date or dateTime designates the date or date and time respectively.

Return Value

The return value has the data type integer.

Example:

 
var d: date, dt: dateTime
dt := str_to_dateTime("31.12.99 23:59:59")
d := str_to_date("1.1.95")
print dayOfYear(dt)  -- 364
print dayOfYear(d)   -- 0
