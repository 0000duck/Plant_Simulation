BatReserve
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > BatReserve

Syntax:

 
MU-path.BatReserve

The attribute BatReserve sets the Reserve charge of the battery of the Transporter designated by MU-path. When the battery charge drops below the value you enter for the BatReserve, Plant Simulation calls the battery charge control (BatChargeCtrl) that may initiate recharging the battery. The unit is ampere hours (A).

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
-- set reserve to 20 % of capacity
@.BatReserve := 0.2 * @.BatCapacity
