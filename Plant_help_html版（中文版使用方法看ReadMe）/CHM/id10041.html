setBlocksize
2D Reference Help > Tools > TransferStation > Methods of the TransferStation > setBlocksize

Syntax:

 
Path.setBlocksize(Blocksize:integer)

The method setBlocksize sets the block size, i.e., the number of parts, which you want the TransferStation to load or to unload en bloc. The default setting is 1.

A block is the number of parts, which is loaded or unloaded all at once, and which you enter into the text box Block size. This means that Plant Simulation does not compute the time or loading and unloading per part, but for the block as a whole.

You can only enter a Block size greater than 1, when the parts station is a ParallelStation for modeling a Loading and a Moving process.

For an Unloading and Reloading process, you can use any of the permitted stations as the block is not removed directly from the station but from a pallet or from a Transporter on the station.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyTransferStation.setBlocksize(3)
