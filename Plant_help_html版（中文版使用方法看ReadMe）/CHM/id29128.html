prompt
SimTalk Reference > General SimTalk Functions > Predefined Functions > Input Functions > prompt

Syntax:

 
prompt(Text:string [,MoreText:string, ...]) → string

The function prompt asks the user to enter data.

Plant Simulation opens a dialog displaying the designated text and a text box. It pauses the simulation run until the user has entered that data and closed the window by clicking OK. Plant Simulation can display a maximum of four lines with a maximum of 45 characters per line. It treats each line as a parameter of data type string of its own.

Parameter

The parameter Text of data type string designates the text you want to show.

Return Value

The return value has the data type string. It is the data the user entered. You can convert the result with the Functions for Converting Data Types.

Example:

 
-- method for querying the number of pallets in a model 
-> integer
var s: string
-- gets the number of pallets
s := prompt("Enter number", "of pallets")
return str_to_num(s)
