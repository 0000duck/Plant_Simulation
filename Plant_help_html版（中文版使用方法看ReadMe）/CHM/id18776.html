Line style
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Color > Line style

To change the properties of the line with which the Chart shows the values, click the down-arrow in the cell whose property you want to change. You can then select another Line weight and another Marker type from the drop-down lists.

When you select Display in frame, you cannot freely combine line styles and Line weights of the Chart.

Select the line style with which the Chart shows the values.

The Chart uses these settings for the Chart types > Conveyor and Spline.
