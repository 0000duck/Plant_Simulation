Context Menu of the Contents of List Objects
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > Context Menu of the Contents of List Objects > Context Menu of the Contents of List Objects

The most important commands for list objects are also available on the context menu that a click with the right mouse button on a cell opens.

Some of the commands are located on the mini toolbar, while others are commands on the context menu itself.

You can also close the context menu and leave the mini toolbar open by clicking on the border of the mini toolbar with the left mouse button.
