ZoomFactor
2D Reference Help > Material Flow Objects > Frame > Attributes of the Frame > ZoomFactor

Syntax:

 
Path.ZoomFactor

A 3D Only model does not provide the attribute ZoomFactor.

The attribute ZoomFactor sets the zoom factor with which you view the contents of the Frame. You can enter a factor from 0.25 (25 percent) to 10 (1000 percent).

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
Frame.ZoomFactor := 2 -- sets the zoom factor to 200 percent
EngineAssembly.ZoomFactor := 0.25 -- sets the zoom factor to 25 percent
