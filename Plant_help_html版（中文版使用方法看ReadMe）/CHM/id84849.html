StatTranspStoppedPortion
2D Reference Help > Material Flow Objects > Drain > Read-Only Attributes of the Drain > StatTranspStoppedPortion

Syntax:

 
Path.StatTranspStoppedPortion

The read-only attribute StatTranspStoppedPortion returns the time portion during which the MU was located on a material flow object of resource type Transport that was stopped by a LockoutZone and which the Drain designated by Path removed from the plant.

Return Value

The return value has the data type real.

Example:

 
print Drain.StatTranspStoppedPortion
