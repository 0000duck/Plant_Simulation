StatStayTimeDelta
2D Reference Help > Resource Objects > Broker > Read-Only-Attributes of the Broker > StatStayTimeDelta

Syntax:

 
Path.StatStayTimeDelta

The read-only attribute StatStayTimeDelta returns the standard deviation of the time during which Exporters stayed at importers who are managed by the Broker designated by Path.

Data Type of the Value You Can Assign

The return value has the data type time.

Example:

 
print MyBroker.StatStayTimeDelta
