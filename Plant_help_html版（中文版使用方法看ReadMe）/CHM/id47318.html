getAttributeName
2D Reference Help > Information Flow Objects > XMLInterface > Methods of the XML Interface > getAttributeName

Syntax:

 
Path.getAttributeName(Position:integer) → string

The method getAttributeName returns the name of the attribute located at the position designated by the parameter of data type integer. It applies when you randomly traverse data.

Parameter

The parameter Positionof data type integer designates the position of the attribute.

Return Value

The return value has the data type string.

Example:

 
MyXMLInterface.getAttributeName(i)+":" + MyXMLInterface.getAttributeValue(i)
