StatPredeposition
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Read-Only Attributes for Statistics of the Material Flow Objects > StatPredeposition

Syntax:

 
Path.StatPredeposition → integer

The read-only attribute StatPredeposition returns the number of MUs that were located on the object at the beginning of the statistics collection.

Return Value

The return value has the data type integer.

Example:

 
print ParallelStation.StatPredeposition
