finishLoading
2D Reference Help > Tools > TransferStation > Methods of the TransferStation > finishLoading

Syntax:

 
Path.finishLoading

The method finishLoading cancels the loading process. You can use this method to send the target, for example a pallet, which is only loaded in part, on to its destination because some station along the way is failed.

Example:

 
MyTransferStation.finishLoading
