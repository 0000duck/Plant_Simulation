Go To Cell
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Go To Cell

The command Go To Cell changes to cell of the list or table which you enter into the dialog Go To.

  • Enter the number of the Column in which the active cell will be located.

  • Enter the number of the Row in which the active cell will be located.

    This cell will then be the starting point for scrolling within the list or table. This makes moving within large tables easy and comfortable.

  • The box Occupied columns shows the number of columns of the table that contain an entry.

  • The box Occupied rows shows the number of rows of the table that contain an entry.
