node
2D Reference Help > Material Flow Objects > Frame > Methods of the Frame > node

Syntax:

 
Path.node(ObjectNumber:integer) → object

The method node calls the object with the number of data type integer in the Frame designated by Path.

Plant Simulation numbers the objects within models in the order in which you insert them.

Parameter

The parameter ObjectNumber of data type integer designates the number of the object.

Return Value

The return value has the data type object.

Example:

 
print .model.node(5)
