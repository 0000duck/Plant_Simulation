Importing Training Data
2D Reference Help > Tools > Artificial Neural Network > Importing Training Data

You can import the training data either from a table or they can be the results of a simulation study, which you ran with the ExperimentManager. Training data are classified as input/output values and as noise values. In order to evaluate the results of a simulation study with a neural network, drag the ExperimentManager onto the NeuralNet and drop it there. For stochastic simulation studies the NeuralNet enters the observed standard deviations of the output values of each experiment as noise values.

Specifying the training data

If you want to import the training data from a TableFile, which you can drop onto the NeuralNet, it will ask you how you want to classify the columns of the table as input values, output values and noise values. The NeuralNet accepts data of data types real, integer, time, money, length, weight, speed, and acceleration as input and output values. It ignores data of data types date and string while importing data.

Classification of the columns of the table into input values, output values and noise values

You can check the imported data by clicking the buttons under Data on the tab Input.
