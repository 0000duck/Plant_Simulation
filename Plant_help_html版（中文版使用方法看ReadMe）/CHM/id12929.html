UserPopupMenu
2D Reference Help > Material Flow Objects > Frame > The Frame Window > General Ribbon Tab > Configure User-defined Context Menu > Attributes of the User-defined Context Menu > UserPopupMenu

Syntax:

 
Path.UserPopupMenu

The attribute UserPopupMenu is assigned a table with two columns: string is the text to be displayed and string is the name of the Method, which the command executes.

Data Type of the Value You Can Assign

You can assign a value of data typetable.

Example:

 
var tab: table[string,string]
tab.create
tab.writeRow(1,1,"Open Component","openComponent")
current.UserPopupMenu := tab
