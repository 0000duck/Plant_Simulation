Curve width
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Curve > Curve width

Enter the width of the length-oriented curved objects in the Frame. This can be a number between 1 and 500 pixels.

The text box curve width will only be displayed when you deactivate Transfer length. When you activate it, Plant Simulation computes the width in meters.

Transfer length on

Transfer length off

SimTalk:

CurveWidth if you want to enter the width in pixels

Width if you want to enter the width in meters
