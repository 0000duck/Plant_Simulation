Transfer length
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Curve > Transfer length

To use the length of the curve in the layout within the Frame as the length of the curved objects, select this. When the dialog of the object is open, the program enters this value into the text box length. When you clear transfer length, you can change the shape of the object in the Frame at will, while the object uses the length which you manually entered.

When you select transfer length, the program adapts the length of the graphic, when you change the length if the object only consists of a single straight segment.

When the length cannot be exactly displayed in pixels, the program adapts the length of the graphic. If you need to use the exact length, clear transfer length.

You can only enter the length of the length-oriented objects, when you deactivate transfer length or when the object consists of a single straight segment.
