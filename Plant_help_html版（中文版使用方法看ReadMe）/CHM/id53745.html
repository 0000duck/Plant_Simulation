Display in frame
Add-Ins Reference Help > ActiveX > Dialog Window of the Object ActiveX > Tab Attributes > Display in frame

To show the active ActiveX control in the Frame, into which you inserted object ActiveX,select this. The program then displays the icon of the ActiveX control on top of the icon of the object.

On

Off

You cannot scale or move the icon of ActiveX controls, which are activated in the Frame.
