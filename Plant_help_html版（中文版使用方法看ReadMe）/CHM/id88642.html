InitCtrl
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > InitCtrl

Syntax:

 
MU-path.InitCtrl

The attribute InitCtrl designates a Method object, the Init control, which Plant Simulation activates during the Init phase of the simulation. It may, for example, initialize the Transporter with suitable values and assign user-defined attributes.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
if .MUs.Transporter:1.InitCtrl = VOID 
   .MUs.Transporter:1.InitCtrl := &myTransporterInitCtrl -- enter
end                                         -- and
.MUs.Transporter:1.InitCtrl := VOID         -- delete
