Plant Simulation Professional License
Setting-Up and Starting > Plant Simulation License Types > Plant Simulation Professional License

The Plant Simulation Professional license is the most powerful program package within the Plant Simulation product family and allows you to even run the most complex simulation studies. The number of objects is unlimited. Numerous mechanisms support creating, validating, and maintaining large projects. This license allows you to interactively create simulation models using the built-in class objects covering material flow objects (Station, ParallelStation, Sorter, Buffer, Store, Track, Conveyors, etc.), mobile objects (Transporter, Containerand Part), display objects (Gauge, Chart, HtmlReport) and lists and tables. You can use the built-in objects as a starting point to create customized objects that meet your exact requirements by employing the programming language SimTalk.

This license also provides a number of features for the advanced user. These features facilitate editing complex projects, increase efficiency, and support the collaboration between a number of colleagues within a project.

With the Plant Simulation Professional license you can:

  • Interactively model with the built-in objects and user-defined objects, controls, and processes with any number of class objects.

  • Insert an unlimited number of objects.

  • Modify, adjust or extend optional libraries as well as work with interactive communication interfaces and with user-defined dialogs.

  • Access tables and lists at any time during the simulation runs.

  • Modify the simulation model and its parameters during the simulation run interactively or via file interfaces, databasesor binding of Socket interfaces.

  • Animate the material flow during the simulation runs in 2D or in 3D.

  • Evaluate and display model sizes by with the built-in analysis and evaluation tools.

  • Access the built-in objects for the material flow and the information flow, as well as other objects for controlling, managing and visualizing your model.

  • Use the HTML integration for standard browsers with access to all Plant Simulation objects for fast visualization of data and simulation results.

  • Use the genetic algorithms to optimize your simulation model.

  • Use the objects Attribute Explorer, DataFit, Dialog, Experiment Manager, FlowControl, Generator, TimeSequence, and Trigger.

  • Use the built-in merge-mechanism for updating classes in the class library in existing models. Use the Profiler for analyzing time usage of the methods contained in the model, enabling you to optimize the SimTalk source code within.

  • Use the integrated interfaces: COM, DWG, DGNand DXF Reader, HTML Interface, and XML Interface.
