count
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods for Defining Failures of the Material Flow Objects > count

Syntax:

 
Path.Failures.count → integer

The method count returns how many failure profiles the object designated by Path has.

Return Value

The return value has the data type integer.

Example:

 
var i : integer
for i := 1 to Machine.Failures.count 
  Machine.Failures.getFailure(i).active := true
