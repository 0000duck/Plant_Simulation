ShiftCalendarObject
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > ShiftCalendarObject

Syntax:

 
MU-path.ShiftCalendarObject

The attribute ShiftCalendarObject sets the ShiftCalendar, which the Transporter designated by MU-path uses.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
forklift.ShiftCalendarObject := SCEngineAssembly
