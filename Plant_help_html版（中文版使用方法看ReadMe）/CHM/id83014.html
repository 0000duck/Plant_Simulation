putValuesIntoTable
2D Reference Help > Display and User Interface Objects > Chart > Methods of the Chart > putValuesIntoTable

Syntax:

 
Path.putValuesIntoTable(CollectedValues:any)

The method putValuesIntoTable writes the collected values into the table designated by the parameter of data type any. This method is especially useful when you selected Category > Histogram or Category > Plotter as then the Chart collects values over time, which you cannot easily access in any other way.

Parameter

The parameter CollectedValues of data type any designates the table.

Example:

 
MyChart.putValuesIntoTable(TableFile)
