Context Menu of the Selected Graphic in the Scene Window
3D Reference Help > 3D Reference > Context Menus in 3D > Context Menu of the Selected Graphic in the Scene Window > Context Menu of the Selected Graphic in the Scene Window

Click a graphic with the right mouse button in the 3D scene to select it. 3D then shows a context menu with these menu commands.

You can also close the context menu and leave the mini toolbar open by clicking on the border of the mini toolbar with the left mouse button.

Cut

Make Animatable Object

Copy

Make Simulation Object

Delete

Optimize Selected Graphic

Rename

Ungroup

Edit 3D Properties

Remove Texture

Show Graphic Structure

Don’t confuse the selected graphic of the object with the selected object in 3D!

A yellowish green denotes that a graphic is selected, meaning that you can use the + key to contract the selection along the graphic structure. Dark green denotes that an object is selected.
