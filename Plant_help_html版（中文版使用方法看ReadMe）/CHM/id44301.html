deleteInterval
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > deleteInterval

Syntax:

 
Path.deleteInterval([BeginningOfInterval:time, EndOfInterval:time)}

The method deleteInterval deletes all values located within the designated time interval.

Parameters

  • The parameter BeginningOfInterval of data type time designates the beginning of the time interval.

  • The parameter EndOfInterval of data type time designates the end of the time interval.

The TimeSequence also deletes entries matching one of the boundaries.

Example:

 
resultList.deleteInterval(0,3600)
