computeSHA1Hash
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > computeSHA1Hash

Syntax:

 
computeSHA1Hash(Text:string) → string

The function computeSHA1Hash returns the 40-digit SHA1 hash code of the passed text.

Parameter

The parameter Text of data type string designates the string for which you would like to know the hash code.

Return Value

The return value has the data type string.

Example:

 
print computeSHA1Hash("MyText") 
-- returns 0e9e68d9402c96044f0f93194f7010bc2e056752
