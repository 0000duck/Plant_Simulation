LowerBound
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Attributes of the Distributions > LowerBound

Syntax:

 
Attribute_path.LowerBound

The attribute LowerBound sets the smallest value to be sampled from a distribution. When it encounters a smaller number, Plant Simulation keeps drawing samples until a value equal to or larger than the LowerBound results. The LowerBound has to be less than the UpperBound.

If both UpperBound and LowerBound are zero, an unlimited distribution results.

Data Type of the Value You Can Assign

You can assign a value of data type time.

You can assign the following attributes/parameters to these distributions:

Distribution

Set of parameters

Range of values for the parameters

Erlang

Mu, Sigma, Lower Bound, UpperBound

Mu > 0, Sigma > 0, Upper Bound > Lower Bound

Gamma

Alpha, Beta, Lower Bound, UpperBound

Alpha > 0, Beta > 0, Upper Bound > Lower Bound

Geometric

p, Lower Bound, UpperBound

0 p 1, Upper Bound > Lower Bound

Lognormal

Mu, Sigma, Lower Bound, UpperBound

Mu > 0, Sigma > 0, Upper Bound > Lower Bound

Negative exponential

Beta, Lower Bound, UpperBound

Beta > 0, Upper Bound > Lower Bound

Normal

Mu, Sigma, Lower Bound, UpperBound

Mu can be any value, Sigma > 0, Upper Bound > Lower Bound

Weibull

Alpha, Beta, Lower Bound, UpperBound

Alpha > 0, Beta > 0, Upper Bound > Lower Bound

Example:

 
MyTrack.proctime.Type := "normal"
MyTrack.proctime.Mu := 60.0
MyTrack.proctime.Sigma := 30.0
MyTrack.proctime.LowerBound := 20.0 -- no value below 20 seconds
