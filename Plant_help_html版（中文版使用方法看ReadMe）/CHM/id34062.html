Control
2D Reference Help > Display and User Interface Objects > Drop-Down List > Attributes of the Drop-Down List > Control

Syntax:

 
Path.Control

The attribute Control sets the Control, which the Drop-Down List designated by Path executes when you select an item in the Frame. The anonymous identifiers ? and @ are set to the Drop-Down List when the method is called.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyDropdownList.Control := "myControl"

Example:

 
switch ?.Value
case 1 
   print "Item 1 selected"
case 2 
   print "Item 2 selected"
end
