getColumnWidth
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > getColumnWidth

Syntax:

 
Path.getColumnWidth(Column:any) → integer

The method getColumnWidth returns the width of a column of the TableFile designated by Path.

Parameter

The parameter Column of data type any designates the column.

Return Value

The return value has the data type integer.

Example:

 
MyTableFile.getColumnWidth(1)
print MyTableFile.getColumnWidth(1)
