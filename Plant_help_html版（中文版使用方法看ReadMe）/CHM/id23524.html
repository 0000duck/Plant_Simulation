Name
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes of All Mobile Objects > Name

Syntax:

 
MU-path.Name

The attribute Name sets the Name of the MU.

You can enter a combination of letters, digits, and underscore (_) characters, for example MyPart, MyPart1, My_Part_1. The name of an object cannot start with a digit, i.e., you cannot enter 1Part.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
print Station.Cont.Name
