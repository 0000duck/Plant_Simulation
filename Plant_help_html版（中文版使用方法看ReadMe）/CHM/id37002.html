Start probability
Add-Ins Reference Help > Genetic Algorithms > GASetAllocation > Dialog Window of the GASetAllocation > Tab Genetic Operators4 > Start probability

xid1188300The start probability refers to different items for the settings Crossover and Mutation.

Crossover

Enter a value for the probability, which refers to the entire individual of the respective genetic operator, which is going to be applied to a newly created chromosome.

We found these probabilities to yield good results.

Mutation

0.01 to 0.1

Crossover

0.5 to 0.8

SimTalk:

CrossoverProb, CrossoverDelta

Mutation

Enter a value for the probability, which refers to each individual item of the respective genetic operator, which is going to be applied to a newly created chromosome.

We found these probabilities to yield good results.

Mutation

0.01 to 0.1

Crossover

0.5 to 0.8

SimTalk:

MutationProb, MutationDelta
