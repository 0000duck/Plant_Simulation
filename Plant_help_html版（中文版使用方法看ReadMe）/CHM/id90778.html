Transparent
2D Reference Help > Display and User Interface Objects > Comment > Attributes of the Comment > Transparent

Syntax:

 
Path.Transparent

The attribute Transparent sets if the Comment, which you inserted into a Frame, shows its background transparent (true) or not (false). Then, Plant Simulation shows the Comment in the color you selected as the Font color on the background of the Frame, i.e., the background color of the Frame shines through the hollow parts of the text.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyComment.Transparent := false
