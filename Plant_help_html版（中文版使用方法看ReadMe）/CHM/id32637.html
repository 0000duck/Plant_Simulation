isAlignmentRow
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Rows > isAlignmentRow

Syntax:

 
Path.isAlignmentRow(Row:any, Alignment:string) → boolean

The method isAlignmentRown returns if the alignment of the designated row of the TimeSequence designated by Path matches the designated value (true) or not (false).

Parameters

  • The parameter Row of data type any designates the row.

  • The parameter Alignment of data type string designates the assignment: Right, Left or Center.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
print timeSequence1.isAlignmentRow(7,"center")
