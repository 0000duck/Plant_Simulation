Built-in Methods for Detailed Access To Graphics
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > Built-in Methods for Detailed Access To Graphics

Plant Simulation provides a number of methods for accessing the object graphics in 3D:

_3D.addGraphicTransformation, _3D.concatenateRotations, _3D.countGraphics, _3D.deleteGraphic, _3D.getGraphicIndex, _3D.getGraphicName, _3D.getGraphicPosition, _3D.getGraphicRotation, _3D.getGraphicScale, _3D.groupGraphics, _3D.removeGraphicMaterial, _3D.setGraphicMaterial, _3D.setGraphicName, _3D.setGraphicPosition, _3D.setGraphicRotation, _3D.setGraphicScale, _3D.setGraphicTransformation, _3D.setGraphicTransformationMatrix, and _3D.ungroupGraphic.
