Illustrating the Results of the Training
2D Reference Help > Tools > Artificial Neural Network > Illustrating the Results of the Training

A trained neural network can compute the respective output values to a given set of input values. On the tab Results you can show the functional correlation between the input values and the output values. The charts show essential properties of the correlation of the individual input and output values. The charts show the input and output values in their natural order, for example according to growing buffer capacity. Viewing the unfiltered training data makes characteristic correlations hard to recognize.
