Show Comment
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Show Comment

The command Show Comment shows or hides the text box for entering descriptive text above the contents of the table proper in the window of the list object.

To show the comment you enter as a tooltip, drag the mouse over the object in the Frame.

When displaying a table, the object HtmlReport shows the text you enter here as heading above the table proper.
