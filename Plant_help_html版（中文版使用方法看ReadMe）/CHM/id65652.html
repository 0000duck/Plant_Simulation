Dismantle mode
2D Reference Help > Material Flow Objects > DismantleStation > Dialog Window of the DismantleStation > Tab Attributes > Dismantle mode

Select how the DismantleStation handles the mounting parts:

  • Detach MUs: It detaches the mounting parts from the main MU and moves them on to the successor which you entered into the Dismantle Table.

  • Create MUs: It creates mounting parts.
