Cut
3D Reference Help > 3D Reference > Context Menus in 3D > Context Menu of the Selected Animatable Object in the 3D Scene > Cut

The context menu command Cut cuts the object you selected with the right mouse button from the scene and places it onto the clipboard. You can then paste it from there, provided that in the meantime you did not cut or copy anything else to the clipboard.

If the object you cut has derivatives, i.e., objects that were derived and inherit settings from it, 3D opens a dialog asking you to confirm that these derivatives will be deleted.
