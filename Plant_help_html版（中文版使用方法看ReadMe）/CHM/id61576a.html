Speed
2D Reference Help > Material Flow Objects > Converter > Attributes of the Converter > Speed

Syntax:

 
Path.Speed

The attribute Speed sets the Speed with which the Converter conveys the parts. You can also enter -1 for an infinite speed.

Data Type of the Value You Can Assign

You can assign a value of data type speed.

Example:

 
MyConverter.Speed := 2
