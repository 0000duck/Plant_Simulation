setBackgroundColorColumn
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > setBackgroundColorColumn

Syntax:

 
Path.setBackgroundColorColumn(Column:any [,Column:any,...], BackgroundColor:integer)

The method setBackgroundColorColumn sets the background color of column(s) of the TableFile designated by Path.

Parameters

  • The parameter Column of data type any designates the column whose background color you want to set.

  • The optional parameters Column of data type any designate additional columns whose background color you want to set.

  • The parameter BackgroundColor of data type integer designates the color. You can enter 1 for Black, 2 for Red, 3 for Green, 4 for Blue, 5 for Magenta, 6 for Yellow or 7 for Cyan.

    You can set the RGB values of the color with the method makeRGBValue.

Example:

 
var rgb: integer
-- sets the background color of column 2
MyTableFile.setBackgroundColorColumn(2,makeRGBValue(120,0,0))
-- sets the background color of columns 3 and 5
MyTableFile.setBackgroundColorColumn(3,5,makeRGBValue(120,10,0))
print MyTableFile.getBackgroundColorColumn(2)

Example:

 
MyTableFile.setBackgroundColorColumn(1,1) -- sets the background color of column 1 to black
MyTableFile.setBackgroundColorColumn(1,2,2) -- sets the background color of columns 1 and 2 to red
