TrackPitch
2D Reference Help > Material Flow Objects > TwoLaneTrack > Attributes of the TwoLaneTrack > TrackPitch

Syntax:

 
Path.TrackPitch

The attribute TrackPitch sets the Track pitch, which is the distance between the center Conveyors of the two lanes of the two-lane track.

Data Type of the Value You Can Assign

You can assign a value of data type length.

Example:

 
MyTwoLaneTrack.TrackPitch := 5
