StatExporterUnplannedTime
2D Reference Help > Resource Objects > Exporter > Read-Only Attributes of the Exporter > StatExporterUnplannedTime

Syntax:

 
Path.StatExporterUnplannedTime

The read-only attribute StatExporterUnplannedTime returns the total time during which the Exporter designated by Path was not planned to work.

Return Value

The return value has the data type time.

Example:

 
print MyExporter.StatExporterUnplannedTime
