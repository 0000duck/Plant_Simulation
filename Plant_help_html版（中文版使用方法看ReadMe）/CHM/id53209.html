Transparent dialogs
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Preferences > User Interface > Transparent dialogs > Transparent dialogs

To make dialogs, which are open in the foreground, transparent, select this. Dialogs and windows, which open behind them, will then be visible and it will thus be easier to select them.

Then, enter a percentage for the Window coverage and for the Transparency level.
