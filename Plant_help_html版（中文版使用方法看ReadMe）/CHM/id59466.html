Display and User Interface Objects
2D Reference Help > Display and User Interface Objects > Display and User Interface Objects

Tecnomatix Plant Simulation provides these display and user interface objects:

  • The Button for showing a button in the Frame. When you click the button, it executes the action which you programmed in a control.

  • The Chart for presenting current data and results of the simulation run.

  • The Checkbox for toggling between the states on and off, for toggling operating modes, etc.

  • The Comment for adding a description to your simulation model.

  • The Dialog for designing dialogs for your own objects.

  • The Display for presenting current data and results of the simulation run.

  • The Drop-Down List for showing a drop-down list in the Frame. When you select one of the items, it executes the action which you programmed in the control.

  • The GanttChart to show the chronological sequence of activities as bars on the time axis.

  • The HtmlReport for presenting current data and results of the simulation run.
