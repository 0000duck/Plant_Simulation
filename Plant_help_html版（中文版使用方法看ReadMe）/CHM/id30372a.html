Failed
2D Reference Help > Material Flow Objects > Frame > Attributes of the Frame > Failed

Syntax:

 
Path.Failed

The attribute Failed sets if the Frame designated by Path is Failed (true) or not (false). The attribute is observable.

Then, the Frame shows a red dot in the LED area of the icon.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

During the reset phase of the simulation Plant Simulation resets all failures.

Example:

 
subFrame.failed := true
