Find Words Similar to the Search Term
Using HTML Help > Working with Full Text Search > Advanced Full Text Search > Find Words Similar to the Search Term

This feature enables you to include minor grammatical variations for the phrase you search. A search for the word "add," for example, will find "add," "adds," and "added."

  • Click the Search tab, enter the word or phrase you want to find, and then select the check box Match similar words.

  • Click List Topics, select the topic you want, and then click Display.

    This feature only locates variations of the word with common suffixes. A search on the word "add," for example, will find "added," but it will not find "additive."
