Create and Work with a Distance Control
Step-by-Step Help > Modeling in 2D > Modeling the Flow of Materials, Advanced > Create and Work with a Distance Control > Create and Work with a Distance Control

To prevent a Transporter, which reduces its speed, from being rear-ended by the following Transporter, the following transporter has to reduce its speed in time as well. You can accomplish this in the Distance control of the Transporter.

In our example the braking distance is 5 meters. The distance control is called in a distance of 6 meters to ensure a safety distance of 1 meter for any stopped Transporter.

To create the sample model, you will:

  • Configure the Source That Produces the Transporters

  • Configure the Track on Which the Transporters Drive

  • Configure the Transporter That Drives on the Track
