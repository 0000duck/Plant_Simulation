Fast index access
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Format > Tab Data Type > Fast index access

When you select the row of the column index, which is located above the first row of cells of the list/table, and then click Format on the List ribbon tab, the dialog List Format > Data Type shows the check box Fast index access.

Select this to allow for quickly accessing the user-defined column/row index. Then, Plant Simulation internally creates a data structure that reduces the search effort, but slightly increases memory usage and the time required for changing the index.

The more columns and rows your table has, the more you benefit from the improved access speed.

If the table contains a large number of entries, you might want to activate Fast index access.
