AutomaticMediation
2D Reference Help > Resource Objects > Exporter > Attributes of the Exporter > AutomaticMediation

Syntax:

 
Path.AutomaticMediation

The attribute AutomaticMediation prevents that the Exporter, which you yourself want to assign, is automatically assigned by the Broker.

To make the Broker assign the Exporter automatically, assign true. To assign the Exporter yourself, assign false.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
param type: integer -- Importer type (0=failure, 1=setup, 2=processing, 3=transport)
var t : table
switch type
case 0 
    ?.failImp.releaseExporters
case 1 
   ?.setupImp.getExporters(t)
   if t.yDim > 0
       t[1,1].AutomaticMediation := true
   end
   ?.setupImp.releaseExporters
case 2 
   ?.imp.getExporters(t)
   if t.yDim > 0 
       t[1,1].AutomaticMediation := true
   end
   ?.imp.releaseExporters
end

Example:

 
.Resources.myExporter:2.AutomaticMediation := true
