promptMessage
SimTalk Reference > General SimTalk Functions > Predefined Functions > Output Functions > promptMessage

Syntax:

 
promptMessage(Message:string[,MoreText:string, ...])

The function promptMessage opens a dialog window displaying a message.

Parameters

  • The parameter Message of data type string designates the message you want to show. While you view this window, Plant Simulation pauses the simulation. Clicking OK will resume it.

  • The optional parameter MoreText of data type string designates additional text you want to show.

Example:

 
promptMessage("All machines in the plant Dinkelsbühl failed!")
