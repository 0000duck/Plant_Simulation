Align to Grid
3D Reference Help > 3D Reference > Edit Ribbon Tab > Align to Grid

The command Align to Grid aligns the selected object or graphic to the grid points of the grid.

The command moves the selected objects to the next projected intersecting point of the grid lines. To move the selected objects on the grid level instead, hold down the Ctrl key.

You can also combine Align to Grid and Snap to Grid.

For Align to Grid you can adjust the coordinate that falls perpendicular to the grid by holding down the Ctrl key while selecting the command.

The commands Align to Grid and Snap to Grid adjust the two coordinates that are placed along the grid plane, which normally are the x-coordinate and the y-coordinate.
