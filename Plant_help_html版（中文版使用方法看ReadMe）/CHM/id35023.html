assignedWorkplaces
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods for the Workplace of the Material Flow Objects > assignedWorkplaces

Syntax:

 
Path.assignedWorkplaces([Workplaces:table])

The method assignedWorkplaces returns the Workplaces assigned to the object designated by Path and writes them to a table (an object or local variable).

Parameter

The optional parameter Workplaces of data type table designates the name of the table.

Return Value

If you do not specify the optional parameter, Plant Simulation returns an array with the assigned Workplaces.

Example:

 
Station13.assignedWorkplaces(myAssignedWorkplaces)

Example:

 
Station.assignedWorkplaces -- might return[*.Models.Frame1.Workplace, *.Models.Frame1.Workplace1]
