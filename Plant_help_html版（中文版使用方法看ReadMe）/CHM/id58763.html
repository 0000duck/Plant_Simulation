Number of columns
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Format > Tab Dimension > Number of columns

Enter the number of columns of the TableFile. If you do not enter a value for the Number of columns and/or the Number of rows, the dimension of the list is not limited.

Plant Simulation activates the text box Number of columns only after you selected Select All on the Home ribbon tab.
