Length
2D Reference Help > Resource Objects > FootPath > Attributes of the FootPath > Length

Syntax:

 
Path.Length

The attribute Length sets the physical Length of the Footpath designated by Path. A Worker enters the FootPath at position 0 and exits it after covering the length you enter.

Data Type of the Value You Can Assign

You can assign a value of data type length.

Example:

 
MyFootpath.Length := 44 -- meters
