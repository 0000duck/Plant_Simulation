append
2D Reference Help > Information Flow Objects > CardFile > Methods of the CardFile > append

Syntax:

 
Path.append(Entry:any)

The method append appends an entry to the end of a CardFile.

Note that Plant Simulation does not allow to exceed the Number of rows you set.

Parameter

The parameter Entry of data type any designates the entry you want to append.

Example:

 
MyCardFile.append("crankshaft")
