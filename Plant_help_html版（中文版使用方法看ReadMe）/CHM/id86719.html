Dialog Window of the ShiftCalendar
2D Reference Help > Resource Objects > ShiftCalendar > Dialog Window of the ShiftCalendar > Dialog Window of the ShiftCalendar

Double-click the icon of the ShiftCalendar, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Resources in the Toolbox. The text boxes Name and Label are described under Dialog Items of the Objects.

Before you can enter data, click the toggle button Inheritance, so that it looks like this .

You can manipulate the contents of the list with the commands of the Context Menu of Embedded Lists.
