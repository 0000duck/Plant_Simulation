Insert Fence
3D Reference Help > 3D Reference > Edit Ribbon Tab > Insert Fence

The command Insert Fence creates and inserts a fence box into the selected Frame.

  • Select the Graphic group to which you want to add the new graphic. The list shows the graphic groups which are defined for the object.

    • Default (external): Designates the standard graphic group of the 3D objects. It initially is external and represents the object towards the outside. When you deactivate Show External Graphics, 3D hides the graphic groups designated as Default.

    • One of the user-defined graphic groups you added on the Tab Graphics. Newly created 3D Frames by default provide the graphic group Deco (internal), which are intended for graphics for decorating the interior view of the Frame.

  • Enter the Width, the Depth, and the Height of the fence you want to create into the dialog Create Fence.

    To only insert a single fence segment instead of a set of four segments, enter 0 as the width or the depth.

  • Click the color box next to Profile material to assign a material to the profiles of the fence.

  • Click in the color box next to Post material to assign a material to the posts of the fence, which hold the mesh panels.

  • Click to copy the current material settings you selected. You can then paste these settings into the graphic of another object.

    The command also copies the corresponding SimTalk code to the clipboard. You can then paste this code into a Method object to set the material of another graphic. The syntax is described under Generate SimTalk Code and Copy to Clipboard.

  • Click to paste the material settings that you copied to the graphic of the object.

  • Click the color box next to Profile material to assign a material to the profile of the fence which holds the wire mesh.

  • Click the color box next to Post material to assign a material to the posts of the fence that hold the wire mesh panels.

  • Select Mesh to insert a wire mesh between the posts of the fence. Or

  • Select Panes to insert window panes between the posts of the fence.

    Click in the color box next to material to assign a material to the window panes of the fence.

    You can also create the fence without wire mesh and without window panes or you can combine both.

  • To create the fence, click Create. 3D attaches the shape to the mouse pointer. Drag the mouse to the position at which you would like to insert the shape and click the left mouse button. To cancel inserting, you can click the right mouse button or press the Esc key.

    Select the graphic group to which you would like to append the fence and click OK.

  • To move the fence to the location of your choice, click it and drag the mouse.

  • To change the settings of the fence, click it with the right mouse button and select Edit 3D Properties > Fence Settings. Here you can also select if the fence is an obstacle for the worker.

You can also edit the dimensions of the fence by clicking Show Manipulators on the Edit ribbon tab.
