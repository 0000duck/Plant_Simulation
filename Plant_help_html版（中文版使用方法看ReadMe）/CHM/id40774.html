AvailableCtrl
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of All Objects > Attributes for Controls of All Objects > AvailableCtrl

Syntax:

 
Path.AvailableCtrl

The attribute AvailableCtrl designates a Method object, which Plant Simulation calls, whenever the state of the Exporter or the Worker changes to available. Within the interaction method you can use the anonymous identifier ? to access the object that triggered the interaction method.

Example:

 
MyExporter.AvailableCtrl := &myAvailableCtrl
