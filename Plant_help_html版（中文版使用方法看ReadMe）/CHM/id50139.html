setFontSizeCells
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for the Format of Lists and Tables > setFontSizeCells

Syntax:

 
Path.setFontSizeCells(Range:listrange [,Range2:listrange, ...], FontSize:integer)

The method setFontSizeCells sets the font size of the range of cells of the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Parameters

  • The parameters Range of data type any designate the cells, whose font color you want to set.

  • The parameter FontSize of data type integer designates the font size. You can enter 1 for Small, 2 for Medium, 3 for Large or 4 for Extra Large.

    You can only use the asterisk * to designate a range, if the object has a Dimension. Plant Simulationconsiders the object to have a dimension if you either entered the number of rows and columns in the dialog or if the object contains entries.

Example:

 
MyTableFile.setFontSizeCells({1,2}..{1,6},3)
