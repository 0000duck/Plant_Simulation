[ ] (read the contents of a cell)
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > [ ] (read the contents of a cell)

Syntax:

 
Path[Column:integer, Row:integer] → contents of the cell

The TimeSequence allows you to randomly access the contents of the cells by employing the column and row indexes. The index statement begins and ends with a bracket. Within the bracket you first enter the column and then the row of the cell to be accessed. The data type of the return value is the same as the data type of the table cell. Plant Simulation just reads the contents of the cell but does not remove it!

Parameters

  • The parameter Column of data type integer designates the column.

  • The parameter Row of data type integer designates the row.

Return Value

The return value is the contents of the cell. An empty cell returns VOID.

Example:

 
print timeSequence[1,2]
print timeSequence["Value",2]
