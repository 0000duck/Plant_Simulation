Width
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Dialog Window of the MultiPortalCrane > Tab Portal > Width

Enter the width of the portal of the MultiPortalCrane. It designates the distance between the rails of the runway. You can also make the portal width greater than the Width of runway. In this case you have to enter an Overhang on the right hand side of the portal in the direction in which you inserted the MultiPortalCrane.
