2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Importer > Sub-tab Failure > Broker

Click and select the Broker, which assigns the repair service.

For Broker hierarchies Plant Simulation only calls the control of this Broker, not the controls of any existing sub Brokers.

To open the dialog of the object, whose name you entered into the text box, click in the text box, and press the F2 key.

Or you can select the Broker in a Frame, drag it to the text box and drop it there.
