Attach a Camera to an Frame and Detach it
Step-by-Step Help > Modeling in 3D > Controlling Your View > Attach a Camera to an Frame and Detach it

You can attach a camera to any object, so that you can view the scene through the lens of this object, and detach a camera from it again, when you do not need it any more. When the object moves through scene, for example because you started the simulation, the camera will move along with the object you attached it to.

In addition, 3D:

  • Visualizes the camera you attached, i.e., inserts an animatable object that represents the camera.

  • Allows you to change between object view and normal view.

  • Allows you to manually transform the camera you attached: By changing the view while in object view, or by manipulating the object, while in normal view.
