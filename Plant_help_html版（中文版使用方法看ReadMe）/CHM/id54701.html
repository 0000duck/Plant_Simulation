setEnv
SimTalk Reference > General SimTalk Functions > Predefined Functions > Operating System Functions > setEnv

Syntax:

 
setEnv(EnvironmentVariable:string, Value:string) → boolean

The function setEnv sets an environment variable.

If you deactivate the safety setting File > Model Settings > General > Access to the computer is permitted Plant Simulation does not execute the function setEnv and shows an error message.

Parameters

  • The parameter EnvironmentVariable of data type string designates the name of the variable.

  • The parameter Value of data type string designates its value.

Return Value

The return value has the data type boolean.

Example:

 
setEnv("ralf","0")
startExtProc("PlantSimulation14.exe")

In the external process you just started you can then query the value you set:

Example:

 
print getEnv("ralf") -- returns the value of the variable

You might, for example, set the environment variable PlantSimUseLFH, which determines that Plant Simulation uses the Low Fragmentation Heap when allocating main memory. This prevents that allocating memory degrades performance with increasing runtime when a simulation model uses a very large amount of main memory.
