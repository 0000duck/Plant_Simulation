deleteMovables
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Methods of All Mobile Objects > Methods for Accessing the Load of Container and Transporter > deleteMovables

Syntax:

 
MU-path.deleteMovables

The method deleteMovables deletes all MUs on the loading space of the MU designated by MU-path.

Example:

 
.Models.container:1.deleteMovables
