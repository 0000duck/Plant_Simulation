subtract
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > subtract

Syntax:

 
Path.subtract(ValueToBeSubtracted:integer)
Path.subtract(ValueToBeSubtracted:real)
Path.subtract(ValueToBeSubtracted:object)

The method subtract either subtracts a numerical value from all entries of the value column of the TimeSequence designated by Path, or it combines the time and value columns of two TimeSequence objects.

Parameters

  • If the parameter to subtract is a numerical value designated by the parameter ValueToBeSubtracted of data type integer or real, Plant Simulation simply subtracts it from each entry of the value column. The time column remains unchanged.

  • If the parameter to subtract is another TimeSequence designated by the parameter ValueToBeSubtracted of data type object, Plant Simulation adds time value pairs in sections. The values in the time columns set the extent of the individual sections. If for a certain time value there are multiple entries in one time sequence, and only one entry in the other time sequence, the single entry will be subtracted from all other entries.

If both TimeSequences contain the same number of multiple entries with a particular time value, Plant Simulation combines the entries in pairs and subtracts them afterwards. If the number of entries with the same time value differs, Plant Simulation duplicates the last entry, before it subtracts it, until the number of entries with the same time matches.

Example:

 
MyTimeSequence.subtract(5)
stock.subtract(missingEntities)
