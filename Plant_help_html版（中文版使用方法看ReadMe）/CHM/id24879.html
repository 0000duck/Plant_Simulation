Open Table
2D Reference Help > Tools > TransferStation > Dialog Window of the TransferStation > Tab Advanced Attributes > Open Table

When you only want to load/unload parts of a certain type onto/from a means of transport, click Open Table. Then, enter the Part Type into the cells of column 1 of the table, enter the Container Type into the cells of column 2. The TransferStation only takes parts from the column Part Type from the parts station; when loading and moving it takes them directly from the parts station, when unloading and reloading it takes them from the means of transport. When loading and reloading parts, you have to enter the type of means of transport into the cells of the column Container Type, onto which you want to load the part. Remember that you define the names of the parts in the column Name of the sequence Table in the object Source.
