New Drop-Down List Box
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New Drop-Down List Box

The user can select a single item from the drop-down list . When it is closed, it displays the current value. When the user clicks the down arrow , it opens and displays more items for changing the value.

  • Enter the Name of the drop-down list.

  • Enter the Callback argument, which will be passed to the Callback method as a parameter.

    It will be executed, when the user closes the drop-down list box.

  • Enter its X and Y position in the Dialog.

  • Enter its Width. The default value of 0 sets the width to the values defined by us.

  • Click Items and enter the items the list box displays into the dialog Items.

  • Select if you want to Enable it or not.

  • To insert the dialog item into your dialog, click OK.
