Direction
Add-Ins Reference Help > Genetic Algorithms > GAOptimization > Attributes of the GAOptimization > Direction

Syntax:

 
Path.Direction

The attributeDirection defines if the optimum is going to be a high (Maximum) or a low (Minimum) fitness value.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
GAOptimization.Direction := "Maximum"
