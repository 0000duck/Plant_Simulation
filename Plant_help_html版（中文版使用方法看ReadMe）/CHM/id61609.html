Feed Parts from a Branch Line into the Main Line
Step-by-Step Help > Modeling in 2D > Conveying Parts Laterally with the Converter > Feed Parts from a Branch Line into the Main Line

In this sample model the Converter feeds parts from a branch line into the main line, once the gap, which you enter, is big enough to accommodate the part.

To create the simulation model, insert the objects so that they match the screenshot below. The finished simulation model looks like this:

Insert and configure the Source which produces the parts moving on the main line:

  • Select when parts are to be produced. We entered a constant interval of 30 seconds, meaning that the Source produces a part every 30 seconds.

  • Select which part type is going to be produced. We selected the part MyPart, to which we added a direction arrow.

To make the Converter need the parts from the branch line into the main line, provided the gap between the parts on the main line is big enough, open the dialog of the Converter.

  • Select the Strategy > Feed in.

  • Enter the size of the gap between the parts. We entered a gap of 5 meters.

Start the simulation and watch how the Converter moves the parts from the branch line into the main line and conveys them with the orientation in which they were fed in.
