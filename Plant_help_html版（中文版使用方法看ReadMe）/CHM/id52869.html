Encrypt Method
2D Reference Help > Information Flow Objects > Method > The Method Window > Tools Ribbon Tab > Encrypt Method

The command Encrypt Method opens the dialog Encrypt. Then, encrypt the source code of the Method to prevent unauthorized access by other users.

Enter the password for encrypting theMethod into the text box Password.

Enter the password for encrypting the Method again into the text box Confirm password.

Make sure to remember the password, as only this password will allow you to decrypt the source code afterwards. After clicking OK, the source code cannot be accessed any more without the password you entered.

After you have encrypted the Method, the command Encrypt changes to Decrypt. In addition, you can encrypt the entire model by selecting File > Options > Encrypt Methods in the Plant Simulation main window.
