New Image
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New Image

An image is a picture, i.e., an icon you defined for the object Dialog. You can enter a number, or a name, such as Icon1.

  • Enter the Name of the image.

  • Enter its X and Y position in the Dialog.

  • Enter its Width and Height in the Dialog.

  • Enter the Image ID, i.e., the number of the icon, or an image name. The image is an icon of the Dialog the image belongs to.
