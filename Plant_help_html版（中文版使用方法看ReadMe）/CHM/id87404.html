Capacity
2D Reference Help > Material Flow Objects > Store > Read-Only Attributes of the Store > Capacity

Syntax:

 
Path.Capacity

The read-only attribute Capacity returns the capacity of the Store designated by Path. It is the product of XDim times YDim times ZDim.

Return Value

The return value has the data type integer.

Example:

 
if MyStore.Capacity >= lotsize 
   @.move(MyStore)
end
