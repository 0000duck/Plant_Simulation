setAlignmentRow
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Rows of the TableFile > setAlignmentRow

Syntax:

 
Path.setAlignmentRow(Row:any [,Row:any,...], Alignment:string)

The method setAlignmentRow sets the alignment of row(s) of the TableFile designated by Path.

Parameters

  • The parameter Row of data type any designates the row whose alignment you want to set.

  • The optional parameters Row of data type any designate additional rows whose alignment you want to set.

  • The parameter Alignment of data type string designates the alignment you want to assign: Left, Right or Centered.

Example:

 
MyTableFile.setAlignmentRow(9,10,"center")
MyTableFile.setAlignmentRow(10,"center")
