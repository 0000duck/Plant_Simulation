Insert Cuboid
3D Reference Help > 3D Reference > Edit Ribbon Tab > Insert Cuboid

The command Insert Cuboid creates and inserts a cuboid into the selected Frame.

  • Select the Graphic group to which you want to add the new graphic. The list shows the graphic groups which are defined for the object.

    • Default (external): Designates the standard graphic group of the 3D objects. It initially is external and represents the object towards the outside. When you deactivate Show External Graphics, 3D hides the graphic groups designated as Default.

    • One of the user-defined graphic groups you added on the Tab Graphics. Newly created 3D Frames by default provide the graphic group Deco (internal), which are intended for graphics for decorating the interior view of the Frame.

  • Enter the Width, the Depth, and the Height of the cuboid which you want to create.

  • To create a flat, non-textured plate, enter 0 into the respective text box:

  • To create the cuboid, click Create. 3D attaches the shape to the mouse pointer. Drag the mouse to the position at which you would like to insert the shape and click the left mouse button. To cancel inserting, you can click the right mouse button or press the Esc key.

  • To move the cuboid to the location of your choice, click it and drag the mouse.

  • To change the size of the cuboid, select it and click 3D Properties > Tab Transformation on the Home ribbon tab. Then change the size under Scale.

  • To set if the cuboid is an obstacle for the worker, select it and then click Edit 3D Properties > Tab Graphic Settings on the Home ribbon tab.

  • To assign a material and a color, select the cuboid and click 3D Properties > Tab Material on the Home ribbon tab.
