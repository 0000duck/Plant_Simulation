User name
Add-Ins Reference Help > Inter-Process Communication Interfaces > Teamcenter > Dialog Window of the Teamcenter Interface > Tab Connection > User name

Enter the user name with which you want to log on to the Teamcenter database. Plant Simulation saves the user name in the model file.

If you do not enter your user name here, you have to enter it in the Teamcenter login dialog when you log on to the Teamcenter database for the first time.
