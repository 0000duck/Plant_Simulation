Creating a List within a QueueFile or StackFile
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Creating Lists within Lists and Tables > Creating a List within a QueueFile or StackFile > Creating a List within a QueueFile or StackFile

To create a sub-list or a sub-table in objects of type QueueFile and StackFile:

  • Add the sub-list to a cell within the object, compare Creating Lists within Lists and Tables.

  • Insert a list object from the Class Library into your simulation model: Drag that object to the active cell of the QueueFile/StackFile and drop it there. This inserts the path to and the name of the list or table into the active cell.

  • Assign a List to a QueueFile or a StackFile in a Method.
