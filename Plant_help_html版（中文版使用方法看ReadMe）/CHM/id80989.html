Use Context Menu Commands
Using HTML Help > General Topics > Use Context Menu Commands

The Help Viewer provides several commands on the context menu for displaying information.

To do this

Click and Select

Open all books or folders on the table of contents. This command only works when the Contents tab is displayed.

Click on the table of contents with the right mouse button, and then select Open All.

Close all books or folders. This command only works when the Contents tab is displayed.

Click on the table of contents with the right mouse button, and then select Close All.

Prints the topic.

Click with the right mouse button, and then select Print.
