Plant Simulation Simulation Package License
Setting-Up and Starting > Plant Simulation License Types > Plant Simulation Simulation Package License

The Plant Simulation Simulation Package license supports running complex simulation studies. This license is especially suited for employing the distributed simulation, for employing neural networks and for the optimization tools, such as genetic algorithms. Using several Simulation licenses allows to run experiments with different parameters, control logics, random number streams or layouts in parallel to arrive at results considerably faster. The Plant Simulation Simulation Package license does not have a graphical user interface. You can neither visualize the simulation model nor can you manually change the model.

The number of objects within the model is unlimited. The Simulation license contains the Plant Simulation interfaces ActiveX, COM, ODBC, SQL for Oracle, Socketand XMLas well as the Assembly libraryand the Shop libraryfor production planning. You can also run models, which contain the Plant Simulation Gantt Chart, but you cannot visualize them with a Simulation license.

With the Plant Simulation Simulation Package license you can:

  • Modify the model and its parameters during the simulation run via interfaces.

  • Execute simulation runs without a graphical user interface, visualization or modeling.

  • Execute the simulation step by step.

  • Execute all Methods.

  • View and save the results of the simulation runs.

  • Use the integrated interfaces ActiveX, COM, ODBC, SQL for Oracle, Socket, and XML Interface.

  • Use the libraries Plant Simulation Assembly, Plant Simulation Shopas well as all libraries which do not require a license.
