Part
2D Reference Help > Mobile Objects > Part > Part

The Part is a moving material flow object without loading capacity that moves through a plant on the material flow objects proper. The Part represents all kinds of parts being produced and transported, but not transporting other work pieces.

To add the Part to your simulation model, click Manage Class Library > Basic Objects > MUs > Part on the Home ribbon tab.
