PredInterface
2D Reference Help > Material Flow Objects > Connector > Read-Only Attributes of the Connector > PredInterface

Syntax:

 
ConnectorPath.PredInterface

The read-only attribute PredInterface returns the direct predecessor object of the Connector designated by ConnectorPath. When this preceding object is a Frame, the read-only attribute PredInterface returns the Interface object, as opposed to the method pred, which returns the Frame.

Example:

 
print Connector2.predinterface
