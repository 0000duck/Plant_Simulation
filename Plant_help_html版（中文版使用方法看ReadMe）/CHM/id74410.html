Inherit Visibilities
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Show 3D Graphic Structure > Inherit Visibilities

The button Inherit Visibilities shows and sets if the selected object or the graphic groups of the object inherits the visibilities of its graphic groups from the origin object or not. When you select a single graphic, the button is grayed out .

Click the button to change inheritance of the visibilities of the graphic.

You can only activate inheritance in derived, i.e., in non-class objects.
