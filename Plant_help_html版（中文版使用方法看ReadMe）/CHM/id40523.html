Percentage
2D Reference Help > Material Flow Objects > FlowControl > Dialog Window of the FlowControl > Tab Exit Strategy > Strategy > Percentage

To move the MUs on to a successor according to a percentage distribution, select Percentage from the drop-down list.

Click Open List to open a table and enter the percentages. The n-th row on the table defines the n-th successor’s portion. The FlowControl moves an MU to that successor, for which the difference between the desired value and the actual value is greatest.

When you enter 0 into the table, the predecessor of the FlowControl will not receive an MU.

  • When you select Blocking , the FlowControl blocks the MU until the successor determined by the distribution can receive the MU

  • When you clear Blocking , the FlowControl moves the MU on to the successor with the second largest difference between the desired value and the actual value, etc.
