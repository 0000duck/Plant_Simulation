Rotate an Object Precisely
Step-by-Step Help > Modeling in 3D > Working with Objects > Rotate an Object > Rotate an Object Precisely

At times manually rotating an object with the mouse will not be accurate enough. Then, you can enter exact values into the text boxes in the dialog 3D Properties > Tab Transformation to fine tune the position, the rotation and the scale of the selected object:

To precisely rotate an object:

  • Select it.

  • Click on the Home ribbon tab or press Ctrl+3.

  • To rotate the object around an arbitrary axis, defined by its x-component, y-component and z-component:

    Enter a value for the x-component of the rotation axis.

    Enter a value for the y-component of the rotation axis.

    Enter a value for the z-component of the rotation axis.

  • Enter the angle of rotation in degrees into the text box.

  • To apply your changes if your satisfied with the results of your changes, click Apply.

    Transformations of an object, such as moving, rotating or scaling it, in one 3D window are not immediately updated in all other 3D windows, but only after you deselect the changed object in its 3D window.
