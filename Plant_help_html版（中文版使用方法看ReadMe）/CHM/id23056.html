pos
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Strings > pos

Syntax:

 
pos(CharactersPosition:string, SourceText:string) → integer

The function pos returns the position at which the designated text occurs for the first time within a string.

Parameters

  • The parameter CharactersPosition of data type string designates the character whose position you would like to determine.

  • The parameter SourceText of data type string designates the string within which string occurs.

The return value 0 indicates that string does not occur in string.

Return Value

The return value has the data type integer.

Example:

 
print pos("b","abcdefb") -- returns 2
