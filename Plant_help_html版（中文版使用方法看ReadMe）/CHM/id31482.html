_3D.hideGraphicGroup
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Graphic Groups > _3D.hideGraphicGroup

Syntax:

 
Path._3D.hideGraphicGroup(Name:string)

The method _3D.hideGraphicGroup sets that the specified graphic group does currently not visually represent the object designated by Path. Instead the graphic group is hidden in all 3D windows.

Parameter

The parameter Name of data type string designates the name of the graphic group.

Example:

 
Station._3D.hideGraphicGroup("MyNewGraphicGroup")
