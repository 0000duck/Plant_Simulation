statBlockingTimePerSuccessor
2D Reference Help > Material Flow Objects > DismantleStation > Methods of the DismantleStation > statBlockingTimePerSuccessor

Syntax:

 
Path.statBlockingTimePerSuccessor(Successor:integer) → boolean

The method statBlockingTimePerSuccessor returns the blocking time in relation to a successor from the table Blocking Times of the DismantleStation designated by Path.

Parameter

The parameter Successor of data type integer designates successor.

Return Value

The return value has the data type boolean.

Example:

 
for i := 1 to MyDismantleStation.NumSucc 
   print MyDismantleStation.statBlockingTimePerSuccessor(i)
next
