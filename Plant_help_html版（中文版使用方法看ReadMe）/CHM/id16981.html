SuccLane
2D Reference Help > Material Flow Objects > Connector > Read-Only Attributes of the Connector > SuccLane

Syntax:

 
ConnectorPath.SuccLane

The read-only attribute SuccLane returns the lane of a TwoLaneTrack, whose starting point is connected with the Connector designated by ConnectorPath. This is the lane to which the Connector leads in the forward direction.

If the successor in not a TwoLaneTrack, the method returns the successor object.

Example:

 
print Connector2.SuccLane
