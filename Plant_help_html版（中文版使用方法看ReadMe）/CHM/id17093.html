insert
2D Reference Help > Information Flow Objects > CardFile > Methods of the CardFile > insert

Syntax:

 
Path.insert(Row:integer, Entry:any)

The method insert adds additional entries to a CardFile.

Parameters

  • The parameter Row of data type integer designates the number of the cell into which Plant Simulation inserts the value.

    The CardFile moves all cells with the same or a greater index as the designated cell down one position. When the number you enter is greater than the highest position in the CardFile, Plant Simulation does not place it at the position designated by integer, but adds it after the last current entry. Remember that Plant Simulation does not allow gaps between entries in a CardFile.

  • The parameter Entry of data type any designates the value, which you want to insert.

    The data type of this parameter has to be the same as the data type of the CardFile.

Example:

 
MyCardFile.insert(2,12.34)
