OpenGL Version
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Preferences > 3D > OpenGL Version

Select the OpenGL version you would like to use for rendering the 3D scenes in your simulation models. When 3D is running while the dialog Preferences is opened, the drop-down list only shows the OpenGL versions that the graphics card driver of the graphics card of your computer supports.

You can check out the features of the individual OpenGL versions on the Internet.

We recommend to use the highest setting available. Sometimes you might have to choose a lower level though to get rid of display bugs in the driver of your graphics card. Be aware that if you go below version 3.1, you might suffer a significant decrease in performance.
