User-defined control for the targets
2D Reference Help > Tools > TransferStation > Dialog Window of the TransferStation > Tab Advanced Controls > User-defined control for the targets

Enter the name of the Method in which you defined the conditions when a target ( Container or Transporter) exits a point-oriented station, which provides the targets ( Container or Transporter) for the TransferStation. For the length-oriented objects the control will be called, when the target continues its way at the sensor. You might, for example, define conditions what will happen with the full container/transporter after the parts have been loaded.

When you enter a boolean parameter in the user-defined control, Plant Simulation also calls the method, when a target arrives at the object.

User-defined controls might for example look like this:

Example:

 
--for point-oriented objects (leavingPart:boolean) 
is 
do
  if leavingPart then 
    @.move
  end
end

Example:

 
--forthelenght-oriented object Conveyor(leavingPart:boolean)
is
do
  if leavingPart then
    @.Stopped:=false
  else
    @.Stopped:=true
  end
end

Or click , navigate to the Frame in which you stored the Method and select it there.
