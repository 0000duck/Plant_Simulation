String
SimTalk Reference > General SimTalk Functions > Constants and Variables > Constants > String

A constant of data type string can contain any combination of letters, numbers and special characters within quotation marks. Enter a backslash \ as a line break character if the contents of the string does not fit on one line. To use quotation marks in the text of the string, enter a backslash followed by a quotation mark, the text that is to appear in quotation marks, and another backslash followed by ending quotation marks.

Example:

 
priority:= "Urgent"
very_long_text := "Frank Jones\
                   Halford Lane 9\
                   Cedar Rapids, IA 52409"
Text := "It is \"very\" urgent." -- outputs It is "very" urgent.
