Tab Point Cloud
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Point Cloud > Tab Point Cloud

On the tab Point Cloud you can:

  • Select the file path.

  • Enter settings for the position.

  • Enter settings for the rotation of the point cloud.

Initially, your 3D window might only show small parts of the point cloud. When you start rotating the view, 3D shows additional parts of the graphic.

The point cloud will load significantly faster if the point cloud file is located on an SSD storage device.
