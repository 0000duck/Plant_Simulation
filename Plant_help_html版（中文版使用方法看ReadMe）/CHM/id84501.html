Make Simulation Object
3D Reference Help > 3D Reference > Edit Ribbon Tab > Make Simulation Object

The command Make Simulation Object converts the selected graphic into a simulation object. A simulation object is a object that has a counterpart in Plant Simulation 2D.

  • Enter a Name for the newly created simulation object.

  • Select the 2D object Class to be used when creating the simulation object. In general these are the material flow objects, some of the resource objects, the object Chart and the tools.

  • Enter or select the x-, y-, and z-coordinates of the position of the animatable simulation object that will be created. Or you can click the spin buttons

    The white lines show a preview of the object position which you specified.

  • Click OK.

    The object will always be created in 2D as well and will always be derived from the selected class.
