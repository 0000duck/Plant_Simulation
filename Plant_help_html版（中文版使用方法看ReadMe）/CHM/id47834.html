Find
2D Reference Help > Information Flow Objects > Method > The Method Window > Edit Ribbon Tab > Find in Method > Find

The command Find finds the expression that you enter into the dialog Find. You can also open it on the toolbar of the Method Debugger. With its settings you can find expressions in your source code and replace them if necessary.

.

  • Enter the expression that you want to find into the text box Find what.

  • When you select the check box Match whole word only Plant Simulation only finds whole words only instead of searching for the expression you entered into the text box inside longer words.

  • When you select the check box Match case Plant Simulation only finds text that has the same pattern of upper and lower case as the expression you entered into the text box Find what.

  • When you select the check box Regular expression Plant Simulation uses regular expressions in your search. Click the button with the arrow pointing to the right . This opens the window Regular Expression where you can pick any of the displayed regular expressions, which Plant Simulation will then enter into the text box Find what.

  • When searching with regular expressions you can use wild cards in the string you are searching for. The period character . for example, is a wild card for any character. When Plant Simulation searches for a.b, it will find all strings consisting of a sequence of the character a, followed by any character, and the character b. Compare the examples under Show Structure.

    Note that Plant Simulation only finds a search term with regular expression when you selected the check box Regular expression.

  • Select Up to search toward the beginning of the Method.

  • Select Down to search toward the end of the Method.

  • Click Find Next, to find the next instance of the expression which you entered into the text box Find what.

  • Click Mark All, to highlight all instances of the expression which you entered into the text box Find what. To perform the action of your choice, use the commands on the Tools ribbon tab (Toggle Bookmarks, Clear All Bookmarks, Next Bookmark, and Previous Bookmark).
