2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Preferences > Modeling > 2D

On the tab 2D you can select the default settings that determine how a Frame shows the objects which you insert into it.

You can override these settings for each individual Frameon the General ribbon tab. The 2D settings are saved to your model file.
