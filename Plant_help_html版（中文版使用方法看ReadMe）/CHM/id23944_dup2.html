Name
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Graphics > Object graphics > Graphic Groups > Add > Name

3D shows the current name of the object here. You can leave it as it is or change it by entering a name of your choice. To change the name, double-click and overtype it. You can also enter a name that is invalid in 2D. If you want to, you can also leave the text box empty.
