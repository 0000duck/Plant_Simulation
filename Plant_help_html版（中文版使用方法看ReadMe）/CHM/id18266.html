Tab Dimension
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Format > Tab Dimension > Tab Dimension

On the tab Dimension you can enter values for the dimension of the list: The Number of rows, for the TableFile also the Number of columns, and the Column width. If you do not enter a value for the Number of columns and/or the Number of rows, the dimension of the list is not limited.

Plant Simulation only shows the Tab Data Type and the Tab Dimension after you selected one or several columns by clicking into the column header!
