Save Object As
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Save Object As

Plant Simulation shows this command, when you select an object other than a folder.

The context menu command Save Object As saves the selected object as an Plant Simulation object file (.obj). Plant Simulation saves the object, and all objects, which that object uses, together with their folders and paths.

Enter a name into the dialog Save As and specify a location.

You can then load this object into other simulation models or share it with your co-workers.

To save a folder as an object file, right-click that folder and select Save Folder As.

If you want your saved object to also contain existing 3D data, 3D has to be running when you save the object.
