Target folder
Add-Ins Reference Help > Inter-Process Communication Interfaces > Teamcenter > Dialog Window of the Teamcenter Interface > Tab Export > Target folder

Enter the Teamcenter target folder in which the Plant Simulation HtmlReport will be stored. This folder has to be a subfolder of the Home folder of the user logged on to Teamcenter.

If you are using Active Workspace, you can click the button and select the target folder.
