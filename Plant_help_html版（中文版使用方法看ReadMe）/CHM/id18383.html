Select Template
2D Reference Help > Information Flow Objects > Method > The Method Window > Edit Ribbon Tab > Select Template > Select Template

The command Select Template , opens a dialog in which you can select one of the templates we provide for a number of commonly used tasks.

  • Select the category of template you want to use in the left list box.

  • Select the template you want to use in the middle list box.

  • View a description of the selected category of the template in the right text box.

You can select the Template directory under Preferences > Editor.

Adding templates which you or a colleague created to the dialog is described under Creating Your Own Templates.
