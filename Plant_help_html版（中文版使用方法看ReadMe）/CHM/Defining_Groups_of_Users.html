Defining Groups of Users
Setting-Up and Starting > Licensing Plant Simulation > Activating Licenses > Defining Groups of Users

In most cases groups of users work with the same applications. For this reason you can define groups in the ugslmd.opt options file and assign applications to these groups.

By default user names in group definitions are case-sensitive. To make them case-insensitive, add this line to the options file:

GROUPCASEINSENSITIVE OFF
