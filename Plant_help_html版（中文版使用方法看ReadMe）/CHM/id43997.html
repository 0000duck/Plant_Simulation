AvailabilityOn
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Defining Failures of the Material Flow Objects > Attributes of the Failure Profiles > AvailabilityOn

Syntax:

 
Path.Failures.NameOfFailureProfile.AvailabilityOn

The read-only attribute AvailabilityOn returns, if the object designated by Path uses the representation Availability and MTTR or the representation Interval and Duration for failures.

Example:

 
print MyStation.Failures.MyFailureProfile1.AvailabilityOn
