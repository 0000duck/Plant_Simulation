Control
2D Reference Help > Display and User Interface Objects > Drop-Down List > Dialog Window of the Drop-Down List > Tab Attributes > Control

Click and select a Method object in the dialog Select Object. Enter the source code of the control which is going to be executed when you select an item from the Drop-Down List in the Frame. The anonymous identifiers ? and @ are set to the Drop-Down List when the method is called.

Instead, you can also right-click in the text box. Then, select Create Control on the context menu to create a control method, which is a user-defined attribute of data type method of this object:

  • Enter a meaningful name into the text box and select Create Control. Plant Simulation then inserts self.Name_you_entered_for_the_control. Or

  • Select Create Control. Plant Simulation then inserts self.OnClicked

    Enter the source code of this control into the Method that opens.

  • To delete this control, delete the user-defined attribute; deleting the name from the text box just deletes the name, but keeps the user-defined attribute.

    To open the dialog of the object, whose name you entered into the text box, press the F2 key.

A control might, for example, look like this:

Example:

 
switch ?.Value
  case 1
    print "Item 1 selected"
  case 2 
    print "Item 2 selected"
end
