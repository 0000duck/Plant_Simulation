StatTraveledDistance
2D Reference Help > Resource Objects > Worker > Read-Only Attributes of the Worker > StatTraveledDistance

Syntax:

 
Path.StatTraveledDistance

The method StatTraveledDistance returns the distance which the instance of the Worker designated by Path covered while walking from the Workerpool to the Workplaces attached to the stations and between the stations in meters.

Return Value

The return value has the data type length.

Example:

 
print .Resources.MyWorker:2.StatTraveledDistance
