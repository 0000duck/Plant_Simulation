_3D.setGraphicMaterial
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.setGraphicMaterial

Syntax:

 
Path._3D.setGraphicMaterial(GraphicPath:any[], DiffuseColor:integer, AmbientColor:integer, SpecularColor:integer, EmissiveColor:integer, Transparency:real, Shininess:real [,GraphicGroupName:string])

The method _3D.setGraphicMaterial sets the material of the graphic of the object designated by Path.

The method deactivates graphic inheritance.

Parameters

  • The parameter GraphicPath is an array of data type any which sets the path to the graphic. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The parameter DiffuseColor of data type integer sets the diffuse color. You can set the color with the function makeRGBValue.

  • The parameter AmbientColor of data type integer sets the ambient color.

  • The parameter SpecularColor of data type integer sets the specular color.

  • The parameter EmissiveColor of data type integer sets the emissive color.

  • The parameter Transparency of data type real sets the transparency. You can specify a value between 0 and 1.

  • The parameter Shininess of data type real sets the shininess. You can specify a value between 0 and 1.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Example:

 
MyStation._3D.setGraphicMaterial([3,3], makeRGBValue(100,100,100), makeRGBValue(200,200,200), makeRGBValue(250,250,250), makeRGBValue(150,150,150), 0.7, 0.3)

Example:

 
MyStation._3D.setGraphicMaterial([3,3], makeRGBValue(100,100,100), makeRGBValue(200,200,200), makeRGBValue(250,250,250), makeRGBValue(150,150,150), 0.7, 0.3, "deco")
