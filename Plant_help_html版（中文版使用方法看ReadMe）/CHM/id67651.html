Hide and Show the Navigation Pane
Using HTML Help > General Topics > Hide and Show the Navigation Pane

To hide or show the Navigation pane:

  • To hide the Navigation pane, which contains the Contents, Index, Search, and Favorites tabs, click Hide on the toolbar.

  • To show the Navigation pane, which contains the Contents, Index, Search, and Favorites tabs, click Show on the toolbar.

    If you close the Help Viewer with the Navigation pane hidden, it will appear that way when you open the Help Viewer again.
