Add-Ins Reference Help > Inter-Process Communication Interfaces > Teamcenter > Dialog Window of the Teamcenter Interface > Tab Export > Export

To log onto the Teamcenter database to export a Plant Simulation HtmlReport to Teamcenter, click this. Then, enter your user name and your password for Teamcenter into the Teamcenter Login dialog that opens.

In case you entered a User name and a Password on the tab Connection, Plant Simulation does not open the login dialog but uses the login data which you entered.
