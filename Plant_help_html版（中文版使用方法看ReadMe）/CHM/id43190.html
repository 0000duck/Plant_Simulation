setFormatString
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > setFormatString

Syntax:

 
Path.setFormatString(Column:any [,Column:any, ...], FormatString:string)

The method setFormatString sets, depending on the data type, the Format string of the designated column(s) of the TableFile designated by Path.

Parameters

  • The parameters Column of data type any designate the columns for which you want to set the format string.

  • The parameter FormatString of data type string designates the format string.

Example:

 
MyTable.setFormatString(1,"-15.3") -- three digits after the comma
