Pick Color
2D Reference Help > The Plant Simulation Program Window > Icon Editor > Edit Ribbon Tab > Pick Color

The command Pick Color activates the Pick Color tool, which sets and applies the drawing color:

  • Place the color dropper on the color in the drawing window, which you would like to use as the drawing color and click the left mouse button.

  • The Icon Editor adds that color to the drawing color field Plant Simulation enters that color into the drawing color field below the Color Palette.
