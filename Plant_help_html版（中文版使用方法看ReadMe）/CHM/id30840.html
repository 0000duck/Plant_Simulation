setCaption
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > setCaption

Syntax:

 
Path.setCaption(DialogItem:string, Caption:string) → boolean

The method setCaption sets the Caption and the settings described below of the dialog item designated by the parameter string to the value designated by the parameter string. Plant Simulation shows changes immediately.

This method applies to these dialog items:

  • New Static Text Box, which changes its caption.

  • New Edit Text Box, which changes the contents of the text box.

  • New Tab Page, which changes the label of the tab.

  • New Button, which changes the label of the button.

  • New Group Box, which changes its caption.

  • New Menu/New Menu Command, which changes its caption.

  • New Check Box, which changes its caption.

  • New Radio Button, which changes its caption.

Parameters

  • The parameter DialogItem of data type string designates the dialog item to which the caption applies.

  • The parameter Caption of data type string designates the text you want to assign as the caption.

Return Value

The return value has the data type boolean.

Example:

 
MyDialog.setCaption("object",current.name)
if Station.failed
   MyDialog.setCaption("state","failed")
else
   MyDialog.setCaption("state","ready")
end
