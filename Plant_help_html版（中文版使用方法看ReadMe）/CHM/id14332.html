Save preview
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Quick Access Toolbar > Save Model As > Save preview

Select the check box Save preview:

  • To save a snapshot of the active Plant Simulation window, when you are working with an Plant Simulation model.

  • To save a snapshot of the active 3D window, when you are working with a 3D model.

    Once you start 3D, saving the 2D model also saves the 3D part of the model to the same *.spp file. As you then cannot tell which 2D model also contains a 3D part, we recommend to select the check box Save Preview in the dialog Save As, to save a snapshot of the active 3D window. Plant Simulation also shows this graphic, when you open the model again, enabling you to see, which model has a corresponding 3D part.

Plant Simulation shows the graphic in the dialog Open, when you select the command Open Model and click in the dialog that opens to show the preview window.
