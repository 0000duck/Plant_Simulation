Methods for the Format of Lists and Tables
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for the Format of Lists and Tables > Methods for the Format of Lists and Tables

Lists and tables provide these methods for setting and getting their format. In the dialog of the objects, you can select these settings under Format > List Format > Tab Settings:

copyContentTo, copyFormatTo, getAlignmentCell, getBackgroundColorCell, getCommonFormatData, getEditorRightsCell, getFontColorCell, getFontSizeCell, getHtmlCode, isAlignment, setAlignmentCells, setBackgroundColorCells, setCommonFormatData, setEditorRightsCells, setFontColorCells, and setFontSizeCells.

To view all of the methods and attributes of the list object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.
