VectorgraphicsActive
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes for Vector Graphics of the Mobile Objects > VectorgraphicsActive

Syntax:

 
MU-path.VectorgraphicsActive

The attribute VectorgraphicsActive activates (true) or deactivates (false) vector graphics mode for the MU designated by MU-path. Then, Plant Simulation shows the graphics of the parts with the correct size and with the correct distance to each other.

The table below illustrates the difference between the vector graphic and the icon of the object.

Object

Vector Graphic

Icon

Part

Container

Transporter

Transporter Tractor

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
.MUs.MyPart.VectorgraphicsActive:=true
