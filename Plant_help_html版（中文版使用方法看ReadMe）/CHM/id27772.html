_3D.StatesScale
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes for State Groups > _3D.StatesScale

Syntax:

 
Path._3D.StatesScale

The attribute _3D.StatesScale scales the state group of the object designated by Path. This changes the size and the position of all state graphics.

The method deactivates graphic inheritance.

Data Type of the Value You Can Assign

You can assign a value of data type real to use uniform scaling.

You can also assign an array with three values to use different scaling factors for the three axes.

Return Value

The return value always is an array with three values.

Example:

 
MyStation._3D.StatesScale := 1 -- uniform scaling
MyStation._3D.StatesScale := [1, 4, 9] -- different scaling factors for the x-axis, y-axis and z-axis
var a := MyStation._3D.StatesScale -- ‘a’ designates an array
