Use set-up matrix
Add-Ins Reference Help > Object Libraries > Value Stream Mapping > Value Stream Mapping Objects > Process_indexed > Dialog Window of the Object Process_indexed > Tab Set-Up > Use set-up matrix

To use a set-up matrix, select this check box. This activates the button Open Set-Up Matrix.
