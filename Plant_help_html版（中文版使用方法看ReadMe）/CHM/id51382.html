mirrorY
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Animating All Objects > mirrorY

Syntax:

 
Path.mirrorY

A 3D Only model does not provide the method mirrorY.

The method mirrorY mirrors the icon of the object designated by Path on the y-axis. This means that Plant Simulation mirrors the curved object horizontally and thus reverses left and right.

You cannot flip icons in the Class Library.

Example:

 
MyStation.mirrorY
