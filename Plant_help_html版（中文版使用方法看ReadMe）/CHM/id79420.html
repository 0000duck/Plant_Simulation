Cumulated Statistics of the Parts Which the Drains Removed From the Plant
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Statistics of the Material Flow Objects > Statistics Report > Resource Statistics of the Drain > Cumulated Statistics of the Parts Which the Drains Removed From the Plant

This topic shows the cumulated statistics of the parts which the Drains removed from the installation.

The column

Shows the

Read-only attribute

Working

mean processing time of the investigated parts. This is the sum of the return values of the methods StatProdWorkingPortion, StatStoreWorkingPortion, and StatTranspWorkingPortion multiplied with the return value of StatAvgLifeSpan.

--

Set-up

mean set-up time of the investigated parts. This is the sum of the return values of the methods StatProdSetupPortion, StatStoreSetupPortion, and StatTranspSetupPortion multiplied with the return value of StatAvgLifeSpan.

--

Waiting

mean Waiting time of the investigated parts. This is the sum of the return values of the methods StatProdWaitingPortion, StatStoreWaitingPortion and StatTranspWaitingPortion multiplied with the return value of StatAvgLifeSpan.

--

Stopped

mean time that the investigated parts were located on a stopped resource. This is the sum of the return values of the methods StatProdStoppedPortion, StatStoreStoppedPortion, and StatTranspStoppedPortion multiplied with the return value of StatAvgLifeSpan.

--

Failed

mean time that the investigated parts were located on a failed resource. This is the sum of the return values of the methods StatProdFailPortion, StatStoreFailPortion, and StatTranspFailPortion multiplied with the return value of StatAvgLifeSpan.

--

Paused

mean time that the investigated parts were located on a paused resource. This is the sum of the return values of the methods StatProdPausingPortion, StatStorePausingPortion, and StatTranspPausingPortion multiplied with the return value of StatAvgLifeSpan.

--

Mean Life Time

mean life time (throughput time) of all investigated parts.

StatAvgLifeSpan

Mean Exit Time

average time interval between exits of the MUs which the Drain removed from the plant.

StatAvgExitInterval

Total Throughput

total throughput of all parts, which passed through the Drain and which it removed from the plant.

StatDeleted

Throughput per Hour

number of MUs that the Drain removed from the plant in an hour over all observed times during which the Drain was available.

StatThroughputPerHour

Throughput per Day

number of MUs that the Drain removed from the plant in a day over all observed times during which the Drain was available. This is the value of the throughput per hour multiplied with 24.

StatThroughputPerDay
