Tab Product
Add-Ins Reference Help > Object Libraries > Value Stream Mapping > Value Stream Mapping Objects > ScheduleBox > Dialog Window of the Object ScheduleBox > Tab Product

On the tab Product you can enter the product and the amount that is to be produced for each day.

Double-click the table in the dialog to open the input table. Instead, you can also click the button Edit Table to open the input table.

Enter the name of the end product to be produced into the first column of a day. Enter the amount of the product to be produced into the second column. You can also enter the same product with different amounts for a single day. The amount of entries can differ from day to day.

At the beginning of a day production orders are produced with these settings for the assigned processes. Note that the production orders will be rounded up to the lot size of the product. The production orders are passed on to the Process object and then entered into the table of the production orders.
