PowerInput
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for the Energy Simulation of the Material Flow Objects > PowerInput

Syntax:

 
Path.PowerInput

The read-only attribute PowerInput shows the current power input of the object designated by Path depending on the current energy state in kilowatts.

Return Value

The return value of data type real.

Example:

 
print Station.PowerInput
