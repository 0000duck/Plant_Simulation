SwitchYWay
Add-Ins Reference Help > Object Libraries > Electrical Overhead Monorail > SwitchYWay > SwitchYWay

You can use the object SwitchYWay to model a switch which distributes the Transporters in two directions. The directions are left or right.

The object SwitchYWay provides the method switchTo to change the position of the switch.
