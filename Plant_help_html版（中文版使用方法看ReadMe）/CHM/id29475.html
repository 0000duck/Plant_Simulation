Attributes of the Curved Objects
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Curved Objects > Attributes of the Curved Objects

The curved objects AngularConverter, Conveyor, Track, TwoLaneTrack, Turntable and FootPath provide the attributes AnimateOnEveryXthPixel, Color, CurveActive, CurveWidth, MidConveyor, ObjectAngle, ObjectAngle, PenColor, PenWidth, RotateMovables, TransferLengthToObject, Transparent, and Width for modifying the settings for curved and straight segments, when you activated Curve mode.

You can also select these settings in the dialogs of the objects on the Tab Curve.

To view all of the methods and attributes of the material flow object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.

To set the value of an attribute, you might, for example, type: 

Station1.Pause := true

To get the value of an attribute, you might, for example, type: 

print Station1.Pause
posit := MyStation.Cont.XPos
