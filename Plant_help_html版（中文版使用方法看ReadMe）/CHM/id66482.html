BatChargeCtrl
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > BatChargeCtrl

Syntax:

 
MU-path.BatChargeCtrl

The attribute BatChargeCtrl designates a Method object, the Charge control, which Plant Simulation calls, when the value you entered as the battery charge (BatCharge) dropped to the reserve charge (BatReserve). It also calls it when the charge drops to 0 or reaches the capacity of the battery (BatCapacity).

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
-- A sample charge control for a Transporter could look like this
var charge: real
charge := ?.BatCharge
if round(charge - ?.BatCapacity) = 0 
   ?.BatCharging := false -- stops charging
elseif charge= ?.BatReserve 
   ?.BatCharging := true -- starts charging
end

Example:

 
.MUs.MyTransporter.BatChargeCtrl := &myBatteryChargeCtrl
