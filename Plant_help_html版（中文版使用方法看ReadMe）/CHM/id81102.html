2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Attributes of the Distributions > n

Syntax:

 
Attribute_path.n

The attribute n sets the number of trials which the distribution attempts.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

You can assign the following attributes/parameters to these distributions:

Distribution

Set of parameters

Range of values for the parameters

Binomial

n, p

integer n > 0, 0 p 1

Hypergeometric

m, n, p

1 > n > m, 0 p 1
