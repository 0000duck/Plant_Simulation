Attributes for Vector Graphics of the Mobile Objects
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes for Vector Graphics of the Mobile Objects > Attributes for Vector Graphics of the Mobile Objects

The MUs provide these attributes for vector graphics:

VectorgraphicsActive, VectorgraphicsAnilineDirection, VectorgraphicsBorderColor, VectorgraphicsBorderWidth, VectorgraphicsColor, VectorgraphicsShowDirectionArrow, and VectorgraphicsShowState.

You can also select these settings in the dialog of the objects on the Tab Graphics.

To view all of the methods and attributesof the mobile object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.
