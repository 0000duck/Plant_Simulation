newLine
2D Reference Help > Information Flow Objects > FileInterface > Methods of the FileInterface > newLine

Syntax:

 
Path.newLine → boolean

The method newLine adds a carriage return to the end of the file and sets the cursor to the end of the entry. The FileInterface inserts new entries into a new line. You can format the protocol file like this.

If the file is not open, the FileInterface opens it and leaves it open after accessing it

Return Value

The return value has the data type boolean.

Example:

 
-- inserts an Empty line into the table Tab every ten lines
var y: integer
for var y := 1 to Tab.YDim 
   MyFileInterface.writeLn(Tab[1,y], Tab[2,y])
   if index mod 10 = 0 
      MyFileInterface.newLine 
   end
