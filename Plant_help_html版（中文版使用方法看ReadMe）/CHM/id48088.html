_3D.AniRotationCenter
SimTalk Reference > 3D-related SimTalk Functions > Built-in Functions for Saved Animations > _3D.AniRotationCenter

Syntax:

 
Path._3D.AniRotationCenter

The attribute _3D.AniRotationCenter sets the rotation center for the function _3D.TypeOfAnimations.scheduleRotation of the object designated by Path.

Parameter

The parameter of data type array of three values of data type real sets the rotation center for the animation.

Example:

 
MyStation._3D.AniRotationCenter := [1,0,0]
-- rotates the object with an offset of 1,0,0
