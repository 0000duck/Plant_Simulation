Unplanned
2D Reference Help > Resource Objects > ShiftCalendar > Read-Only Attributes of the ShiftCalendar > Unplanned

Syntax:

 
Path.Unplanned

The read-only attribute Unplanned returns, if the ShiftCalendar designated by Path is Unplanned (true) or not (false). The attribute is observable.

Return Value

The return value is of data type boolean.

Example:

 
MyShiftCalendar.ShiftPlan := shiftTimesTable
