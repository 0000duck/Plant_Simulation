inheritPosition
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Animating All Objects > inheritPosition

Syntax:

 
Path.inheritPosition

The method inheritPosition turns Inheritance of the Coordinate3D of the selected object on again and places it at the same position in the Frame as the position of the object from which it was derived.

Example:

 
MyStation.inheritPosition
