2D Reference Help > Information Flow Objects > Variable > Dialog Window of the Variable > Tab Display > Transparent

To make the background of a Variable, which you inserted into a Frame, transparent, select this.

Then, the Variable shows the space around the text in the color of the background image or it shows the grid points of the Frame. If you clear Transparent, Plant Simulation shows the space around the text white.
