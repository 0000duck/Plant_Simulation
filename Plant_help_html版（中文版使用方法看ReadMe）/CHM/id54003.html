Tab Vehicle Attributes
2D Reference Help > Tools > CrossSlidingCar > Dialog Window of the CrossSlidingCar > Tab Vehicle Attributes > Tab Vehicle Attributes

The tab Vehicle Attributes provides these dialog items:

Length

Load by capacity

Speed

Load bay width

Acceleration

Load bay speed

Acceleration

Damping time

Deceleration

Take preferred position

The attributes are saved in the underlying Track and are propagated to the vehicle/transporter that is created during the initialization of the model.

The settings on the tab Vehicle Attributes correspond to the settings on the Tab Load Bay of the Transporter.
