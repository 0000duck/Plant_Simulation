Attributes for Controls of All Objects
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of All Objects > Attributes for Controls of All Objects > Attributes for Controls of All Objects

The objects provide some or all of these attributes to access control methods:

ChangePathCtrl, CloseCtrl, ConnectCtrl, ConstructorCtrl, DestructorCtrl, DragDropCtrl, FailCtrl, MoveInFrameCtrl, MoveToFolderCtrl, OpenCtrl, PauseCtrl, PermitDeleteCtrl, PlausibilityCtrl, RelabelCtrl, RenameCtrl, SelectCtrl, and UnplannedCtrl.

The data type of the method, which you assign, is object.

The expressions &Method or PathToMethod.&Method both assign an object reference to a method. Use a string to assign the relative path to a method. Assign void to disable a control.

To view all of the methods and attributes of the material flow object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.

To set the value of an attribute, you might, for example, type: 

Station1.Pause := true

To get the value of an attribute, you might, for example, type: 

print Station1.Pause
