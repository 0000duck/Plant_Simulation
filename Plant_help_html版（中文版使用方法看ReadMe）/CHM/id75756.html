[ ] (writing to a cell)
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > [ ] (writing to a cell)

Syntax:

 
Path[Column:integer, Row:integer]

The method write writes data to the designated cell.

Parameters

  • The parameter Column of data type integer designates the column.

  • The parameter Row of data type integer designates the row.

First you access the cell by its index and then you assign a value. The index statement begins and ends with a bracket. In the bracket you first enter the column and then the row of the cell to be accessed. The data type of the new value has to be the same as the data type of the table cell.

Example:

 
timeSequence[2,3] := 12.34
