Satisfied Importers
2D Reference Help > Resource Objects > Broker > Dialog Window of the Broker > The View Menu > Satisfied Importers

The menu command Satisfied Importers opens a table showing all objects whose importers were assigned Exporters.

The table shows the Importers in column 1, and their Type in column 2: The number 0 designates the failure-importer, 1 the set-up-importer, 2 the processing-importer, and 3 the transport-importer.
