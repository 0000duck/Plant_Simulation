repeatTo
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > repeatTo

Syntax:

 
Path.repeatTo(TimeSequence:object, Subrange:time, RangeToBeFille:time)

The method repeatTo removes all entries of the TimeSequence designated by Path and replaces them with copies of a subrange of the data of the designated TimeSequence.

Parameters

  • The parameter TimeSequence of data type object designates the TimeSequence whose data will replace the existing data.

  • The parameter Subrange of data type time designates the subrange. It starts with the time zero and ends at the time designated by this parameter.

  • The parameter NumberOfCopies of data type time designates the range of the TimeSequence designated by Path that is to be filled with copies of the subrange.

    Consequently, the range of the TimeSequence designated by Path must be larger than the subrange of the TimeSequence designated by the parameter of data type object. Otherwise, it would remain Empty.

Example:

 
weekPeriod := str_to_time("5:00:00:00.00")
dayPeriod := str_to_time("1:00:00:00.00")
weekPlan.repeatTo(dayPlan,weekPeriod,dayPeriod)

MyTimeSequence.or(true) -- all values := true
