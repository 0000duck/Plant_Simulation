Create the Processing Stations
Step-by-Step Help > Modeling in 2D > Modeling the Flow of Materials, Basics > Producing and Processing Parts with a Work Plan > Create the Processing Stations > Create the Processing Stations

To be able to assign the operations and the stations in the work plan later on, we start out with creating the stations which execute the operations.

  • To create the class of the processing station, right-click the object Station in the folder Materialflow and select Derive on the context menu.

  • To move the derived Station to the folder in which we are going to create the model, hold down Shift and drag it so that folder.

  • Insert it from there into the model three times. Rename the stations so that they match the names of the operations. In our example we named them Milling, Drilling_A, Drilling_B and Packing.

As the processing stations share a number of features, we set these in their class, MyStation in our example. The class object passes these properties on to its instances. We

  • Define Times in the Class of the Processing Stations

  • Define Set-up Behavior in the Class of the Processing Stations

  • Enter the Name of the Exit Control in the Class of the Processing Stations
