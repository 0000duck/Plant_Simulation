Analyzing the Results of the Optimization
2D Reference Help > Tools > GAWizard > Analyzing the Results of the Optimization

On the tab Run you can observe the optimization process. It shows the currently best fitness value at any time. Click Show Evolution on the tab Evaluate to open a diagram displaying the best, the average and the worst fitness values of each generation. You can, for example, detect, if additional optimization runs with the active settings produce better results or not.

Click Show in the group HTML Report on the tab Evaluate to show the results and the course of the optimization in a Report. When a very great number of individuals have been created, we recommend to clear the check box Show detailed HTML report.

You can, for example, detect, if the genetic algorithm is searching for the best results on the level of almost identical fitness values or if the fitness values of the offspring is still fluctuating quite a bit. For a stochastic simulation the chart shows Min-Max intervals of the fitness values of an individual an. When the fitness values fluctuate greatly, we recommend to check if an optimization of your model makes sense or not. To analyze the statistics table of the fitness values, click the respective buttons. Column 7 of the statistics table shows if individuals were created multiple times but evaluated once only.

Click Show next to Data of the report on the tab Evaluate to show the results and the history of the optimization. When the program created a large number of individuals, we recommend to not select the check box Show detailed report.
