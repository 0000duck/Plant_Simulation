Edit Controls
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit Controls

The command Edit Controls opens the dialog Edit Controls.

Click this . In the dialog Select Object, navigate to the folder or Frame, in which the Method or the object is located, select it, and click OK.

Or you can select Create Control on the context menu of the text box to create a user-defined attribute of data type method, called self.OnBuilt_in_name_of_control, for example self.OnCreate. Enter your source code here.

Plant Simulation calls the control/Method, when the user performs the actions to which you assigned the Method. The control methods are described under Edit Controls.

In addition, the Frame provides the Close control and the Permit-delete control.
