Toggle Bookmarks
2D Reference Help > Information Flow Objects > Method > The Method Window > Tools Ribbon Tab > Toggle Bookmarks

The command Toggle Bookmarks adds a bookmark for the current row or the currently selected range of the source code. Plant Simulation highlights the bookmark in the bar to the left. Bookmarks facilitate navigation in source code that extends over many lines.

We mark temporary bookmarks in the source code of a Method in a brighter color than saved bookmarks. This might, for example, be the case when you changed the source code of a Method and set a bookmark after the original end of the source code. These will be lost when you do not apply the source code.
