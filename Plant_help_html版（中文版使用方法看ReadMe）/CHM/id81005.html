CurrentMaterial
2D Reference Help > Fluid Objects > Pipe > Read-Only-Attributes of the Pipe > CurrentMaterial

Syntax:

 
Path.CurrentMaterial

The read-only attribute CurrentMaterial returns the Current material that currently flows through the Pipe designated by Path.

Return Value

The return value has the data type string.

Example:

 
print Pipe1.CurrentMaterial
