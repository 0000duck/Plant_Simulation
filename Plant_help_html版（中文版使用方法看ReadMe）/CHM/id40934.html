promptListN
SimTalk Reference > General SimTalk Functions > Predefined Functions > Input Functions > promptListN

Syntax:

 
promptListN(List:list [,Text:string, ...]) → list

The function promptListN allows the user to pick one or several entries out of the designated list, which is displayed in the dialog window.

Parameters

  • The parameter List of data type list designates the list.

The user can select one or more entries by holding down the Ctrl key and clicking the entry. OK closes the dialog and accepts the choice.

  • The optional parameter(s) Text of data type string designate the text above the list box. You can enter any number of optional parameters of data type string with a maximum length of 45 characters.

Return Value

The return value has the data type list[integer]. When the user clicks Cancel, Plant Simulation returns zero. Plant Simulation saves the indexes of the selected entries unsorted.

Example:

 
-- A part entering the plant may be processed by more than one machine.
var machPark: list[string]  -- You are going to enter the machines that can 
var indices: list[integer]  -- process it
machPark.create
machPark[1] := "M7"; machPark[2] := "M12"
machPark[3] := "M4"; machPark[4] := "M19"
indices := promptListN(machPark,
   "Select one or several of the machines that can", "process the part:")
for var i := 1 to Indices.Dim 
   switch indices[i]
   case 1 
   -- M7 selected
   case 2 
      -- M12 selected
   case 3 
      -- M4 selected
   case 4 
      -- M19 selected
   end -- switch
next

Example:

 
promptlistN(MyCardFile, "Select one or several entries from the list below",
"by Shift+clicking or Ctrl+clicking the objects.","Additional text, text, text, text, ...")
-- the entries are contained in the object MyCardFile
