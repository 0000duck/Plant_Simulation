encrypt
2D Reference Help > Information Flow Objects > Method > Methods of the Method > encrypt

Syntax:

 
&Method.encrypt(Key:string) → boolean

The method encrypt encrypts the source code of the Method designated by ref(Path).

Parameter

The parameter Key of data type string designates the password.

Return Value

The return value has the data type boolean.

Example:

 
&MyMethod.encrypt("zTrqYa8%e")
