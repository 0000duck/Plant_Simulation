setParam
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Methods of the Distributions > setParam

Syntax:

 
Path.Attribute.setParam(Type:string [,SetOfParameters:any, ...])

The method setParam sets the type of distribution and assigns a complete set of parameters to the distribution.

Parameter

The parameter Type of data type string designates the type of the distribution.

Distribution

Name in Plant Simulation English/German

Constant number

Const/Konst

Uniform distribution

Uniform/Gleich

Normal distribution

Normal/Normal

Lognormal distribution

Lognorm/Lognorm

Negative exponential distribution

Negexp/Negexp

Geometric distribution

Geom/Geom

Hypergeometric distribution

Hypgeo/Hypgeo

Erlang-distribution

Erlang/Erlang

Weibull-distribution

Weibull/Weibull

Triangular distribution

Triangle/Dreieck

Binomial distribution

Binomial/Binomial

Poisson-distribution

Poisson/Poisson

Gamma distribution

Gamma/Gamma

Beta distribution

Beta/Beta

Discrete Empirical (dEmp) distribution

dEmp/dEmp

Continuous Empirical (cEmp) distribution

cEmp/cEmp

Primitive Empirical (Emp) distribution

Emp/Emp

Formula

Formula/Formel

Type-dependent distribution

List (Type)/Liste(Typ)

List-dependent distribution (ParallelStation)

List (Place)/Liste(Platz)

Matrix distribution for setting-up objects

Matrix(Type)/Matrix(Typ)

Depending on the type, you will have to enter additional parameters, such as LowerBound, Mu, Sigma, Type, and UpperBound.

Example:

 
MyStation.proctime.setParam("normal",30,10)
MyStation.proctime.setParam("normal",30,10,1,100)
MyStation.setuptime.setParam("formula","Variable1/20")
MyGenerator.duration.setParam("dEmp",2,TableEmp)
