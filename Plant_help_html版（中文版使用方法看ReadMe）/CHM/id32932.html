_3D.TypeOfAnimations.AnimationPathName.IsSpline
SimTalk Reference > 3D-related SimTalk Functions > Built-in Functions for Saved Animations > _3D.TypeOfAnimations.AnimationPathName.IsSpline

Syntax:

 
Path._3D.MUAnimations.AnimationPathName.IsSpline → boolean
Path._3D.SelfAnimations.AnimationPathName.IsSpline → boolean
Path._3D.CameraAnimations.AnimationPathName.IsSpline → boolean

The read-only attribute _3D.TypeOfAnimations.AnimationPathName.IsSpline returns if the designated animation is of type Spline or not.

Return Value

The return value has the data type boolean.

Example:

 
var b : boolean := .MaterialFlow.Station._3D.MUAnimations.Default.IsSpline
