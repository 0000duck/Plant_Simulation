StatServicesEnRouteToJobTime
2D Reference Help > Resource Objects > Worker > Read-Only Attributes of the Worker > StatServicesEnRouteToJobTime

Syntax:

 
Path.StatServicesEnRouteToJobTime

The read-only attribute StatServicesEnRouteToJobTime returns the total time during which the Worker was en-route to a Workplace , for which he was mediated to do a job at the associated station. When you check Get job orders in the pool only, Plant Simulation also counts the time during which the Worker walked back to the WorkerPool to get the next order.

Return Value

The return value has the data type time.

Example:

 
print .Resources.MyWorker:1.StatServicesEnRouteToJobTime
