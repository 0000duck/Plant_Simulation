Speed
2D Reference Help > Material Flow Objects > Turnplate > Attributes of the Turnplate > Speed

Syntax:

 
Path.RotationTimePer90Degrees

The attribute Speed sets the speed, with which the Turnplate transports the part, while it is located on the plate. You can also enter -1 for an infinite speed.

Data Type of the Value You Can Assign

You can assign a value of data type speed.

Example:

 
MyTurnplate.RotationTimePer90Degrees := 0.75
