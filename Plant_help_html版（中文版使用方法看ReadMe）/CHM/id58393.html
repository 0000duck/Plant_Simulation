Exported Services
2D Reference Help > Resource Objects > Worker > Dialog Window of the Worker > The View Menu > Exported Services

The menu command Exported Services opens a list showing the names of all services, which the Worker provides at this time. Double-click the name of a service to open a sub-table, which shows the name of the Importer, the Type of the importer and the Amount of services provided.
