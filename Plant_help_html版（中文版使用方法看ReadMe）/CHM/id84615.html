UseInputChannels
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > UseInputChannels

Syntax:

 
Path.UseInputChannels

The attribute UseInputChannels sets if the Chart takes data it displays from input channels (true ) or from a data table you defined (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyChart.UseInputChannels := true
