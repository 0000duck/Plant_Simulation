startSetup
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods of the Importer > startSetup

Syntax:

 
Path.setupImp.startSetup([StartAnewContinue:boolean]) → boolean

The method startSetup starts setting-up the object designated by Path, for the MU for which the Exporter was requested. Then the appropriate Set-up end event is registered, compare list of scheduled events.

Parameter

The optional parameter StartAnewContinue of data type boolean sets if setting-up starts anew after an interruption (true) or if setting-up continues where it stopped before the interruption (false).

Only the Station, the ParallelStation, the AssemblyStation, and the DismantleStation, the Mixer, the Portioner, and the Tank, can import services, which you define on the Tab Importer.

Return Value

The return value has the data type boolean. It is false when set-up cannot be started, because not all of the required services are available.

Example:

 
ParallelStation.setupImp.startSetup
dismantle.setupImp.startSetup(true)
