Ignore Errors in Formulas
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Debugger Ribbon Tab > Ignore Errors in Formulas

The command Ignore Errors in Formulas prevents Plant Simulation from opening the Debugger when a runtime error occurs in a formula.

You can also select Ignore Errors in Formulas on the Debug Menu of an open Method, which only applies to that Method.
