Height
2D Reference Help > Display and User Interface Objects > Button > Dialog Window of the Button > Tab Attributes > Height

Enter the height in meters with which the object is displayed in the Frame. To change the height, you can also hold down Ctrl+Shift and drag the top or the bottom of the icon.

When you enter a height of more than 30 pixels, Plant Simulation uses a larger font to display the label of the button.
