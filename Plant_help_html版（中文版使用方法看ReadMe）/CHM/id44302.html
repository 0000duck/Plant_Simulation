Connect 2D and 3D positions
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Transformation > Settings for the Position > Connect 2D and 3D positions

To match the position of the object in the Plant Simulation part of the simulation model with the position of the respective object in the 3D part, select this.

The example illustrates this:

  • Insert a Station into the Frame into the 3D part of a simulation model. Plant Simulation automatically inserts the Station into the Frame in its part of the model.

    Plant Simulation maps the x-y coordinates of the 3D object to the respective position of the Station in the Frame in the Plant Simulation simulation model. Expert users can edit how coordinates are mapped with the command File > Scaling Factor in the Plant Simulation Frame.

  • Select Connect positions. When you move the Station in the 3D part of the model now, the Station in the Plant Simulation part of the model also moves to a position that matches the position of the Station in the 3D window. The position of the object in Plant Simulation will be coupled to the position of the object in the 3D window until you clear the check box again!
