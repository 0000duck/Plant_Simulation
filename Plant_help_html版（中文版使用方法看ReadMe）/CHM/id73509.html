Find a Condition of an Object
Step-by-Step Help > Getting to Know Tecnomatix Plant Simulation > Working with the Program, Basics > Finding Objects and Text in Your Simulation Model > Find a Condition of an Object

To find a condition of an object you inserted into your simulation model:

  1. Select Condition from the left drop-down list.

  2. Enter any SimTalk expression whose attributes or methods are identical to the expression you enter in the drop-down combo box. You might, for example, enter the name of an attribute or a method and the expression you are looking for, such as Proctime = 100, or OpenCtrl = void, etc.

  3. Repeat steps 4 to 7 described under Find the Name of an Object

  4. Click Find.
