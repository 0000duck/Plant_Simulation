num_to_time
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Values without Units into Physical Data Types with Units > num_to_time

Syntax:

 
num_to_time(real) → time

The function num_to_time converts the number without a unit into a value of data type time.

Parameter

The parameter Number of data type real designates the number.

Return Value

The return value has the data type time.

Example:

 
var s: speed
s := Track.length / num_to_time(30)
