Get job orders in the pool only
2D Reference Help > Resource Objects > WorkerPool > Dialog Window of the WorkerPool > Tab Attributes > Get job orders in the pool only

To make the Worker get new work orders for performing a service at a station in the WorkerPool only, select this. When you clear the checkbox, new orders can be assigned to the Worker anywhere, for example, while he walks on a FootPath.
