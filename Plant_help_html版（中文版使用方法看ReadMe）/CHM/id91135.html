StatWaitingResMU
2D Reference Help > Material Flow Objects > AssemblyStation > Read-Only Attributes of the AssemblyStation > StatWaitingResMU

Syntax:

 
Path.StatWaitingResMU

The read-only attribute StatWaitingResMU returns the average duration of a period of time during which the AssemblyStation designated by Path was Waiting for mounting parts and/or for Exporters/services.

Return Value

The return value has the data type real.

Example:

 
print MyAssembly.StatWaitingResMU
