DismantleStation
2D Reference Help > Material Flow Objects > DismantleStation > DismantleStation

The DismantleStation removes mounting parts from the main MU or it creates new MUs. You can use it to model dismantle processes in your plant. To model assembly operations, you can use the AssemblyStation.

To add the DismantleStation to your simulation model, click Manage Class Library > Basic Objects > MaterialFlow > DismantleStation on the Home ribbon tab.

Compare the sample models: Click the Window ribbon tab, click Start Page > Getting Started > Example Models and click Small Examples. Then, select the respective Category, the Topic, and the Example in the dialog Examples Collection and click Open Model.
