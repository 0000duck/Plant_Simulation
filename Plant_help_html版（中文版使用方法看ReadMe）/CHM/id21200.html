setEpsilon
SimTalk Reference > General SimTalk Functions > Predefined Functions > Mathematical Functions > Basic Mathematical Functions > setEpsilon

Syntax:

 
setEpsilon(Value:real) → real

The function setEpsilon sets the value for which Plant Simulation considers values to be about equal.

Parameter

The parameter Value of data type real designates the value.

Return Value

The return value has the data type real.

Example:

 
setEpsilon(0.0001)
-- the comparison 1.0003 ~= 1.0 returns false
-- the comparison 1.00003 ~= 1.0 returns true

Use the about equal comparison to prevent rounding errors when using real values.

Enter print (1/3+1000-1000)*3 and see which number your computer returns.
