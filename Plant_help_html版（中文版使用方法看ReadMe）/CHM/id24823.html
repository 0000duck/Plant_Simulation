findFloor
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for the Order of Cells within Lists and Tables > findFloor

Syntax:

 
Path.findFloor([Range:listrange, ...,] Value:any) → boolean

The method findFloor searches in the list for the value less than or equal to the value you specify.

When Plant Simulation finds a value, the method returns true and sets the cursor onto that entry. It returns false when it did not find the value. When several values exist, Plant Simulation sets the cursor to the cell that is closest to the value of data type any.

The search starts at the current cursor position. This holds only true when the cursor is located within one of the ranges specified. When that is not the case, the search begins in the first range specified and, if finished without success, is continued in the next range, etc.

When the cursor is located in one of the ranges specified, Plant Simulation does not search the preceding ranges, not even after the search was negative for the other ranges. When the cursor is located in the middle of the range, Plant Simulation only searches the region after the cursor, but not the region before. When you do not enter a range, Plant Simulation searches the list starting at the position of the cursor to the end of the list.

The method only applies to lists and tables of data types integer, realand string, and to data types, which can be converted into the data type real, such as dateTime, weight, etc.

Parameters

  • The parameter Value of data type any designates the value you want to find.

  • The parameter Range of data type listrange designates the range between which Plant Simulation searches to the position of the cursor, for tables the position of both cursors.

Return Value

The return value has the data type boolean.

When Plant Simulation finds a value, the method returns true and sets the cursor onto that entry. It returns false when it did not find the value. When several values exist, Plant Simulation sets the cursor to the cell that is closest to the value of data type any.

Suppose that a CardFile C contains the entries 20, 40, 0, 30, 10, 50. We then apply the method findFloor to it:

Example:

 
c.Cursor := 1 -- prepares the search
c.findFloor({2}..{5},25)

As the result, Plant Simulation will set the cursor to the entry 10. Note that the Cursor is located in cell 5. The entry 10 is the value smaller or equal to the search value 25, being closest to 25 and that is located in the given range.
