_3D.exportAsJt
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > _3D.exportAsJt

Syntax:

 
Path._3D.exportAsJt(FilePath:string)

The method _3D.exportAsJt exports the designated graphic or the designated 3D object as a .JT file. It exports only graphics which are part of the object.

Parameter

The parameter FilePath of data type string designates the path to the target file.

Example:

 
.MaterialFlow.Frame.exportAsJt("D:\MyS3D\Frame.jt")
