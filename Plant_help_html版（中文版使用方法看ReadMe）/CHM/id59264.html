DialogX
2D Reference Help > Display and User Interface Objects > Dialog > Attributes of the Dialog > DialogX

Syntax:

 
Path.DialogX

The attribute DialogX sets the position on the x-axis where Plant Simulation shows the dialog window of the Dialog you designed, when you open it. The unit is pixels, the zero point is the left top corner of the screen or the dialog window. Enter -1 for either DialogX or DialogY to center the Dialog on screen.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyDialog.DialogX := 350
