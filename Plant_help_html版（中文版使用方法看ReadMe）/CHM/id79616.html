2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > Units > Daylight saving time > Day End

Enter the day when daylight saving time ends. For the U.S. and the EU, enter 0 for Sunday. 0 designates Sunday, 1 Monday, 2 Tuesday, 3 Wednesday, 4 Thursday, 5 Friday, and 6 Saturday.
