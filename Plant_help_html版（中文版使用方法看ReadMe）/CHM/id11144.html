Additional Statistical Data of the MUs
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Statistics of the Mobile Objects > Additional Statistical Data of the MUs

The statistical data listed below only apply to MU classes, as statistics collection includes all instances of the MU.

Item

Shows

Method

Number

how many MU instances of this class existed during the statistics collection period. Plant Simulationonly counts those MUs whose product statistics was active during the simulation run. The MUs may be located in different Frame hierarchies.

StatCount

Deleted

how many MU instances of this class were removed from the plant during the statistics collection period. Plant Simulation only counts those MUs whose product statistics was active during the simulation run.

StatDeleted

Mean Life Time

the average life-span of the MUs, which were created and removed from the plant during the statistics collection period. Plant Simulation only counts those MUs whose product statistics was active during the simulation run.
