hideBBL
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > hideBBL

Syntax:

 
hideBBL(MinimizeWindow:boolean [,HideWindow:boolean]) → boolean

The function hideBBL minimizes Plant Simulation to an icon in the status area of taskbar.

Parameters

  • Setting the parameter MinimizeWindow of data type boolean to true minimizes Plant Simulation into the taskbar status area. The application window can be restored by double-clicking the icon in the taskbar status area.

Setting the parameter to false restores the application window (if it has been minimized into the taskbar status area).

The parameter HideWindow of data type boolean is optional and only relevant, when the parameter boolean has the value true.

  • Setting the parameter HideWindow of data type boolean to true hides the application window without showing an icon in the taskbar status area, so the user cannot restore the application window.

Setting the parameter to false shows an icon in the taskbar status area, when Plant Simulation is minimized in the taskbar status area. This is the default.

Return Value

The return value has the data type boolean.

Example:

 
hideBBL(true) -- hides the Plant Simulation window
