Target is on
2D Reference Help > Tools > TransferStation > Dialog Window of the TransferStation > Tab Attributes > Target is on

Enter onto which target the TransferStation places the MUs. When loading or reloading parts, the MUs will be located on a means of transport on the target station. You can also drag that object, i.e., the target station, over the icon of the TransferStation and drop it there.

The target station can be any one of these material flow objects: Drain, Conveyor, TwoLaneTrack, Track, ParallelStation, Station, DismantleStation, Buffer, and PlaceBuffer.

A TransferStation, which you inserted into a Frame, shows the station type, the part station, and the target station in the Tooltip.
