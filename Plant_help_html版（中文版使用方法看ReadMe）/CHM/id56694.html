Setting a Station Up
Step-by-Step Help > Modeling in 2D > Modeling the Flow of Materials, Basics > Setting a Station Up > Setting a Station Up

You can define how to set up the objects Station , ParallelStation , AssemblyStation , DismantleStation , and Drain to process another type of MU.

You can:

  • Select Set-up Options on the tab Set-Up.

  • Select the Set-Up Time on the tab Times.

Compare the sample models: Click the Window ribbon tab, click Start Page > Getting Started > Example Models, and click Small Examples. Then, select the respective Category, the Topic, and the Examplein the dialog Examples Collectionand click Open Model.
