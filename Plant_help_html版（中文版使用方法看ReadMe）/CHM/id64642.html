calendarWeek
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Managing Date and Time > calendarWeek

Syntax:

 
calendarWeek(Date:dateTime) → integer

The function calendarWeek returns the calendar week of a date according to DIN 1355/ ISO 8601. Here the first calendar week of a year is the first week that contains at least four days of the new year.

A new week always starts on Monday.

Parameter

The parameter Date of data type dateTime designates the date.

Return Value

The return value has the data type integer.

Example:

 
print calendarWeek(str_to_date("11.21.2012")) -- returns 45
