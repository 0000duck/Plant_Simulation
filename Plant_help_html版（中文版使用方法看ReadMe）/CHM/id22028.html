drawLine
2D Reference Help > Material Flow Objects > Frame > Methods of the Frame > drawLine

Syntax:

 
Path.drawLine(Layer:integer, X1:integer, Y1:integer, X2:integer, Y2:integer, RGB:integer, lineWidth:integer [,lineStyle:integer])

The method drawLine draws a line on the background of the Frame designated by Path.

To draw the line in the foreground, i.e., in front of the objects in the Frame, enter a negative number for the layer, designated by the parameter integer.

Parameters

The parameter of data type integer

Designates the

Layer

the layer.

X1

the starting point of the line on the x-axis.

Y1

the starting point of the line on the y-axis.

X2

the end point of the line on the x-axis.

Y2

the end point of the line on the y-axis.

RGB

the color of the line.

lineWidth

the width of the pen in pixels of the line.

When you use the line width 0, Plant Simulation always draws the line with a width of 1. This also applies when you zoom the contents of the Frame.

lineStyle

the line style of the line.

-1 looks like this .

-2 looks like this .

0 stands for a solid line .

1 stands for a dashed line .

2 stands for a dotted line .

3 stands for a dash dot line .

4 stands for a dash dot dot line .

For the line styles with the number 1 to 4 the operating system only allows a line width of 1 or 0. Note that for a line width of 1 Plant Simulation always draws the line style as a solid line, when you zoom the contents of the Frame.

Example:

 
.Models.Frame.drawLine(1,8,15,58,17,166,4,0)
