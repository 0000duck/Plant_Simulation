Stop
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Defining Failures of the Material Flow Objects > Attributes of the Failure Profiles > Stop

Syntax:

 
Path.Failures.NameOfFailureProfile.Stop

The attribute Stop sets the Stop time that has to pass before a new failure for the object designated by Path takes place.

When you enter 0, Plant Simulation creates new failures infinitely.

When you change failure settings, this does not affect the already computed failure events. If you would like the changed settings to take effect immediately, you first have to assign the value false to the attribute Active, and then reassign the value true again to the attribute after setting the stop time of the failure. This causes a new failure event to be calculated.

Data Type of the Value You Can Assign

You can assign a value of data type time.

Example:

 
MyStation.Failures.MyFailureProfile1.Stop.setParam("Const", str_to_time("2:30:00"))
