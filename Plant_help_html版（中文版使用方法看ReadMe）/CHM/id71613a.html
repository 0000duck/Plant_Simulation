Use LEDs
2D Reference Help > The Plant Simulation Program Window > Icon Editor > General Ribbon Tab > State Icons/LEDs > Use LEDs

The command Use LEDs activates or deactivates the display LEDs instead of state icons of the material flow object during your simulation runs. This is the default setting.

Default LED

Sample state icons

By default, the material flow objects do not provide predefined state icons. If you want to use icons for the different states of the object, you have to create these icons yourself.
