MaxDim
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Miscellaneous Attributes of Lists and Tables > MaxDim

Syntax:

 
Path.MaxDim

The attribute MaxDim sets the maximum number of cells for a list with one column. Enter -1 for an infinite size.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyStackFile.MaxDim := -1 -- unlimited dimension
MyQueueFile.MaxDim := 20 -- 20 entries
MyCardFile.MaxDim := 2*MyCardFile.MaxDim -- double the number of entries
