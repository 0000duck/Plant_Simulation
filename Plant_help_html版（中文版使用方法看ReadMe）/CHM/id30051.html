Time reference
2D Reference Help > Information Flow Objects > Trigger > Dialog Window of the Trigger > Tab Period > Time reference

Select the time reference which the Trigger uses during the simulation run.

  • For a Relative time reference enter a point in time ( time) into the text box Start time.

  • For an Absolute time reference enter a point in time on a certain date (dateTime) into the text box Start date.
