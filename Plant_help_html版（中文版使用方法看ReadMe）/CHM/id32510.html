Plant Simulation Interface Package
Setting-Up and Starting > Plant Simulation Options > Plant Simulation Interface Package

The Plant Simulation Interface Package enables you to:

  • Exchange data with databases and other applications

  • Control Plant Simulation from other applications or vice versa

The Plant Simulation Interface Package provides these interfaces:

  • The ActiveX® interface to work with ActiveX controls

  • The C-programming interface to call C functions from inside Plant Simulation

  • The ODBC interface to connect to databases and applications which support ODBC drivers

  • The OPC Classic interface to connect to an OPC server to exchange signals with PLCs

  • The OPC UA interface to read and write control variables and the signals of PLC controls

  • The Oracle11g SQL interface to exchange data with Oracle databases via an Oracle client

  • The PLCSIM interface to exchange signals and data between Plant Simulation and virtual PLCs

  • The SIMIT interface to connect to process control devices, such as PLC controls, via SIMIT (Simulation & Testing)

  • The Socket interface to exchange data with other applications via an TCP/IP connection
