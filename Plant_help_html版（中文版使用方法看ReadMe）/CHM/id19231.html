getAlignmentColumn
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > getAlignmentColumn

Syntax:

 
Path.getAlignmentColumn(Column:any) → string

The method getAlignmentColumn returns the alignment of a column of the TableFile designated by Path.

Parameter

The parameter Column of data type any designates the column whose alignment you want to get.

Return Value

The return value has the data type string.

Example:

 
print MyTableFile.getAlignmentColumn(4)
