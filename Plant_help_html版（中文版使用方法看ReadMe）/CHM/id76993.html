getColumnNo
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > getColumnNo

Syntax:

 
Path.getColumnNo(ColumnIndex:any) → integer

The method getColumnNo searches for the column which contains a user-defined column index of the TableFile designated by Path. It returns the column number. If the column index does not exist, Plant Simulation returns -1.

If the table contains a large number of entries, you might want to activate Fast index access.

The method only works for simple data types, such as string, integer, etc.

Parameter

The parameter ColumnIndex of data type any designates the column index which Plant Simulation searches.

Return Value

The return value has the data type integer.

Example:

 
var column : integer
column := MyTableFile.getColumnNo("Urgent")
