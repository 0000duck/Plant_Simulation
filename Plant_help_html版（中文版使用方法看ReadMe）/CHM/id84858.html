_3D.VisibleGraphics
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > _3D.VisibleGraphics

Syntax:

 
Path._3D.VisibleGraphics

The attribute _3D.VisibleGraphics sets the graphic groups of the object designated by Path that will be visible.

Data Type of the Value You Can Assign

You can assign a value of data type array.

The visibilities you set here apply for everything except for walking Workers on a Footpath.

Example:

 
MyStation._3D.VisibleGraphics := ["default"]
