Duration
2D Reference Help > Information Flow Objects > Generator > Attributes of the Generator > Duration

Syntax:

 
Path.Duration

The attributeDuration sets the time span between calling the Interval control and the Duration control.

Data Type of the Value You Can Assign

You can assign a value of data type time.

Example:

 
MyGenerator.Duration.Type := "uniform"
MyGenerator.Duration.Start := 600 -- from 10 minutes
MyGenerator.Duration.Stop := 1200 -- to 20 minutes
