Category
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Display > Category

Select one of these categories:

  • Chart shows a number of values or several sets of a number of values.

  • Histogram shows the frequency distribution for one or several input channels. For this setting you can select the check box Accumulated.

  • Plotter shows the progression of one or more values over time. For this setting you can select the check box Step curve.

  • XY Graph shows a number of x-y pairs of values or several sets of x-y pairs of values. This setting is active only when you select Table File as the Data Source.

The XY Graph shows the range like this:

  • The x values are contained in the first row or column.

  • The range determines the y values. The first row or column, which contains the x values, will automatically be excluded.

When you select another Category, the Chart automatically shows the icon of this category in the Frame.
