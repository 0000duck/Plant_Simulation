View
2D Reference Help > Information Flow Objects > Method > The Method Window > Tools Ribbon Tab > View > View

The commands on the submenu View set what Plant Simulation in the Frame window. Settings you select here override those you selected under File > Preferences > Editor in the Plant Simulation window.

A check mark to the left of a command means that this command is active, i.e., that Plant Simulation shows the selected item. To not show an item, click the command so that it does not show a check mark.

When you change one of this settings, the change applies during the entire Plant Simulation session for all newly opened Methods.

  • Show Outlining

  • Highlight Current Conveyor

  • Show Tooltip with Completion

  • Tooltip includes Type Info
