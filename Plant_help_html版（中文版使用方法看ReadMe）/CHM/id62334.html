DecimalPlaces
2D Reference Help > Information Flow Objects > Variable > Attributes of the Variable > DecimalPlaces

Syntax:

 
&Variable.DecimalPlaces

The attribute DecimalPlacessets the number of decimal places which the object Variable shows in the Frame.

It can show up to 15 decimal places. The preset -1 means that all existing places are shown. When you enter a positive value, the Variable shows the number of decimal places you entered in the Frame.

This setting only applies for the Data types > real, length, weight, time, speed, and acceleration.

The number of decimal places for the data type time affects values differently than for the data type real. The value -1 for the data type time means that the default time display format is displayed with four decimal places. A positive value means that Plant Simulation maximally shows the number of decimal places that you entered.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
&MyVariable.DecimalPlaces := 12
