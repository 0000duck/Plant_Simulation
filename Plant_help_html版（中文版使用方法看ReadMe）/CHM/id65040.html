Empirical Distributions
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Empirical Distributions > Empirical Distributions

Plant Simulation provides these empirical distributions:

Primitive Empirical (Emp)

Continuous Empirical (cEmp)

Discrete Empirical (dEmp)

You can use the empirical distributions when your data cannot be represented properly by a mathematical distribution.

According to a set of observed data x_1, …, x_n a random variable with a distribution following the observed frequency of the data is computed. You have to enter the input data into a frequency table.
