The Method Window
2D Reference Help > Information Flow Objects > Method > The Method Window > The Method Window

Double-click the icon of the Method, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Information Flow in the Toolbox. It provides access to the method editor and to the Method Debugger.
