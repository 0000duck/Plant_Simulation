closeConsole
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > closeConsole

Syntax:

 
closeConsole → boolean

The function closeConsole closes the open Console window. All messages that were output to the Console will be lost.

We still provide the optional parameter of data type boolean for compatibility reasons. It does not have any effect from this version on.

Return Value

The return value has the data type boolean.

Example:

 
closeConsole
