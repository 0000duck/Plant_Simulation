DestCtrl
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > DestCtrl

Syntax:

 
MU-path.DestCtrl

The attribute DestCtrl designates a Method object, the Destination control, which Plant Simulation activates after the Transporter has covered the object you entered as the Destination. It may, for example delete the load and execute new orders.

As soon as the Transporter has reached the destination and covered it, it calls the destination control . When you entered a front-triggered exit control into the Destination object, this does not activate the destination control.

After setting the destination in the destination control, you have to make the Transporter drive on by moving it with the method move.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
@.DestCtrl := &myDestinationCtrl -- enter
@.DestCtrl := VOID -- and delete again
