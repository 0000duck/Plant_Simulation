Tank
2D Reference Help > Fluid Objects > Tank > Tank

The Tank is a type of buffer for temporarily storing a single material before or after processing. The material can flow into the Tank from several predecessors and can flow out of the Tank to several successors.

To add the Tank to your simulation model, click Manage Class Library > Basic Objects > Fluids > Tank on the Home ribbon tab.
