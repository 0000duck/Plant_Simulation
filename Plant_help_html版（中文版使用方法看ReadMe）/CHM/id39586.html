week
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Managing Date and Time > week

Syntax:

 
week(Date:date) → integer
week(DateTime:dateTime) → integer

The function week returns the number of weeks that started since January first of a year.

The week always begins on a Monday. Note that this week does not correspond to the calendar week.

Parameter

The parameter Date of data type date designates the date.

Return Value

The return value has the data type integer.

Example:

 
print week(str_to_date("1.1.99"))             -- 1
print week(str_to_date("1.2.99"))             -- 6
print week(str_to_dateTime("1.1.99 1:00:00")) -- 1
