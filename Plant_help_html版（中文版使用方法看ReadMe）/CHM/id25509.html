Transparency
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Appearance of the MUs > Transparency

To set the Transparency, i.e., how opaque or clear the object is, drag the slider. Water, for example, might be more clear than opaque.

You can also enter a value between 0.0, for opaque, and 1.0, for completely transparent, into the text box.
