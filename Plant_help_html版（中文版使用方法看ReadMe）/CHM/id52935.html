initialize
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Miscellaneous Methods of the TableFile > initialize

Syntax:

 
Path.initialize([Range:listrange, ...,] Value:any)

The method initialize initializes the designated ranges within a table with the designated value. Plant Simulation overwrites any existing data.

If the designated range contains the index and the content, Plant Simulation only initializes the content.

Parameters

  • The parameter Range of data type listrange designates the range.

  • The parameter Value of data type any designates the value.

    The data type of the parameter any has to be identical with the data types of the ranges to be initialized.

Example:

 
MyTable.initialize({1,0}..{5,*},{7,*},5)
MyTable.initialize({*,*},"Empty")
