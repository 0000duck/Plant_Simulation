Tab Calendar
2D Reference Help > Resource Objects > ShiftCalendar > Dialog Window of the ShiftCalendar > Tab Calendar > Tab Calendar

On the tab Calendar you can define the days on which your plant does not work at all (Date From, and Date To) or only works part of the time (Reduce Time To), and a Comment describing the event.

Before you can use the date picker, click the toggle button Inheritance, so that it looks like this

You can also import a calender that you saved. Or you can export a calender you defined and then import it into another simulation model. Click into the list field with the right mouse button and select Import or Export.

To manipulate the contents of the list, use the commands on the Context Menu of Embedded Lists.
