Creating a List within a TableFile
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Creating Lists within Lists and Tables > Creating a List within a TableFile > Creating a List within a TableFile

To create a list or table in an object of type TableFile.

  • Add the sub-list to a cell within the object, compare Creating Lists within Lists and Tables.

  • Insert a List into a TableFile from a Frame, drag that object to the active cell of the TableFile and drop it there. This inserts the path to and the name of the list or table into the active cell.

  • Assign a List to a TableFile in a Method.
