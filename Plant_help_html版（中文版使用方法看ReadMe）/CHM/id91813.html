setTargetStation
2D Reference Help > Tools > TransferStation > Methods of the TransferStation > setTargetStation

Syntax:

 
Path.setTargetStation(TargetStation:<object>,SensorPosition:<length,ActivateSensor:string,Lane:string,UserControl:string)

The method setTargetStation sets the target station of the TransferStation.

Parameters

  • The parameter TargetStation of data type object designates the target station as such.

  • The parameter SensorPosition of data type of data type length designates the position of the sensor, when the target station is a Conveyor, a Track or a TwoLaneTrack.

  • The parameter ActivateSensor of data type string designates when the TransferStation Activates the sensor, when the target station is a Track or a TwoLaneTrack. You can enter "Always" or "Only when Destination" .

  • The parameter Lane of data type string designates the Lane on which the sensor will be inserted, when the target station is a TwoLaneTrack . You can enter "A" , "B", "A+B", or "B+A".

  • The parameter UserControl of data type string designates the name of the user-defined control, which you can enter for all objects serving as target station.

If the target station does not exist at all or if you entered an object, which is not accepted, the method returns false.

Note that when entering the sensor position you have to make sure that the target station provides enough space to accommodate the incoming MU so that it can be transferred. The sensor position 0.0 is an exception: Although the target station theoretically does not provide enough space, the MU can still move onto the target station.

Example:

 
var obj : object
obj := .Tools.TransferStation.derive(.Models.MyFrame)
obj.setTargetStation(MyTwoLaneTrack,4,"","","B+A")
