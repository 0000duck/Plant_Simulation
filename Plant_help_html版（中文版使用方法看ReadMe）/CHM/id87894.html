Select an Object to Display its Current Icon, or an Object and a Specific Icon to be Displayed on the HTML Page
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Select an Object to Display its Current Icon, or an Object and a Specific Icon to be Displayed on the HTML Page

To select an object to display its current icon, or an object and an icon to be displayed on the HTML page, click on the toolbar .

Select the icon you want to use in the dialog that opens and click OK.

Enter the caption which the HtmlReport shows below the picture.

Enter the Image size in percent to which the HtmlReport scales the picture. Instead you can also enter the Image width and the image height. You can also enter an asterisk (*) for either the width or the height to ensure that the image will not be distorted. The relative size takes precedence over the absolute size.

The settings above result in this statement:

[!MyMethod, User, My picture of MyMethod, 110%]

When the text cursor is located within a picture reference ([!…]), Plant Simulation first opens the dialog Open File and then the dialog Object Parameters for the path within brackets when you click Add Icon .

When the text cursor is located within a picture reference ([!…]), Plant Simulation first opens the dialog Open File and then the dialog Object Parameters in which the path is preselected that was detected within brackets when you hold down the Shift key while clicking Add Icon .

Instead, you can also directly enter an opening square bracket, followed by an exclamation mark, the path to the object and a closing square bracket, for example [!MyObject]. To not show the current icon, but a certain icon of the object, enter the name or the number of the icon, separated by a comma. 

[!Station1]
[!Station1, Default]
[!Station1, 0]

In addition you can set the size of the icon as follows:

  • Enter an asterisk * to also zoom the icon out when you zoom the report window out. When zooming in, the icon is never enlarged to more than its original size.

  • Enter a number and the percent sign % to scale the icon with this value.

  • Enter the width and the height of the icon in millimeters to show the icon with these measurements. You can also replace one of the two values with an *. 

[!Station1, "",  *]
[!Station1, "", 150%]
[!Station1, "", 40, 40]
[!Station1, "", 100, *]
[!Station1, "", *, 150]
