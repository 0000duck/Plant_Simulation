You cannot use any of these keywords as names for objects and variables. Keywords are not case-sensitive IntEgeR and inTEGer are considered to be the same. When you add letters or numbers to the beginning or the end of a keyword, Plant Simulation does not treat it as a keyword any more. Consequently, a keyword modified like this is allowed: While using the keyword list as a name for a user-defined method is not allowed, you can use alist or list0.
