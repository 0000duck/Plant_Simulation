Tab Failures
2D Reference Help > Tools > TransferStation > Dialog Window of the TransferStation > Tab Failures > Tab Failures

By default, the Availability of the TransferStation is activated. If you do not want to use the default settings, enter the values of your choice.

If you do not want to work with the Availability, but want to enter the Interval and the Duration, click Open Station on the tab Advanced Controls and clear the check box Availability on the tab Failures of the ParallelStation. Plant Simulation propagates any changes you make to the respective other object.
