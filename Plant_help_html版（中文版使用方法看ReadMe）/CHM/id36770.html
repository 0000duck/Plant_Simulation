Select the Output Values for the Experiments
2D Reference Help > Tools > ExperimentManager > Defining the Experiments > Select the Output Values for the Experiments

Before executing your simulation study you have to define the output values. You cannot start the ExperimentManager without output values. You can view the output values after a simulation run in your model. To open the table Output, click the button Define Output Values . If you want to, you can also enter any explanatory text which describes the output value in plain words. All results are then represented by this text.

When you drag a material flow object onto the ExperimentManager , it offers all attributes and methods of this object, which can be used as output values in an experiment study in a selection dialog. Hold down the Ctrl key and select one or several of these attributes. When you click OK or Apply, the ExperimentManager executes a plausibility check of the output values.

Defining the output values

In addition, you can specify a Plant Simulation TableFile as the output value. The ExperimentManager determines mean values, standard deviations, and the minimum and the maximum for all numeric values within this table. This enables you to represent result data according to your ideas.
