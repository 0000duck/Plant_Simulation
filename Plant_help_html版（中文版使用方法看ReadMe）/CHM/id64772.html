Tab Statistics
2D Reference Help > Material Flow Objects > Turntable > Dialog Window of the Turntable > Tab Statistics

Statistics is described under the Tab Statistics.

In addition, the Turntable collects these statistical values:

Rotation empty

Shows the portion of the statistics collection period during which the object was rotating Empty, i.e., without moving a part.

StatRotationEmptyPortion

Rotation loaded

Shows the portion of the statistics collection period during which the object was rotating while moving a part.

StatRotationLoadedPortion

To view the Rotation Time in the Statistics Report, click the object in the Frame, and press F6 (Show Statistics Report), or click Show Statistics Report on the Home ribbon tab.

You can also click the object in the Frame with the right mouse button and select Show Statistics Report on the context menu.
