numNodes
2D Reference Help > The Plant Simulation Program Window > Toolbox > Attributes and Methods of the Folder in the Class Library > numNodes

Syntax:

 
Path.numNodes → integer

The method numNodes returns the number of objects in the Folder designated by Path. Each additional folder you insert into the selected folder counts as a single object. Plant Simulation does not count the objects contained within the folders.

Return Value

The return value has the data type integer.

Example:

 
print .Models.MyPlant.numNodes
