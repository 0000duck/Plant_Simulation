_3D.MaterialActive
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for the Appearance of the MUs in 3D > _3D.MaterialActive

Syntax:

 
MU-path._3D.MaterialActive

The attribute _3D.MaterialActive sets if the color of the auto graphics overrides the color of the vector graphics of the respective 2D object (true). If you enter false, the MU uses the color of the vector graphics of the respective 2D object.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
.MUs.Part._3D.MaterialActive := true
