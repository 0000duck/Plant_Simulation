MidLine
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Curved Objects > MidLine

Syntax:

 
Path.MidLine

The attribute MidLine sets the style for the line in the middle of the curved object designated by Path. The icons of the MUs move on this line across your tracks or conveyors.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

1

stands for no line.

2

stands for a solid line .

3

stands for a dashed line .

4

stands for a dotted line .

5

stands for a dash dot line .

6

stands for a dash dot dot line .

Example:

 
MyTrack.MidLine := 3
