Work in Process (WIP)
Add-Ins Reference Help > Object Libraries > Value Stream Mapping > Value Stream Mapping Tools > Work in Process (WIP)

You can use the object WIP to analyze the current number of products in Inventories and Supermarkets. Insert the object WIP into your model and double-click its icon.

This opens a table which shows all defined products as the column index of the table. The rows show the material flow objects from your model. The cells show how many products are stored in which object.

The table will not be updated during the simulation run. To update the values, double-click the object again.
