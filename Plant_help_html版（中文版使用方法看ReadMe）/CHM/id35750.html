Enter a detailed description appending the short note you entered into the Text box on the Tab Display.

The formatting properties only apply when you activate .

  • Apply formatting properties to the selected text with the buttons of the Formatting Toolbar or the commands of the Context Menu for Formatting the Comment.

  • For each character format property you can:

    • Apply a typeface, a Font size, and a font color.

    • Apply Bold face, Underline and italicize text (Italic).

    • For each paragraph format property you may:

      • Set the alignment: Align Left, Center or Align Right.

      • Set an indentation with Bullets.

  • Or copy text you formatted in a word processing application that works with rich text format (.rtf), such as the MS Windows WordPad or MS Word, to the clipboard and paste that text, including all of its formatting into the text box.

  • Or enter text into the text box, copy it back to the word processing application, apply formatting and paste it back to the text box. Press Ctrl+A to select all of the text into the text box, Ctrl+C to copy the text, and Ctrl+V to paste it.

To open a window that only shows the comment you entered, without formatting and editing options, click the Comment object with the right mouse button and select Open Comment Window on the context menu.
