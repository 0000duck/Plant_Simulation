Add-Ins Reference Help > ActiveX > Dialog Window of the Object ActiveX > Name

Enter a name of your choice for the object ActiveX or accept the name which Plant Simulation suggests.

You can enter a combination of letters, digits, and underscore (_) characters, for example MyActiveX, MyActiveX1, My_ActiveX_1.The name of an object cannot start with a digit, i.e., you cannot enter 1ActiveX.
