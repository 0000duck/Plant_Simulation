Functions for Converting Physical Data Types with Units into Data Types without Units
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Physical Data Types with Units into Data Types without Units > Functions for Converting Physical Data Types with Units into Data Types without Units

SimTalk provides the functions below for converting physical data types with units into data types without units. This is especially handy when you want to assign incompatible physical data types, which would normally output a warning to the Console.

acceleration_to_num

time_to_num

length_to_num

weight_to_num

speed_to_num

Example:

 
var w: weight
-- assign a time to a weight without showing a warning
w := time_to_num(EventController.simTime)
