setDataTypeDefault
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > setDataTypeDefault

Syntax:

 
Path.setDataTypeDefault(Column:any, ...)

The method setDataTypeDefault resets the data type of the designated column(s) of the TableFile designated by Path to the default data type of the table.

Parameter

The parameters Column of data type any designate the columns.

Example:

 
-- resets the data type of the columns 3 to 4, and of column 10 to 
-- the standard data type of the table
MyTableFile.setDataTypeDefault({3,*}..{4,*},10)
