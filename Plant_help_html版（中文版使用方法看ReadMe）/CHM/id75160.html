closeImg
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Animating All Objects > closeImg

Syntax:

 
Path.closeImg(ApplyChanges:boolean) → boolean

The method closeImg closes the Icon Editor of the object designated by Path.

Parameter

The parameter ApplyChanges of data type boolean determines if Plant Simulation accepts your changes (true) or discards them (false).

Return Value

The return value has the data type boolean.

Example:

 
ParallelStation.closeImg(false) -- discards any changes
MyGenerator.closeImg(true) -- applies your changes
