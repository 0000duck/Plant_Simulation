findAttr
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Indirectly Accessing Lists and Tables > findAttr

Syntax:

 
Path.findAttr([Range:listrange,] AttributeName:string, Value:any) → boolean

The method findAttr searches for an object, which contains the attribute with the passed name and passed attribute value, in a List/Table.

The method findAttr only searches in columns of data type object, i.e, it ignores columns with other data types. The method finds built-in attributes as well as user-defined attributes. If you just want to search for an object, which has this attribute, independent of the attribute value, then enter VOID as the value to be searched for.

The method starts its search at the position of the internal pointer of the List/Table. When the method finds such an object, then the method sets the internal pointer to the respective cell and returns true. When the method does not find such an object starting at the position of the internal pointer, it returns false.

Parameters

  • The optional parameter Range of data type listrange designates the range which you would like to search. If you do not pass this parameter, Plant Simulation searches the entire Table. You can also specify several list ranges, all of which the method then searches.

  • The parameter AttributeName of data type string designates the name of the attribute to be searched for.

  • The parameter Value of data type any designates the value of the attribute for which you are searching. When Plant Simulation finds this value, it terminates the search. When you enter VOID as value, the search ends when an attribute with the name of the parameter string is found. The method sets the internal pointer to the entry of the object and returns true. When the value does not match, the method continues searching. When it does not find a matching entry, it returns false.

    The search for attribute names and attribute values is not case sensitive.

Return Value

The return value has the data type boolean.

Example:

 
var wanted: object,
var row,column: integer 
MyCardFile.Cursor := 1
if MyCardFile.findAttr("color", "red") 
   wanted := MyCardFile.read(MyCardFile.Cursor)
end
table.CursorX := 1 table.CursorY := 1
if table.findAttr({1,1}..{4,4},"order",VOID) 
   column := table.CursorX 
   row := table.CursorY print table[column,row].order
end
