StatHomeDrivingPortion
2D Reference Help > Mobile Objects > Transporter > Read-Only Attributes of the Transporter > StatHomeDrivingPortion

Syntax:

 
MU-path.StatHomeDrivingPortion

The read-only attribute StatHomeDrivingPortion returns the portion of its lifetime during which the Transporter designated by MU-path was returning to its base.

Return Value

The return value has the data type real.

Example:

 
print MyTrack.Transporter:1.StatHomeDrivingPortion
