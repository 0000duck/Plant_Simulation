Model a Cross-sliding Car with an Application-specific Strategy
Step-by-Step Help > Modeling in 2D > Modeling Transport Systems > Modeling a Transport System with Passive Objects of Type Track > Model with the Library CrossSlidingCar > Model a Cross-sliding Car with an Application-specific Strategy

This sample model uses a similar layout as the model shown in Model a Cross-sliding Car that Distributes and Brings Together Parts. The main difference is that the CrossSlidingCar CSC1 is controlled by an application-specific strategy.

This strategy, which we programmed in the method myStrategy, ensures that the cross-sliding car CSC1 first unloads the arriving parts from the Conveyor AA, then from the Conveyor BB and finally from the Conveyor CC. To do so, we use the internal order list of the cross-sliding car, which is shown below. The order list is a user-defined attribute of the object cross-sliding car.

It contains the order date and time, the source, i.e., the Conveyor at which the cross-sliding car picks up the part, the destination, i.e., the Conveyor st which it is to deliver the part, the priority of the order, which we do not use yet, and the part to be transported.

The method myStrategy dorts this table ascending according to column 2 ( source) and returns the row with the number 1. Therefore the first order on the table will be executed.

The source code looks like this:

-> integer
var tab: table
-- get the order list of the cross-sliding car
tab := CSC1.OrderList
if tab.YDim=1 
   return
else
   -- sort the order list for the source objects
   tab.sort(2, "up")
   -- now the order list is sorted so that AA, BB, CC use the first entry
   -- so we first serve conveyor AA, then conveyor BB, and finally CC
   return 1
end

Creating a 3D model from our 2D model allows us to visualize the flow of parts even better. To do so:

  • Click Activate 3D on the Window ribbon tab.

  • Click the tab Class Library, open the folder Models, and click our model CSC_Demo_3 with the right mouse button. Select Open in 3D Window.

  • This opens the model in a new 3D window.
