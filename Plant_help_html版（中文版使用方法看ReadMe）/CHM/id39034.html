Dwell time
2D Reference Help > Material Flow Objects > Buffer > Dialog Window of the Buffer > Tab Times > Dwell time

For the Buffer you can only enter a constant dwell time. For the PlaceBuffer you can only enter a constant processing time. Statistics counts the dwell time as waiting time and not as processing time. The dwell time thus is not prolonged by failures or pauses.

Each MU remains at least for the time in the Buffer, which you entered as the dwell time. For the buffer type > Stack an MU can only exit the Buffer, when all parts which entered the Buffer after the current part, have already exited the Buffer, i.e., when the MU is located at the very top of the stack.
