setBackgroundColorRow
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Rows > setBackgroundColorRow

Syntax:

 
Path.setBackgroundColorRow(Row:any [,Row:any,...], BackgroundColor:integer)

The method setBackgroundColorRow sets the background color of row(s) of the TimeSequence designated by Path.

Parameters

  • The parameter Row of data type any designates the row whose background color you want to set.

  • The optional parameters Row of data type any designate additional rows whose background color you want to set.

  • The parameter BackgroundColor of data typeinteger designates the background color. You can enter 1 for Black, 2 for Red, 3 for Green, 4 for Blue, 5 for Magenta, 6 for Yellow or 7 for Cyan.

You can set the RGB values of the color with the method makeRGBValue.

Example:

 
var rgb: integer
-- sets the background color of row 3
MyTimeSequence.setBackgroundColorRow(3,makeRGBValue(0,200,0))
-- sets the background color of rows 4 and 8
MyTimeSequence.setBackgroundColorRow(4,8,makeRGBValue(60,100,0))
print MyTimeSequence.getBackgroundColorRow(3)

Example:

 
MyTimeSequence.setBackgroundColorRow(1,1) 
-- sets the background color of row 1 to black
MyTimeSequence.setBackgroundColorRow(1,2,2) 
-- sets the background color of rows 1 and 2 to red
