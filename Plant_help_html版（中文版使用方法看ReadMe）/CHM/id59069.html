Import Bitmap File
2D Reference Help > The Plant Simulation Program Window > Icon Editor > Edit Ribbon Tab > Import > Import Bitmap File

The command Import Bitmap File opens the dialog Open. Select the graphics file that will be imported. Plant Simulation can open . png, .gif, .bmp, .dgn, .dwg, .dxf, and . ico files.

Before you are going to import a .dwg file into the Icon Editor, adjust the icon’s size according to the size of the .dwg file by clicking Set Icon Size on the Edit ribbon tab.
