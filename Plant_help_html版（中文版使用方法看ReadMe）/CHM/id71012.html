FluidSource
2D Reference Help > Fluid Objects > FluidSource > FluidSource

The FluidSource produces a single free-flowing material continuously. You will use the FluidDrain to remove the processed and mixed products from the installation.

To add the FluidSource to your simulation model, click Manage Class Library > Basic Objects > Fluids > FluidSource on the Home ribbon tab.
