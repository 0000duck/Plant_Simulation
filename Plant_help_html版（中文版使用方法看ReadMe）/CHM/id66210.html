shGetFolderPath
SimTalk Reference > General SimTalk Functions > Predefined Functions > Operating System Functions > shGetFolderPath

Syntax:

 
shGetFolderPath(Path:integer) → string

The function shGetFolderPath returns the path of a folder.

Parameter

The parameter Path of data type integer designates the path.

You can use this function if you need the temporary path of a user or if you need a path which all users can access.

Different versions of Microsoft Windows return different folder names and folder paths.

Return Value

The return value has the data type string.
