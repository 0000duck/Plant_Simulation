StatNumOut
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for Statistics of the Mobile Objects > StatNumOut

Syntax:

 
MU-path.StatNumOut

The read-only attribute StatNumOut returns the number of MU instances that exited the MUclass designated by MU-path.

Return Value

The return value has the data type integer.

Example:

 
print .MUs.Container:1.StatNumOut
