ExitLength
2D Reference Help > Material Flow Objects > AngularConverter > Attributes of the AngularConverter > ExitLength

Syntax:

 
Path.ExitLength

The attribute ExitLength sets the length of the second leg of the AngularConverter designated by Path. It covers the distance from the point at which the AngularConverter switches the conveying direction to the point at which the MU exits the object.

Data Type of the Value You Can Assign

You can assign a value of data type length.

Example:

 
MyAngularConverter.ExitLength := 2
