Range X
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Axes > Range X

Enter the first value of the range the Chart is to display scaled on the x-axis in the left text box.

Enter the last value of the range the Chart is to display scaled on the x-axis in the right text box. The default setting 0 … 0 means that the program shows the entire range of definition.

When you are only displaying part of the range, the Chart shows a horizontal scrollbar, allowing you to scroll through the entire range of definition.

When you select the Category > Histogram, the Range X directly defines the range of definition. Enter 0 in the right text box, so that the Chart dynamically adjusts the upper bound. Enter the size of the intervals, into which the Chart divides the frequency distribution, into the text box Step size.

When you select the Category > Plotter, the Range X defines the time span, which the Chart shows in the display. You can also define the Feed rate by which the Chart scrolls when it reaches the right hand side of the display. You can enter a number between 0.1 and 12. The number 0.1 stands for smooth scrolling, while 12 scrolls the entire page.

Use the asterisk * to designate open or semi-open ranges along the x-axis. You can also use it to designate dynamic upper limits for the Histogram. Compare these examples:

Enter

To make the program show on the x-axis

 
* … *

the values between the smallest and the greatest displayed value.

 
0 … *

the values between 0 and the greatest displayed value on the x-axis.

 
* … 7.5

the values between the smallest displayed value and 7.5 on the x-axis.

 
-20 … 20

the values between -20 and +20.

Note that negative values and 0 are useful for charts of category XY Graph.
