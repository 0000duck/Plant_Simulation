Width
2D Reference Help > Display and User Interface Objects > Button > Dialog Window of the Button > Tab Attributes > Width

Enter the width in meters with which the object is displayed in the Frame. If you enter long text, you have to adjust the width so that the text fits into the object boundaries without being cut off. If you check Use icon , the object is shown with the width of the icon.

To change the width, you can also hold down Ctrl+Shift and drag the left or the right side of the icon.
