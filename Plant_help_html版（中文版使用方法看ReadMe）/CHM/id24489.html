getPasswordMasking
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > getPasswordMasking

Syntax:

 
Path.getPasswordMasking(DialogItem:string) → boolean

The method getPasswordMasking returns the state of the check box Password , designated by the parameter of data type string, for a New Edit Text Box. When it is selected (true) the password the user enters into the text box is masked. When it is cleared (false) the password will be shown as clear text.

Parameter

The parameter DialogItem of data type string designates the name of the text box.

Return Value

The return value has the data type boolean.

Example:

 
print MyDialog.getPasswordMasking("MyEditTextBox")
