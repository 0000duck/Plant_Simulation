Set the Main Directions
Step-by-Step Help > Modeling in 3D > Controlling Your View > Set the Main Directions

Independent of the active camera setting, you can adjust the view to any main direction of the view or of an object at any time. When you selected an object, 3D zooms out this object and adapts the direction of the camera to the selected main direction of the coordinate system of the object.
