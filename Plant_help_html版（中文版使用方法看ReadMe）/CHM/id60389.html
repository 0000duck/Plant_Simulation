delAniPoints
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Animating All Objects > delAniPoints

Syntax:

 
Path.delAniPoints → boolean

A 3D Only model does not provide the method delAniPoints.

The method delAniPoints deletes all animation points in the active icon of the class of the object designated by Path. You might have to change the current icon of the class before.

Return Value

The return value has the data type boolean.

Example:

 
.example.delAniPoints
