Attach Camera
3D Reference Help > 3D Reference > View Ribbon Tab > Attach Camera

The command Attach Camera attaches the camera to the object you selected, so that you can view the scene "through the lens" of that object. Note that the camera moves along with the object during a simulation run. 3D represents the attached camera with an object. Once you have attached a camera, you can toggle the object camera and the main camera with:

  • The command Toggle Camera.

  • The F4 key.

To attach a camera to an object:

  • Select a single object.

  • Click Attach Camera on the View ribbon tab.

By default the object camera faces on the positive x-axis using the positive z-axis as Up direction.

As soon as the object with the camera attached moves, the camera moves along with it automatically.

Note that you cannot change the view as long as the view of the object camera is active. To return to interactive navigation mode, you have to remove the camera by clicking Detach Camera
