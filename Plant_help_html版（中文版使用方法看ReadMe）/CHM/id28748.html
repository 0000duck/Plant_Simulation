Default angle
2D Reference Help > Material Flow Objects > PickAndPlace Robot > Dialog Window of the Pick-and-Place Robot > Tab Attributes > Default angle

Enter the angle in degrees to which the pick-and-place robot rotates when you selected Go to default position. If you would like the pick-and-place robot to rotate to the side on which the start position is located, add 180° to this angle. You can enter a value between 0° and 360°.

The default angle is also used as the start position after a Reset, no matter if Go to default position is activated or not.
