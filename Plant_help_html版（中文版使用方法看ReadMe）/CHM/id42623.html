TransitionTimeStandbyToOperational
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for the Energy Simulation of the Material Flow Objects > TransitionTimeStandbyToOperational

Syntax:

 
Path.TransitionTimeStandbyToOperational

The attribute TransitionTimeStandbyToOperational sets the transition time which passes when the object designated by Path changes its energy state from standby to operational.

Data Type of the Value You Can Assign

You can assign a value of data type real or a formula.

Example:

 
Station.TransitionTimeStandbyToOperational := 4:00
