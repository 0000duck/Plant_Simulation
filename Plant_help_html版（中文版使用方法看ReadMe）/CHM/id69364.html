Beta
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Attributes of the Distributions > Beta

Syntax:

 
Attribute_path.Beta

The attribute Beta sets the mean time in seconds between two events. It is a positive number.

Data Type of the Value You Can Assign

You can assign a value of data type time.

You can assign the following attributes/parameters to these distributions:

Distribution

Set of parameters

Range of values for the parameters

Gamma

Alpha, Beta, LowerBound, UpperBound

Alpha > 0, Beta > 0, Upper Bound > Lower Bound

Loglogistic Distribution

Stream, Alpha, Beta, LowerBound, UpperBound

Alpha > 0, Beta > 0, Upper Bound > Lower Bound

Negative exponential

Beta, LowerBound, UpperBound

Beta > 0, Upper Bound > Lower Bound

Weibull

Alpha, Beta, LowerBound, UpperBound

Alpha > 0, Beta > 0, Upper Bound > Lower Bound
