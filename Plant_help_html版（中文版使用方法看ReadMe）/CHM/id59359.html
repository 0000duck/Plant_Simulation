createImage
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > createImage

Syntax:

 
Path.createImage(Name:string, X:integer, Y:integer [,Parent:string]) → boolean

The method createImage creates a New Image in the Dialog object designated by Path.

Parameters

  • The parameter Name of data type string designates the Name of the image.

  • The parameter X of data type integer designates the position of the image on the X-axis in the Dialog.

  • The parameter Y of data type integer designates the position of the image on the Y-axis in the Dialog.

  • The optional parameter Parent of data type string designates the parent of the image, for example a New Group Box or a New Tab Page.

Return Value

The return value has the data type boolean.

Example:

 
MyTabbedDialog.createImage("ImageItem", 40, 1, "MyTab4")
MyTabbedDialog.setIcon("ImageItem", "CompanyLogo")
