Forecasting
2D Reference Help > Tools > Artificial Neural Network > Forecasting

When optimizing the model, you will often look for a combination of input values for which an output value is located as close as possible to a given value. You can select an output value under the group Forecast on the tab Forecast for which you want to compute a forecast. The object shows the observation interval of the training data after you selected the output value.

Values located within this interval will produce realistic forecasts. Then, enter the desired output value.

Optimizing the input values for a desired output value

For smaller simulation studies you can evaluate the complete table of all combinations of the input values. Within this table you can compare the simulation results and results of the neural network. The columns of the output values are marked with the suffixes "_S" (training data, which may be the results of a simulation) and "_NN" (for neural network). The columns of the den simulation results may contain empty cells, when you reduced the size of the input values in the dialog Configuration . The artificial neural network then computes approximate values for these empty cells.

Table containing all input and output values as well as the results of the neural network

You can open this table and the corresponding visualization of the data under the group Comparison on the tab Forecast. The table is sorted according to the selected output value. The range around the desired output value, which you enter into the text box Range for cost minimization, is marked light green. The row with the most cost-efficient input values is marked dark green.

In addition, all rows, in which the desired output value appears exactly as expected, are marked orange. If this value is not arrived at, the NeuralNet marks the row, which comes closest to the desired output value.

Optimization on the table of all combinations of the input values

When a very large number of combinations of the input values exists, the program cannot create such a table in a short amount of time. In addition it is hard to manually evaluate very large tables. In these cases the most cost-efficient combinations of the input values are calculated by an optimization algorithm. To do so, click the button Start under the group Forecast. This opens the table with the input values, which approximate the desired output value as closely as possible and which cause the least amount of costs. Although the optimization algorithm does not search within a limited range, it can only detect a local minimum of the costs. To get various solutions for a certain problem, click Start repeatedly.
