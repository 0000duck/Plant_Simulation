Tab Controls
2D Reference Help > Mobile Objects > Transporter > Dialog Window of the Transporter > Tab Controls > Tab Controls

Click and select a Method object, into which you entered the source code of the respective control, in the dialog Select Object.

You can select or enter the Driving control, the Routing control, the Destination control, the Collision control, the Distance control, and the Speed control.

You can also select the Shift calendar, which controls the shifts during which the object works.

To open the dialog of the object, whose name you entered into the text box, press the F2key.
