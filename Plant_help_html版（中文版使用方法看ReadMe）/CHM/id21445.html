New Group Box
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New Group Box

A group box graphically groups a set of dialog items or controls by drawing a frame around it.

  • Enter the Name of the group box.

  • Enter the Caption the group box displays.

  • Enter its X and Y position in the Dialog.

  • Enter its Width and its Height.

  • Select if you want to Enable it or not.

  • To insert the dialog item into your dialog, click OK.
