Starting Plant Simulation with Different Kinds of Licenses
Setting-Up and Starting > Licensing Plant Simulation > Activating Licenses > Starting Plant Simulation with Different Kinds of Licenses

If you are using Plant Simulation with a license other than Professional, you can use the start parameter -L parameter> or /L parameter> to start Plant Simulation with one of these licenses.

For this license

Enter this start parameter

Application

-L Application or /L Application

Educational

-L Educational or /L Educational

Professional

-L Professional (or no parameter) or /L Professional

Research

-L Research or /L Research

Runtime

-L Runtime or /L Runtime

Simulation

-L Simulation or /L Simulation

Standard

-L Standard or /L Standard

Student

-L Student or /L Student

Viewer

-L Viewer or /L Viewer

You might, for example, enter:

C:\Program Files\Siemens\Tecnomatix Plant Simulation 14.1\PlantSimulation14_1.exe /-L Standard

into the text box Target of a shortcut.

Or you might enter the information in the screenshots below to start Plant Simulation English or German.

Naturally, you can also select the License type in Plant Simulation itself under File > Preferences > License.

For licenses, which limit the number of objects you can use, the root object (Basis) in the Class Library shows how many objects are used and how many are allowed. For a Student license it looks like this:

SimTalk

licenseName

numOfLimitedObjects
