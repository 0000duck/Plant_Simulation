Edit User-defined Attributes
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit User-defined Attributes

The command Edit User-defined Attributes opens the dialog User-defined Attributes. Here you can define your own attributes as described under the Tab User-defined.

In a Method, which you defined as a user-defined attribute for a Frame, the keyword current refers to Frame itself.

This table lists, where the relative path for a user-defined attribute you define for a Frame starts.

Object

User-defined attribute defined for a Frame

Object variable

Within that Frame.

Method (current)

Within that Frame.

Table with column of data type object

In the Frame into which you inserted that Frame.
