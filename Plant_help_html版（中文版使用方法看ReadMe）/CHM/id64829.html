To create a new observer, click New . This only works if the attribute or the method of the object is observable.

  • Select the observable attribute or the observable method, whose value you want to watch.

  • To check whichattributes and methods of an object are observable, select the object and press F8. observableattributes and methods are marked with an asterisk (*) in the column Watchable in the dialog Show Attributes and Methods. SimTalk:attributeWatchable.

  • Select the Method , which is going to be executed, when the value of the Attribute or of the Method changes. The name of the attribute/method and the previous value of the attribute/method are passed to the executed Method as parameters.

    To employ a user-defined attribute of data type method as the Method, click the right mouse button in the field Method and select the command Create Control. Then, the program automatically formats the Method correctly.

  • Click OK to add the observer.

    The list field shows the name of the Attribute or the Method, the name of the Method that is going to be executed, and if the object inherits the observer from its class.

    When you add the observer to the instance of the object, which you inserted into a Frame, it shows an asterisk * in the column Created here.

To execute several Methods for the same attribute or method, click New again, select the same attribute and add an additional method.
