FastAccessRowIndex
2D Reference Help > Information Flow Objects > TableFile > Attributes of the TableFile > FastAccessRowIndex

Syntax:

 
Path.FastAccessRowIndex

The attribute FastAccessRowIndex sets if Plant Simulation uses the faster access to the user-defined row index (true) or not (false).

The more columns and rows your table has, the more you benefit from the improved access speed.

If the table contains a large number of entries, you might want to activate Fast index access.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTableFile.FastRowColumnIndex := true
