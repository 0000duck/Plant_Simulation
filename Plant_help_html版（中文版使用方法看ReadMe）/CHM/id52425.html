Start at successor 1
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Exit > Strategy > Start at successor 1

To always move the MU to the material flow object’s successor with the number 1, select Start at successor 1 from the drop-down list.

  • When you select Blocking , the object moves the MU to the first successor in the sequence of successors.

  • When you clear Blocking , the object moves the MUto the first successor in the sequence of successors, which can receive the MU.

Note that a previous search for a successor does not affect the strategy Start at successor 1.
