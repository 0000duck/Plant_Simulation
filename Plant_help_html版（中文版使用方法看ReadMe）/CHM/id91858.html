Support DDE hotlinks
2D Reference Help > Information Flow Objects > Variable > Dialog Window of the Variable > Tab Communication > Support DDE hotlinks

To activate monitoring the global variable Variable via DDE, select Support DDE hotlinks. To deactivate it, clear the check box.

You might, for example, write the value of the Plant Simulation Variable to Microsoft Excel, which in turn could display the value in a dynamic chart, etc.

Note that--for performance reasons--the feature is deactivated by default.
