Width
2D Reference Help > Display and User Interface Objects > Drop-Down List > Dialog Window of the Drop-Down List > Tab Attributes > Width

Enter the width in meters with which the Drop-Down List is shown in the Frame. If you enter long names for the Items, you have to adjust the width so that the names of the items in the drop-down list are not cut off. If you check Use icon , the drop-down list is shown with the width of the icon of the drop-down list. To change the width, hold down Ctrl+Shift and drag the left or right side of the icon.
