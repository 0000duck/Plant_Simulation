getCommonFormatData
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for the Format of Lists and Tables > getCommonFormatData

Syntax:

 
Path.getCommonFormatData(TargetList:list)
Path.getCommonFormatData(TargetTable:table)

The method getCommonFormatData gets format data of the column of data type List/Table for which you activated common format and writes it to the target table.

The method applies to the objects StackFile, QueueFile, and CardFile.

Parameter

The parameter TargetTable of data type table designates the target table.

The parameter TargetList of data type list designates the target list.

Example:

 
SourceStack.getCommonFormatData(MyStack)
SourceStack.getCommonFormatData(MyTable)
