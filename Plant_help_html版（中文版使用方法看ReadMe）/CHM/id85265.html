Show External Graphics
3D Reference Help > 3D Reference > View Ribbon Tab > Show External Graphics

The command Show External Graphics shows or hides all graphic groups of the object in the active 3D window that are not marked as Internal. Graphics are objects without parameters relevant to the simulation. Normally graphics are part of a object providing it with graphic data. Displaying this graphic enables you to edit the graphic data of the current root of the scene. You can then edit, extend or replace an existing graphic to transform it or you can create, import or copy another graphic.

When you create and insert a 3D shape for the first time, Plant Simulation shows the graphic of the Frame in the center of the 3D scene window and attaches the shape you defined to the mouse pointer. The reason for this is that 3D automatically activates Show External Graphics to show the graphics in the scene window. If this would not be the case, the shape that we inserted, which is a graphic after all, would not be shown as well. To show the shape, just delete the graphic of the Frame.

When you open a Frame or a folder in a 3D window, Show External Graphics is deactivated by default. When you open any other object, it is activated by default.

3D does not save if Show External Graphics is activated or deactivated for the individual Frames. When you save your model, Plant Simulation does not save if Show External Graphics is activated or deactivated for the individual 3D window, but returns to the default setting.

Showing external graphic groups is useful to edit them:

You can transform and replace any displayed graphic group by selecting, moving, rotating, scaling, or deleting it in the 3D window. In addition, you can add new created 3D shapes or imported graphics into any graphic group of the opened object by inserting into the 3D scene.

When you start to insert a new created shape into an external graphic group, 3D automatically activates Show External Graphics until you click the destination position so that you can align the input graphic at the mouse pointer with all other graphics of the destination graphic group.

When you imported or dropped graphics to an external graphic group, the Plant Simulation also automatically activates Show External Graphics so that the inserted graphics do not disappear after inserting them.
