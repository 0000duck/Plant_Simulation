Edit 3D Properties
3D Reference Help > 3D Reference > Context Menus in 3D > Context Menu of the Selected Graphic in the Scene Window > Edit 3D Properties

The context menu command Edit 3D Properties opens the dialog Edit 3D Properties. Depending on the type of graphic and the context it shows the Tab Transformation, the Tab Material, and the Tab Graphic Settings. A number of shapes show their settings on shape-specific tabs.
