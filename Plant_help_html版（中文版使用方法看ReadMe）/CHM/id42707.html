switch-case-end
2D Reference Help > Information Flow Objects > Method > The Method Window > Edit Ribbon Tab > Insert Control Structure > switch-case-end

The command switch-case-end inserts a switch-statement at the position of the cursor into the source code. Type in the condition of the switch-statement. Press Tab to jump to the location within the source code, at which you can enter the statement that is to be executed, when the condition becomes true.

You can also highlight several statements before you select the command switch-case-else. Then Plant Simulation inserts a switch-statement around these statements in the source code.

When you drag the mouse over the sections marked in red, Plant Simulation shows a Tooltip.
