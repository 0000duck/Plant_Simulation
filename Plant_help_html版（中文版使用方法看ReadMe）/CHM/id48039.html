Dialog Window of the Oracle11g Interface
Add-Ins Reference Help > Database Interfaces > Oracle11g > Dialog Window of the Oracle11g Interface > Dialog Window of the Oracle11g Interface

Double-click the icon of the object Oracle11g, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Information Flow in the Toolbox. The text boxes Name and Label are described under Dialog Items of the Objects.
