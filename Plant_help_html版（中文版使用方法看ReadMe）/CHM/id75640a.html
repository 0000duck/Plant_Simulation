Display a FileLink
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Select an Object or an Object Attribute to be Displayed > Display a FileLink

To display a bitmap file or an SVG file, which you imported with a FileLink, on the HTML page, select the bitmap file or the SVG file in the dialog of the FileLink. Click OK.

Click on the toolbar .

Enter the caption which the HtmlReport shows below the FileLink image file.

Enter the Image size in percent to which the HtmlReport scales the screenshot of the FileLink. Instead you can also enter the Image width and the image height. You can also enter an asterisk (*) for either the width or the height to ensure that the image will not be distorted. The relative size takes precedence over the absolute size.
