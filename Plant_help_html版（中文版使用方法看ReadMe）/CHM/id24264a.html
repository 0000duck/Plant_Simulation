BlockingAngle
2D Reference Help > Material Flow Objects > PickAndPlace Robot > Attributes of the Pick-and-Place Robot > BlockingAngle

Syntax:

 
Path.BlockingAngle

The attribute BlockingAngle sets the Blocking angle in degrees which the pick-and-place robot cannot cross. This prevents the robot from taking the shortest way. The default setting -1 means that no blocking angle applies and that the robot thus takes the shortest way. You can enter an angle between 0 and 360 degrees.

The blocking angle only affects the animation, the simulation is controlled by the rotation times which you defined in the Times Table.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
MyPickAndPlace.BlockingAngle := 5
