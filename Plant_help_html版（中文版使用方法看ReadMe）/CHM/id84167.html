To get acquainted with Plant Simulation you also can move along at your own pace by working through the lessons of the Tutorial. It shows how to build a simulation model for a testing line for TVs and VCRs.

To open the tutorial:

  • Start Plant Simulation.

  • Click Tutorial on the Start Page.

  • Open one of the model files for the tutorial in the Plant Simulation installation folder > Plant Simulation > Tutorial, compare the example below.
