Anonymous Identifiers
SimTalk Reference > General SimTalk Functions > Names > Anonymous Identifiers > Anonymous Identifiers

Normally, the name of a control you programmed in a Method does not change in your model. The path to the Method changes from model to model, though. If a control requires the current path, you can query it by using anonymous identifiers. Anonymous identifiers, such as @, basis, current, ?, root, RootFolder, and self make using a Method more flexible and independent of its context. You can insert it into different models without having to modify its source code.
