MaxXDim
2D Reference Help > Information Flow Objects > TableFile > Attributes of the TableFile > MaxXDim

Syntax:

 
Path.MaxXDim

The attribute MaxXDim sets the maximum number of columns. Enter -1 for an infinite size.

This attribute applies to the TableFile and the TimeSequence.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyTableFile.MaxXDim := 10 -- 10 columns
tableFile1.MaxXDim := -1 -- unlimited
