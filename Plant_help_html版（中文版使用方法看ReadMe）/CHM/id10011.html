2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Appearance of the Length-oriented Objects > Segments

Click Segments to show the values of the individual segments of the inserted length-oriented objects. You can edit some of the values while you can only view others.

The settings of the segments table are described in detail under Segments in the 2D part of the Online Help.
