Waiting
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Statistics of the Material Flow Objects > Resource Statistics > Waiting

A material flow resource is waiting when:

  • It is not Paused.

  • It is not Failed.

  • It is not Stopped.

  • It is not fully occupied.

  • An occupied station is not working, for example, when it is waiting for an Exporter.

  • It is not Setting-Up.

    The read-only attribute ResWaiting returns the state waiting of the resource.

    The LED of a station that is waiting for an Exporter/Worker is orange .

    The Statistics Report shows the Waiting Time below Resource Statistics of Stationary Resources.
