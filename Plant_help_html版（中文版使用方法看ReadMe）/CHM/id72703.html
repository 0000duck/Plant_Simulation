copyToTableColumn
SimTalk Reference > General SimTalk Functions > Data Types > Array > Methods and Attributes of Arrays > copyToTableColumn

Syntax:

 
copyToTableColumn(TargetTable:table, Column:integer)

The method copyToTableColumn copies the contents one-dimensional array to the specified column of a table.

Parameters

  • The parameter TargetTable of data type table designates the table to which you would like to copy data.

  • The parameter Column of data type integer designates the column into which you would like to paste data.

Example:

 
var array: string[2] := ["hello", "world"]
array.copyToTableColumn(TableFile, 3)
