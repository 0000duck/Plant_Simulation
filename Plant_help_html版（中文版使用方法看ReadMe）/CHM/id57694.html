addObject
2D Reference Help > Resource Objects > LockoutZone > Methods of the LockoutZone > addObject

Syntax:

 
Path.addObject(NameOfObject:path) → boolean

The method addObject assigns a single object to the LockoutZone designated by Path.

Parameter

The parameter NameOfObject of data type path designates the name of the object you want to add.

Return Value

The return value has the data type boolean.

Example:

 
MyLockoutZone.addObject(MyParallelStation)
