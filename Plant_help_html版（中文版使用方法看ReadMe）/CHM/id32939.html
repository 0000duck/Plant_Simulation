delete
SimTalk Reference > General SimTalk Functions > Data Types > Array > Methods and Attributes of Arrays > delete

Syntax:

 
delete([Index:integer])

The method delete without a parameter deletes the contents of the array.

Parameter

The parameter Index of data type integer deletes the item located at this index position.

Example:

 
a := [current,pi,e]
a.delete(2)
print a.dim
