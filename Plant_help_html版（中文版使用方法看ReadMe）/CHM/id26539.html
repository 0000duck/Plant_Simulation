Tab Statistics
Add-Ins Reference Help > Object Libraries > Value Stream Mapping > Value Stream Mapping Objects > Customer > Dialog Window of the Customer > Tab Statistics > Tab Statistics

On the tab Statistics you can view a number of statistics values of the created orders. To do so, click Order protocol, select Use product groups, click Show Delivery Delay, enter the Update interval, Orders Within Threshold, enter the Order threshold, Open Statistics Table, and Chart of Lifetime.
