Accumulating
2D Reference Help > Mobile Objects > Transporter > Dialog Window of the Transporter > Tab Load Bay > Accumulating

When you select the loading space Type > Line, Plant Simulation enables the check box Accumulating.

To make the MUs accumulate on the loading space of the Transporter, select this. This allows the MUs to move front to end to each other, when the exit of the loading space is blocked.

To make the parts retain their distance to each other, i.e., to make all succeeding parts stop moving when the preceding part cannot exit the loading space, clear the check box. When the loading space is blocked, it stops, i.e., it reduces its Speedto 0.
