Corner brightness
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Background > Corner brightness

To define a color gradient, i.e., the brightness of the background color, first select the Base color. Then drag the slider for each of the four corners of the Frame window to make the respective corner darker or brighter. The top left slider sets the settings of the top left corner of the Frame window, etc.
