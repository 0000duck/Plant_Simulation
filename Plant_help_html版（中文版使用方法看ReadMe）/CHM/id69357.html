standardDeviationAttr
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Indirectly Accessing Lists and Tables > standardDeviationAttr

Syntax:

 
Path.standardDeviationAttr(Range:listrange, ... AttributeName:string) → real

The method standardDeviationAttr returns the standard deviation of the designated attribute of all objects in the designated range of the list.

The method applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Parameters

  • The parameters Range of data type listrange designate the range of the list.

  • The parameter AttributeName of data type string designates the name of the attribute.

Return Value

The return value has the data type real.

Example:

 
MyCardFile.Cursor := 1
print MyCardFile.standardDeviationAttr({*},"ProcTime")
