Width
2D Reference Help > Material Flow Objects > Converter > Attributes of the Converter > Width

Syntax:

 
Path.Width

The attribute Width sets the Width of the Converter.

When the Converter is to convey parts perpendicular, it has to provide enough room for the length and the width of the MU to completely fit on the Converter, when the booking point is located on the change over point in the center of the Converter.

Data Type of the Value You Can Assign

You can assign a value of data type length.

Example:

 
MyConverter.Width := 2
