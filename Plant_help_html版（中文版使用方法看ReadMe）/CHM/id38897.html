memUsage
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > General Methods of All Objects > memUsage

Syntax:

 
Path.memUsage → integer

The method memUsage computes how much memory the object designated by Path uses.

To show how much memory the individual objects in a Frame consume, you can also select the menu command Show Memory Usage on the context menu in the Structure window in the windows Show Inheritance and Show Structure.

Return Value

The return value has the data type integer.

Example:

 
print.MyStation.memUsage
