Height
2D Reference Help > Display and User Interface Objects > Drop-Down List > Dialog Window of the Drop-Down List > Tab Attributes > Height

Enter the height in meters with which the Drop-Down List is shown in the Frame. To change the height, hold down Ctrl+Shift and drag the top or the bottom of the icon.

When you enter a height of more than 30 pixels, Plant Simulation uses a larger font to display the label of the drop-down list.
