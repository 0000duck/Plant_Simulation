openColorSelectBox
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > openColorSelectBox

Syntax:

 
openColorSelectBox(Color:integer) → integer

The function openColorSelectBox opens the dialog Colors provided by Windows.

Parameter

The parameter Color of data type integer designates the preset color value. Typically this will be a color, which you define with the function makeRGBValue.

Return Value

The return value has the data type integer. This is the RGB color value of the color selected by the user.

Example:

 
colorVar := openColorSelectBox(colorVar)
openColorSelectBox(makeRGBValue(255,150,0)) -- current color is orange
