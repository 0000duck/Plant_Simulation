refillDialog
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Accessing Lists and Tables > refillDialog

Syntax:

 
Path.refillDialog

The method refillDialog fills an open dialog box of a list or a table with the current values. This is especially helpful, when you changed values via information flow and you want to show these values in the list or table, which you opened as a dialog box. When you did not open the table as a dialog box, but as a regular list window, Plant Simulation will always show the current values anyway. In this case the method recalculates any formulas contained in the list window.

Values you changed in the dialog box but did not apply will be discarded.

Example:

 
MyStackFile.refillDialog
MyQueueFile.refillDialog
MyCardFile.refillDialog
MyTableFile.refillDialog
