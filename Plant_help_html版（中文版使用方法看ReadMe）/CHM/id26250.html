Navigate Menu
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Navigate Menu > Navigate Menu

The Navigate menu provides these menu commands:

Open Location

Go to Class

Open Origin

Open 3D Location

Go to Origin

Edit 3D Properties

Open Class

Some of the commands are available only after you insert the object into a Frame, i.e., they are deactivated when you open the object in the Class Library!

The Context Menu of the Selected Object in the Frame and the Home Ribbon Tab also provide some of these commands.
