deleteMovables
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods for Accessing the Contents of the Material Flow Objects > deleteMovables

Syntax:

 
Path.deleteMovables

The method deleteMovables deletes all MUs which are located on the object designated by Path. If Path designates a Frame, the method deletes all MUs on all objects, which are located in the Frame or in sub-Frames.

Example:

 
ParallelStation.deleteMovables
