unhitchFront
2D Reference Help > Mobile Objects > Transporter > Methods of the Transporter > unhitchFront

Syntax:

 
MU-path-Transporter-rear.unhitchFront → boolean

The method unhitchFront unhitches the front of the active Transporter from the rear of the Transporter in front of it in the train.

The active Transporter can either be one of the trailers or the tractor or a Transporter which is not part of a train yet.

Return Value

The return value has the data type boolean. true tells that unhitching was successful, false that it failed.

Example:

 
.MUs.Transporter:4.unhitchFront
