VectorgraphicsColor
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes for Vector Graphics of the Mobile Objects > VectorgraphicsColor

Syntax:

 
MU-path.VectorgraphicsColor

The attribute VectorgraphicsColor sets the fill color of the vector graphic of the MU designated by MU-path. The fill color fills the interior of the built-in vector graphic with the color that you select.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
.MUs.MyPart.VectorgraphicsColor := 128 -- dark red
.MUs.MyPart.VectorgraphicsColor := makeRGBValue(200,200,100)
