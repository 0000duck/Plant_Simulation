Basic Mathematical Functions
SimTalk Reference > General SimTalk Functions > Predefined Functions > Mathematical Functions > Basic Mathematical Functions > Basic Mathematical Functions

The parameter for the basic mathematical functions has the data type real or integer.

Function

Result

abs(x)

absolute value of |x|

beta(x,y)

beta function B(x,y), 0 x,y

betai(x,y,z)

incomplete beta function Iz (x,y), 0< x,y 1, 0 u 1

ceil(x)

round up

exp(x)

exponential function ex The parameter is any number of data type real.

floor(x)

round down

gamma(x)

gamma function Γ (x), x > 0

log(x)

natural logarithm

log10(x)

logarithm to the base of 10

max(x,y)

maximum

min(x,y)

minimum

pow(x,y)

x yWhen the basis is 0, the exponent has to be greater than 0. When the exponent is not an integer, for example -0.5, the basis has to be greater than or equal to 0.

round(x)

rounds up or down to integers.

round(x,y)

rounds to the decimal place y.

sqrt(x)

square root. The parameter of data type real has to be equal to or greater than 0.

The square root function only accepts parameters equal to or greater than zero.

When using the functions exp, pow, gamma and the arithmetic operators in SimTalk, you have to make sure that the result is located within the range of numbers your computer can show. When the result is outside of the range that you computer can show, Plant Simulation will create an emergency copy of your simulation model and terminate. Use the gamma function to compute the factorial of a number n: n! = 1 • 2 • … • n = Γ (n+1)

Return Value

The return value has the data type real.
