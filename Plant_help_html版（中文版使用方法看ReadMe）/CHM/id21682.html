Attributes for Printing Lists and Tables
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Attributes for Printing Lists and Tables > Attributes for Printing Lists and Tables

Lists and tables provide the attributes GenerateColumnWidth, PrintColumnNumber, PrintDataType, PrintInternalLists, PrintRowNumber, RepeatColumnIndex, and RepeatRowIndex for printing their contents. In the dialog of the objects, you can select these settings by clicking Print >Print Setup on the List ribbon tab.

To view all of the methods and attributes of the list object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.

To set the value of an attribute of a list or table, you might, for example, type: 

MyTableFile.MaxXDim := -1

To get the value of an attribute a list or table, you might, for example, type: 

print MyTableFile.MaxXDim
