StatAvgLifeSpan
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for Statistics of the Mobile Objects > StatAvgLifeSpan

Syntax:

 
MU-path.StatAvgLifeSpan

The read-only attribute StatAvgLifeSpan returns the average life-span of MU instances, which were created and removed from the plant during the statistics collection period.

Return Value

The return value has the data type time.

Example:

 
print .MUs.Transporter.StatAvgLifeSpan
