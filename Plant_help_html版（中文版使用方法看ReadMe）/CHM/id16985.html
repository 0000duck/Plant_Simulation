The geometric distribution is a discrete distribution. The realizations are non-negative integer numbers. The parameter p defines the probability that an event occurs.

The parameters of the distribution are described under Attributes of the Distributions.

You can use it to determine how many parts are produced before the next reject.

The parameter p designates the probability of a reject. The value for p is located between 0 and 1, f(k) = p • (1-p)k is the probability that the (k+1)th part is a reject.

Figure 6-5. Probabilities of the geometric distribution with the parameter p=0.5

You can also create the geometric distribution with this SimTalk function: 

z_geom(Stream:integer,p:real[,LowerBound:real,UpperBound:real])

The parameter Stream (integer) designates the random number stream.

Parameter

The parameter p designates the probability of a reject. It has the data type real. The value for p is located between 0 and 1.
