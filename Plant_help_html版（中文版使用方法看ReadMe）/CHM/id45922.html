BwEntranceCtrl
2D Reference Help > Material Flow Objects > TwoLaneTrack > Attributes of the TwoLaneTrack > BwEntranceCtrl

Syntax:

 
Path.A.BwEntranceCtrl
Path.B.BwEntranceCtrl

The attribute BwEntranceCtrl designates a Method object, that is called, when a Transporter enters the designated lane of the TwoLaneTrack via its exit and/or when the Transporter moves towards its entrance after entering.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
MyTwoLaneTrack.B.BwEntranceCtrl := &MyBackwardEntranceMethod
