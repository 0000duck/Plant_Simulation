Template directory
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Preferences > Editor > Template directory

Click the button and navigate to the folder in which you store the templates for the source code of your methods. You can then select one of the templates in the Method object by clicking Select Template on the Edit ribbon tab and thus load it into the Method you are programming.

The text box shows the path to the folder in which you store the templates for the source code of your methods.

You can also accept the default folder Templates.
