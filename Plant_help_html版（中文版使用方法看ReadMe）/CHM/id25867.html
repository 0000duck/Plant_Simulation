Shift calendar
2D Reference Help > Resource Objects > Exporter > Dialog Window of the Exporter > Tab Controls > Shift calendar

Click and select the ShiftCalendar in the dialog Select Object. This ShiftCalendar object contains the data of the shifts in your installation and controls during which shifts the Exporter provides services.

Or you can select the ShiftCalendar in a Frame, drag it to the text box and drop it there.
