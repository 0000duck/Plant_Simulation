Showing a Display Panel
Add-Ins Reference Help > Object Libraries > Value Stream Mapping > Shared Properties of the Value Stream Mapping Objects > Showing a Display Panel

During the simulation run you can display a Display Panel below the value stream mapping objects in the Frame containing your simulation model. To define which values you want to show, click Display Panel on the tab Settings. Depending on the object type it opens the dialog which shows different settings which you can select.

Plant Simulation only creates or updates the Display Panel after the model was initialized.
