Exiting Tecnomatix Plant Simulation
Setting-Up and Starting > Exiting Tecnomatix Plant Simulation

To exit Plant Simulation:

  • Click Close in the title bar of the program window. Plant Simulation prompts you to save your simulation model.

  • Select the command File > Exit. Plant Simulation prompts you to save your simulation model.
