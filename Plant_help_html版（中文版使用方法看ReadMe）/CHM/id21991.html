and/or
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > and/or

Syntax:

 
Path.and(Operand:any)
Path.or(Operand:any)

You can use boolean operations, such as and and or, on a TimeSequence of data type boolean. The boolean operation may be executed with a single boolean parameter or another TimeSequence object. In the latter case, the sequence of values in the time column will be the union of the time columns of both time sequences involved in the operation.

Parameter

The parameter Operand of data type any designates the operand which will be applied.

Example:

 
failureA.and(failureB)
workerA.or(workerB)
