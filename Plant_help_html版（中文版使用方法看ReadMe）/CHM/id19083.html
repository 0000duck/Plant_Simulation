The storage places of the Store are organized in a net of coordinates whose dimension you can set in the X-dimension, Y-dimension, and the Z-dimension. You can access the various storage places using their coordinates.

In 3D you can edit the dimensions of the Store by clicking Show Manipulators on the Edit ribbon tab and selecting and dragging a manipulator.
