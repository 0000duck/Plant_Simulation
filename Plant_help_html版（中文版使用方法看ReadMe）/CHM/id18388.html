AvailableForMediation
2D Reference Help > Resource Objects > Worker > Read-Only Attributes of the Worker > AvailableForMediation

Syntax:

 
Path.AvailableForMediation

The read-only attribute AvailableForMediation returns if the Worker can be brokered (true) or not (false).

Data Type of the Value You Can Assign

The return value has the data type boolean.

The attribute returns true, when the Worker is not failed, not paused, does not carry parts, and when he is not brokered.

Example:

 
print .Resources.MyWorker:2.AvailableForMediation
