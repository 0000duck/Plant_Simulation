Common format
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Format > Tab Data Type > Common format

For the data types Table, List, Stack and Queue, Plant Simulation shows the check box Common format instead of the text box Format string and adds the Tab Contents.

To force all lists in the range displayed into the same format, select Common format. Then click the Tab Contents and create a template list here.

Lists you create after this will take all formatting properties you selected here, compare Creating Lists within Lists and Tables.
