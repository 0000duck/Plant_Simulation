VectorgraphicsBorderWidth
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes for Vector Graphics of the Mobile Objects > VectorgraphicsBorderWidth

Syntax:

 
MU-path.VectorgraphicsBorderWidth

The attribute VectorgraphicsBorderWidth sets the border width of the vector graphic of the MU designated by MU-path. Plant Simulation surrounds the fill color with the border width in the Border color that you select.

A border width of 2 looks like this , a border width of 4 looks like this .

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
.MUs.MyPart.VectorgraphicsBorderWidth := 2
