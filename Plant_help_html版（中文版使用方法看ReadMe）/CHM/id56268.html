include
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > include

Syntax:

 
Path.include(TimeSequence:object)

The method include takes all entries of the TimeSequence designated by the parameter of data type object and inserts them in the TimeSequence designated by Path. This may result in a single time sequence containing multiple entries with the same time value. If this happens, the order of the entries with identical time values, with respect to each other, is undefined.

The data types of the involved time sequences must be identical or compatible.

Parameter

The parameter TimeSequence of data type object designates the TimeSequence whose values will be included.

Example:

 
timeSequence1.include(timeSequence2)
