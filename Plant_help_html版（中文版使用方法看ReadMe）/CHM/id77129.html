Incremental Search
2D Reference Help > Information Flow Objects > Method > The Method Window > Edit Ribbon Tab > Incremental Search

The command Incremental Search or Ctrl+I finds the expression which you start typing. Plant Simulation highlights the character combination while you type in the text window.

  • If you mistype a word, you can delete the last character you typed by pressing the Backspace key. Then, type in the correct character.

  • Press the F3 key to find the next instance of the expression.

  • Press the Esc key to terminate Incremental Search.
