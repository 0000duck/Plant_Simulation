day
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Managing Date and Time > day

Syntax:

 
day(Date:date) → integer
day(DateTime:dateTime) → integer

The function day extracts the day from a dateTime value.

Parameter

The parameter Date or Datetime of data type date or dateTime designates the date or date and time respectively.

Return Value

The return value has the data type integer.

Example:

 
print day(str_to_date("24.12.99"))           -- 24
print day(str_to_dateTime("1.1.99 1:00:00")) -- 1
