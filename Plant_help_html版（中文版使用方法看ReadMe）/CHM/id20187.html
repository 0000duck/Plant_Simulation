Accumulating
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > Accumulating

Syntax:

 
MU-path.Accumulating

In case the first MU cannot exit the loading space of type Line, the attribute Accumulating sets if succeeding MUs may move up, so that the front of the succeeding MU touches the rear of the preceding MU (true) or if they will retain their distance to each other (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
.MUs.Transporter:1.LoadBayType := "Line"
.MUs.Transporter:1.Accumulating := true
