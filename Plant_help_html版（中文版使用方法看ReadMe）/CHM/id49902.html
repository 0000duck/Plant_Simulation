Show summary report
2D Reference Help > Material Flow Objects > EventController > Dialog Window of the EventController > Tab Settings > Show summary report

To show a summary report of the parts which the Drain deleted, select this. At the end of the simulation run the report checks all Drain objects within the entire simulation model which removed parts from the plant.

The column

Shows the

Method

Object

names of the Drain objects within the entire simulation model that removed parts.

--

Name

name of the part that passed through the plant.

--

Mean Life Time

mean life time (throughput time) of all investigated parts.

StatAvgLifeSpan

Total Throughput

total throughput of all parts, which passed through the Drain and which it removed from the plant.

StatDeleted

Throughput per Hour

number of MUs that the Drain removed from the plant in an hour over all observed times during which the Drain was available.

StatThroughputPerHour

Production

time portion which the part spent on a resource object of type production.

--

Transport

time portion which the part spent on a resource object of type transport.

--

Storage

time portion which the part spent on a resource object of type storage.

--

Value Added

time portion which the part was processed and was located on resource objects of type production, i.e., the time during which the value of the parts was increased.

StatProdWorkingPortion

Portion

accumulated values of the columns production (green), transport (orange), and storage (red) as bar segments.

--

You can also show the drain statistics table of a Drain in an HtmlReport, compare Display a Drain.
