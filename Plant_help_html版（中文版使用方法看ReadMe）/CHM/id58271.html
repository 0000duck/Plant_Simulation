Use a Random Experimental Design
2D Reference Help > Tools > ExperimentManager > Controlling the Experiment Runs > Automatically Generating Experiments > Use a Random Experimental Design

If you want to create a large number of experiments whose input values are not distributed according to certain patterns, you can do this with the random experimental design. The results of such an experiment study are ideally suited for training an Artificial Neural Network.

The menu command Tools > Random Experimental Design opens the table RandomDesign. Enter the values defining the random experimental design. All input values are determined independently of each other via a uniform distribution between the bounds by throwing the dice.
