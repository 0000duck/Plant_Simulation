_3D.NumObjects
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Objects in 3D > _3D.NumObjects

Syntax:

 
Path._3D.NumObjects

The read-only attribute _3D.NumObjects returns the number of objects located on the object designated by Path. You can then access these objects with the function _3D.getObject.

Return Value

The return value has the data type integer.

Example:

 
.Models.Frame.Station1._3D.InternalGraphics := ["deco1", deco2"]
var a : string[] := .Models.Frame.Station1._3D.InternalGraphics
