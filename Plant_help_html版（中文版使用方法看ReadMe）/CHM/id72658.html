Width
2D Reference Help > Material Flow Objects > AngularConverter > Attributes of the AngularConverter > Width

Syntax:

 
Path.Width

The attribute Width sets the width of the AngularConverter designated by Path. The Width only applies when the setting Transfer length/TransferLengthToObject is deactivated.

You can only enter the CurveWidth when you deactivate TransferLengthToObject. When you activate it, Plant Simulation computes the width in meters.

Transfer length on

Transfer length off

Example:

 
MyAngularConverter.Width := 4
