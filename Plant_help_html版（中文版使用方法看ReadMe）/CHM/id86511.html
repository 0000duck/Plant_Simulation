Export Graphics
3D Reference Help > 3D Reference > Edit Ribbon Tab > Export Scene > Export Graphics

The command Export Graphics exports the graphics that are contained in the active 3D window:

  • If you selected an object or a graphic in the active 3D window, Plant Simulation exports the selected object or the selected graphic.

  • If you did not select anything, Plant Simulation exports all graphics that are visible within the 3D window.

The command Export Graphics opens the dialog Export Scene. You can export the scene as a graphics file in the format JT Files (*.jt).

Select drive and folder from the drop-down list and enter a name for the scene you are going to export into the text box next to File name.

You can open these files again and continue modeling with them. Note that a scene encompasses all objects that are descendants of the current root of the scene.
