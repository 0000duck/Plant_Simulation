Inherit Format
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Inherit Format

The command Inherit Format activates or deactivates inheritance of the format of the list.

When you insert a list into a Frame, it automatically inherits the format from its class. You can only change the format of the new, instantiated list after you deactivate Inherit Format. Deactivating this command also deactivates Inherit Contents.
