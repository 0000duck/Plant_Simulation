Functions for Managing Date and Time
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Managing Date and Time > Functions for Managing Date and Time

SimTalk provides these functions for managing date and time information:

calendarWeek

setDaylightSavingTime

day

sysDate

dayOfWeek

timeOfDay

dayOfYear

week

getDate

year

month

Plant Simulation uses the date format of the Model language you selected: Year, month, day for English; day, month, year for German.
