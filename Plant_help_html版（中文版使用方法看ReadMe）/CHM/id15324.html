Most recently used
2D Reference Help > Material Flow Objects > FlowControl > Dialog Window of the FlowControl > Tab Exit Strategy > Strategy > Most recently used

To move the MUs on to the successor, which has Waiting the shortest time for an MU, select Most recently used from the drop-down list. The FlowControl always moves the MUs to the same successor, as long as this successor can receive the MUs.
