User-defined Index
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Accessing a Range of Cells with a Method > Ranges in Lists with One Column > User-defined Index

For a user-defined index of data type integer you have to add the number sign # in front of the identifier, as the system index, the numbers of rows and columns Plant Simulation assigns, also has the data type integer. Without the number sign, Plant Simulation would not be able to distinguish whether to use the system index or the user-defined index.

Example:

 
[2]  -- system index theme
[#2] -- user-defined index
