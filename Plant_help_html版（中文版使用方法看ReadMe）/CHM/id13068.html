Dialog Window of the ODBC Interface
Add-Ins Reference Help > Database Interfaces > ODBC > Dialog Window of the ODBC Interface > Dialog Window of the ODBC Interface

Double-click the icon of the object ODBC, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Information Flow in the Toolbox. The text boxes Name and Label are described under Dialog Items of the Objects.
