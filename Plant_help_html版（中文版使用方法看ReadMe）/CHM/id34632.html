MaxConnections
Add-Ins Reference Help > Inter-Process Communication Interfaces > Socket > Attributes of the Socket Interface > MaxConnections

Syntax:

 
Path.MaxConnections

The attribute MaxConnections sets the maximum number of client connections (sessions), which can connect to the active server socket. Note that each and every connection uses a handle of the operating system and the number of these handles is limited.

This setting applies when you are not running the socket connection in Server socket

Plant Simulation can handle a maximum of 2000 socket connections at any one time.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
Socket.MaxConnections := 7
