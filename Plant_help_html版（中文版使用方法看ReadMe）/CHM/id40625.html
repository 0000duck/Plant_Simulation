_3D.setGraphicTransformation
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.setGraphicTransformation

Syntax:

 
Path._3D.setGraphicTransformation(GraphicPath:any[], Position:length[3], RotationAngle:real, RotationAxis:real[3], RotationCenter:length[3], Scale:any [,GraphicGroupName:string])

The method 3D.setGraphicTransformation sets the transformation of the designated graphic of the object designated by Path-

The method deactivates graphic inheritance.

Parameters

  • The parameter GraphicPath of data type any sets the path to the graphic. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The parameter Position is an array of data type length which sets the position of the graphic.

  • The parameter RotationAngle of data type real sets its rotation angle of the graphic.

  • The parameter RotationAxis is an array of data type real which sets the rotation axis of the graphic.

  • The parameter RotationCenter is an array of data type length which sets the rotation center of the graphic.

  • The parameter Scale of data type real sets its scale.

    You can assign a value of data type real to achieve uniform scaling.

    You can also assign an array with three values of data type real to use different scaling factors for the three axes.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Example:

 
MyStation._3D.setGraphicTransformation([1,4], [3,2,1], 30, [0,0,-1], [2,0,0],2)
 -- uses uniform scaling with factor 2

Example:

 
MyStation._3D.setGraphicTransformation([1,4], [3,2,1], 30, [0,0,-1], [2,0,0],2, "deco")
 -- uses uniform scaling with factor 2

Example:

 
MyStation._3D.setGraphicTransformation([1,4], [3,2,1], 30, [0,0,-1], [2,0,0], [1,2,3])
 -- uses individual scaling along each axis with factors 1,2,3

Example:

 
MyStation._3D.setGraphicTransformation([1,4], [3,2,1], 30, [0,0,-1], [2,0,0], [1,2,3], "deco")
 -- uses individual scaling along each axis with factors 1,2,3
