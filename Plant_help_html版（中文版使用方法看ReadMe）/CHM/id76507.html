Reverse Segments
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Curve > Context Menu of Curved Objects > Reverse Segments

To change the direction of motion of the curved object, i.e., to make the last point of the object the first point, right-click the object and select this command.

This is especially handy if you want to mirror the layout of a production line by copying it and then reversing the direction of motion of the mirrored line to ensure a circular flow of materials.
