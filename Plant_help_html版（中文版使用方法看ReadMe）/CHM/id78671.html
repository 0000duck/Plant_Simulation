sendSMTPMail
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > sendSMTPMail

Syntax:

 
sendSMTPMail(Mail-Server:string, Receiver:string, Subject:string, MessageText:string)

The function sendSMTPMail sends an e-mail via the denoted mail server to the designated address.

If you deactivate the safety setting File > Model Settings > General > Access to the computer is permitted Plant Simulation does not execute the function loadLibrary and shows an error message.

Parameters

  • The parameter Mail-Server of data type string designates the mail server.

  • The parameter Receiver of data type string designates the receiver of the mail.

  • The parameter Subject of data type string designates the subject of the mail.

  • The parameter MessageText of data type string designates the message text of the mail.

Example:

 
sendSMTPMail("mail.company.com", "John.Smith@abc.org", "Test", "Hello John, how are you?")
