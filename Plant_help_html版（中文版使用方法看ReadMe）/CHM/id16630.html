Change the Font Size
Using HTML Help > Modifying Help Viewer Settings > Change the Font Size

To change the font size the Help Viewer uses to display the topics:

  • Click in the topic pane with the left mouse button, hold down the Ctrl key and roll the mouse wheel of your wheel mouse.

  • If you do not have a wheel mouse, close the Help, change to the Internet Explorer, select View > Text Size, and then select a size. Open the Help again to display the topics with the text size you selected.
