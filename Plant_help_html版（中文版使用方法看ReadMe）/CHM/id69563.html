Tab Background
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Background > Tab Background

Only the Frame and the Folder provide this tab.

On the tab Background you can define the background color of the selected Frame.

You can:

  • Select to Assign a background color of its own. Once you select the check box, 3D activates the other settings on the tab.

  • Select the Base color.

  • Select settings for the Corner brightness of the four corners of the Frame window.

    If you do not define a background color, 3D uses the background color of the parent object of the selected object.
