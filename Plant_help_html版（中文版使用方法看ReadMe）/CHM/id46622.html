IsSetUpFor
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Miscellaneous read-only attributes of the Material Flow Objects > IsSetUpFor

Syntax:

 
Path.IsSetUpFor

The read-only attribute IsSetUpFor returns the name of the MU which the object designated by Path is set up for or is setting-up for. Plant Simulation returns an empty string ("") when the object is not set up for an MU.

Return Value

The return value has the data type string.

Example:

 
if machine.IsSetUpFor = @.name AND NOT machine.setUp
  @.move(machine)
else
  machine.setUpFor(@)
end
