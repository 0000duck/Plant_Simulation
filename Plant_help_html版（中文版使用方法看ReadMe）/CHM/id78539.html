CurveActive
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Curved Objects > CurveActive

Syntax:

 
Path.CurveActive

The attribute CurveActive activates (true) curve mode for the curved object designated by Path or deactivates it (false).

Then, you can insert the length-oriented objects that contain curves into your simulation model. With the attribute AnimateOnEveryXthPixel you can set how often the MUs are animated on the length-oriented objects.

When you activate RotateMovables, Plant Simulation also activates CurveActive.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTrack.CurveActive := true
