Sequence
Add-Ins Reference Help > Object Libraries > Kanban Objects > KanbanSingleProc > Dialog Window of the KanbanSingleProc > Tab Advanced2 > Sequence

To create orders with a sequence table, select the check box Sequence. When you select Sequence, the KanbanStation also activates the check box Cyclical and the button Open Open Kanban Sequence Table.
