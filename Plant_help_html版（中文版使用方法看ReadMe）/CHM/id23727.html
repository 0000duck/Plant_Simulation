_3D.optimizeByPruningTinyGraphics
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > _3D.optimizeByPruningTinyGraphics

Syntax:

 
_3D.optimizeByPruningTinyGraphics(GraphicPath:any[], BoundingCubeSideLength:length[, GraphicGroupName:string])

The method _3D.optimizeByPruningTinyGraphics optimizes the graphic designated by Path by pruning its tiny graphics, i.e., it removes all component graphics which individually fit within the cube designated by the argument.

Parameter

  • The parameter GraphicPath is an array of data type any which sets the path to the graphic. The parameter is described underThe Parameters GraphicPath and GraphicGroupName.

  • The parameter boundingCubeSizeLength is an array with a single value of data type real. It designates the size of the bounding cube.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described underThe Parameters GraphicPath and GraphicGroupName.

Example:

 
.Models.Frame._3D.optimizeByPruningTinyGraphics([2], 0.001)

Example:

 
.Models.Frame._3D.optimizeByPruningTinyGraphics([2], 0.001, "deco")
