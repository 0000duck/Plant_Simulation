AttributeType
2D Reference Help > Material Flow Objects > Converter > Attributes of the Converter > AttributeType

Syntax:

 
Path.AttributeType

The attribute AttributeType sets the Attribute type of the Strategy > MU Attribute of the Converter.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
Converter.AttributeType := "String"
