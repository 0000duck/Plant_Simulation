MovingTime
2D Reference Help > Material Flow Objects > Converter > Attributes of the Converter > MovingTime

Syntax:

 
Path.MovingTime

The attribute MovingTime sets the Moving time which the Converter needs to lift the part onto a different conveying level and to then lower itself and move back to the default position.

Data Type of the Value You Can Assign

You can assign a value of data type time.

Example:

 
MyConverter.MovingTime := 2
