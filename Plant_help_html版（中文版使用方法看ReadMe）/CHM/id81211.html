appendLine
2D Reference Help > Display and User Interface Objects > HtmlReport > Methods of the HtmlReport > appendLine

Syntax:

 
Path.appendLine(Text:string)

The method appendLine appends a line of text to the content of the HtmlReport designated by Path.

Parameter

The parameter Text of data type string designates the text of the line that you want to append.

Example:

 
MyReport.Content := "#Heading" + chr(10)
for var i := 1 to numNodes 
   MyReport.appendLine(object(i).name)
