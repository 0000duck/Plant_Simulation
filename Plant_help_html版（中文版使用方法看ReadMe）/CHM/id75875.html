loadLibrary
Add-Ins Reference Help > Inter-Process Communication Interfaces > C > Using the C Interface > loadLibrary

Syntax:

 
loadLibrary(NameOfDll:string) → integer

The function loadLibrary loads a DLL.

If the setting File > Model Settings > General > Access to the computer is permitted is deactivated, which is the default setting for new models, Plant Simulation does not execute the function loadLibrary and shows an error message.

Parameter

The parameter NameOfDll of data type string designates the name of the DLL.

When the function fails, the return value will be less than 0, when the function succeeds, the program will return a value greater than 0, which you can use to access the DLL. This return value is the handle of the DLL.

Return Value

The return value has the data type integer.

Example:

 
loadLibrary("PlantSimCore.DLL")
