Context Menu of Embedded Lists
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > Context Menu of Embedded Lists > Context Menu of Embedded Lists

Embedded lists provide the commands listed below on the context menu.

Examples of embedded lists are:

  • The Services list on the Sub-tab Failure of the material flow objects.

  • The Services for Setting-Up and Processing list on the Tab Importer of the material flow objects.

  • The Services list on the tab Attributes of the Exporter.

  • The Creation Table on the tab Attributes of the WorkerPool.

  • The lists for the strategies Percentage and Sequence of the FlowControl.

  • The Attributes and Methods lists on the tab Actions of the Trigger.

  • The lists on the tabs of the ShiftCalendar.

  • The lists on the tabs of the AttributeExplorer.

    Not all embedded lists provide all of the commands listed below.

Open Object

Append Row

Select Object

Sort Ascending

Cut

Sort Descending

Copy

Show Comment

Paste

Import

Delete

Export

Select All

Import a Services List, a List of Shifts, etc. into an Object

Delete Row

Open a List as a Dialog Window in the Foreground

Insert Row
