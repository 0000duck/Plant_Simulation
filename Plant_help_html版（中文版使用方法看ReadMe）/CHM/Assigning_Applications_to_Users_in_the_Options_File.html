Assigning Applications to Users in the Options File
Setting-Up and Starting > Licensing Plant Simulation > Activating Licenses > Assigning Applications to Users in the Options File

To assign users (using the login names) to products (using the package names), use the keyword INCLUDE in the ugslmd.opt options file. The syntax is: 

INCLUDE package name USER login name1 login name2 login name3 …

Our example for the login names BobM, VladimirK, and ManoM looks like this: 

INCLUDE CAR_PLANTSIM_RESEARCH USER BobM VladimirK ManoM

If you would like to define groups, use the keyword GROUP followed by a group name of your choice, followed by the login names. The syntax is: 

GROUP group name> login name1 login name2 login name3 …

Our example looks like this:

GROUP SimulationExperts BobM VladimirK ManoM

Then you can assign products to groups:

INCLUDE CAR_PLANTSIM_RESEARCH GROUP SimulationExperts
