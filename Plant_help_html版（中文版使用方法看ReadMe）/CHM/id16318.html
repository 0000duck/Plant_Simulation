max
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Querying Statistics Values of List and Tables > max

Syntax:

 
Path.max([Range:listrange, ...])

The method max returns the greatest value in the range of cells designated by the parameters of data type listrange. The method then sets the cursor into the cell in which it found the value.

The method only applies to entries of data types real or integer. Plant Simulation ignores entries of other data types in the ranges to be searched or considers their value to be 0.

The method applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Return Value

The return value has the data type real.

Example:

 
print MyStackFile.max({*})
print MyQueueFile.max({1}..{*})
print MyCardFile.max( {*} )
print MyTableFile.max({3,3}..{3,*}) -- maximum number of entries
