Value
2D Reference Help > Display and User Interface Objects > Drop-Down List > Dialog Window of the Drop-Down List > Tab Attributes > Value

Enter which item in the Drop-Down List will be selected when you open it. This is one of the items which you entered in the list of Items. The value is identified by its index number.

If you localize the items to another language, you will usually use the Value instead of the attribute Item. The value is a number which is language-independent, while the Item is a string which differs from language to language.
