num_to_weight
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Values without Units into Physical Data Types with Units > num_to_weight

Syntax:

 
num_to_weight(real) → weight

The function num_to_weight converts the number without a unit into a value of data type weight.

Parameter

The parameter Number of data type real designates the number.

Return Value

The return value has the data type weight.

Example:

 
var w: weight
w := num_to_weight(Track4.Length)
