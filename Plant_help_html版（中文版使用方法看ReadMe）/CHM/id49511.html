Accessing a Range of Cells with a Method
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Accessing a Range of Cells with a Method > Accessing a Range of Cells with a Method

Plant Simulation can process a contiguous range of cells. A number of methods, such as min, max, only apply to a range. When you access a range of cells, you have to use a special syntax.
