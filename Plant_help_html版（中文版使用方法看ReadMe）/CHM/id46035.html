2D Reference Help > Resource Objects > LockoutZone > Dialog Window of the LockoutZone > Tab Objects > Objects

You can assign resources to the LockoutZone on the tab Objects:

  • Enter the path to and the name of the resource object in a cell. Or

  • Drag the resource object to the icon of the LockoutZone and drop it there. You can also select and then drag-and-drop multiple objects at the same time!

You can manipulate the contents of the list with the commands of the Context Menu of Embedded Lists.
