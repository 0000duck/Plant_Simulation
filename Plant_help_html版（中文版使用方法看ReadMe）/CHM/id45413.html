Automatic
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Set-Up > Automatic

To automatically set the object up, select this.

  • When you activate Automatic, Plant Simulation triggers the set-up process of objects as soon as the respective differing MU class intends to move onto the object.

  • When you clear Automatic, you have to define the set-up process yourself, for example with the Method setUpFor.

    If the object cannot be set up at the current point in time, for example because it is failed, Plant Simulation does not attempt to set the object up anew.
