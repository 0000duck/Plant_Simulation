InheritContents
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Miscellaneous Attributes of Lists and Tables > InheritContents

Syntax:

 
Path.InheritContents

The attribute InheritContents sets if the list designated by Path inherits its contents from the class (true) or not (false).

The attribute applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

User-defined attributes and local variables do not provide this attribute.

Example:

 
MyCardFile.InheritContents := false

Plant Simulation does not deactivate Inherit Contents, when you instantiate the class of a list containing a sub-list, and then write to the instance with a method, or when you write to the sub-list with a method. If you model like that, you have to deactivate Inherit Contents yourself in the window of the list object.
