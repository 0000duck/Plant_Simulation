Width
2D Reference Help > Material Flow Objects > Conveyor > Dialog Window of the Conveyor > Tab Attributes > Width

Enter the width of the length-oriented object. The width only applies when the setting Transfer length is activated.

You can only enter the Curve width in pixels when you deactivate Transfer length. When you activate it, Plant Simulation computes the width in meters.
