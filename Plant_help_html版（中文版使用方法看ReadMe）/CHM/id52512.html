2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Defining Failures of the Material Flow Objects > Attributes of the Failure Profiles > Name

Syntax:

 
Path.Failures.NameOfFailureProfile.Name

The attribute Name sets the Name of the failure profile. of the object designated by Path.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyStation.Failures.MyFailureProfile1.Name := "MyFailure2"
