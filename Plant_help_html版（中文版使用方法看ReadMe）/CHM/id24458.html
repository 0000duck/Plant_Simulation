Edit Icons
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Edit Icons

The context menu command Edit Icons opens the Icon Editor. Here you edit, delete, and add new icons to objects and models. The context menu that opens, when you click an object with the right mouse button, also provides this command.
