Insert Text
3D Reference Help > 3D Reference > Edit Ribbon Tab > Insert Text

The command Insert Text creates and inserts text into the selected Frame.

  • Select the Graphic group to which you want to add the new graphic. The list shows the graphic groups which are defined for the object.

    • Default (external): Designates the standard graphic group of the 3D objects. It initially is external and represents the object towards the outside. When you deactivate Show External Graphics, 3D hides the graphic groups designated as Default.

    • One of the user-defined graphic groups you added on the Tab Graphics. Newly created 3D Frames by default provide the graphic group Deco (internal), which are intended for graphics for decorating the interior view of the Frame.

  • Enter the text which you would like to add to the scene into the text box Text in the dialog Create Text.

  • Select the text color, the background color, and the alignment of the text in the scene window.

  • To create the text, click Create. 3D attaches the text to the mouse pointer. Drag the mouse to the position at which you would like to insert the shape and click the left mouse button. To cancel inserting, you can click the right mouse button or press the Esc key.

  • To move the text box to the location of your choice, click it and drag the mouse.

  • To change the size of the text, select it and select 3D Properties > Tab Transformation. Then change the size under Scale.

  • To set if the cuboid is an obstacle for the worker, select it and then click Edit 3D Properties > Tab Graphic Settings on the Home ribbon tab.

  • To assign a material, select the text and click 3D Properties > Tab Material on the Home ribbon tab.

Plant Simulation 2D determines which text will be shown via the object type in 2D:

  • Folder: 3D shows the name of the folder.

  • Comment: 3D shows the text you enter on the tab Display.

  • Variable: 3D shows the Name of the Variable and the Value of the Variable.

You cannot edit text once you have inserted it into the scene window. If the text does not meet your needs any more, you have to delete it and create and insert new text.
