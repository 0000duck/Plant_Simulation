2D Reference Help > Information Flow Objects > Trigger > Dialog Window of the Trigger > Tab Values > Values

To open the TimeSequence object that contains the current sequence of values of the Trigger, click this. Enter the default value of the Trigger on the tab Start Values > Default value.

Instead, you can also right-click the Trigger object in the Frame and select Open Values Table on the context menu.
