StatTspPausingTime
2D Reference Help > Mobile Objects > Transporter > Read-Only Attributes of the Transporter > StatTspPausingTime

Syntax:

 
MU-path.StatTspPausingTime

The read-only attribute StatTspPausingTime returns the sum of all pause-times of the Transporter designated by MU-path.

Return Value

The return value has the data type time.

Example:

 
print .MUs.Transporter:1.StatTspPausingTime
