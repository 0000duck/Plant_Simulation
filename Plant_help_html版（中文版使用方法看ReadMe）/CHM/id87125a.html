Duration
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Defining Failures of the Material Flow Objects > Attributes of the Failure Profiles > Duration

Syntax:

 
Path.Failures.NameOfFailureProfile.Duration

The attribute Duration sets the Duration of a failure for the object designated by Path.

When you change failure settings, this does not affect the already computed EndFailure event.

When a station fails, and a pause occurs during the failure, the failure time of the station will still be consumed, although the station is paused.

Statistics counts the times during which these two states overlap as pause time.

Data Type of the Value You Can Assign

You can assign a value of data type time.

Example:

 
MyStation.Failures.MyFailureProfile1.Duration.setParam("Normal",1,30,10)
