GetLastStation
2D Reference Help > Material Flow Objects > Cycle > Read-Only Attributes of the Cycle > GetLastStation

Syntax:

 
Path.GetLastStation

The read-only attribute GetLastStation returns the last station of the balanced line which the Cycle station designated by Path sets.

Return Value

The return value has the data type object.

Example:

 
print MyCycleObject.GetLastStation
