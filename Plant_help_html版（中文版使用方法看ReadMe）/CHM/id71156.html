RoutingCtrl
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > RoutingCtrl

Syntax:

 
MU-path.RoutingCtrl

The attribute RoutingCtrl designates a Method object, the Routing control, which the Transporter designated by MU-path activates, when Plant Simulation automatically predicted a route to the destination. Within this control you might, for example, read out the newly predicted route using the method getRouteToDestination. You might then manipulate this route and assign the changed route with the method setRoute. You might also check if the entrance or the exit of one of the Tracks along the predicted route is closed/locked and then drive around this Track.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
.MUs.Transporter.RoutingCtrl := &myRoutingControl
