Real
SimTalk Reference > General SimTalk Functions > Constants and Variables > Constants > Real

Constants of data type real contain a fractional part after the decimal point. In exponential notation the constant consists of a mantissa followed by the letter E and an exponent. You can, for example, write the number 100 also as 1.0e2 (1.0 times the second power of 10). The value range of the exponent is limited to the interval [307, -307].

Example:

 
contents gas tank := 115.0
Avogadro := 6.5e23
epsilon := 1.0e-6 -- 0.000001

You recognize real constants by the decimal point. If it is missing, the constant has the data type integer with a limited value range.

The data types time, date and dateTime do not support constants. Instead, use real or integer constants, or the data type conversion functions str_to_time, str_to_date, and str_to_dateTime.
