InFrame
Add-Ins Reference Help > ActiveX > Attributes of the Object ActiveX > InFrame

Syntax:

 
Path.InFrame

The attribute InFrame sets if the active ActiveX control will be displayed inside the Frame into which you inserted the ActiveX object is located (true) or not (false).

On

Off

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyActiveX.InFrame := true
