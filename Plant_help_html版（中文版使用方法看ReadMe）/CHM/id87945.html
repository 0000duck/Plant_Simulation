Dialog Window of the Comment
2D Reference Help > Display and User Interface Objects > Comment > Dialog Window of the Comment > Dialog Window of the Comment

Double-click the icon of the Comment, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab User Interface in the Toolbox.

To make the dialog window larger or smaller, grab any of its sides or corners, and drag the mouse.

To open a window that only shows the comment you entered, without formatting and editing options, click the Comment object with the right mouse button and select Open Comment Window on the context menu.

Plant Simulation opens the comment window with the size the tab Comment in the dialog window had, when you entered the comment and with the formatting you applied.

To close the comment window, click Close in the title bar.
