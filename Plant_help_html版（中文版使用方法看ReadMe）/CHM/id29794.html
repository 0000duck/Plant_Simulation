Bounding Probability Distribution Functions
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Bounding Probability Distribution Functions

Using random numbers may cause Plant Simulation to roll unrealistic values. Enter upper and lower bounds to prevent that from happening. Plant Simulation will then discard numbers outside of these bounds and throw a new value.

If the interval between lower bound and upper bound is very small, the values will have to be determined quite often before they are located within the interval. Enter the maximum number of samples under File > Model Settings/Preferences > Simulation > Maximum number of samples.

Within Methods, Plant Simulation accesses the bounds with the sub-attributes UpperBound and LowerBound.

You can use the attributes LowerBound and UpperBound to set or query the type of distribution of an attribute.
