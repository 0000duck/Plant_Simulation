QueueFile and StackFile
2D Reference Help > Information Flow Objects > QueueFile and StackFile > QueueFile and StackFile

The objects StackFile and QueueFile are lists with one column. They share all methods and attributes, but differ in their built-in properties.

List Ribbon Tab

The List Ribbon Tab of the list objects provides the following commands. Not all list objects provide all of the commands shown below.

To add the StackFile and QueueFile to your simulation model, click Manage Class Library > Basic Objects > InformationFlow > StackFile or QueueFile on the Home ribbon tab.
