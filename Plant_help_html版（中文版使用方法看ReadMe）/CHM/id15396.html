deleteValue
SimTalk Reference > General SimTalk Functions > Data Types > Array > Methods and Attributes of Arrays > deleteValue

Syntax:

 
deleteValue(Value:any) → boolean

The method deleteValue deletes a value from the array.

Parameter

The parameter Value of data type any deletes the first occurrence of specified value from the one-dimensional array.

Return Value

The return value has the data type boolean. The method returns true, if the value is part of the array, false if it is not part of the array.

Example:

 
a: string[] := ["Alice", "Bob", "Charlie"]
a.deleteValue("Bob")
print a -- prints ["Alice", "Charlie"]
