HasRouteToDestination
2D Reference Help > Mobile Objects > Transporter > Read-Only Attributes of the Transporter > HasRouteToDestination

Syntax:

 
MU-path.HasRouteToDestination

The read-only attribute HasRouteToDestination returns if the Transporter did find a route to its destination (true) or if it did not find one (false). The method is observable.

Return Value

The return value has the data type boolean.

Example:

 
print .MUs.Transporter:1.hasrouteToDestination
