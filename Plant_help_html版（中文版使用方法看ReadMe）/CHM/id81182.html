Exchange Graphics
3D Reference Help > 3D Reference > Edit Ribbon Tab > Exchange Graphics

The command Exchange Graphics assigns different graphic and animation data to the selected object. By default it opens the dialog Exchange Graphics. By default it opens to the folder where you installed Plant Simulation, namely C:\Program Files\Siemens\Tecnomatix Plant Simulation XX\3D\s3D-graphics.

Select the .s3D graphics file that is to replace the current graphic groups and the graphic attributes of the object.

To replace existing graphics in a model, which you created in a previous version of the program, with the new graphics, select the respective graphic and select Exchange Graphics. Navigate to the folder 3D > BasicObjects and select the new graphic of the object.

You can also select a graphic which you created in 3D with the command Export Scene > Export Object.

The command is only active when you open the folder containing the object in 3D in the Class Library, i.e. MaterialFlow, Resources, InformationFlow, MUs, etc.

When you call the command for a mobile object (MU), Plant Simulation shows the following message:

Click Yes to use the dimensions of the loaded graphic as the new dimensions of the MU, i.e., as the MU size.

Click No to retain the existing dimensions of the MU despite its changed graphic.
