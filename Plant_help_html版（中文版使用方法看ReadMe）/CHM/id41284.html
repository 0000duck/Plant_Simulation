incl
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Strings > incl

Syntax:

 
incl(TextToBeInserted:string, Text:string, Position:integer) → string

The function incl inserts the designated text into another string and returns the combined text of both.

Parameters

  • The parameter TextToBeInserted of data type string designates the text, which you want to insert into the other string.

  • The parameter Text of data type string designates the text into which the string will be inserted.

  • The parameter Position of data type integer designates the position in front of which string will be inserted.

The first character of string has the index 1. Plant Simulation does not display an error message when the parameter integer is less than 1 or integer is greater than the length of the string. string will then be inserted before or after string respectively.

Return Value

The return value has the data type string.

Example:

 
print incl("XYZ","abcdef",3)   -- returns "abXYZcdef"
print incl("XYZ","abcdef",-1)  -- returns "XYZabcdef"
print incl("XYZ","abcdef",1)   -- returns "XYZabcdef"
print incl("XYZ","abcdef",20)  -- returns "abcdefXYZ"
