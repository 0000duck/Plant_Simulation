Restart Simulation
2D Reference Help > Information Flow Objects > Method > The Method Window > Method Debugger > Menu Bar of the Method Debugger > Debug Menu > Restart Simulation

The menu command Restart Simulation starts the simulation again after you stopped it. It does the same as clicking Reset Simulation and then clicking Start/Stop Simulation in the EventController does, i.e., it terminates the active simulation run and resets and the starts the simulation again.
