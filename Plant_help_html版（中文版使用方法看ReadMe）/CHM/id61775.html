Length
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > Units > Length

Select the unit for the length. Plant Simulation uses m (meters) as unit within methods. When you change the unit while modeling, it converts all values, which you already entered into objects, to the new unit! This may cause rounding errors.
