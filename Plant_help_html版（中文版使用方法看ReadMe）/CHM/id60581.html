Methods for the Workplace of the Material Flow Objects
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods for the Workplace of the Material Flow Objects > Methods for the Workplace of the Material Flow Objects

The material flow objects that support an importer, i.e., the Station, the ParallelStation, the AssemblyStation, the DismantleStation, the Mixer, the Portioner, the DePortioner, and the Tank provide the method assignedWorkplaces for the Workplace which you can assign to them.

To view all of the methods and attributes of the material flow object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.
