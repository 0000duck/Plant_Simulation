Combination Table
2D Reference Help > Information Flow Objects > Trigger > Dialog Window of the Trigger > Tab Values > Combination Table

To open the Combination Table, click this and enter the Trigger objects, which the program combines into the combination trigger.

  • Enter the name of a TimeSequence in the column Time Sequence. Using this name, you can access the TimeSequence in the connection Formula. Click one of the entries to open the respective value list.

  • Enter the path to the source Trigger in the column Origin.

  • Enter when the Trigger starts working in the column Start Time.

  • Enter how often the Trigger is going to repeat in the column Number.

  • Enter when the Trigger stops working, independently of repeated patterns, in the column End Time. If you do not enter anything, no end time applies.
