HtmlReport
2D Reference Help > Display and User Interface Objects > HtmlReport > HtmlReport

You can create reports with the object HtmlReport which Plant Simulation displays as a HTML page. You can also save the HtmlReport as an .htm file and open this file in an HTML-Browser, such as Internet Explorer, Firefox, Google Chrome, etc. Plant Simulation is not required to open the HtmlReport in an external Browser, meaning that the HtmlReport can be displayed on any computer on which an HTML-Browser is present.

The easiest way to configure the contents of the HtmlReport is to drag an object from the Frame to the tab Content and to drop it there. You can enter additional content and formatting settings in the built-in markup language on the Tab Content.

You can also show another HtmlReport within an HtmlReport, compare Display anHtmlReport.

To add the HtmlReport to your simulation model, click Manage Class Library > Basic Objects > UserInterface > HtmlReport on the Home ribbon tab.

Compare the sample models: Click the Window ribbon tab, click Start Page > Getting Started > Example Models and click Small Examples. Then, select the respective Category, the Topic, and the Example in the dialog Examples Collection and click Open Model.
