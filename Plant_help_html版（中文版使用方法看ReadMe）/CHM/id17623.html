StatWaitingResCount
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Read-Only Attributes for Statistics of the Material Flow Objects > StatWaitingResCount

Syntax:

 
Path.StatWaitingResCount → integer

The method StatWaitingResCount returns how often the object designated by Path was Waiting for MUs and for Exporters/ Services.

Return Value

The return value has the data type integer.

Example:

 
print MyStation.StatWaitingResCount
