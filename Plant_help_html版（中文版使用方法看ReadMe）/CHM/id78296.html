AttributeRepresentation
2D Reference Help > Information Flow Objects > AttributeExplorer > Attributes of the AttributeExplorer > AttributeRepresentation

Syntax:

 
Path.AttributeRepresentation

The attribute AttributeRepresentation sets if the AttributeExplorer shows the attributes with their Name or their Alias in the Explorer window.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyAttributeExplorer.AttributeRepresentation := "Alias"
