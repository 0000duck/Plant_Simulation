Tab Content
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Tab Content

On the tab Content you define the content which the HtmlReport is to display.

The built-in report provides examples of the most important items that you might use for your standard reports.

The toggle button to the right of the canvas activates or deactivates inheritance.

  • A green button means that inheritance is active. Then the object inherits, i.e., uses, the value from the parent object from which it was derived. When you modify the value of the parent object, the program also modifies the value of the child object.

  • An orange button with a minus means that inheritance is not active. Values you enter only apply to the selected object.

You can enter the content of the HtmlReport as text or you can enter and select settings in the dialog Object Parameters. You define special elements, such as headings, tables, and pictures in purely textual form. For this purpose the HtmlReport provides a markup language which is designed for easy readability. The syntax is a superset of HTML, i.e., you can also enter the text in HTML syntax. We advise against doing this as HTML syntax is complicated to type and hard to read. You can also drag an object from the Frame to the tab Contentand drop it there.

To open the dialog Object Parameters for an object, which you added to the tab Content with drag and drop, click the mouse between the name of the object and the closing bracket, and click .

Then, select the settings you need in the dialog Select Object.

You can:

  • Define a Heading

  • Create a Bulleted List

  • Create a Numbered List

  • Highlight Text by Bolding and/or Italicizing It

  • Indent Text

  • Center Text

  • Select an Object or an Object Attribute to be Displayed

  • Select an Object to Display its Current Icon, or an Object and a Specific Icon to be Displayed on the HTML Page

  • Select a Bitmap File to Create an Icon in the HtmlReport

  • Insert a Comment into Your Report

  • Insert a Horizontal Conveyor

  • Insert HTML Tags

  • Protect Characters

  • Integrate a String Unchanged into the HTML Code

You can also add some of the formatting options by clicking the respective button on the toolbar.
