Init control
2D Reference Help > Mobile Objects > Transporter > Dialog Window of the Transporter > The Tools Menu > Init control

Click and select a Method object in the dialog Select Object. Enter the source code of the init control.

Plant Simulation calls the Method once at the beginning of the simulation run during the init phase. The Method may, for example, initialize important data structures.

To open the dialog of the object, whose name you entered into the text box, press the F2 key.
