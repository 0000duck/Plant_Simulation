deleteTriggeredAttr
2D Reference Help > Information Flow Objects > Trigger > Methods of the Trigger > deleteTriggeredAttr

Syntax:

 
Path.deleteTriggeredAttr(Object:object, Attribute:string)

The method deleteTriggeredAttr deletes the designated attribute of the designated object from the table Triggered Attributes of the Trigger designated by Path.

Parameters

  • The parameter Object of data type object designates the object.

  • The parameter Attribute of data type string designates the attribute to be controlled.

Return Value

The return value has the data type boolean.

Example:

 
MyTrigger.deleteTriggeredAttr(MyTrack,"pause")
