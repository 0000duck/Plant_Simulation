PowerInputSettingUp
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for the Energy Simulation of the Material Flow Objects > PowerInputSettingUp

Syntax:

 
Path.PowerInputSettingUp

The attribute PowerInputSettingUp sets the power input of the state setting-up of the object designated by Path.

Data Type of the Value You Can Assign

You can assign a value of data type real or a formula.

Example:

 
Station.PowerInputSettingUp := 0.001
