_3D.getGraphicRotation
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.getGraphicRotation

Syntax:

 
Path._3D.getGraphicRotation(GraphicPath:any [,GraphicGroupName:string]) → real[3]

The method _3D.getGraphicRotation returns the rotation of the designated graphic of the object designated by Path.

Parameters

  • The parameter GraphicPath of data type any designates the path of the graphic. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Return Value

When you specify a rotation center when setting the rotation, it will only be converted into a position of the graphic, 3D does not save it separately.

The return value is an array of data type real.

Example:

 
var a : real[3]  := .MaterialFlow.Assembly._3D.getGraphicRotation(["Graphic1", "Subgraphic"])
var a : real[3]  := ParallelStation._3D.getGraphicRotation([1, 1])

Example:

 
var a : real[3] := .MaterialFlow.Assembly._3D.getGraphicRotation(["Graphic1", "Subgraphic"], "deco")
var a : real[3] := ParallelStation._3D.getGraphicRotation([1, 1])
