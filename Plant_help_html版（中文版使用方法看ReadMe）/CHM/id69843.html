Tooltip Includes Type Info
2D Reference Help > Information Flow Objects > Method > The Method Window > Tools Ribbon Tab > View > Tooltip Includes Type Info

The command Tooltip Includes Type Info activates or deactivates if Plant Simulation shows the data type of the parameters, of the return value of methods, and of the parameter of attributes with the tooltip, when the Auto Complete feature reaches a point from which on the name is unique.

This command only works if you also activate Show Completion Tooltip.

The notation is the same as in the window Show Attributes and Methods.

--

Type info on

Type info off

Method

Attribute

When you change this setting, the change applies during the entire Plant Simulation session for all newly opened Methods.
