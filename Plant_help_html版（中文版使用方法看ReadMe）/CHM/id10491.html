_3D.ungroupGraphic
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.ungroupGraphic

Syntax:

 
Path._3D.ungroupGraphic(GraphicPath:any[] [,GraphicGroupName:string])

The method _3D.ungroupGraphic ungroups the designated group of graphics of the object designated by Path.

The method deactivates graphic inheritance.

Parameter

  • The parameter GraphicPath of data type any designates the path of the grouped graphic. The parameter is described under Built-in Methods for Detailed Access To Graphics.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Example:

 
MyFrame._3D.ungroupGraphic([1,2,3])

Example:

 
MyFrame._3D.ungroupGraphic([1,2,3], "deco")
