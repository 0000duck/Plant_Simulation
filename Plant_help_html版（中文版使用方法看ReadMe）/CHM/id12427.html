getItemsList
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > getItemsList

Syntax:

 
Path.getItemsList(Table:table)

The method getItemsList returns the full path within the dialog, the Name and the type of dialog item of the Dialog object designated by Path and writes it into a table.

Parameter

The parameter Table of data type table designates the name of the table.

Example:

 
MyDialog.getItemsList(myItemsTable)
