Number of Animation Events
2D Reference Help > Material Flow Objects > PickAndPlace Robot > Dialog Window of the Pick-and-Place Robot > Tab Attributes > Number of Animation Events

Click the icon of the pick-and-place robot with the left mouse button and press Ctrl+I.

Go to the General ribbon tab of the Icon Editor and select Set Animation Events. Enter a number into the text box that determines how often Plant Simulation draws the pick-and-place robot while it rotates.
