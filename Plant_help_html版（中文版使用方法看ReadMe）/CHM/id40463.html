BwExitCtrlRear
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Controls of the Material Flow Objects > BwExitCtrlRear

Syntax:

 
Path.BwExitCtrlRear

The attribute BwExitCtrlRear sets if the length-oriented objects Conveyor, Track, or TwoLaneTrack designated by Path activate the Backward exit control, when the rear of the MU exits via their entrance and/or when it moves towards their entrance when exiting (true) or not (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTrack.BwExitCtrlRear := false
MyTwoLaneTrack.A.BwExitCtrlRear := true
MyConveyor.BwExitCtrlRear := true

Example:

 
.MUs.Transporter:4.LoadBayType := "Line"
.MUs.Transporter:4.BwExitCtrlRear := false
