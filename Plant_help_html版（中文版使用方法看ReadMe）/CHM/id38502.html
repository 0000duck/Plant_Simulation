Show Data Type
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Show Data Type

The command Show Data Type shows or hides the data type of the columns and rows. Note that Plant Simulation shows the data type abbreviated when the name of the data type is too long for the text box. Plant Simulation also shows the data types of user-defined indexes.
