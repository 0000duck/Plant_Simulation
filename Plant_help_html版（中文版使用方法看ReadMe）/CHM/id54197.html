Working with Paths
Step-by-Step Help > Modeling in 3D > Working with Paths > Working with Paths

3D uses a set of points in space as paths for animating objects and extruding shapes.

  • Animation paths describe the route an animated mobile object takes when it moves on objects or through the scene. The cameras, through which you view the scene in a fly-through, also use animation paths.

  • Length-oriented Objects are objects with a shape built from extrusion segments and an extrusion configuration. The length-oriented objects Track, TwoLaneTrack, Conveyor and FootPath in the standard library are length-oriented.

You can create and edit a path in the dialog 3D Properties on the Tab MU Animation, the Tab Self Animation, or on the Tab Self Animation.
