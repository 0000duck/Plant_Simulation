Frame grid spacing
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > Modeling > Frame grid spacing

Enter the distance between the individual grid points from each other on the x-axis, i.e., the horizontal axis of the Frame.

Enter the distance between the individual grid points from each other on the y-axis, i.e., the vertical axis of the Frame.

The grid points are the points at which the imaginary grid lines intersect.

The default setting is 20 x 20 pixels.
