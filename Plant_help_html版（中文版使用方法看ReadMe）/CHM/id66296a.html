typeStatistics
2D Reference Help > Fluid Objects > FluidDrain > Methods of the FluidDrain > typeStatistics

Syntax:

 
Path.typeStatistics(TypeStatisticsTable:table)

The method typeStatistics returns the Detailed Statistics Table of the FluidDrain designated by Path and writes it into a table.

Name

Shows the

Material

name of the material that the FluidDrain drained from the plant.

Throughput

total amount of the material that the FluidDrain drained from the plant.

Parameter

The parameter TypeStatisticsTable of data type table designates the name of the table.

Example:

 
var MyFluidDrainStatisticsTable: table
MyFluidDrain.typeStatistics(MyFluidDrainStatisticsTable)

MyFluidDrain.typeStatistics(MyFluidDrainStatisticsTable)
