ResourceType
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Statistics Values of the Material Flow Objects > ResourceType

Syntax:

 
Path.ResourceType

The attribute ResourceType sets the type of resource for the object designated by Path: Production, Storage, or Transport. The attribute only affects the statistics collection of the MUs located on the object. Compare Additional Statistical Data of the MUs.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyTrack.ResourceType := "storage"
