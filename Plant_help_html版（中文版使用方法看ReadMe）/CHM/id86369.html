Clipart Library
2D Reference Help > The Plant Simulation Program Window > Icon Editor > General Ribbon Tab > Clipart Library

The command Clipart Library opens the clip-art library that is part of your Plant Simulation program package. The graphics in the folder Clipart provide different icons, which you can use in your simulation model.

To insert an icon from the Clip Art Library into the Plant Simulation Icon Editor, select an icon in the Windows Explorer window, drag it to the open window of the Icon Editor, and drop it there.
