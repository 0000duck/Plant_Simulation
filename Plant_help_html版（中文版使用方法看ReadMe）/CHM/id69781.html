checkArguments
2D Reference Help > Information Flow Objects > Method > Methods of the Method > checkArguments

Syntax:

 
&Method.checkArguments([Argument1:any, ...])

The method checkArguments checks if the parameters passed have a compatible data type. You can use the method checkArguments to make sure that values, which you or a user of a model you create enters into a dialog, may be passed to methods.

Parameter

The parameters Argument of data type any designate the arguments that are to be checked.

Example:

 
&MyMethod.checkArguments(42,void)
