str_to_date
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Times and Dates > str_to_date

Syntax:

 
str_to_date(DateStatement:string) → date

The function str_to_date converts the date-statement of the parameter designated by the parameter of data type string into a statement of data type date. Years either have two or four digits. The function also handles leap years.

Conversion depends on the setting you selected under File > Model Settings/Preferences > Units > Time scale.

Parameter

The parameter DateStatement of data type string designates the date that is going to be converted.

Return Value

The return value has the data type date.

Example:

 
print str_to_date("13.11.99") -- outputs 1999/01/13
print str_to_date("01.04.00") -- outputs 2000/04/01
print str_to_date("99/12/24") -- outputs 1999/12/24
