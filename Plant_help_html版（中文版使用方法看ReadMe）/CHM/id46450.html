getUserDialogXYWH
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > getUserDialogXYWH

Syntax:

 
Path.getUserDialogXYWH(byRef X:integer, byRef Y:integer, byRef Width:integer, byRef Height:integer)

The method getUserDialogXYWH returns the position of the user-defined dialog of the object designated by Path, and assigns them to the local variables, which you define.

Local Variables

  • The local variable X of data type integer designates the x-coordinate of the dialog of the object.

  • The local variable Y of data type integer designates the y-coordinate of the dialog of the object.

  • The local variable Width of data type integer designates its width.

  • The local variable Height of data type integer designates its height.

Return Value

When the user-defined dialog is closed, the method returns -1 for all parameters.

Example:

 
MyDialog.getUserDialogXYWH(x,y,w,h) 
print x," ",y," ",w," ",h
