Evaluation control
Add-Ins Reference Help > Genetic Algorithms > GAOptimization > Dialog Window of the GAOptimization > Tab Controls > Evaluation control

Click and select a Method object in the dialog Select Object. It contains the source code on how to evaluate the newly generated individuals. The program calls it, when the individuals of a new generation have been created. These newly generated individuals are going to be passed to the method as parameter of data type table. Within the method or within a simulation run started by the method, the fitness value is going to be calculated for each newly created individual of the generation. The simulation model computes the fitness value and writes it into column 2 of the table.
