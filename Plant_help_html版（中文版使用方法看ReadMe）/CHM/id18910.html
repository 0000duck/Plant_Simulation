Attributes of Lists and Tables
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Attributes of Lists and Tables

All lists and tables have predefined attributes controlling their behavior or representing their state.

You can set the value of an attribute and you can get its value, either with the check boxes, the text boxes and drop-down lists in the windows or by assigning values to the respective attributes.

  • To set the valueof an attribute of a list or table, you might, for example, type: 

    MyTableFile.MaxXDim := -1

  • To get the valueof an attribute a list or table, you might, for example, type: 

    print MyTableFile.MaxXDim
