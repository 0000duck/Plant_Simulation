Add-Ins Reference Help > Genetic Algorithms > GARangeAllocation > Dialog Window of the GARangeAllocation > Dialog Window of the GARangeAllocation

Double-click the icon of the GARangeAllocation , which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Genetic Algorithms in the Toolbox. The dialog items and menus are described under Dialog Items of the Objects.
