ExplorerTable
2D Reference Help > Information Flow Objects > AttributeExplorer > Attributes of the AttributeExplorer > ExplorerTable

Syntax:

 
Path.ExplorerTable

The attribute ExplorerTable sets the table the AttributeExplorer shows, when you click Show Explorer. The AttributeExplorer enters the attributes you enter into row 0 of the table into the AttributeTable.

  • The AttributeExplorer enters the objects you enter into column 0 of the table into the ObjectTable.

  • The AttributeExplorer enters the attributes you enter into any of the columns of the table into the Explorer table.

Data Type of the Value You Can Assign

You can assign a value of data type table.

Example:

 
MyAttributeExplorer.ExplorerTable := MyExplorerTable
