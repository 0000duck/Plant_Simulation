ExitBlocking
2D Reference Help > Material Flow Objects > FlowControl > Attributes of the FlowControl > ExitBlocking

Syntax:

 
Path.ExitBlocking

The attribute ExitBlocking sets whether the FlowControl uses Blocking behavior, when an MU exits the FlowControl (true) or not (false).

The non-blocking strategy of the exit strategy > MU Attribute moves the part on when any of the desired successors can receive it.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyFlowControl.ExitBlocking := true
