setTab
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > setTab

Syntax:

 
Path.setTab(DialogItem:string, NewTabPage:string) → boolean

The method setTab moves a dialog item onto a tab.

Parameters

  • The parameter DialogItem of data type string designates the dialog item.

  • The parameter NewTabPage of data type string designates the tab (New Tab Page) to which the dialog item is moved.

    Do not apply this method for dynamically creating tabs in your dialog!

    You cannot move a dialog item within a group with this method.

Return Value

The return value has the data type boolean

Example:

 
MyDialog.setTab("button","Properties.Settings")
