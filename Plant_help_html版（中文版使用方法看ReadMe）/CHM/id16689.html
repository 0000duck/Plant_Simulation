BrokerStatOn
2D Reference Help > Resource Objects > Broker > Attributes of the Broker > BrokerStatOn

Syntax:

 
Path.BrokerStatOn

The attribute BrokerStatOn activates (true) or deactivates (false) the collection of statistics of the Broker.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyBroker.BrokerStatOn := true
