Dialog Window of the GASetAllocation
Add-Ins Reference Help > Genetic Algorithms > GASetAllocation > Dialog Window of the GASetAllocation > Dialog Window of the GASetAllocation

Double-click the icon of the GASetAllocation, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Genetic Algorithms in the Toolbox. The dialog items and menus are described under Dialog Items of the Objects.
