_3D.TypeOfAnimations.AnimationPathName.IsCurve
SimTalk Reference > 3D-related SimTalk Functions > Built-in Functions for Saved Animations > _3D.TypeOfAnimations.AnimationPathName.IsCurve

Syntax:

 
Path._3D.MUAnimations.AnimationPathName.IsCurve → boolean
Path._3D.SelfAnimations.AnimationPathName.IsCurve → boolean
Path._3D.CameraAnimations.AnimationPathName.IsCurve → boolean

The read-only attribute _3D.TypeOfAnimations.AnimationPathName.IsCurve returns if the designated animation is of type Polycurve or not.

Return Value

The return value has the data type boolean.

Example:

 
var b : boolean := .MaterialFlow.Station._3D.MUAnimations.Default.IsCurve
