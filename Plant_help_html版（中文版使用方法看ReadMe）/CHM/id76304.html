Optimize Selected Graphic
3D Reference Help > 3D Reference > Edit Ribbon Tab > Optimize Selected Graphic > Optimize Selected Graphic

The command Optimize Selected Graphic optimizes the graphic of the selected object. During this process it discards information no longer required, such as unnecessary group objects.

You cannot restore discarded data, so we advise to optimize the structure of the object after you are done modeling the object. Optimizing the 3D graphic improves performance.

Proceed as follows to open the dialog Optimize Graphic:

  • Select the graphic in the scene window and click the button Optimize Graphic.

    If you would like to optimize an external graphic, click Show External Graphics on the View ribbon tab to display the graphics in the scene window.

  • Or click the object with the right mouse button and select Optimize Graphic on the context menu.

In the dialog you can optimize the selected graphic in three steps:

  • Select Step 1: Prune tiny graphics to remove component graphics which individually fit within the defined bounding cube.

  • Select Step 2: Visibility filter to remove graphic components which are not visible from outside.

  • Select Step 3: Flatten structure to flatten the structure of the object while preserving its contents.

  • Click Generate SimTalk Code and Copy to Clipboard to execute these actions.

  • Click Preview Result to preview your changes before accepting them.

  • Click Revert to undo your changes and to revert to the settings before you made changes.

  • Click Accept Result to accept your changes.

  • View information about the Graphic complexity of the graphic.

    Compare the number of nodes and the number of polygons before and after optimizing in our example below.

When you are done with optimizing the graphic, close the dialog by clicking in the title bar of the dialog.
