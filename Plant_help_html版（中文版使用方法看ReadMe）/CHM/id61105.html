GenerationSize
Add-Ins Reference Help > Genetic Algorithms > GAOptimization > Attributes of the GAOptimization > GenerationSize

Syntax:

 
Path.GenerationSize

The attributeGenerationSize sets the number of parent individuals of a generation. This value matches the number of families in a generation. Note that not every parent individual will have a family (compare also ParentSelection). The number of children in a generation is twice the value of GenerationSize.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
GAOptimization.GenerationSize := 30
