Tab Set-Up
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Set-Up > Tab Set-Up

On the tab Set-Up you can define how Plant Simulation sets the objects Station, ParallelStation, AssemblyStation, DismantleStation, Drain, and DePortioner up.

Note that you enter the Set-up time on the tab Times.

Compare the sample models: Click the Window ribbon tab, click Start Page > Getting Started > Example Models and click Small Examples. Then, select the respective Category, the Topic, and the Example in the dialog Examples Collection and click Open Model.
