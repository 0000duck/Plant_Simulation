Delete All Expressions
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Debugger Ribbon Tab > Delete All Expressions

The command Delete All Expressions deletes all expressions of all methods, which you entered into the Watch Window on the Tab Expressions.

A large number of saved expressions increases the size of the model file and makes loading and saving the model slower. We recommend to only save expressions while you are developing the model and delete them afterwards.
