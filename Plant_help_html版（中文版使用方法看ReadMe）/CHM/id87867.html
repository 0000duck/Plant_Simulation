print
SimTalk Reference > General SimTalk Functions > Predefined Functions > Output Functions > print

Syntax:

 
print ...

The function print outputs any amount of text. There are no limitation for the type.

Parameter

The parameters passed are going to be output in the Plant Simulation Console, which you open by clicking the Console button on the Window ribbon tab. You can use this function to show error messages or other hints for the user.

Example:

 
print "machine",M12.getName,"failed"
