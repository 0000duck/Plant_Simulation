2D Reference Help > Material Flow Objects > Source > Dialog Window of the Source > Tab Attributes > MU selection > Constant

The Source produces one type of MU only. Enter the path to this type of MU into the text box next to MU.

Or you can click and select an MU class, Part, Container or Transporter, in the dialog Select Object.

Or you can select the MU in the Toolbox or the Class Library, drag it to the text box and drop it there.
