Dialog Window of the Object Dialog
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Dialog Window of the Object Dialog

Double-click the icon of the Dialog, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab User Interface in the Toolbox. The text boxes Name and Label and the menus are described under Dialog Items of the Objects.
