_3D.createText
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating 3D Shapes > _3D.createText

Syntax:

 
_3D.createText(BasePoint:length[3], Text:string, Width:length [,GraphicGroupName:string]) → integer

The method _3D.createText creates text in the scene window.

Parameters

  • The parameter BasePoint is an array of data type length which sets the base point of the text.

  • The parameter Text of data type string sets the actual text being shown.

  • The parameter Width of data type length sets the width of the text.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Return Value

The return value has the data type integer and specifies the numerical index of the text, compare _3D.getGraphicIndex.

Example:

 
_3D.createText([2.0,2.0,1]," My text!",6) -- width 6m

Example:

 
_3D.createText([2.0,2.0,1]," My text!",6,"deco") -- width 6m

Example:

 
var index := _3D.createText([2.0,2.0,1]," My text!",6,"deco")
_3D.setGraphicMaterial([index], makeRGBValue(255,0,0), 0, 0, 0, 0, 0, "deco")
