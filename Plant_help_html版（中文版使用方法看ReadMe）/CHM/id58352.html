_3D.Position
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Setting the Transformation > _3D.Position

Syntax:

 
Path._3D.Position

The attribute _3D.Position sets the current position of the object designated by Path. This does not apply to MU instances and to Connector instances.

The attribute _3D.Position shows the current position. For MU instances this position is calculated by the animation.

Data Type of the Assignment Value

The parameter of data type array sets the position.

Example:

 
MyFrame._3D.Position := [5,3,0]
