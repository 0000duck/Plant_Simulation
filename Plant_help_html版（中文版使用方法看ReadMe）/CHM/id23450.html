Rotation point
2D Reference Help > Material Flow Objects > Turntable > Dialog Window of the Turntable > Tab Attributes > Rotation point

Enter the position of the fulcrum around which the Turntable rotates. You can enter a value between 0 and the length of the Turntable. 0 designates the point at which you start inserting the Turntable.

The center of rotation is the fulcrum around which the table rotates. The start point is the point which you click, when you start inserting the Turntable. The end point is the point which you click, when you finish inserting the Turntable.
