_3D.getGraphicScale
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.getGraphicScale

Syntax:

 
Path._3D.getGraphicScale(GraphicPath:any [,GraphicGroupName:string]) → real[3]

The method _3D.getGraphicScale returns the scale of the designated graphic of the object designated by Path.

Parameters

  • The parameter GraphicPath of data type any designates the path of the graphic. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Return Value

The return value is an array of data type real.

Example:

 
var a : array := Station._3D.getGraphicScale([2])
var a : array := .MaterialFlow.AssemblyStation._3D.getGraphicScale(["Graphic2"])

Example:

 
var a : array := Station._3D.getGraphicScale([2], "deco")
var a : array := .MaterialFlow.AssemblyStation._3D.getGraphicScale(["Graphic2"], "deco")
