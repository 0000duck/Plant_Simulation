makeRGBValue
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > makeRGBValue

Syntax:

 
makeRGBValue(Red:integer, Green:integer, Blue:integer) → integer

The function makeRGBValue generates RGB values, which you can then use in the methods drawEllipse, drawConveyor, drawRectangle and drawText, the attribute BackgroundColor and the attribute Color of the objects Connector, Variable, Comment, and Display to efficiently define colors.

Parameters

  • The parameter Red of data type integer designates its red component of the color.

  • The parameter Green of data type integer designates its green component.

  • The parameter Blue of data type integer designates its blue component.

You can enter an integer number between 0 and 255.

Instead of directly typing the values for the color components into the source code of the method, you can also press the Ctrl key and the spacebar to open the dialog Colors.Then, select the color in the dialog, and click OK. Plant Simulation then inserts the color values into the method.

When you already defined a color by setting its color values, Plant Simulation recognizes it, selects it in the dialog Colors, and transfers its values for red, green, and blue into the source code of your method. Compare this example: 

var rgb: integer := makeRGBValue(0,0,255)

When you hold down the Ctrl key and press the spacebar in the method editor, Plant Simulation recognizes the color values, and selects the color blue in the dialog Colors.

When you then select the color red in the dialog Colors, for example, and click OK, Plant Simulation automatically enters the color values of the color red into the source code of your method, namely: 

var rgb : integer := makeRGBValue(255,0,0)

Return Value

The return value has the data type integer.

Example:

 
.Models.Frame.BackgroundColor.Color := makeRGBValue(110,0,200)
.Models.Frame.Connector.Color := makeRGBValue(100,0,200)
.Models.Frame.Comment.Color := makeRGBValue(110,0,200)
.Models.Frame.Display.Color := makeRGBValue(110,0,200)
.Models.Frame.&Variable.Color := makeRGBValue(110,0,200)
