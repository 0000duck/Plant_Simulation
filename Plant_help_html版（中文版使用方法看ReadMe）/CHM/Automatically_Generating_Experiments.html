Automatically Generating Experiments
2D Reference Help > Tools > ExperimentManager > Controlling the Experiment Runs > Automatically Generating Experiments > Automatically Generating Experiments

The Tools menu provides commands for automatically creating experiments. The ExperimentManager assigns values to all input values; this is called experimental design. It automatically fills the experiment table ExpTable. Note that all experiments and their evaluations will be deleted when you employ an experimental design.

You can:

  • Use a Multi-level Experimental Design

  • Use a Random Experimental Design

  • Use a Two-level Experimental Design and Use Factorial Analysis
