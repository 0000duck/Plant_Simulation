Built-in Read-Only Attributes and Methods for Accessing Objects in 3D
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Objects in 3D > Built-in Read-Only Attributes and Methods for Accessing Objects in 3D

Plant Simulation provides these methods for accessing objects 3D:

_3D.addLengthOrientation, _3D.addObject, _3D.delete, _3D.Exists, _3D.existsObject _3D.getPositionOfObject, _3D.hasAttribute, _3D.NumObjects, _3D.openWindow, and _3D.removeLengthOrientation.

You can view and access the methods for accessing objects in 3D by clicking Auto Complete on the Edit ribbon tab of the method editor.
