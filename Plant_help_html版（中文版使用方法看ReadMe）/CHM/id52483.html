Settings for Rotating
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Transformation > Settings for the Rotation > Settings for Rotating

To set the axis around which to rotate the object, enter a value for each of the three components (x-y-z) that define the rotation axis.

Instead, you can also click the button to the right of the respective axis to set the rotation axis. Plant Simulation then enters the respective values into the text boxes.

Enter the angle of rotation in degrees into the text box.

When you set a rotation around the z-axis for an object that is length-oriented in 2D and in 3D, Plant Simulation converts the rotation into a polycurve and resets the angle to 0.

The toggle button activates or deactivates inheritance of the rotation .

  • A green button means that inheritance is active. Then the object inherits, i.e., uses, the rotation of the parent object from which it was derived. When you modify the rotation of the parent object, the program also modifies the rotation of the child object.

  • An orange button with a minus means that inheritance is not active.
