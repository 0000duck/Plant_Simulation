To define a new failure profile and to open a dialog, click New.

Enter the Name of the failure profile and select if it is Failed and if it is Active or not. You can define the Start time and the Stop time, the Interval and the Duration of this failure profile. You can also select the time which the Failure relates to.

The Lognormal distribution, the Erlang distribution and the Negative exponential distribution are especially suited for modeling failures.

When you select the check box Availability, Plant Simulation grays the dialog items Interval and Duration out, and you can enter the Availability and the MTTR instead. You can also select if the failure relates to the simulation time, the operating time or the processing time.
