Methods for Indirectly Accessing Lists and Tables
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Indirectly Accessing Lists and Tables > Methods for Indirectly Accessing Lists and Tables

Lists and tables provide the methods asObject, asString, findAttr, maxAttr, meanValueAttr, minAttr, standardDeviationAttr, and sumAttr for indirectly accessing them. Note that these methods only apply to ranges of data type object. The contents of the cells are references to objects whose attributes are to be processed.

To view all of the methods and attributes of the list object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.
