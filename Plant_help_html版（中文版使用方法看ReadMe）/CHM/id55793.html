Rotate movables
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Curve > Rotate movables

To rotate the part on the length-oriented curved objects in the direction of motion when the part turns in a curve, select this.

We advise to not deactivate Rotate movables as this setting does not affect the behavior in 3D.

Rotate movables active

Rotate movables not active

A rotation triggered by the setting Rotate MUs on a length-oriented object only has an effect as long as the part is located on this object. When leaving the object the part again assumes the rotation it had before. When entering a new object, the part considers the rotation of this new object when Rotate MUs is active.

When Plant Simulationshows a length-oriented object in curve mode, Plant Simulation evaluates the setting Rotate MUs (RotateMovables).

When Plant Simulation shows the object as an icon , Plant Simulation evaluates the setting Same as the Object (RotateMUs).
