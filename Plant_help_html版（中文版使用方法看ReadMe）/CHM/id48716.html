Dialog Window of the Trigger
2D Reference Help > Information Flow Objects > Trigger > Dialog Window of the Trigger > Dialog Window of the Trigger

Double-click the icon of the Trigger, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Information Flow in the Toolbox. The text boxes Name and Label are described under Dialog Items of the Objects.
