Entering the Identifier of a Cell
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Accessing a Range of Cells with a Method > Ranges in Tables with Several Columns > Entering the Identifier of a Cell

The direct statement consists of the identifier of the column and the identifier of the row, separated by a comma. The identifier for the column and row may be a number greater than 0, a user-defined index or an asterisk *, which stands for the entire table.

The left curly brace { designates the start of the range, followed by the identifier of the column, a comma, the identifier of the row, and a right curly brace }.

Example:

 
{1,3}
{"vehicle","door"}
{2,*}

To access

Notation

The cell in column 1, row 3

{1,3}

All rows in column 1

{1,*}

All columns in row 3

{*,3}

The entire table

{*,*}
