Tab Advanced
Add-Ins Reference Help > Object Libraries > Kanban Objects > KanbanSource > Dialog Window of the KanbanSource > Tab Advanced

On the tab Advanced you can extend and modify the properties of the Kanban system. You can access the tables, and the Station, on which the KanbanSource is based. In addition, you can manipulate the order process.

The tab Advanced provides these dialog items:

Show List of Open Requests

Open Workplace

Open User-defined Method sortList
