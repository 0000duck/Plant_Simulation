Dialog Window of the AttributeExplorer
2D Reference Help > Information Flow Objects > AttributeExplorer > Dialog Window of the AttributeExplorer > Dialog Window of the AttributeExplorer

Double-click the icon of the AttributeExplorer, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Information Flow in the Toolbox. The text boxes Name and Label are described under Dialog Items of the Objects.
