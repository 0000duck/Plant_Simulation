getFontColorColumn
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Columns > getFontColorColumn

Syntax:

 
Path.getFontColorColumn(Column:any) → integer

The method getFontColorColumn returns the font color of a column of the TimeSequence designated by Path.

Parameter

The parameter Column of data type any designates the column whose font color you want to get.

Return Value

The return value has the data type integer.

Example:

 
print timeSequence1.getFontColorColumn(3)
