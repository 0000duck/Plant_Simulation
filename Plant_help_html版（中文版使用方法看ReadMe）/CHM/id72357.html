Least Recent Demand
Step-by-Step Help > Modeling in 2D > Modeling the Flow of Materials, Basics > Transferring Parts from Station to Station > Select an Exit Strategy > Least Recent Demand

To move the part to the successor that has been Waiting the longest for an MU, select Least recent demand from the drop-down list.

  • When you select Blocking , the object moves the MU to the next successor.

  • When you clear Blocking , the object moves the MU the next successor, which can receive the MU.
