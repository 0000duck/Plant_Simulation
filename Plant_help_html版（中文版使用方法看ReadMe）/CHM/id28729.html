Show direction arrow
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Dialog Window of the Mobile Objects > Tab Graphics > Show direction arrow

To show the direction arrow on the vector graphic of the mobile object to illustrate the conveying direction, select this. The Border color, which you select, also sets the color of the direction arrow.
