_3D.removeLengthOrientation
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Objects in 3D > _3D.removeLengthOrientation

Syntax:

 
Path._3D.removeLengthOrientation

The method _3D.removeLengthOrientation makes a object out of the length-oriented object designated by Path which will not be length-oriented in 3D.

The method only applies to classes which do not inherit settings from another class.

Example:

 
MyStation._3D.removeLengthOrientation
