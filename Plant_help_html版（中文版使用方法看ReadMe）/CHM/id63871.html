AsString
2D Reference Help > Information Flow Objects > Variable > Read-Only Attributes of the Variable > AsString

Syntax:

 
&Variable.AsString

The read-only attribute AsString returns the relative path to the value of a Variable of data type object, as opposed to a normal access, which returns an absolute path.

Return Value

The return value has the data type string.

Example:

 
Variable := "~.ProdMgr.prodplan"
print Variable -- returns .facility.ProdMgr.prodplan
print &MyVariable.AsString -- returns ~.ProdMgr.prodplan

For a Variable with a compound data type, such as stack, queue, list and table, you have to tell Plant Simulation to create the data structure first, as it may not contain the actual data, but a reference to the corresponding data structure. Then you can access the Variable like any other data.
