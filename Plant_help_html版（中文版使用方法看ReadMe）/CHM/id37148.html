2D Reference Help > Material Flow Objects > Frame > The Frame Window > Icons Ribbon Tab > Icons Ribbon Tab

The Icons ribbon tab of the Frame provides these commands for editing the icons of the objects which you inserted into your simulation model.

As long as vector graphics mode is active, you cannot manipulate any objects in the Frame. To deactivate it, and to return to normal mode, click another tab.
