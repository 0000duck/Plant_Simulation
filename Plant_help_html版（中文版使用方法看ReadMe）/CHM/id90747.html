Plant Simulation Application License
Setting-Up and Starting > Plant Simulation License Types > Plant Simulation Application License

The Plant Simulation Application license allows you to work with user-defined object libraries which do not require any programming with the programming language SimTalk. The Plant Simulation Application license allows to interactively create simulation models using customized objects from object libraries as well as using the built-in material flow objects (Station, ParallelStation, Sorter, Buffer, Store, Track, Conveyor, etc.), mobile objects (Transporter, Containerand Part), display objects (Gauge, Chart, HtmlReport), and listsand tables. This license can visualize Plant Simulationmodels in 2D as well as in 3D. The programming language SimTalkis not available in the Plant Simulation Application license.

With the Plant Simulation Application license you can:

  • Model interactively and enter and select settings of the class objects in the dialogs of the objects.

  • Create models with application-specific user-defined objects, as well as access the built-in objects that are part of the user-defined objects.

  • Execute simulation runs with 2D and 3D graphics and animate simulation models, record statistics, analyze simulation runs, and save the results and the statistics tables.

  • Optimize the simulation results by modifying the model parameters with file interfaces or databases.

  • Automatically optimize the simulation model with genetic algorithms.

  • Use pre-defined application objects from object libraries available for many industries.
