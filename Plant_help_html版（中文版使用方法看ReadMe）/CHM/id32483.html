Button
2D Reference Help > Display and User Interface Objects > Button > Button

The object Button inserts a button into the Frame. The Button executes the action, which you programmed in the Control, when you click it. If you enter a label for the button, Plant Simulation shows that label on the button in the Frame.

By default Plant Simulation does not show the name of the button as a tooltip in the Frame when you roll the mouse over it. To show your own tooltip, create a user-defined attribute and name it Tooltip.

Plant Simulation can show the button in the Frame in a number of ways:

  • With the label you type in .

  • With an icon you draw in the icon editor. Then, you can show text you enter as the label on that button, for example.

  • With a picture you pasted into a new icon in the icon editor, and for example. You could name the icons play and play_down for example.

To open its dialog window, click the button with the right mouse button and select Open on the context menu.

To select the button in the Frame window, you can also drag a marquee over its icon.

To add the Button to your simulation model, click Manage Class Library > Basic Objects > UserInterface > Button on the Home ribbon tab.
