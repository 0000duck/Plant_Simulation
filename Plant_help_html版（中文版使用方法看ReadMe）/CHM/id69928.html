StockPlotter
Add-Ins Reference Help > Object Libraries > Value Stream Mapping > Value Stream Mapping Tools > StockPlotter

You can use the object StockPlotter to create a Chart that shows the current stock of the selected products of selected Inventories or Supermarkets.

Double-click the icon of the StockPlotter, which you inserted into you model, to open its dialog. The list on the left hand side of the tab Stores shows all Inventories and Supermarkets of your simulation model in the column Stock. Double-clicking one of the Inventories or Supermarkets shows the products defined for that Inventory or Supermarket in the list on the right hand side.

Double-clicking one of the products on the list on the right hand side changes that value in the column Selected from true to false and vice versa.

Select the check box Active to show the Chart of the selected products of the selected Inventory or Supermarket.

On the tab Settings you can define the time span (x-range) of the Chart and the number of values to be stored for the Chart.
