Display a Check Box
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Select an Object or an Object Attribute to be Displayed > Display a Check Box

To display a Checkbox, which you inserted into your simulation model, on the HTML page, click on the toolbar .

Select the Checkbox which you would like to show.

To show the label of the object Checkbox to the right of the check mark on the HTML page, select this .

To display the label of the check box to the right of the icon , you can also type in the following: 

[Checkbox] or 
[Checkbox, true]

To display the label of the check box to the left of the icon , enter: 

[Checkbox, false]
