View Output Values of Interest
2D Reference Help > Tools > ExperimentManager > Viewing Evaluations > View Output Values of Interest > View Output Values of Interest

In models with random components the output values usually fluctuate around a mean value. You can select to show min-max intervals or confidence intervals for the output values. Type in the confidence level for the confidence intervals into the text box Confidence level (%) on the tab Settings.

To select the output values for this evaluation, click Variable on the tab Evaluation. This opens a selection dialog. You can select several values by holding down Ctrl and clicking the values. To ensure readability, the output values should have a similar order of magnitude.

Selecting output values of interest

To close the dialog, click OK. To open the chart with the selected output values, click Show on the tab Evaluation. The output values for the individual experiments are connected with each other. The chart draws the confidence intervals or the min-max intervals around the mean values of the output values. In addition to the min-max interval, the median or the 50 percent quartile, the 25 percent quartile, and the 75 percent quartile can be shown. An x percent quartile is a certain value so that x percent of the observations of an output value of an experiment are less than or equal to this value. x designates one of these percentages, 25, 50, or 75.

Selecting Tools > Analysis of Variance allows you to test the hypothesis if the mean values of a selected output values in two experiments are the same or if they are significantly different. To check this, the program runs a t test.

You might ask yourself how many observations are required so that the confidence interval for an output value reaches the desired width. In general you can expect halving of the width of the confidence interval after the multiplication of the number of observations. Selecting Tools > Number of Observations shows the functional correlation between the number of observations and the width of the confidence interval.

The chart shows how the number of observations on the x axis influences the width of the confidence interval on the y axis for the selected output value.
