Interval
2D Reference Help > Information Flow Objects > Generator > Attributes of the Generator > Interval

Syntax:

 
Path.Interval

The attributeInterval sets the time between two activations of the Interval control method (IntervalCtrl).

Data Type of the Value You Can Assign

You can assign a value of data type time.

Example:

 
-- exponentially distributed intervals between 5 and 50 seconds 
-- with a mean value of beta = 15 seconds
MyGenerator.Interval.setParam("negexp",1,15,5,50)
