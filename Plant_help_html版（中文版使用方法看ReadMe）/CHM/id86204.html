ObjectMirrored
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Curved Objects > ObjectMirrored

Syntax:

 
Path.ObjectMirrored

The attribute ObjectMirrored sets if the curved object designated by Path will be mirrored on the y-axis (true) or not (false). Plant Simulation does not re-compute the anchor points, but retains the original curve and saves the inversion to the attribute ObjectMirrored.

When you assign a value in the segments table, Plant Simulation changes the original anchor points.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyConveyor.ObjectMirrored := true
print MyConveyor.ObjectMirrored
