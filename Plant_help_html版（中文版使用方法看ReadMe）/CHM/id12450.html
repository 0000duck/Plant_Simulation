createNestedList
2D Reference Help > Information Flow Objects > CardFile > Methods of the CardFile > createNestedList

Syntax:

 
Path.createNestedList([Row:integer,Name:string])

The method createNestedList creates a nested list in the CardFile designated by Path, containing lists and tables. The CardFile has to be of data type list, stack, queue, or table.

Plant Simulation overwrites the existing sublist without moving existing list one row downwards.

Parameters

The optional parameter Row of data type integer designates the row of the cell. If you do not specify a row, Plant Simulation adds the list to the end of the CardFile.

The optional parameter Name of data type string designates the name of the sublist that is to be created.

Example:

 
MyCardFile.createNestedList(1,"MyNestedlist")
