Evaluating Simulation Studies with the Neural Network
2D Reference Help > Tools > Artificial Neural Network > Evaluating Simulation Studies with the Neural Network

The experiments of a simulation study assign a set of output values to a set of input values. The interrelation between the input values and the output values can be described and modeled through regression or with an artificial neural network. The artificial neural network uses the biological principles of information processing and information representation in the neurons of the brain. For correlating the input values and the output values, the artificial neural network uses a distributed representation of the information. This leads to a considerably higher error tolerance and stability than traditional methods of regression analysis. This enables a neural network to create prognoses for not yet analyzed combinations of parameters. Modeling the data with the artificial neural network also takes adequate care of the noise, which is introduced to almost each and every simulation study through random processes and through systematic measurement errors of the output values.

The learning process of the neural network is called training. A trained neural network can then approximately determine the corresponding set of output values to a given set of input values without having to run a simulation. The neural network can only determine the interrelation between the input values and the output values if a sufficient number of data sets for the input values and the output values exist. The number of data sets for the training is the number of experiments.
