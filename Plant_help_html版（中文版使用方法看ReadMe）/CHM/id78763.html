Group ID
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Defining Properties of a Dialog Item > Group ID

For a dialog item of Type New Radio Button the Dialog shows the text box Group ID.

Enter a group id, i.e., a number, for grouping any number of radio buttons to denote, which radio buttons belong together.

Group ID 1 might stand for one set of radio buttons, while Group ID 2 might stand for another set. Note that the user can only select one radio button of a group.
