Save Library As
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Save/Load > Save Library As

The context menu command Save Library As saves the selected library as a Plant Simulationlibrary file (.lib).

  • Click the library folder with the right mouse button and select Save Library As.

  • This opens the dialog Library Information. If necessary you can change information about the library here, you can, for example, increase the Library version number. Then click OK.

  • Enter a name into the dialog Save As and specify a location where the object file will be saved. If you would like the library to be available in the dialog Manage Class Library later on, you have to select a location within the directory which you entered under File > Preferences > General > Libraries directories.

Plant Simulation saves the library to the specified library file. You can then load this folder into other simulation models and share it with your co-workers.

When you deactivate the 3D part of your model, by clicking Activate 3D and when you then make a library out of the model, the saved library does not contain the 3D part any longer.

When you then import this library into another model, Plant Simulation creates the 3D part from scratch and does not use the 3D part of the original library for which you deactivated 3D.

When you need the 3D part of the library in the future, 3D has to be active when you save the library.
