Availability
2D Reference Help > Tools > TransferStation > Dialog Window of the TransferStation > Tab Failures > Availability

Enter the availability of the TransferStation into the text box. This is a number between 0 and 100 percent. Availability is the proportion of time during which a machine is in an operable state. Availability is based on a combination of MTBF and MTTR.

You can enter the availability here or on the tab Failures of the ParallelStation. Plant Simulation propagates any changes you make to the respective other object.
