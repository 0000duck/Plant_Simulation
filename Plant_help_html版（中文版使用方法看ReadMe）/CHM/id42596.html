Path
2D Reference Help > Display and User Interface Objects > Display > Attributes of the Display > Path

Syntax:

 
Path.Path

The attribute Path sets the path to the value which the Display shows.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
Display1.Sampler := false
Display1.Path := "MyConveyor.NumMU"
Display2.Sampler := true
Display2.SmpInterval := 300
Display2.Path := "root.MyConveyor.utilization"
