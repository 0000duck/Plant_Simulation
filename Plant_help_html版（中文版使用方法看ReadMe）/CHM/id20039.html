findPart
2D Reference Help > Material Flow Objects > ParallelStation > Methods of the ParallelStation > findPart

Syntax:

 
Path.findPart(PartType:string) → object

The method findPart finds the part with the specified name on the ParallelStation designated by Path and returns it.

Parameter

The parameter PartType of data type string designates the part type.

Return Value

The return value has the data type object.

Example:

 
var o: object := MyParallelStation.findPart("Container") -- assigns for example .MUs.Container:1
