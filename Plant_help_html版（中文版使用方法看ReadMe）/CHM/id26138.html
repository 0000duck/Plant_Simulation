_3D.createIndexedLineSet
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating 3D Shapes > _3D.createIndexedLineSet

Syntax:

 
_3D.createIndexedLineSet(BasePoint:length[3], Nodes:length[3], LineSequences:any[][,Color:integer, GraphicGroupName:string]) → integer

The method _3D.createIndexedLineSet creates an indexed line set in the window of the scene.

Parameter

  • The parameter BasePoint is an array of data type length which sets the base point of the indexed line set.

  • The parameter Nodes is a two-dimensional array of data type length which sets the positions of the nodes, which follow, in space.

  • The parameter LineSequences is an array of arrays of data type any which sets the sequence of lines which follow. Each inner array contains a sequence of node indexes which set the line.

  • The optional parameter Colors of data type any sets the color.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Return Value

The return value has the data type integer and specifies the numerical index of the indexed line set, compare _3D.getGraphicIndex.
