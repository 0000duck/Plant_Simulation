Name
2D Reference Help > Information Flow Objects > TableFile > Attributes of the TableFile > Name

Syntax:

 
Path.Name

The attribute Name sets the name of the TableFile designated by Path to the designated expression.

Data Type of the Value You Can Assign

You can use the attribute Name to access the name of the sub-list via information flow.

Example:

 
TableFile.createNestedList(1, 1)
TableFile[1,1].Name := "MyName"

Example:

 
var str : string
str := .Models.MyPlantAnytown.SteeringTypes["Jacks","9149"].Name 
-- returns the value of the cell that contains the sub-table, 4 in the example
