Processing time
2D Reference Help > Material Flow Objects > PlaceBuffer > Dialog Window of the PlaceBuffer > Tab Times > Processing time

The processing time/dwell time is the time during which the part remains in the PlaceBuffer.

You can only define the processing time for the PlaceBuffer as whole, not for each individual station. It divides the processing time equally among its stations, i.e., when you set a processing time of 60 seconds for 3 stations, each station processes the part for 20 seconds.

As opposed to the other material flow objects, you can only enter a constant processing time for the PlaceBuffer.
