Entering a Range of Cells
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Accessing a Range of Cells with a Method > Ranges in Tables with Several Columns > Entering a Range of Cells

The from-to statement consists of two cells located in two columns, separated by two periods. The first identifier designates the top left cell identified by the number of the column and the number of the row, the second the bottom right cell identified by the number of the column and the number of the row. When the second identifier is {*,*}, Plant Simulation processes the table up to the highest valid column index and row index.

Example:

 
{2,2}..{3,3}
{"front","door"}..{"rear","door"}
{3,1}..{*,*}

To access

Enter

The cells starting with column 1, row 2 to column 3, row 5

{1,2}..{3,5}

All rows in columns 1 to 4

{1,*}..{4,*}

All columns in row 3 starting with column 2

{2,3}..{*,3}

All columns starting with column 2, row 3 to the highest valid cell

{2,3}..{*,*}

All columns up to column 3 and all rows until row 5

{*,*}..{3,5}
