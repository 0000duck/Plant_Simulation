BitOR
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Individual Bits > BitOR

Syntax:

 
BitOR(Value1:integer, Value2:integer) → integer

The function BitOR executes a bitwise OR of value 1 and value 2 and returns the result.

Parameters

  • The parameter Value1 of data type integer designates value 1.

  • The parameter Value2 of data type integer designates value 2.

Return Value

The return value has the data type integer.
