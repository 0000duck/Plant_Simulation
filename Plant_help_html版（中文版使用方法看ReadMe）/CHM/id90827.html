endFailure
2D Reference Help > Mobile Objects > Transporter > Methods of the Transporter > endFailure

Syntax:

 
MU-path.endFailure

The method endFailure terminates the failure of the Transporter designated by MU-path. It also sets the attribute Failed to false.

Example:

 
@.MUs.Transporter:4.endFailure
