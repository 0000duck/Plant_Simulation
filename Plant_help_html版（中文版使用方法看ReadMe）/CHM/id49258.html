StatTspFailDelta
2D Reference Help > Mobile Objects > Transporter > Read-Only Attributes of the Transporter > StatTspFailDelta

Syntax:

 
MU-path.StatTspFailDelta

The read-only attribute StatTspFailDelta returns the standard deviation from the mean value of a failure of the Transporter designated by MU-path.

Return Value

The return value has the data type time.

Example:

 
print .MUs.Transporter:1.StatTspFailDelta
