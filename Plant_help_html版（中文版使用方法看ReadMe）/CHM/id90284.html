StatProdFailPortion
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for Statistics of the Mobile Objects > StatProdFailPortion

Syntax:

 
MU-path.StatProdfailPortion

The read-only attribute StatProdFailPortion returns the time portion during which the MU instance designated by MU-path was located on a material flow object of type Production that was failed.

Return Value

The return value has the data type real.

Example:

 
print .MUs.Transporter.StatProdfailPortion
