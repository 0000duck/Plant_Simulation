Gravity mode
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab MU Animation > Gravity mode

To activate gravity mode, select this. To deactivate it, clear the check box.

In gravity mode the part is always placed perpendicular to the floor, either hanging from a rail or standing up on a rail.

This setting only applies to the length-oriented objects.
