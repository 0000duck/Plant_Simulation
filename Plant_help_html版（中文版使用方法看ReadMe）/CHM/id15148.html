IsTractor
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > IsTractor

Syntax:

 
MU-path.IsTractor

The attribute IsTractor sets if the Transporter designated by MU-path is the tractor (Is Tractor) of the tugger train, which pulls or pushes the trailers (true) or not (false). The Transporter then shows the icon tractor instead of the icon Operational. As a vector graphic the icon of the tractor looks like this .

To prevent the tractor from loading parts, enter 0 as the XDim and the YDim for the loading space type > Store. For the loading space types Track and Line, enter 0 as the Capacity.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
.MUs.Transporter:1.IsTractor := true
