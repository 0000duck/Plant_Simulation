IncludeSubframes
2D Reference Help > Information Flow Objects > AttributeExplorer > Attributes of the AttributeExplorer > IncludeSubframes

Syntax:

 
Path.IncludeSubframes

The attribute IncludeSubframes sets if the AttributeExplorer also searches Frames located within the active Frame (true), or not (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyAttributeExplorer.IncludeSubframes := true
