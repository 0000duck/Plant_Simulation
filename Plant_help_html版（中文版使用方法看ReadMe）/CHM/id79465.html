Dialog Window of the Variable
2D Reference Help > Information Flow Objects > Variable > Dialog Window of the Variable > Dialog Window of the Variable

Double-click the icon of the Variable, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Information Flow in the Toolbox.
