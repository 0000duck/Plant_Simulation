ShowStandardButtons
2D Reference Help > Display and User Interface Objects > Dialog > Attributes of the Dialog > ShowStandardButtons

Syntax:

 
Path.ShowStandardButtons

The attribute ShowStandardButtons sets if the Dialog shows the standard buttons OK, Cancel and Apply (true) or not (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
Dialog.ShowStandardButtons := false
