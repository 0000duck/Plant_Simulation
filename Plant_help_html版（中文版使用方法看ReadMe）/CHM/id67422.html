Tab Color
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Color > Tab Color

Select a Color for the Grid, the Background, the Desk, the Text, for the plotted data (numbers 1 through 14), and for additional colors on the tab Color. You can Add or Delete a color. You can also select the Opacity, a Line style, and a Line weight. For certain chart types you can also select a Marker size.

If you would like the Chart to inherit color attributes, make sure that the toggle button for inheritance on the right hand side of the tab is selected .
