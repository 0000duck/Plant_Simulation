ColumnIndexContents
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Attributes for the Format of Lists and Tables > ColumnIndexContents

Syntax:

 
Path.ColumnIndexContents

The attribute ColumnIndexContents sets that the column index belongs to the contents of the TableFile (true ). Then the column index is inherited together with the contents of the TableFile when Inherit Contents is active.

When you enter false, the column index belongs to the format of the TableFile. Then the column index is inherited together with the format of the TableFile when Inherit Format is active.

Only the TableFile provides this setting.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTableFile.ColumnIndexContents := true
