DataBase
Add-Ins Reference Help > Database Interfaces > ODBC > Attributes of the ODBC Interface > DataBase

Syntax:

 
Path.DataBase

The attribute DataBase designates the name of an ODBC server within the computer network. SQL*Net determines the format. If you would like to work with the default server, you do not have to enter anything. The default server usually is the local server.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
ODBC.Database := "ODBC"
