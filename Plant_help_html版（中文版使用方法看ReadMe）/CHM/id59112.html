load
2D Reference Help > Information Flow Objects > Method > Methods of the Method > load

Syntax:

 
&Method.load(FileName:string) → boolean
 
The method load loads the contents of a text file in ASCII format as source code into the Method. 
         

         
Parameter

         
The parameter FileName of data type string designates the name of the text file.
         

         
Return Value

         
The return value has the data type boolean.
         

         

            
            
               
               
               
            
            
            
               
               

                  
                  
                     
                     

                        
                        
                           
                           

                              
                              
                                 
                                 
Example:
