_3D.getGraphicName
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.getGraphicName

Syntax:

 
Path._3D.getGraphicName(GraphicPath:any[] [,GraphicGroupName:string]) → string

The method _3D.getGraphicName returns the name of the designated graphic of the object designated by Path.

Parameter

  • The parameter GraphicPath is an array of data type any into which the method writes the name of the graphic. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Return Value

The return value has the data type string.

Example:

 
var name : string := MyStation._3D.getGraphicName([1,2,4]) -- the variable contains the name of the graphic
