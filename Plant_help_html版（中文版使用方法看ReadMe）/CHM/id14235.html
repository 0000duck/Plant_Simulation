openImg
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Animating All Objects > openImg

Syntax:

 
Path.openImg

A 3D Only model does not provide the method openImg.

The method openImg opens the Icon Editor with the icon of the object designated by Path. If the Icon Editor is already open, Plant Simulation prompts you to save your previous work. Only then does it open the other icon. If you do not save the previous changes, the first icon will remain open and the method will have no effect.

The objects Comment, Connector, and Variable do not have icons.

Example:

 
MyStation.openImg
table.openImg
