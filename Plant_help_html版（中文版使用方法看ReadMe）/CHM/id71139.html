Insert Cone Frustum
3D Reference Help > 3D Reference > Edit Ribbon Tab > Insert Cone Frustum

The command Insert Cone Frustum creates and inserts a cone or a cone frustum into the selected Frame.

  • Select the Graphic group to which you want to add the new graphic. The list shows the graphic groups which are defined for the object.

    • Default (external): Designates the standard graphic group of the 3D objects. It initially is external and represents the object towards the outside. When you deactivate Show External Graphics, 3D hides the graphic groups designated as Default.

    • One of the user-defined graphic groups you added on the Tab Graphics. Newly created 3D Frames by default provide the graphic group Deco (internal), which are intended for graphics for decorating the interior view of the Frame.

  • Enter the Top radius of the cone you want to create into the dialog Create Cone Frustum.

    A top radius of 0 creates the cone with a tip.

    A top radius of greater than 0 cuts off the tip of the cone. This way you can, for example, model a bottle quickly and easily.

  • Enter the Bottom radius of the cone you want to create.

    Type in a greater bottom radius than the top radius to create a funnel-type truncated cone.

  • Enter the Height of the cone you want to create.

  • Enter the Wall thickness of the cone.

    If you do not specify the wall thickness, Plant Simulation creates a closed cone.

    If you specify a negative value, Plant Simulation interprets this as if you did not specify a value.

    If you enter 0, Plant Simulation creates the mantle without a thickness.

    If you enter a positive value, Plant Simulation creates the cone with the specified wall thickness.

  • Select Create covers to create the frustum with a top and a bottom. Clear Create covers to leave the top and the bottom of the frustum open.

  • To create the cone, click Create. 3D attaches the shape to the mouse pointer. Drag the mouse to the position at which you would like to insert the shape and click the left mouse button. To cancel inserting, you can click the right mouse button or press the Esc key.

  • To move the cone to the location of your choice, click it and drag the mouse.

  • To change the size of the cone, select it and select 3D Properties > Tab Transformation. Then change the size under Scale.

  • To set if the cone is an obstacle for the worker, select it and then click Edit 3D Properties > Tab Graphic Settings on the Home ribbon tab.

  • To assign a material and a color, select the cone and click 3D Properties > Tab Material on the Home ribbon tab.
