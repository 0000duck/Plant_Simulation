Remark
Add-Ins Reference Help > Object Libraries > Value Stream Mapping > Value Stream Mapping Tools > Remark

You can use the object Remark to create any comment of your own. Plant Simulation shows this comment in the value stream mapping model. Double-click the icon of the object Remark to open its dialog.

Enter any text you want to show and which you deem helpful for anybody working with the model. If you want to show several lines of text, you can force a line break by entering a forward slash (/).
