copyFile
SimTalk Reference > General SimTalk Functions > Predefined Functions > Operating System Functions > copyFile

Syntax:

 
copyFile(Source:string, Destination:string) → boolean

The function copyFile copies a file from one location to another.

If you deactivate the safety setting File > Model Settings > General > Access to the computer is permitted the function copyFile can copy data from another folder and write them to the model folder or its sub-folders. The cannot copy files from the model folder or its sub-folders.

Parameters

  • The parameter Source of data type string designates the source, i.e., the file you want to copy.

  • The parameter Destination of data type string designates the destination, i.e., the folder into which you want to copy it.

Return Value

The return value has the data type boolean: true if the copy action was successful, false if it failed.

Example:

 
copyFile("C:\file.txt", "C:\temp\file.txt")
