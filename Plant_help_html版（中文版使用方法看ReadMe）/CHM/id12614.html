cutRow
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Rows of the TableFile > cutRow

Syntax:

 
Path.cutRow(Row:any)

The method cutRow cuts a row, including all of its data, from the TableFile designated by Path.

Parameter

The parameter Row of data type any designates the row. Plant Simulation shifts all rows with an index equal to or greater than any one position up.

Example:

 
MyTableFile.cutRow(8)
