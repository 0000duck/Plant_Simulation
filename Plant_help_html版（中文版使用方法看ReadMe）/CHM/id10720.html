Vector graphics active
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Dialog Window of the Mobile Objects > Tab Graphics > Vector graphics active

To activate vector graphics mode, select this, and click OK or Apply. Then, Plant Simulation shows the graphics of the parts with the correct size and with the correct distance to each other while animating the simulation model.

The table below illustrates the difference between the vector graphic and the icon of the object.

Object

Vector Graphic

Icon

Part

Container

Transporter

Transporter Tractor

To also show the graphic in 3D that corresponds to the vector graphic in 2D, select Scale automatically on the Tab Transformation in the dialog Edit 3D-Properties.
