The while-loop
SimTalk Reference > General SimTalk Functions > Control Flow Statements > Loops > The while-loop

Plant Simulation executes the instruction within the while-loop several times as long as the condition is met. Plant Simulation checks the condition before each loop iteration. When the condition is already not met on the first iteration, the instructions within the loop will not be executed a single time.

When the condition never yields the value false, the loop will be executed infinitely. Press Ctrl+Alt+Shift to stop the endless loop. This opens the Method-Debugger.

Example:

 
param n: integer -> integer  -- computes the factorial of n 
result := 1
while n > 1 
   result := n * result
   n -= 1
end
