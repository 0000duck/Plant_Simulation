pe, [X,Y]
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Methods of All Mobile Objects > Methods for Accessing the Load of Container and Transporter > pe, [X,Y]

Syntax:

 
MU-path.pe(X:integer, Y:integer) → object
MU-path[X:integer, Y:integer] → boolean

The method pe(X:integer, Y:integer) accesses the storage place of the matrix-based loading space. [X:integer,Y:integer] accomplishes the same.

Parameters

  • The parameter X of data type integer designates the X-position of the MU.

  • The parameter Y of data type integer designates the Y-position of the MU.

Example:

 
print .MUs.Transporter:1[2,3].Cont
@.pe(2,3).Cont.move(machine)
