Step 2: Visibility filter
3D Reference Help > 3D Reference > Edit Ribbon Tab > Optimize Selected Graphic > Step 2: Visibility filter > Step 2: Visibility filter

This setting removes graphic components which are not visible from outside. You can select the cull granularity and the simplification level.

It scans the graphic for hidden parts which will then be deleted. The cull granularity sets the finest structure that Plant Simulation checks.

The reduction level sets how closely Plant Simulation checks the graphic. The higher this setting, the more graphics are kept. When you need the optimized graphic to be as small as possible, select Low.
