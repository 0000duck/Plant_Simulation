Unique index key
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Format > Tab Data Type > Unique index key

When you select the row of the row index located to the left of the first column of cells in the list/table, and then select Format > Format > List Format > Data Type, Plant Simulation shows the check box Unique index key.

Select it to only allow unique entries in user-defined indexes, depending on the data type. Plant Simulation issues a warning when you attempt to assign an entry that already exists. Note that when you assign an entry for an index that already exists in a Method, Plant Simulation does not issue a warning message!
