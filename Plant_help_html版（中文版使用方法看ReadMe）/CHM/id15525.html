Argument for open
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Method > Argument for open

Enter the callback argument, i.e., a string, for the Open action, which the Dialog executes, when it executes the Callback method. The default setting is Open.

The Open section of the callback method initializes the contents of the dialog window or sets the dialog items to values of your choice.
