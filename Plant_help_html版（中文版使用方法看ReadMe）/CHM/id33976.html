Change Values in the Dialog of the Objects
Step-by-Step Help > Getting to Know Tecnomatix Plant Simulation > Working with the Program, Basics > Changing the Settings of the Objects > Change Values in the Dialog of the Objects

As a rule you are going to select the settings of the individual objects in your simulation model in the dialogs of the respective objects.

To do so, you either enter a value into a text box.

Or you select a value from a drop-down list.

Or you check or clear a check box to activate or to deactivate a setting.

All of these settings correspond to attributes which you can also access with the programming language SimTalk. The What’s This help for the respective dialog item shows the name of the attribute in the bottom row next to SimTalk.
