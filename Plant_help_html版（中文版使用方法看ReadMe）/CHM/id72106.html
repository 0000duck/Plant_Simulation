Edit Grid Settings
3D Reference Help > 3D Reference > View Ribbon Tab > Edit Grid Settings > Edit Grid Settings

The command Edit Grid Settings allows you to edit grid settings.

You can:

  • Show or hide the Base plate on which the grid is placed in the scene window.

  • Select the color of the x- and of the y- Axes in the scene window.

  • Select settings for the Grid lines.

  • To apply and display your changes of the grid settings in the scene window, click Apply.

  • To apply your changes to the settings and close the dialog, click OK.

  • To close the dialog without applying any of the settings you modified, click Cancel.
