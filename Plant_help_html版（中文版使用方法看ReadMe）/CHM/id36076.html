_3D.setGraphicRotation
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.setGraphicRotation

Syntax:

 
Path._3D.setGraphicRotation(GraphicPath:any, RotationAngle:real, RotationAxis:real[3] [,RotationCenter:length[3], GraphicGroupName:string])

The method _3D.setGraphicRotation sets the rotation of the designated graphic of the object designated by Path.

The method deactivates graphic inheritance.

Parameter

  • The parameter GraphicPath of data type any sets the path to the graphic. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The parameter RotationAngle of data type real sets the rotation angle of the graphic.

  • The parameter RotationAxis is an array of data type real, which sets the rotation axis of the graphic.

  • The optional parameter RotationCenter is an array of data type length, which sets the rotation center of the graphic.

    3D does not save the rotation center as a separate value, but converts it into another position among others.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Example:

 
obj._3D.setGraphicRotation([1,7],45, [0,1,0])

Example:

 
obj._3D.setGraphicRotation([1,7],45, [0,1,0], "deco")
