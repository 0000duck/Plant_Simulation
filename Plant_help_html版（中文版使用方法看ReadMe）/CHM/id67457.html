Embed file
2D Reference Help > Information Flow Objects > FileLink > Dialog Window of the FileLink > Tab Attributes > Embed file

To embed the file, which you drag-and-drop onto the Frame, into the Plant Simulation model, select this. Then that version of the file will become part of the simulation model, meaning the program does not care, when you move the original file to another location. Changes to the file after embedding it will naturally not be reflected if you embed it. Embedding may significantly increase the file size of your .spp model file.

When you clear the check box, the program creates a link to the file in the file system of your computer. If you move or delete the file, the link will become invalid and Plant Simulation will not be able to find the file and open it.
