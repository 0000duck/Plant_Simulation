Tab Query
2D Reference Help > Information Flow Objects > AttributeExplorer > Dialog Window of the AttributeExplorer > Tab Query > Tab Query

Before you can enter data, click the toggle button Inheritance, so that it is deselected .

Define a query that finds objects according to the criteria you enter into the cells of the list on the tab Query. Select the number of opening Parentheses, enter an Attribute, select a Condition, a Value, select the number of closing Parentheses, select an Operator, and enter a Comment. Select the Frame that the AttributeExplorerqueries and select the check box Include subframes to also query any Frames located within the selected Frame. Select Include MUs to also include the parts (MUs) in your model in your query.

Spelled out our query reads like this:

Name Expr Station.* and ((XPos > 100 and YPos = 200) or ExitStrategy = Cyclic)

To manipulate the contents of the list, use the commands on the Context Menu of Embedded Lists.
