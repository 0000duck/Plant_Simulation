Built-in Methods for Accessing Graphic Groups
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Graphic Groups > Built-in Methods for Accessing Graphic Groups

Plant Simulation provides these methods for accessing graphic groups in 3D:

_3D.addGraphicGroup, _3D.deleteGraphicGroup, _3D.hideGraphicGroup, _3D.isGraphicGroupGenerated, _3D.isGraphicGroupInternal, _3D.isGraphicGroupInternal, _3D.isGraphicGroupLocked, _3D.isGraphicGroupVisible, _3D.lockGraphicGroup, _3D.renameGraphicGroup, _3D.setGraphicGroupExternal, _3D.setGraphicGroupInternal, _3D.showGraphicGroup, and _3D.unlockGraphicGroup.

You can view and access the methods for accessing graphic groups by clicking Auto Complete on the Edit ribbon tab of the method editor.
