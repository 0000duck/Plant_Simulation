Viewing Evaluations
2D Reference Help > Tools > ExperimentManager > Viewing Evaluations > Viewing Evaluations

You can view the input values and the output values of the individual experiments on the tab Evaluation. It provides a complete overview over all experiments.

Evaluations

You can:

  • Click Results open the table Results. It shows all input values and all output values for all experiments.

  • Click Detailed Results to open the table DetailedResults. It shows the mean value, the standard deviation, the minimum, the maximum, and die interval bounds of the confidence intervals for all output values. If a table is an output value, the detailed results show sub-tables containing the mean values, the standard deviations, the minimum values, and the maximum values of the numerical values of the output table.

    You can use the attribute ResultsPrecision to set the number of decimal places of the results in the tables. This setting also determines how the values are shown in the HtmlReport. If you want to show the results with a suitable number of decimal places, you have to set the value of the attribute ResultsPrecision of the ExperimentManager to a value between 1 and 14. The default setting -1 shows the results values with maximally 15 decimal places. Compare this example:

    ExperimentManager.ResultsPrecision := 3

  • Show the Results in a Report.

  • View Output Values of Interest.

Statistical evaluation for the output values

The results of all simulation runs are contained on the sub-tables of the columns Observations. To open the sub-table, click in the respective cell and press the F2 key.
