Attributes of the TimeSequence
2D Reference Help > Information Flow Objects > TimeSequence > Attributes of the TimeSequence > Attributes of the TimeSequence

The TimeSequence provides:

  • The attributes Absolute, Active, DataType, DefaultValue, Path, ReferenceDate, ReferenceTime, Sample, ShowColumnIndex, ShowRowIndex, and SmpPeriod.

  • The Attributes of Lists and Tables.

  • The Attributes of All Objects.

To view all of the methods and attributes of the TimeSequence, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.

You can set the value of an attribute and you can get its value, either with the check boxes, the text boxes and drop-down lists in the dialog windows or by assigning values to the respective attributes.

  • To set the value of an attribute of a list or table, you might, for example, type: 

    ts1.ReferenceTime := str_to_time("1:00:00:00.00")

  • To get the value of an attribute a list or table, you might, for example, type: 

    print MyTimeSequence.Active
