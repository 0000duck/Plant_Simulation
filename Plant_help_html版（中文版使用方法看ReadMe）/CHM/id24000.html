doStandardExport
2D Reference Help > Resource Objects > Broker > Methods of the Broker > doStandardExport

Syntax:

 
Path.doStandardExport(Exporter:object)

You can use the method doStandardExport in the exporter request control of the Broker.

Parameter

The parameter Exporter of data type object designates the Worker/Exporter.

Example:

 
MyBroker.doStandardExport(MyExporter)
