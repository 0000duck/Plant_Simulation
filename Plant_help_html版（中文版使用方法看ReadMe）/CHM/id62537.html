startElement
2D Reference Help > Information Flow Objects > XMLInterface > Methods of the XML Interface > startElement

Syntax:

 
Path.startElement(Element:string) → boolean

The method startElement designates the start of the item designated by the parameter of data type string. It applies when you sequentially access data.

Parameter

The parameter Element of data type string designates the start of the item.

Return Value

The return value has the data type boolean.

Example:

 
MyXMLInterface.startElement("catalog")
   MyXMLInterface.startElement("book")
