Theta
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Attributes of the Distributions > Theta

Syntax:

 
Attribute_path.Theta

The attribute Theta designates an unknown parameter in the following distributions. It is a number greater than 0.

Data Type of the Value You Can Assign

You can assign a value of data type real.

You can assign the following attributes/parameters to these distributions:

Distribution

Set of parameters

Range of values for the parameters

Cauchy Distribution

Stream, Mu, Theta, LowerBound, UpperBound

Mu can be any value, Theta > 0, Upper Bound > Lower Bound

Fréchet Distribution

Stream, Mu, Theta, LowerBound, UpperBound

Mu > 0, Theta > 0, Upper Bound > Lower Bound

Gumbel Distribution

Stream, Mu, Theta, LowerBound, UpperBound

Mu > 0, Theta > 0, Upper Bound > Lower Bound

Paralogistic Distribution

Stream, Alpha, Theta, LowerBound, UpperBound

Alpha > 0, Theta > 0, Upper Bound > Lower Bound

Pareto Distribution

Stream, Alpha, Theta, LowerBound, UpperBound

Alpha > 0, Theta > 0, Upper Bound > Lower Bound
