Capacity
2D Reference Help > Mobile Objects > Transporter > Dialog Window of the Transporter > Tab Load Bay > Capacity

When you select the loading space of Type Track or Line, Plant Simulation enables the text box Capacity.

Enter the capacity, i.e., the maximum number of MUs that may be located on the loading space in their entirety or in part. Enter -1 for an infinite capacity.
