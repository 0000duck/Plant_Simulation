Browse the Help File
Using HTML Help > General Topics > Browse the Help File

To browse the help file:

  • Click in a page in the Display pane to move to the Next page in the sequence of pages on the table of contents.

  • Click in a page in the Display pane to move to the Previous page in the sequence of pages on the table of contents.

  • Click to show the last topic you viewed.

  • Click to show the next topic in a previously displayed sequence of topics.
