openWrite
2D Reference Help > Information Flow Objects > XMLInterface > Methods of the XML Interface > openWrite

Syntax:

 
Path.openWrite → boolean

The method openWrite opens the XML file, which you want to write sequentially. It applies when you sequentially access data.

The method encodes the xml file in UTF-8 instead of in UTF-16. This allows them to be treated as ASCII files provided only characters from the ASCII character set are used.

Return Value

The return value has the data type boolean.

Example:

 
MyXMLInterface.filename := "D:\MSXML 4.0\writeSequ.xml"
    -- opens the XML document for sequential writing
    MyXMLInterface.openWrite
    MyXMLInterface.startElement("catalog")
         MyXMLInterface.startElement("book")
