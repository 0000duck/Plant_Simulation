Show Profile
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Debugger Ribbon Tab > Show Profile > Show Profile

The command Show Profile opens the Profile window, where you can inspect the results of a Profiler run.

  • It displays the date and time of the simulation run in the first line.

  • It displays the name and the path of the model with which the simulation run was executed in the second line.

  • The row Total elapsed time shows the entire time consumed by the simulation run, taking into account all activities of Plant Simulation. As other programs are also executed by your computer simultaneously, the time shown here is less than the time that really elapsed.

  • The row Total elapsed time for method execution shows the time, which the Interpreter used, i.e., the time for executing the methods in the simulation model. Only this time can be reduced by optimized programming of your Methods.

  • The table Top 50 Most Relevant Methods shows a global overview over all methods activated during the simulation.

  • The table Top 50 Call Cycles lists every instance, including the time consumed for the caller and the methods called.

  • To print the profile, click Print All.

  • To save the profileas an HTML file or as a text file, click Save.

  • To update the profilewith the most up-to-date data from the simulation run, click Refresh.
