Shift
2D Reference Help > Resource Objects > ShiftCalendar > Dialog Window of the ShiftCalendar > Tab Shift Times > Shift

Enter the names of the different shifts into these cells. You might, for example, enter Morning, Evening, Graveyard, etc.

The ShiftCalendar displays this icon for the day shift, and this icon for the night shift. It displays this icon when it is waiting, and this icon when it is paused.
