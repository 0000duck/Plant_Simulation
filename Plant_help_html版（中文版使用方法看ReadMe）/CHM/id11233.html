autoexec
SimTalk Reference > General SimTalk Functions > Names > Predefined Names > autoexec

Plant Simulation calls the method named autoexec immediately after opening the model. if it is located in the Class Library, immediately after opening the simulation model. You can create and use any number of autoexec methods.

To make the ExperimentManager run models in a batch run, you can start the experiment in the experiment study in the autoexec method with the method startExperiment. Create the autoexec method in the Class Library, and enter the following source code: 

-- the name of the method is autoexec
.Models.MySimulation.ExperimentManager.startExperiment
