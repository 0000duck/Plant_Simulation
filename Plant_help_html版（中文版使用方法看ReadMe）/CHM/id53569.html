Y
SimTalk Reference > General SimTalk Functions > Data Types > Array > Methods and Attributes of Arrays > Y

The attribute Y applies to one-dimensional arrays, which are of data type integer, real, or length and have a fixed dimension of three elements. The attribute Y points to the element of the array with the index 2.

The attribute Y applies to one-dimensional arrays, which are of data type integer, real, or length and have a fixed dimension of two elements. The attribute Y points to the element of the array with the index 2.

Example:

 
var c: real[3] := Station.Coordinate3D
print c.Y -- prints c[2]
c.Y := 42  -- c[2] := 42
