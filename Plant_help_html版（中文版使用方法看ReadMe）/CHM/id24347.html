LoadBayAcceleration
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > LoadBayAcceleration

Syntax:

 
MU-path.LoadBayAcceleration

The attribute LoadBayAcceleration sets the speed with which the loading space of type Line accelerates on the loading space of the Transporter.

Data Type of the Value You Can Assign

You can assign a value of data type acceleration.

Example:

 
.MUs.Transporter:1.LoadBayType := "Line"
.MUs.Transporter:1.LoadBayAcceleration := 10
