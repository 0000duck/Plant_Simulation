Insert Shape

When inserting a graphic into an external graphic group, and when external graphics were not visible before, 3D shows external graphics. When the graphic group was not visible before, it also shows the graphic group.

3D automatically displays external graphics:

  • As soon as inserting the newly generated shape into an external graphic group starts, i.e., as soon as the shape is attached to the mouse pointer and its insertion position still has to be determined. When you cancel inserting the shape by pressing the Esc key or clicking the right mouse button, 3D hides external graphics again, provided they were invisible before as well.

  • When you imported graphics from a file into an external graphic group, i.e., after you determined the insertion position and the coordinate system, and an external graphic group.

  • When you dropped a graphics file from the File Explorer into an external graphics group, i.e., after you determined the insertion position and the coordinate system, and an external graphic group.

The basic 3D shapes are graphics, which you can, for example, use to replace the standard graphic of your stations.
