Direction of the animation line
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Dialog Window of the Mobile Objects > Tab Graphics > Direction of the animation line

Select the direction of the animation line on the vector graphic of the object Transporter: Lengthwise or Crosswise in relation to the direction of movement.

This setting is only available for the load space Types Track and Line.
