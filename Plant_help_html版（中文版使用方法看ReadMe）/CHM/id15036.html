Active
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Defining Failures of the Material Flow Objects > Attributes of the Failure Profiles > Active

Syntax:

 
Path.Failures.NameOfFailureProfile.Active

The attribute Active sets if this failure profile designated by NameOfFailureProfile is Active (true) or not (false).

The failure of the object is only active, when you activate this attribute, i.e., the failure profile itself, and the attribute FailureActive of the object as such!

Data Type of the Value You Can Assign

You can assign a value of data typeboolean.

Example:

 
MyStation.Failures.MyFailureProfile1.Active := true
