moveToObject
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Methods of the MultiPortalCrane > Methods of the Portals > moveToObject

Syntax:

 
Portal-path.moveToObject(TargetObject:object) → integer

The method moveToObject moves the portal to the designated target object. The position is defined by the position of the object. You can, for example, get it by querying the x-position and the y-position of the target object.

Parameter

The parameter TargetObject of data type object designates the target object.

Return Value

The return value has the data type integer. The value is 0 if the designated position can be reached. If the designated position cannot be reached the value will be greater than 0.

The portal changes its state to driving. When it has reached the designated position, the state of the portal changes to waiting.

Example:

 
MyPortal.moveToObject(MyObject,1)
