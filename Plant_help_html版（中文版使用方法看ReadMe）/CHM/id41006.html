Tab Display
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Display > Tab Display

The tab Display shows several drop-down lists from which you select a Category, the Chart type for displaying the records, a 3D effect, if the Chart shows data as a Graph/table, if it shows Data in rows or in columns, and if it shows the contents of the Chart window instead of the icon of the object in the Frame (Display in frame).

With the check box Gap when null you can set, if the Chart interrupts the display of the data, when a value is not defined.

When you select the Category Histogram, you can select to show the values Accumulated.

When you select the Category Plotter, you can select to show the values as a Step curve.
