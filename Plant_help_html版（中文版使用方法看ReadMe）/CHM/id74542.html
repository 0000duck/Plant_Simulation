An example of multiple drag-and-drop might look like this:

param a: object[]
for var i := 1 to a.Dim 
   print "i=", i, " : ", a[i]
next

When we select the objects Station, Station1, and ParallelStation in the Frame window, drag it onto the icon of the Method and drop it there, the Console shows this: 

i=1 : .Models.Frame.Station
i=2 : .Models.Frame.ParallelStation
i=3 : .Models.Frame.Station1
