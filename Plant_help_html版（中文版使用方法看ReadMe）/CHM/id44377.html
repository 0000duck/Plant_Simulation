Rear
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Sensors of the Length-Oriented Objects > Rear

Syntax:

 
Path.Sensor.Rear

The attribute Front sets if the object designated by Path calls the Method you entered in the attribute Ctrl, when the rear of the MU passes the sensor (true) or not (false).

Creating and manipulating sensors via SimTalk for the Transporter only works if you select the loading space Types Track or Line!

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTrack.sensorID(2).Rear := true
MyTwoLaneTrack.A.sensorID(2).Rear := true

Example:

 
.MUs.Transporter:1.sensorID(2).Front := false
.MUs.Transporter:1.sensorID(2).Rear := true
