Stopped
2D Reference Help > Resource Objects > LockoutZone > Attributes of the LockoutZone > Stopped

Syntax:

 
Path.Stopped

The attribute Stopped sets if the object designated by path is stopped (true) or if it is not stopped (false). Stopped means that all of the processing operations of the objects, which the LockoutZone controls, come to a halt.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyLockoutZone.Stopped:=true
