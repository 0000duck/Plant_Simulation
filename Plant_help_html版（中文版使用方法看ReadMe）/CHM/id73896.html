2D Reference Help > Display and User Interface Objects > Display > Dialog Window of the Display > Tab Data > Path

Click this and select the attribute or method in the dialog Select Object, whose value the Display displays.

If you want to show the value of a global variable, you can enter the path to the Variable itself. You can also enter an expression, such as buffer.NumMU+1.

The Display supports the data types boolean, integer, real, string, object, time, money, length, weight, speed, date and dateTime. When the path you enter is invalid, the background of the text box changes to red.
