setCurrentDirectory
SimTalk Reference > General SimTalk Functions > Predefined Functions > Operating System Functions > setCurrentDirectory

Syntax:

 
setCurrentDirectory(WorkingFolder:string) → boolean

The function setCurrentDirectory sets the current working folder for Plant Simulation.

If you deactivate the safety setting File > Model Settings > General > Access to the computer is permitted the function setCurrentDirectory can copy data from another folder and write them to the model folder or its sub-folders. The cannot copy files from the model folder or its sub-folders.

Parameter

The parameter WorkingFolder of data type string designates the working folder.

Return Value

The return value has the data type boolean and tells if the working folder was set (true) or not (false).

Example:

 
print setCurrentDirectory("C:\users\hank")
