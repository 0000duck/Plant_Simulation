Built-in Attributes and Read-Only Attributes for Accessing Graphic Groups
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Read-Only Attributes for Accessing Graphic Groups > Built-in Attributes and Read-Only Attributes for Accessing Graphic Groups

Plant Simulation provides these attributes for accessing graphic groups in 3D:

_3D.GraphicGroupNames, _3D.LockedGraphics, and _3D.InheritGraphicVisibilities.

You can view and access the methods for accessing graphic groups by clicking Auto Complete on the Edit ribbon tab of the method editor.
