Paused
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Statistics of the Mobile Objects > Product Statistics of the MUs > Paused

The MUis paused, when it is located on a Paused or an Unplanned resource.

Do not confuse the paused state of the MU with the paused state of the drive control of the Transporter. The read-only attribute ProdOnPaused, returns the state paused of the MU (boolean).
