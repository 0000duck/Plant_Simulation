_3D.setGraphicName
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.setGraphicName

Syntax:

 
Path._3D.setGraphicName(GraphicPath:any, Name:string [,GraphicGroupName:string])

The method _3D.setGraphicName sets the name of the graphic of the object designated by Path.

The method deactivates graphic inheritance.

Parameters

  • The parameter GraphicPath of data type any sets the path to the graphic. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The parameter Name of data type string designates the name of the graphic group you are working with.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Example:

 
var j:integer := 3
obj._3D.setGraphicName([1,7],to_str("P",j))

Example:

 
var j:integer := 3
obj._3D.setGraphicName([1,7],to_str("P",j), "deco")
