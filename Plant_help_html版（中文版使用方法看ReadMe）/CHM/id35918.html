Traffic
2D Reference Help > Material Flow Objects > TwoLaneTrack > Attributes of the TwoLaneTrack > Traffic

Syntax:

 
Path.Traffic

The attribute Traffic sets the side of the TwoLaneTrack designated by Path on which the bidirectional traffic moves: Right-hand traffic or left-hand traffic.

Right-hand traffic

Left-hand traffic

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyTwoLaneTrack.traffic := "left-hand traffic"
