Insert a Comment into Your Report
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Insert a Comment into Your Report

Enter two hyphens -- to start a comment which extends to the end of the line. The comments will be ignored when generating the HTML page, meaning they do not affect the look of the HTML page. The comment is not part of the .htm file when you save the HtmlReport.
