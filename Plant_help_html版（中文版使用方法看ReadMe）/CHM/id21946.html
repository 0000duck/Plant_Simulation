copyFromTable
SimTalk Reference > General SimTalk Functions > Data Types > Array > Methods and Attributes of Arrays > copyFromTable

Syntax:

 
copyFromTable([SourceRange:listrange,] SourceTable:table)

The method copyFromTable copies the table designated by the parameter of data type table into an array.

When the array has a fixed size, the dimension of the table or of the range of the table has to match the size of the array. When you copy the table into a one-dimensional array, the values have to be contained in a row of the table.

Parameters

  • The parameter SourceTable of data type table designates the table from which you would like to copy data.

  • The optional parameter SourceRange of data type listrange designates the range which you want to copy from the table.

Example:

 
targetArray.copyFromTable(MyTableFile)
