GetLastDestination
2D Reference Help > Material Flow Objects > PickAndPlace Robot > Read-Only Attributes of the PickAndPlaceRobot > GetLastDestination

Syntax:

 
Path.GetLastDestination

The read-only attribute GetLastDestination returns the last destination object on which the pick-and-place robot designated by Path deposited the part or picked up the part.

Return Value

The return value has the data type object.

Example:

 
print MyRobot.GetLastDestination 
-- might return .Models.MyRobot.Source
