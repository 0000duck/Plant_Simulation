ParentSelection
Add-Ins Reference Help > Genetic Algorithms > GAOptimization > Attributes of the GAOptimization > ParentSelection

Syntax:

 
Path.ParentSelection

The attribute ParentSelection sets how the GAOptimization selects the parents for the next generation: Deterministic or Random. For deterministic parent selection the parents are selected strictly according to their fitness values. For stochastic parent selection the fitness value set the probability used for determining the parents randomly.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
GAOptimization.ParentSelection := "Random"
