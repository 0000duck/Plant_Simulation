State Transition
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Energy > State Transition > State Transition

To open the dialog in which you can select the state transition settings at the Start and the End of a Pause and if you want to Power up early or not.

You can also select the state transition settings at the Start and the End of the Unplanned state and if you want to Power up early or not.
