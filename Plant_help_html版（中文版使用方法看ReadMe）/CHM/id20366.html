getRouteLength
2D Reference Help > Resource Objects > Workplace > Methods of the Workplace > getRouteLength

Syntax:

 
Path.getRouteLength(ToWorkplace:path [, FreelyWithinArea:boolean:=false) → length

The method getRouteLength computes the length of the route which a Worker would have to cover to get from a Workplace to another Workplace or to the Workerpool.

Parameters

  • The parameter ToWorkplace of data type path designates the Workplace, which is the starting point of the route.

  • The optional parameter FreelyWithinArea of data type boolean sets if the Worker gets to the Workplace by walking freely within the area or not. If false or if the parameter is not passed, Plant Simulation calculates the route length between the two Workplaces via Footpaths connected with Connectors.

    If true is passed, Plant Simulation calculates the route length in the area, i.e., the route of the Worker, who is moving freely within the area, would take.

Data Type of the Value You Can Assign

The return value has the data type length.

Example:

 
var len := Workplace1.getRouteLength(Workplace2)
