Show Outline
2D Reference Help > Information Flow Objects > Method > The Method Window > Tools Ribbon Tab > View > Show Outline

The command Show Outline shows the outlining of a sequence of source code lines, such as control and loop structures, as an expansible and collapsible structure. To show the collapsed structure again, reselect the command.

When you change this setting, the change applies during the entire Plant Simulation session for all newly opened Methods.
