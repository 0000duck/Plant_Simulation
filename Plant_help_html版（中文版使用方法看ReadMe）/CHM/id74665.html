Exiting MUs
2D Reference Help > Material Flow Objects > DismantleStation > Dialog Window of the DismantleStation > The View Menu > Exiting MUs

The menu command Exiting MUs opens a list displaying the MUs located at present on the DismantleStation, and which it deletes when it moves the main part onto the successor contained in column 2 of the table.
