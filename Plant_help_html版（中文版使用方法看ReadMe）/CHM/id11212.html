CurveWidth
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Curved Objects > CurveWidth

Syntax:

 
Path.CurveWidth

The attribute CurveWidth sets the width of the curved object designated by Path. You can enter a number between 1 and 500 pixels.

You can only enter the CurveWidth when you deactivate TransferLengthToObject. When you activate it, Plant Simulation computes the width in meters.

Transfer length on

Transfer length off

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyFootpath.TransferLengthToObject := false
MyFootpath.CurveWidth := 44 -- pixels
