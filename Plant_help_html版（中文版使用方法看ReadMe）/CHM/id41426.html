Powering up/down
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Statistics of the Material Flow Objects > Resource Statistics > Powering up/down

A material flow resource is powering up or down, when it changes its energy state.

Plant Simulation only records this value if the station is not failed, not paused, not stopped, or not unplanned.

The LED of a station, which changes its energy state, is purple .

The Statistics Report shows the Powering up/down Time below Resource Statistics of Stationary Resources.
