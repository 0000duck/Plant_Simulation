maxAttr
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Indirectly Accessing Lists and Tables > maxAttr

Syntax:

 
Path.maxAttr(Range:listrange, ..., AttributeName:string) → real

The method maxAttr returns the maximum of the designated attribute of all objects in the list to the designated value. Plant Simulation sets the file cursor into the cell in which the object with the maximum attribute value is located.

The method applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Parameters

  • The parameters Range of data type listrange designate the range of the list.

  • The parameter AttributeName of data type string designates the name of the attribute.

Return Value

The return value has the data type real.

Syntax:

 
-- search all entries within the specified range on the table and determine
-- the maximum of the user-defined attribute Time
MyTableFile.setCursor(1,1)
print MyTableFile.maxAttr({1,2}..{3,5},"time")
