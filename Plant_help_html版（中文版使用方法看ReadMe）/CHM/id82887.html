GetFrontWagon
2D Reference Help > Mobile Objects > Transporter > Read-Only Attributes of the Transporter > GetFrontWagon

Syntax:

 
MU-path.GetFrontWagon

The read-only attribute GetFrontWagon returns the Transporter which is hitched to the front of the active Transporter, which can be a trailer or the tractor.

Return Value

The return value has the data type object.

Example:

 
print .MUs.Transporter:2.GetFrontWagon -- returns .MUs.Transporter:1
