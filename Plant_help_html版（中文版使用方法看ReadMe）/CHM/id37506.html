setEditorRightsRow
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Rows > setEditorRightsRow

Syntax:

 
Path.setEditorRightsRow(Row:any [Row:any, ...], ReadOnly:boolean)

The method setEditorRightsRow sets the editing permissions of the designated row(s) of the TimeSequence designated by Path to the designated value.

Parameters

  • The parameter Row of data type any designate the row(s).

  • The parameter ReadOnly of data type boolean designates read-only access (true) or read and write access (false).

Example:

 
timeSequence1.setEditorRightsRow(1,true)
