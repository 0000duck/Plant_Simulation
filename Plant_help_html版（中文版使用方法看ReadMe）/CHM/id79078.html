AttributeType
2D Reference Help > Material Flow Objects > Turnplate > Attributes of the Turnplate > AttributeType

Syntax:

 
Path.AttributeType

The attribute AttributeType sets the Attribute type of the Turnplate for the Strategy > MU Attribute. This attribute sets the rotation angle by which the Turnplate rotates the part.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyTurnplate.AttributeType := "String"
