User-defined control
2D Reference Help > Tools > TransferStation > Dialog Window of the TransferStation > Tab Advanced Controls > User-defined control

Enter the name of the Method , which will be called before the parts are moved onto the TransferStation and before the parts leave the TransferStation again. Or click , navigate to the Frame in which you stored the Method and select it there. A user-defined control might look like this:

Example:

 
(i : integer)
is
do
   if i = 1 then
      -- before starting the transfer to the TransferStation
      -- parts/target are ready
   elseif i = 2 then
      -- before parts leaving the TransferStation
   end
end
