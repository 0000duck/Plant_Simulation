_3D.exportAsS3D
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > _3D.exportAsS3D

Syntax:

 
Path._3D.exportAsS3D(FilePath:string)

The method _3D.exportAsS3D exports the graphic data and the animation data of the designated 3D object as an .s3D (3D File) file. It exports graphics which are part of the object as well as embedded graphics.

Parameter

The parameter FilePath of data type string designates the path to the target file.

Example:

 
.MaterialFlow.Frame._3D.exportAsS3D("D:\MyS3D\Frame.s3d")
