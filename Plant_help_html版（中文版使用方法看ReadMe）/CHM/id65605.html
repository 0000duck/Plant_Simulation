MainMU
2D Reference Help > Material Flow Objects > DismantleStation > Attributes of the DismantleStation > MainMU

Syntax:

 
Path.MainMU

The attribute MainMU sets the number of the successor to which the DismantleStation moves the main MU.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyDismantleStation.MainMU := 1
