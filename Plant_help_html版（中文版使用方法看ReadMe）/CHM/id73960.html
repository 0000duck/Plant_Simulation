_3D.countGraphics
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.countGraphics

Syntax:

 
Path._3D.countGraphics(GraphicPath:any[] [,GraphicGroupName:string])

The method _3D.countGraphics counts the direct child graphics of the graphic of the object designated by Path.

Parameter

  • The parameter GraphicPath is an array of data type any into which the method writes the child graphics. Here you can also enter an empty path, i.e., the expression []. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Example:

 
var count : integer := MyStation._3D.countGraphics([1,2,4]) 
for var i := 1 to count 
   MyStation._3D.setGraphicName([1,2,4,i],to_str("P",i))
next

Example:

 
var count : integer := MyStation._3D.countGraphics([1,2,4], "deco") 
for var i := 1 to count 
   MyStation._3D.setGraphicName([1,2,4,i],to_str("P",i))
next
