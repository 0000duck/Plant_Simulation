BackgroundColor
2D Reference Help > The Plant Simulation Program Window > Display Panel > Attributes of the Display Panel > BackgroundColor

Syntax:

 
Path.DisplayPanel.BackgroundColor

A 3D Only model does not provide the attribute DisplayPanel.BackgroundColor.

The attribute BackgroundColor sets the background color of the display panel of the object designated by Path.

You can set the RGB values of the color with the method makeRGBValue. To not show the background color, you can make it transparent by assigning the RGB value 0, 128, 128.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
Station.DisplayPanel.BackgroundColor := makeRGBValue(100,100,100)
Station.DisplayPanel.BackgroundColor := 6579300 -- is the same as the color above
