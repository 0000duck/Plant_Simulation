str_to_dateTime
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Times and Dates > str_to_dateTime

Syntax:

 
str_to_dateTime(string) → dateTime

The function str_to_dateTime converts the date-statement of the parameter designated by the parameter of data type string into a statement of data type dateTime. The year either has two or four digits. The function also takes leap years into account.

Conversion depends on the setting you selected under File > Model Settings/Preferences > Units > Time scale.

Parameter

The parameter DateTimeStatement of data type string designates the date and the time that are going to be converted.

Return Value

The return value has the data type dateTime.

Example:

 
print str_to_dateTime("01.12.98 14:00:00.00")
print str_to_dateTime("01.01.2000 0:00:00.0000")
