onLaneB
2D Reference Help > Mobile Objects > Transporter > Methods of the Transporter > onLaneB

Syntax:

 
MU-path.onLaneB([Position:length]) → boolean

The method onLaneB returns if the Transporter is located on lane B of a TwoLaneTrack (true) or not (false).

Parameter

The optional parameter Position of data type length designates the position starting from the front of the Transporter. When you do not enter the parameter, Plant Simulation uses the reference point. This parameter is of importance, when the Transporter stretches across several Tracks.

Return Value

The return value has the data type boolean.

Example:

 
print .MUs.Transporter:1.onLaneB
