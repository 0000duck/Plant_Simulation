setTable
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods for Defining Failures of the Material Flow Objects > setTable

Syntax:

 
Path.setTable(FailuresTable:table)

The method setTable sets the failure profiles of the object designated by Path.

You can also define them on the Tab Failures.

The List of Failure Profiles looks like this:

Parameter

The parameter FailuresTable of data type table designates the TableFile in which you defined the failure profiles.

Example:

 
MyMachine.Failures.setTable(TableFile1)
