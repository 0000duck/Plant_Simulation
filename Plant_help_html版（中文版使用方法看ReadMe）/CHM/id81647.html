Start time
2D Reference Help > Material Flow Objects > EventController > Dialog Window of the EventController > Dialog Window of the Event Debugger > Breakpoints > The Dialog Breakpoint > Start time

Enter the time, at which the breakpoint will be activated.

If you do not enter a value for the start time and/or the stop time, no limits apply for the interval during which the breakpoint is active.
