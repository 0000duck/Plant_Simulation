Import Graphics
3D Reference Help > 3D Reference > Edit Ribbon Tab > Import Graphics

The command Import Graphics imports graphics. It opens the dialog Import 3D Graphics. Select drive and folder from the drop-down list and select or enter the name of a 3D object or of a 3D graphic which you created in a 3D modeling program or of 3D model file.

This command imports the file you select into the open scene into the displayed level of your model file, making the imported file part of your active model.

3D uses the file name of the graphic as the name of the graphic node.

3D can import these file formats:

  • All 3D Graphic Files

  • JT Files (*.jt)

  • Parasolid Text Files ( *.x_t , *.xmt_txt )

  • Parasolid Binary Files ( *.x_b ; *.xmt_bin )

  • SolidEdge files (*.asm, *.par, *.psm)

  • Catia V4 Files ( *.exp ; *.model )

  • IGES Files ( *.igs ; *.iges )

  • STEP Files ( *.stp ; *.step )

  • VRML Files (*.wrl)

  • CAD Layout Files ( *.dgn, *.dwg, *.dxf )

You can only use these 3D graphics formats if you have a Professional, an Educational, a Research, an Application, or a Standard License.

You can also convert the 3D graphics formats to JT files using the program xt2jt that is installed in the Plant Simulation installation folder.

As soon as you have selected a graphic that you want to import, Plant Simulation attaches the graphic to the mouse pointer. Drag the mouse to the position at which you would like to insert the graphic. When you release the left mouse button, you can:

  • Select the graphic group to which you want to add the graphic, provided that the object has several graphics.

  • Select the positioning of the graphic:

    • As shown in the view: The graphic is positioned as shown in the preview when you drag the mouse with the attached graphic in the scene before you click the left mouse button.

    • As saved in the file: The graphic is positioned as it was saved in the imported file.

    • Axes origin: The graphic is positioned with the center of the floor of its bounding box onto the axes origin of the scene.

      Plant Simulation places the center of the bounding box of the entire robot onto the axes origin. For this reason the center of the foot of the robot does not coincide with the 3D axes origin of the Frame.

  • Select the orientation of the coordinate system of the imported graphic:

    • Z-axis of points up. This is the default setting in Plant Simulation

    • Y-axis points up.

  • Click OK to insert the graphic at the selected position with the selected orientation.
