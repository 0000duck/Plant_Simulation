Before you can enter data, click the toggle button Inheritance, so that it looks like this .

To view or to edit the attributes of an object, drag that object from the Frame window over the open dialog of the AttributeExplorer and drop it on a cell in the list. The program enters the absolute path to and the name of the object(s) into a cell of the list.

You can also select and then drag-and-drop multiple objects at the same time!

To just enter the name of the object, drag that object from the Frame window over the icon of the AttributeExplorer and drop it there.

To open the dialog of the object, press the F2 key.

To manipulate the contents of the list, use the commands on the Context Menu of Embedded Lists.
