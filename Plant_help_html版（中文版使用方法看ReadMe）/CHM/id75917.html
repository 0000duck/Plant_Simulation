DataType
Add-Ins Reference Help > Genetic Algorithms > GASequence > Attributes of the GASequence > DataType

Syntax:

 
Path.DataType

The attribute DataType sets the data type of the definition set.

Data Type of the Value You Can Assign

You can assign a value of data type string: Acceleration, Any, Boolean, Date, DateTime, Integer, Length, List, Money, Object, Queue, Real, Speed, Stack, String, Table, Time, or Weight.

Example:

 
GASequence.DataType := "integer"
