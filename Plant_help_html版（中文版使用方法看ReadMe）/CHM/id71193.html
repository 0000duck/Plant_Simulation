language
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > language

Syntax:

 
language → integer

The function language returns the model language which you selected under File > Model Settings/Preferences > General > Model language. You can use it to show language-specific messages.

0 designates German, 1 designates English, 2 designates Japanese, 3 designates Chinese, 4 designates Russian, and 8 designates Hungarian.

Return Value

The return value has the data type integer.

Example:

 
print language
