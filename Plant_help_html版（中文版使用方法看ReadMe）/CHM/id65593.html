Add-Ins Reference Help > Inter-Process Communication Interfaces > Teamcenter > Dialog Window of the Teamcenter Interface > Tab Import > Import

To log onto the Teamcenter database, click this.

In case you entered your User name and your Password on the tab Connection, Plant Simulation uses this information to login. Otherwise Plant Simulation opens the dialog Teamcenter Login. Then enter user name and your password for Teamcenter into the dialog that opens.

In case you entered an Application interface and a Sync, Plant Simulation uses this information to import data from Teamcenter. Otherwise Plant Simulation opens a dialog which shows all available application interfaces. Select one and then select a sync that belongs to this application interface.

  • If you are using Active Workspace, Plant Simulation opens a dialog, in which you can navigate to the application interface.

  • If you are not using Active Workspace, Plant Simulation opens a dialog which shows all available application interfaces. Select one of them.

    Double-click an entry in the list application interface to show the respective sync.

Click OK to retrieve the respective data from the Teamcenter database.

In case you only entered an application interface and no sync, Plant Simulation only opens the dialog in which you can select the sync.

Plant Simulation imports the data into the TableFile, which you entered as the Table. Note that the table contains the data in a nested structure. To open the next level of data in a sub-table, double-click an entry in the respective cell.
