getEditorRightsRow
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Rows > getEditorRightsRow

Syntax:

 
Path.getEditorRightsRow(Row:any) → boolean

The method getEditorRightsRow returns the editor rights of a row of the TimeSequence designated by Path.

Parameter

The parameter Row of data type any designates the row whose background color you want to get.

Return Value

The return value has the data type boolean: true for read-only access, false for read and write-access.

Example:

 
print timeSequence1.getEditorRightsRow(1)
