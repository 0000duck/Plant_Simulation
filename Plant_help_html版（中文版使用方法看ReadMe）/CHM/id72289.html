Position
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Dialog Window of the Mobile Objects > The View Menu > Position

The menu command Position opens a table with one or three columns. Column 1 shows the object on which the MU is located, column 2 shows the position of the front, column 3 shows the position of the rear of the MU. Remember that, depending on its length, the MU may be located on several length-oriented material flow objects--Conveyors or Tracks--at the same time.
