Welcome to Tecnomatix Plant Simulation, the standard software for object-oriented, graphical, and integrated modeling, for simulating and visualizing systems and business processes.

You can extend the functionality of Plant Simulation by purchasing a number of add-in programs and application object libraries, which cover common modeling tasks.

Plant Simulation runs on the 64-bit version of the operating systems Windows 7 with service pack 1, Windows 8.1, and Windows 10.

The current Plant Simulation version can open simulation models (.spp files) that were created in SiMPLE++ 7.0 or higher.

In models that you created in a version older than Plant Simulation 9.0 the 3D part of the model will not be loaded any more. You can load the model in version 9.0, 10.0, or 10.1 and then save it there though.
