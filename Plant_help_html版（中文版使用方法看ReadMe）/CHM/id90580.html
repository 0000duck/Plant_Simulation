open
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > open

Syntax:

 
Path.open
Path.open(X:integer, Y:integer)

The method open opens the dialog window of the Dialog you defined at the position designated by the parameters integer and integer. If you do not enter the parameters, Plant Simulation opens the dialog window at the position you entered as Position > X-position and Y-position.

Example:

 
MyDialog.open
MyDialog.open(100,100)
