Show Completion Tooltip
2D Reference Help > Information Flow Objects > Method > The Method Window > Tools Ribbon Tab > View > Show Completion Tooltip

The command Show Completion Tooltip activates or deactivates if Plant Simulation shows a tooltip, when the Auto Complete feature reaches a point from which on the name is unique.

This way you can press Ctrl+Spacebar without having to type in too many letters.

When you change this setting, the change applies during the entire Plant Simulation session for all newly opened Methods.
