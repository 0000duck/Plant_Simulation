NumberOfPortals
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Attributes of the MultiPortalCrane > Attributes of the MultiPortalCrane Proper > NumberOfPortals

Syntax:

 
Path.NumberOfPortals

The attribute NumberOfPortals sets the number of portals that can move on the same runway at the same time.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyPortalcrane.NumberOfPortals := 2
