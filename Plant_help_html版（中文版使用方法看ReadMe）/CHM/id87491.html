getLibraryFiles
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > getLibraryFiles

Syntax:

 
getLibraryFiles(FolderPath:string) → string[]

The function getLibraryFiles returns the file paths to .lib files and all versions.

Parameter

The parameter FolderPath of data type string is the path of a library folder. The library does not have to exist in the currently loaded model.

Return Value

The return value is an array of data type string. It contains all file paths to .lib files and all versions. The array always contains a files path and the associated library version alternating. The array is sorted descending according to version number. In case no .lib file is found for the library, the returned array is empty.

Example:

 
var arr: string[] := getLibraryFiles(".Tools.BottleneckAnalyzer")
print "Number of library files: ", arr.dim / 2
print "Newest file: ", when arr.dim > 0 then arr[1] else "(not found)"
var i : integer := 1
while i  arr.dim
   print "File: ", arr[i], "  Version: ", arr[i+1]
   i += 2
end
