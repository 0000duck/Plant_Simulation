ShowDataType
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Attributes for the Showing Settings of Lists and Tables > ShowDataType

Syntax:

 
Path.ShowDataType

The attribute ShowDataType sets if Plant Simulation shows the data type of the list object (true) or hides it (false).

The attribute applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyQueueFile.ShowDataType := false
