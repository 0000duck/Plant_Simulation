Export
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > Context Menu of Embedded Lists > Export

To export the contents of the list to a tab-delimited text file, select this. Then, select one of the following formats:

  • ANSI Text Files saves the list with ANSI encoding.

    ANSI is an 8-bit character set that allows you to represent up to 256 characters (0 through 255). The ANSI character set is a superset of the 7-bit ASCII character set.

  • UTF-8 Text Files saves the list with UTF-8 encoding.

    UTF-8 is another encoding of the Unicode character set. Each character is represented by one to three bytes.

  • Unicode Text Files saves the list with Unicode encoding.

    Unicode is a 16-bit character set that includes almost all of the written languages of the world. Plant Simulation saves Unicode in UTF-16 encoding. Each character is represented by two bytes.

  • Excel Workbooks saves the list as an Excel file with the extension .xls.

You can then Import this file into another list.
