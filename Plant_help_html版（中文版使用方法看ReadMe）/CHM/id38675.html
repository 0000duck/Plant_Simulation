setExitList
2D Reference Help > Material Flow Objects > FlowControl > Methods of the FlowControl > setExitList

Syntax:

 
Path.setExitList(ExitList:any)

The method setExitList sets the exit list of the FlowControl designated by Path for the Exit strategies Percentage or Cyclic sequence.

Parameter

The parameter ExitList of data type any designates the path to a list or a variable of the same data type. Plant Simulation then copies the contents of the passed list to the internal exit list of the FlowControl.

Example:

 
MyFlowControl.setExitList(cardFile)
MyFlowControl.setExitList(cardFile.copy)
