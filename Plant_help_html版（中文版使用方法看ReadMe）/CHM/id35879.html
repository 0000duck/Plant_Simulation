User-defined control for the parts
2D Reference Help > Tools > TransferStation > Dialog Window of the TransferStation > Tab Advanced Controls > User-defined control for the parts

Enter the name of the Method in which you defined the conditions when a part exits a point-oriented station, which provides the parts for the TransferStation. For the length-oriented objects the control will be called, when the part continues its way at the sensor. You might, for example, define conditions what will happen with the Empty container/transporter after the parts have been unloaded.

When you enter a boolean parameter in the user-defined control, Plant Simulation also calls the method, when a part arrives at the object.

User-defined controls might for example look like this:

Example:

 
-- for point-oriented objects 
(leavingPart : boolean)
if leavingPart then
   @.move
end

Example:

 
-- for the lenght-oriented object Conveyor
(leavingPart : boolean)
if leavingPart then
   @.Stopped := false
else
   @.Stopped:= true
end

Or click , navigate to the Frame in which you stored the Method and select it there.
