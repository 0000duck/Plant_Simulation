InheritFormat
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Miscellaneous Attributes of Lists and Tables > InheritFormat

Syntax:

 
Path.InheritFormat

The attribute InheritFormat sets if the list inherits its format from the class (true) or not (false ).

The attribute applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

User-defined attributes and local variables do not provide this attribute.

Example:

 
MyStackFile.InheritFormat:= true
