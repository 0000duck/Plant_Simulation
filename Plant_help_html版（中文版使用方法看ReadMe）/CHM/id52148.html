Methods for Importing and Exporting Data in Text Format
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Importing and Exporting Data in Text Format > Methods for Importing and Exporting Data in Text Format

Lists and tables provides the methods readExcelFile, readFile, readXMLFile, readXMLString, writeExcelFile, writeExcelXMLFile, writeFile, writeObjectFile, and writeXMLFile for working with list objects in text format, instead of in table (.obj) format.

To view all of the methods and attributes of the list object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.
