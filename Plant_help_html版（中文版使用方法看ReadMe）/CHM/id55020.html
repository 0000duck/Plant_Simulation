DataInColumn
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > DataInColumn

Syntax:

 
Path.DataInColumn

The attribute DataInColumn sets if the Chart shows the data in Columns (true) or in Rows (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyChart.DataInColumn := true
