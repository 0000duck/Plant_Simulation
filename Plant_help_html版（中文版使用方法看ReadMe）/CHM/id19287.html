Tab Statistics
2D Reference Help > Mobile Objects > Transporter > Dialog Window of the Transporter > Tab Statistics > Tab Statistics

The tab Statistics provides the check box Driving statistics and shows the Statistics Values of the Transporter on the tab itself.

To view Driving Statistics of the Transporter and the Statistics of the Transporters in the Statistics Report, click the object in the Frame, and press F6 (Show Statistics Report), or click Show Statistics Report on the Home ribbon tab.

You can also click the object in the Frame with the right mouse button and select Show Statistics Report on the context menu.
