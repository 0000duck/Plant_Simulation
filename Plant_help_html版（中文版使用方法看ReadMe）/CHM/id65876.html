getAlignmentRow
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Rows > getAlignmentRow

Syntax:

 
Path.getAlignmentRow(Row:any) → string

The method getAlignmentRow returns the alignment of a row of the TimeSequence designated by Path.

Parameter

The parameter Row of data type any designates the row whose alignment you want to get.

Return Value

The return value has the data type string.

Example:

 
print timeSequence1.getAlignmentRow("BurnIn")
