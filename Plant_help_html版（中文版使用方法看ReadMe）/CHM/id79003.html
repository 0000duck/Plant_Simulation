serviceStat
2D Reference Help > Resource Objects > Broker > Methods of the Broker > serviceStat

Syntax:

 
Path.serviceStat([ServiceStatistics:table]) → boolean

The method serviceStat returns Service Statistics data of the Broker designated by Path, and writes it into a table.

When a service is fulfilled by several Workers or Exporters, Plant Simulation records this time interval only once. The same applies for the count. For the time during which the Worker was en-route to his job, statistics counts the time of the Worker who was en-route the longest.

Parameter

The parameter ServiceStatistics of data type table designates the name of the table.

The column

Shows the

Services

services, which the Broker procures.

Dwelling Time: Count

how often the service in this row was procured, i.e., how often the Exporter/Exporters stayed at the Importer/importers.

Sum

sum of the times during which the service in this row was procured.

Mean Value

mean duration of the times during which the service in this row was procured.

Standard Deviation

standard deviation from the mean value of the times during which the service in this row was procured.

Min

minimum time during which the service in this row was procured.

Max

maximum time during which the service in this row was procured.

Mediation Time: Count

how often the Importer/importers were waiting for the procurement of the service in this row.

Sum

sum of the times during which the Importer/importers were waiting for the procurement of the service in this row.

Mean Value

mean duration of the times during which the Importer/importers were waiting for the procurement of the service in this row.

Standard Deviation

standard deviation from the mean value of the times during which the Importer/importers were waiting for the procurement of the service in this row.

Min

minimum time during which the Importer/importers were waiting for the procurement of the service in this row.

Max

maximum time during which the Importer/importers were waiting for the procurement of the service in this row.

Data Type of the Value You Can Assign

The return value has the data type boolean.

  • The return value is true if statistics collection is activated. It is false if it is deactivated. Then the table remains unaltered.

  • If you do not enter the optional parameter, the return value is the table containing the statistics data if statistics collection is activated. The return value is void if statistics collection is deactivated.

Example:

 
MyBroker.serviceStat(MyServicesTable) -- Writes the statistics values of the Broker to the table named MyServicesTable. 
- The return value is true if statistics collection is activated. It is false if it is deactivated. The table remains unchanged.

Example:

 
MyBroker.serviceStat -- Writes the statistics values to a table. 
- The return value is the statistics table if statistics collection is activated. It is void if statistics collection is deactivated.
