save
2D Reference Help > Display and User Interface Objects > HtmlReport > Methods of the HtmlReport > save

Syntax:

 
Path.save(FileName:string)

The method save saves the HtmlReport designated by Path as an HTML file. You can also specify the location of the file on your hard drive.

Parameter

The parameter FileName of data type string designates the path to and the name of the HTML file you want to save.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyReport.save("C:\temp\MyReport.html")
