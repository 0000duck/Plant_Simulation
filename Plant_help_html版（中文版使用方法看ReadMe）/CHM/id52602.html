getFormatString
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Columns > getFormatString

Syntax:

 
Path.getFormatString(Column:any) → string

The method getFormatString returns the Format string of the column of the TimeSequence designated by Path.

Parameter

The parameter Column of data type any designates the column whose format string you want to get.

Return Value

The return value has the data type string.

Example:

 
timeSequence1.getFormatString("paint")
print timeSequence1.getFormatString(1)
