appendRow
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Rows of the TableFile > appendRow

Syntax:

 
Path.appendRow(Value:any [,Value:any, ..., Value:any])

The method appendRow adds a new row to the TableFile designated by Path and writes the passed values to the individual columns.

If the row has a row index, Plant Simulation writes the first value into the row index.

Parameters

The parameter Value of data type any designates the value that Plant Simulation is to write to the first cell in the first free row.

The optional parameters Value of data type any designates additional values that Plant Simulation is to write to the remaining cells in the first free row.

Example:

 
MyTableFile.appendRow("My value", "My value in row 2")
