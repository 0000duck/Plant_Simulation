minAttr
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Indirectly Accessing Lists and Tables > minAttr

Syntax:

 
Path.minAttr(Range:listrange, ... AttributeName:string) → real

The method maxAttr returns the minimum of the designated attribute of all objects in the list to the designated value. Plant Simulation sets the file cursor into the cell in which the object with the maximum attribute value is located.

The method applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Parameters

  • The parameters Range of data type listrange designate the range of the list.

  • The parameter AttributeName of data type string designates the name of the attribute.

Return Value

The return value has the data type real.

Example:

 
-- cycle through all cells in the CardFile, query their processing time 
-- and return the minimum of the processing times as result 
MyCardFile.Cursor := 1
print MyCardFile.minAttr({*},"ProcTime")
table.setCursor(1,1)
print MyTableFile.minAttr({1,2}..{3,5},"time")
