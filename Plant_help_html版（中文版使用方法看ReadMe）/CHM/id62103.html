setBackgroundColorCells
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for the Format of Lists and Tables > setBackgroundColorCells

Syntax:

 
Path.setBackgroundColorCells(Range:listrange[, Range2:listrange, ...],BackgroundColor:integer)

The method setBackgroundColorCells sets the background color of the cells in a range.

The method applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Parameters

  • The parameters Rangeof data type any designate the cells, whose background color you want to set.

    You can only use the asterisk * to designate a range, if the object has a Dimension. Plant Simulation considers the object to have a dimension if you either entered the number of rows and columns in the dialog or if the object contains entries.

  • The parameter BackgroundColor of data typeinteger designates the color. You can enter 1 for Black, 2 for Red, 3 for Green, 4 for Blue, 5 for Magenta, 6 for Yellow or 7 for Cyan.

    You can set the RGB values of the color with the method makeRGBValue.

Example:

 
MyTableFile.setBackgroundColorCells({2,2}..{2,5},makeRGBValue(0,0,155))
