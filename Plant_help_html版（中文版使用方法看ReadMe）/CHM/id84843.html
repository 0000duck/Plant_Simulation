Dialog Window of the FileLink
2D Reference Help > Information Flow Objects > FileLink > Dialog Window of the FileLink > Dialog Window of the FileLink

The text boxes Name and Label are described under Dialog Items of the Objects.

Double-click the icon of the FileLink, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Information Flow in the Toolbox.

To open the dialog of the FileLink, which is linked to a file, right-click its icon and select Open on the context menu.

To open the linked file in the application that is assigned to the respective file type, double-click the icon in the Frame. This assumes that this application is installed on the computer on which you open the file.
