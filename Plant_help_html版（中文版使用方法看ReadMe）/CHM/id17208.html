setPasswordMasking
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > setPasswordMasking

Syntax:

 
Path.setPasswordMasking(DialogItem:string, Selected:boolean) → boolean

The method setPasswordMasking sets if the check box Password of a dialog item of type New Edit Text Box will be selected or cleared.

Parameters

  • The parameter DialogItem of data type string designates the check box Password .

  • The parameter Selected of data type boolean sets if the check box Password will be selected (true) or cleared (false).

When it is selected the password the Dialog how text that the user enters into the text box with superscripted lower case xes . When it is cleared it shows the password as clear text .

Return Value

The return value has the data type boolean.

Example:

 
MyDialog.setPasswordMasking("Password",true)
