_3D.setGraphicGroupInternal
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Graphic Groups > _3D.setGraphicGroupInternal

Syntax:

 
Path._3D.setGraphicGroupInternal(Name:string)

The method _3D.setGraphicGroupInternal sets that the specified graphic group of the object designated by Path is an internal group.

This selects the check box Internal on the Tab Graphics.

The method deactivates graphic inheritance.

Parameter

The parameter Name of data type string designates the name of the graphic group.

Example:

 
MyStation._3D.setGraphicGroupInternal("MyGraphicGroup")
