Select a Bitmap File to Create an Icon in the HtmlReport
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Select a Bitmap File to Create an Icon in the HtmlReport

To select a bitmap file to create an icon in this HtmlReport and to display this icon on the HTML page, click on the toolbar .

Select the bitmap file in the dialog Open.

In our example we selected the bitmap file named acceptScale. It is 31 mm wide and 6 mm high. 

[!self, "acceptScale", 31, 6]
