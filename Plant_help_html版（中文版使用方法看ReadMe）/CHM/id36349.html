Built-in Attributes, Read-Only Attributes, and Methods for Accessing Graphics
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > Built-in Attributes, Read-Only Attributes, and Methods for Accessing Graphics

Plant Simulation provides these methods for accessing graphics in 3D:

_3D.deletePlainGraphics, _3D.deletePlainGraphics, _3D.exchangeGraphic, _3D.exportAsBitmap, _3D.exportAsJt, _3D.exportAsS3D, _3D.getObject, _3D.importGraphics, _3D.InheritGraphics, _3D.InternalGraphics, _3D.optimizeByPruningTinyGraphics, _3D.optimizeByStructureFlattening, _3D.optimizeByVisibilityFilter, _3D.ShowContent, and _3D.VisibleGraphics.

You can query the values of the read-only-attributes, but you cannot set them as Plant Simulation calculates the value for the point-in-time at which you query it.

You can view and access the attributes and methods for accessing graphics by clicking Auto Complete on the Edit ribbon tab of the method editor.
