Open Start Page
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Window Ribbon Tab > Open Start Page

The command Open Start Page hides or shows the start page of Tecnomatix Plant Simulation.

On the start page you can:

  • Open the simulation model you worked with last by clicking Load Last Model.

  • Create a new simulation model by clicking Create New Model.

  • Open an existing simulation model by clicking Open Model.

  • Open one of the eight simulation models you recently worked with by clicking Recent Models. Clicking the name of a model closes the start page. Closing the model displays the start page again.

    When you drag the mouse over one of the recent models, Plant Simulation shows the path to that model.

  • Open a new, empty simulation model.

  • Open the example and information pages which contains links to information about the program. Among others you can open sample models showing how to solve modeling tasks.

  • Open one of the videos.

  • Open the online tutorial and work your way through the exercises.

  • Open the Online Help to view the list of new and changed features.

  • Open the web page of the Siemens PLM Software Community.

  • Open the Plant Simulation web page on Twitter.

  • Open the Plant Simulation web page on the Siemens web site.

When you open any of the web pages under Example Models, Plant Simulation adds them to the window tab bar. You can then click through the tabs to switch between windows.

To close one of the web pages, click the Close button on the title bar of the window.
