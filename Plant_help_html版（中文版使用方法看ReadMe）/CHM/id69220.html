Host name
Add-Ins Reference Help > Inter-Process Communication Interfaces > OPCClassic > Dialog Window of the OPC Classic Interface > Tab Attributes > Host name

Enter the name or the IP address of the host computer on which the OPC server runs. If the OPC server runs on the same computer as the OPC Classic Interface or as the PLC, you can leave this field empty.

When the OPC server is running remote on another host, you have to configure DCOM as described under Configuring DCOM for Remote Access.

The connection string that you enter here depends on your database provider. Consult the documentation of your database provider for more information.
