setColumnWidth
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > setColumnWidth

Syntax:

 
Path.setColumnWidth(Column:any [,Column:any, ...], ColumnWidth:integer)

The method setColumnWidth sets the width of one or several column of the TableFile designated by Path to the designated value.

Parameters

  • The parameters Column of data type any designate the columns.

  • The parameter ColumnWidth of data type integer designates the width in character widths of a non-proportional font.

Example:

 
MyTableFile.setColumnWidth("column12",10)
