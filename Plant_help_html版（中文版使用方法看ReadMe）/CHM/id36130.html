Feed rate
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Axes > Feed rate

For the Category > Plotter you can enter a feed rate in grid units, which sets the number of grid units by which the Chart scrolls, when it reaches the right border of the display.

You can enter a number between 0.1 and 12. Enter 0.1 for smooth scrolling, enter 12 to scroll the entire page.
