Services
2D Reference Help > Material Flow Objects > Station > Dialog Window of the Station > The View Menu > Services

The menu command Services opens a table showing the names of all imported services. The name of the failure service is identical with the name of the sub-table.

Double-click the name of the service to open the sub-table. Column 1 contains the Name of the Exporter, the column Amount the number of services provided.
