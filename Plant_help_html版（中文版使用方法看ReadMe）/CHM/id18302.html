XPos
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes of All Mobile Objects > XPos

Syntax:

 
MU-path.XPos

The attribute XPos returns the x-coordinate of the MU designated by MU-path in the Frame into which you inserted it.

You cannot set the x-position of the MU, you can only get it!

We advise against using this attribute as it only returns a meaningful value when the MUs actually is drawn, i.e., when MU Animation is active and when the Frame window is open.

Return Value

The return value has the data type integer.

Example:

 
var posit: integer
posit := Station.Cont.XPos

Example:

 
x : integer
x := .MUs.Part:1.XPos
