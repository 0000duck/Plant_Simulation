CurrentSpeed
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for the State of the Mobile Objects > CurrentSpeed

Syntax:

 
MU-path.CurrentSpeed

The read-only attribute CurrentSpeed returns the actual speed with which the MU designated by MU-path moves.

When the MU is located on a Conveyor, it always has the sign (+ or -) that corresponds to the direction of movement of the Conveyor.

Although this value can continually change because of the objects accelerating, CurrentSpeed is observable for certain values. By observing this value you might, for example, recognize when the speed of an MU changes, because it collided with the MU in front of it, because it stops at the end of a Conveyor, or because of other outside influences.

Return Value

The return value has the data type speed.

Example:

 
var MyPart: object
MyPart := MyConveyor.MU(1)
waituntil MyPart.CurrentSpeed = 0 prio 1
print "Stopped at this time:", EventController.simTime
