TransferLengthToObject
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Curved Objects > TransferLengthToObject

Syntax:

 
Path.TransferLengthToObject

The attribute TransferLengthToObject sets if Plant Simulation uses the length of the curved object designated by Path which you inserted in the layout within the Frame as the sength of the object (true) or not (false). When the dialog is open, Plant Simulation enters this value into the text box length.

When you select transfer length, the program adapts the length of the graphic, when you change the length if the object only consists of a single straight segment.

When the length cannot be exactly displayed in pixels, the program adapts the length of the graphic. If you need to use the exact length, clear transfer length.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyConveyor.TransferLengthToObject := true
