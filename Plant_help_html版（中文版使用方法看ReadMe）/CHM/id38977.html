Data type
2D Reference Help > Information Flow Objects > Variable > Dialog Window of the Variable > Tab Value > Data type

Select a data type for the Variable you are creating, compare Data Types.

  • When you select the data type object, you have to enter the path to and the name of the object into the text box Value. Or you can click and select the object in the dialog Select Object.

  • If you would like to use the data type time with a random number distribution, you have to select the data type randtime, and enter the parameters of the distribution function, as the data type Time only accepts constant values. You then have to initialize the randtime value before accessing it, for example by employing the method rollDice, which determines the next random number the Variable uses.

    For list data types (table, list, stack, queue) you can use the method unshare.
