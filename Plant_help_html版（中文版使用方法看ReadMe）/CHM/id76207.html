repeat
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Accessing the TimeSequence > repeat

Syntax:

 
Path.repeat(TimeSequence:object, Subrange:time, NumberOfCopies:real)

The method repeat deletes all entries of the TimeSequence designated by Path and replaces them with copies of a subrange of the data of another TimeSequence.

Parameters

  • The parameter TimeSequence of data type object designates the TimeSequence whose subrange will be pasted.

  • The parameter Subrange of data type time designates the subrange. It starts with the time zero and ends at the time designated by this parameter.

  • The parameter NumberOfCopies of data type real designates the number of copies of the data are to be inserted.

You might, for example, use this to create a week’s schedule from a day’s schedule by repeating it five times.

Example:

 
LengthOfDay := str_to_time("1:00:00:00.00")
weekPlan.repeat(dayPlan,lengthOfDay,5)
