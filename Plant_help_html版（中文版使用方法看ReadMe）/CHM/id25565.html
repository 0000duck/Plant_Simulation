_3D.BackgroundColor
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes for the Background Color of the Frame > _3D.BackgroundColor

Syntax:

 
 Path._3D.BackgroundColor

The attribute _3D.BackgroundColor sets the background color of the Frame designated by Path.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

If you do not want to define a background color, assign the value -1.

Example:

 
MyFrame._3D.BackgroundColor := makeRGBValue(0,0,255) -- bright blue
print MyFrame._3D.BackgroundColor -- returns 16711680
