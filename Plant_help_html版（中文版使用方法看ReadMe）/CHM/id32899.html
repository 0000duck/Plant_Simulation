_3D.TypeOfAnimations.AnimationPathName.schedule
SimTalk Reference > 3D-related SimTalk Functions > Built-in Functions for Saved Animations > _3D.TypeOfAnimations.AnimationPathName.schedule

Syntax:

 
Path._3D.SelfAnimations.AnimationPathName.schedule([,Backwards:boolean])
Path._3D.CameraAnimations.AnimationPathName.schedule([,Backwards:boolean])

The method _3D.TypeOfAnimations.AnimationPathName.schedule schedules a saved animation.

Parameter

The optional parameter Backwards of data type boolean sets if the animation on the path is to be played backwards (true) or forwards (false).

If you do not enter the parameter, the animation on the path will be played forwards.

If you want to run the animation, you have to call the function _3D.TypeOfAnimations.playAnimation.

Example:

 
self.~._3D.SelfAnimations.VerticalAni.schedule
self.~._3D.SelfAnimations.playAnimation
