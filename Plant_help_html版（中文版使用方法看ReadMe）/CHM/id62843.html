setFormula
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Rows of the TableFile > setFormula

Syntax:

 
Path.setFormula([Range:listrange, ...,] Formula:string)

The method setFormula enters the formula into the table cell(s) of the designated range of the TableFile designated by Path.

Parameters

  • The parameter Range of data type any designates the range of cells for which you want to set a formula. You can also enter several ranges.

  • The parameter Formula of data type string designates the formula.

Example:

 
MyTableFile.setFormula({1,2}..{2,3},"myFormula")
-- ‘myFormula’ is a method whose return value is the result of the formula
MyTableFile.setFormula({3,1}..{3,*}, "?[1,yself] - ?[2,yself]") 
-- column 3 results from the difference of column 1 and column 2
