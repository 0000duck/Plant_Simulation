Search for regular expressions
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Preferences > Editor > Search for regular expressions

To also find regular expressions, select this. This means that you can use wild cards in the term you want to find. You can, for example, use the period character . as a wild card for any character.

If the editor searches for a.b, it will find all strings consisting of a sequence of the character a, any character, and the character b.

Compare Show Structure for additional examples.
