_3D.SelfAnimations.deleteNextAnimationBlock
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods and Attributes for Scheduling and Playing Animations > _3D.SelfAnimations.deleteNextAnimationBlock

Syntax:

 
Path._3D.SelfAnimations.deleteNextAnimationBlock

The method _3D.SelfAnimations.deleteNextAnimationBlock deletes the entire next scheduled animation block for the object designated by Path.

The method does not change any saved animations, but only affects scheduled animations.

Example:

 
self.~._3D.SelfAnimations.deleteNextAnimationBlock
