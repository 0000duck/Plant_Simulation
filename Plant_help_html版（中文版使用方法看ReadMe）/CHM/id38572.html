_3D.ShowContent
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > _3D.ShowContent

Syntax:

 
Path._3D.ShowContent

The attribute _3D.ShowContent shows (true) or hides (false) the representing graphics of the objects that are contained within the object designated by Path. This means that all visible external graphics and the state group of all visible objects, which are located one level lower in the hierarchy than the selected object, are visible.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
Station._3D.ShowContent := true
