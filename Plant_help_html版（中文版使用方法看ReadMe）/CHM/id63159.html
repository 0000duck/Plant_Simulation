Setting Default Values in the Windows Registry
Setting-Up and Starting > Starting Plant Simulation > Setting Default Values in the Windows Registry

When you start Plant Simulation for the first time after installing it, it sets the preferences to default values.

If you are an Administrator and you would like to specify different default values, for example a certain model language for new models, you can define your company’s own preferences for all users on the computer with the registry key HKEY_LOCAL_MACHINE\SOFTWARE\Tecnomatix\eM-Plant\14.1\Preferences. Here you can then set the exact same registry values, which Plant Simulation creates in the registry key HKEY_CURRENT_USER\Software\Tecnomatix\Tecnomatix Plant Simulation 14.1\Preferences when a user changes the preferences.

The best way to set the desired settings is to select them in the dialog Preferences in Plant Simulation and to close this dialog by clicking OK.

Then start regedit.exe and export the key HKEY_CURRENT_USER\Software\Tecnomatix\Tecnomatix Plant Simulation 14.1\Preferences. Then start notepad.exe and open the .reg-file. Delete all lines containing values which you do not want to use. Then change the line [HKEY_CURRENT_USER\Software\Tecnomatix\Tecnomatix Plant Simulation 14.1\Preferences] to [HKEY_LOCAL_MACHINE\SOFTWARE\Tecnomatix\eM-Plant\14.1\Preferences] and save the file. When you double-click this file an any Windows Computer the default values for the preferences will be entered into the Registry. Plant Simulation always uses the default values when the registry key HKEY_CURRENT_USER\Software\Tecnomatix\Tecnomatix Plant Simulation 14.1\Preferences does not exist, for example when Plant Simulation is started for the first time.
