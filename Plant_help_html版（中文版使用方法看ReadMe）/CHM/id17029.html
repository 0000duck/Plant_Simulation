Capacity
2D Reference Help > Material Flow Objects > Buffer > Attributes of the Buffer > Capacity

Syntax:

 
Path.Capacity

The attribute Capacity sets the Capacity, i.e., the maximum number of MUs that may be located in the Buffer. The value -1 designates an infinite capacity.

Remember that the capacity is not implemented in a matrix, so that you cannot access any individual locations! You can only reduce the capacity, when the new capacity you enter is greater than or equal to the actual number of MUs located in the Buffer.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
Buffer.Capacity := 12
