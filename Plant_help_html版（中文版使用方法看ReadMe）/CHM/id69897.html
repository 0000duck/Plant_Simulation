Time
2D Reference Help > Material Flow Objects > Conveyor > Attributes of the Conveyor > Time

Syntax:

 
Path.Time

The attribute Time sets the Time which the MU spends moving on the Conveyor. Time divided by the Length form the speed of the object.

Data Type of the Value You Can Assign

You can assign a value of data type time.

Example:

 
MyConveyor.Time := 60
