Edit Grid Lines
Step-by-Step Help > Modeling in 3D > Working with the Grid > Edit Grid Lines

You can edit existing grid lines or add new grid lines in the group box Grid lines.

  • To add another grid line, click Add.

    Enter the distance between two grid lines, which you want to add; we entered 2 meters.

    Select a color for the grid line, we selected a bright blue.

  • To show this grid line in the grid in the scene, select Visible.

  • Then, select if objects you insert snap to this grid line. Making objects snap to grid points allows you to precisely place objects with the mouse.

  • To edit an existing grid line, select it in the list and edit the interval between the grid lines, their color, what objects, which you insert, snap to, and if this grid line is visible or not.
