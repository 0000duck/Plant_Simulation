appendPlainText
2D Reference Help > Display and User Interface Objects > HtmlReport > Methods of the HtmlReport > appendPlainText

Syntax:

 
Path.appendPlainText(PlainText:string)

The method appendPlainText appends a special character as plain text to the content of the HtmlReport designated by Path.

Parameter

The parameter PlainText of data type string designates the special character that you want to append as plain text.

Example:

 
MyReport.Content := "#Heading" + chr(10)
MyReport.appendPlainText("##**")
