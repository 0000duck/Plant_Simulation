Save/Load
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Save/Load > Save/Load

Plant Simulation only shows this command, when you select a folder.

The context menu command Save/Load provides these subcommands:

Save Folder As

Save Library As

Load Object into Folder

Load Object

Update Class Library

To save an object, right-click that object and select Save Object As .
