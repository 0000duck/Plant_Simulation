acceleration_to_num
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Physical Data Types with Units into Data Types without Units > acceleration_to_num

Syntax:

 
acceleration_to_num(acceleration) → real

The function acceleration_to_num converts the data designated by the parameter of data type acceleration into a value without a unit. This function is handy if you want to assign a local variable to another local variable which has a different unit. Normally this is forbidden. Using acceleration_to_num you can discard the unit.

Parameter

The parameter Data has the data type acceleration.

Return Value

The return value has the data type real.

Example:

 
var t: time, a: acceleration
t := acceleration_to_num(a)
