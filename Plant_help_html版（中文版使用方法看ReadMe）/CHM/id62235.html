_3D.TypeOfAnimations.EstimatedTotalAnimationTime
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods and Attributes for Scheduling and Playing Animations > _3D.TypeOfAnimations.EstimatedTotalAnimationTime

Syntax:

 
Path._3D.SelfAnimations.EstimatedTotalAnimationTime
Path._3D.CameraAnimations.EstimatedTotalAnimationTime

The attribute _3D.TypeEstimatedTotalAnimationTime returns the time which the animation of all blocks for the object designated by Path will most likely require. The time measurement relates to the simulation time

Example:

 
var t : time := self.~._3D.SelfAnimations.EstimatedBlockAnimationTime
