View the Results in a Chart
Step-by-Step Help > Modeling in 2D > Executing Simulation Experiments > Execute Experiments with the ExperimentManager > View the Results in a Chart

The ExperimentManager can generate a chart of the selected output values.

You can select to show min-max intervals or confidence intervals for the output values. Under Tools > Advanced Settings > Settings you can select of the ExperimentManager show the 25 percent, the 50 percent, and the 75 percent quartiles for the min-max intervals.

This type of display is called a box plot. It shows the distribution of the observations of an output value of an experiment.
