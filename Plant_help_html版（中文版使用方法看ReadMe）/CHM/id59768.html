ascii
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Strings > ascii

Syntax:

 
ascii(String:string [,PositionOfCharacter:integer]) → integer

The function ascii returns the Unicode character code of the n-th character of a string. If you do not enter the optional parameter PositionOfCharacter, the function returns the character code of the first character of the string.

Parameters

  • The parameter String of data type string designates the string.

  • The optional parameter PositionOfCharacter of data type integer designates the position of the character whose character code you want to know. The number 1 is the first character. The range of values varies from 1 to the length of the string.

Return Value

The return value has the data type integer. It is a number between 1 and 65535.

Example:

 
var s: string
var i,l: integer
s := "Hello World!"
l := strlen(s)
for i := 1 to l
   print chr(ascii(s, i)), ", ", ascii(s, i)
next
