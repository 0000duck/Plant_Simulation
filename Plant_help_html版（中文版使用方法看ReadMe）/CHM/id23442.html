getAttr
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Methods of the Distributions > getAttr

Syntax:

 
Path.Attribute.getAttr

The method getAttr returns the parameters to the distribution.

The return value is the contents of the text box of the respective distribution on the Tab Times of the material flow objects.

Return Value

The return value has the data type string.

Example:

 
type := Station.ProcTime.typ
SetOfParameters := Station.ProcTime.getAttr
ParallelStation.ProcTime.setTypeAndAttr(type,SetOfParameters)

The method getAttr returns the set of parameters in the format that the method setTypeAndAttr understands.
