New Static Text Box
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New Static Text Box

A static text box shows text that the user can view, but not edit.

  • Enter the Name of the static text box.

  • Enter the text the static text field displays (Caption).

  • Enter its X and Y position in the Dialog.

  • Select if you want to Enable it or not.

  • To insert the dialog item into your dialog, click OK.
