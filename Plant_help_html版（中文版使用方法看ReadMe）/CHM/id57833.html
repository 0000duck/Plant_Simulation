Stop
2D Reference Help > Material Flow Objects > Source > Attributes of the Source > Stop

Syntax:

 
Path.Stop

The attribute Stop sets the simulation time at which the Source does not produce any more MUs. A Stop time of zero indicates that the Source will keep producing MUs infinitely.

The attribute Start sets the simulation time at which the Source starts producing MUs, when you set the TimeOfGeneration to Interval adjustable.

The attribute Interval sets the time between creation events.

Data Type of the Value You Can Assign

You can assign a value of data type time.

Example:

 
MySource.MuSelection := "Interval adjustable"
MySource.Start := 3600
MySource.Stop := str_to_time("1:00:00:00.00")
MySource.Interval.setParam("Normal",500,100,200,900)
