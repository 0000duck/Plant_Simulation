_3D.TypeOfAnimations.play
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods and Attributes for Scheduling and Playing Animations > _3D.TypeOfAnimations.play

Syntax:

 
Path._3D.SelfAnimations.play
Path._3D.CameraAnimations.play

The method _3D.TypeOfAnimations.play plays all scheduled animations of the object designated by Path.

Example:

 
self.~._3D.SelfAnimations.scheduleRotation(0, 90, 15)
self.~._3D.SelfAnimations.play
