Step 3: Flatten structure
3D Reference Help > 3D Reference > Edit Ribbon Tab > Optimize Selected Graphic > Step 3: Flatten structure

This setting flattens the structure of the object while preserving its contents. It reduces the graphic structure to a minimal structure that only keeps the visualization running, i.e. the remaining structure levels are those that are needed to hold a material or a texture. If you want to keep grouping information of the selected graphic, select Keep grouping.

Be aware that the result of the optimization process might contain more than one graphic node, provided the graphic contained more than one material/texture combination.

Keep grouping

Select this to keep grouping information of the selected graphic although it might not have a special purpose.
