StatServicesSetupTime
2D Reference Help > Resource Objects > Worker > Read-Only Attributes of the Worker > StatServicesSetupTime

Syntax:

 
Path.StatServicesSetupTime

The method StatServicesSetupTime returns the total time the services of the Worker designated by Path were setting the station up.

Return Value

The return value has the data type time.

Example:

 
print .Resources.MyWorker:1.StatServicesSetupTime
