Run to Cursor
2D Reference Help > Information Flow Objects > Method > The Method Window > Method Debugger > Menu Bar of the Method Debugger > Debug Menu > Run to Cursor

The menu command Run to Cursor, or Shift+F10, continues executing the Method until it reaches the statement at which the cursor is located.

The Method Debugger shows the elapsed time in the status bar of when you execute one or several instructions in a Method with the commands Step Over, Step Into, Step Out, or Run to Cursor and when executing the instructions takes 10 milliseconds or longer.
