Syntax-controlled Indentation
2D Reference Help > Information Flow Objects > Method > The Method Window > Tools Ribbon Tab > Syntax-controlled Indentation

The command Syntax-controlled Indentation activates or deactivates the indentation of the source code you enter according to the predefined SimTalk syntax.

When it is active, Plant Simulation automatically indents the cursor when starting a new line of code according to the delimiting keywords. Plant Simulation indents, for example, the text between while and end, making it easier to follow the logic behind the source code.

When you change this setting, the change applies during the entire Plant Simulation session for all newly opened Methods.
