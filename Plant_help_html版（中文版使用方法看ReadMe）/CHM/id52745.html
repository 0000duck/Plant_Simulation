_3D.StatesPosition
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes for State Groups > _3D.StatesPosition

Syntax:

 
Path._3D.StatesPosition

The attribute _3D.StatesPosition sets the reference point of the state group of the object designated by Path.

For a newly generated state group the position setting is (-1,1,1) for horizontally arranged state graphics and (-1,1,0) for vertically arranged state graphics.

The method deactivates graphic inheritance.

Data Type of the Value You Can Assign

You can assign a value of data type array.

Example:

 
MyStation._3D.StatesPosition := [-1, 1, 1]
