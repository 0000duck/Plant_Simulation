Highlight Words in Searched Topics
Using HTML Help > Working with Full Text Search > Highlight Words in Searched Topics

To highlight words in searched topics:

  • When searching for words in help topics, the Help Viewer can highlight each occurrence of the word or phrase in the topics that it found.

  • Select Options > Search Highlight On in the toolbar to highlight all instances of a search word or phrase,

    Select Options > Search Highlight Off in the toolbar to deactivate this feature. If you are viewing a long topic, the Help Viewer will only highlight the first 500 instances of a search word or phrase.
