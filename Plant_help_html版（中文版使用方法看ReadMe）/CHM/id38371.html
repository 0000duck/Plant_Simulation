Delete objects
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Preferences > User Interface > Delete objects

To open a dialog prompting you to confirm, when you delete objects or user-defined attributes in your model with the Delete key or the menu command Delete on the context menu, select this. It is especially useful to deactivate this feature when you are cleaning up a large model.

Plant Simulation deletes Connectors connected with a material flow object without prompting you to confirm when you delete the material flow object!
