QueryTable
2D Reference Help > Information Flow Objects > AttributeExplorer > Attributes of the AttributeExplorer > QueryTable

Syntax:

 
Path.QueryTable

The attribute QueryTable sets the query table. This table matches the attributes you enter on the Tab Query.

Data Type of the Value You Can Assign

You can assign a value of data type table.

Example:

 
MyAttributeExplorer.QueryTable := myQueryTable
