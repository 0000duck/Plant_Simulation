Insert Plate
3D Reference Help > 3D Reference > Edit Ribbon Tab > Insert Plate

The command Insert Plate creates and inserts a plate into the selected Frame.

  • Select the Graphic group to which you want to add the new graphic. The list shows the graphic groups which are defined for the object.

    • Default (external): Designates the standard graphic group of the 3D objects. It initially is external and represents the object towards the outside. When you deactivate Show External Graphics, 3D hides the graphic groups designated as Default.

    • One of the user-defined graphic groups you added on the Tab Graphics. Newly created 3D Frames by default provide the graphic group Deco (internal), which are intended for graphics for decorating the interior view of the Frame.

  • Select the Type of plate which you want to create and insert into the scene into the dialog Create Plate.

  • Depending on your choice, 3D activates or deactivates one of the three text boxes for the Width and the Height of the plate on the respective axis. Enter a value into the active text box.

  • To create the plate, click Create. 3D attaches the shape to the mouse pointer. Drag the mouse to the position at which you would like to insert the shape and click the left mouse button. To cancel inserting, you can click the right mouse button or press the Esc key.

  • To move the plate to the location of your choice, click it and drag the mouse.

  • To change the size of the plate, select it and select 3D Properties > Tab Transformation. Then change the size under Scale.

  • To set if the plate is an obstacle for the worker, select it and then click Edit 3D Properties > Tab Graphic Settings on the Home ribbon tab.

  • To assign a material and a color, select the plate and click 3D Properties > Tab Material on the Home ribbon tab.
