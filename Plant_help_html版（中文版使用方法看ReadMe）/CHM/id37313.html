Broker
2D Reference Help > Resource Objects > Exporter > Dialog Window of the Exporter > Tab Attributes > Broker

Click this and select the Broker, who assigns the services to the importer, in the dialog Select Object. You might also, for example, type in .Models.Frame.MyBroker.

To open the dialog of the object, whose name you entered into the text box, press the F2 key.

Or you can select the Broker in a Frame, drag it to the text box and drop it there.

The program enters the Exporter into the list of administered Exporters, which this Broker manages. To view them, select View > Exporters.
