Tab Material
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Material > Tab Material

On the tab Material you can define the surface qualities of the selected graphic: how well it reflects light, what color it reflects, and what color it emits.

Only graphics provide this tab.

You can:

  • Make the Material active or inactive.

  • Select the Diffuse color.

  • Select the Ambient color.

  • Select the Specular color.

  • Select the Emissive color.

  • Set the Transparency.

  • Set the Shininess.

  • Copy the Current Material Settings.

  • Paste a Previously Copied Material.

  • Remove the Materials in Nested Graphics.

To activate the settings for the material, select Material active . To reset all settings of the selected object to the values the text boxes had, when you opened the dialog, clear the check box.

To apply and display your changes of the settings to the selected object in the scene window and leave the dialog window open, click Apply.

To apply your changes of the settings to the selected object and close the dialog window, click OK.

To close the dialog without applying any of the settings you modified, click Cancel.

When you define materials for a graphic in the graphic structure at different levels in the structure, 3D always uses the material that is located furthest down in the structure for the actual rendering.
