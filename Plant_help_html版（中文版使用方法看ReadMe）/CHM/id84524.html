Blocking angle
2D Reference Help > Material Flow Objects > PickAndPlace Robot > Dialog Window of the Pick-and-Place Robot > Tab Attributes > Blocking angle

Enter the blocking angle in degrees which the pick-and-place robot cannot cross while rotating. This prevents the robot from taking the shortest way. The default setting -1 means that no blocking angle applies and that the robot thus takes the shortest way. You can enter an angle between 0 and 360 degrees.

The blocking angle only affects the animation, the simulation is controlled by the rotation times which you defined in the Times Table.
