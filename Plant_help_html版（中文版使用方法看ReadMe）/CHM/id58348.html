2D Reference Help > Resource Objects > LockoutZone > Dialog Window of the LockoutZone > Tab Objects > Tab Objects

You can assign resource Objects to the LockoutZone on the tab Objects. A resource is any of the built-in Material Flow Objects, or a Frame in which you modeled a machine, whose Working hours you would like to control with the LockoutZone. In addition, you can also add a Worker or the Workerpool

You can manipulate the contents of the list with the commands of the Context Menu of Embedded Lists.
