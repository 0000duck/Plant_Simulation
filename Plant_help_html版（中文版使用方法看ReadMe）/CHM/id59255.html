[ ] (write the contents of a cell)
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Accessing the TableFile > [ ] (write the contents of a cell)

Syntax:

 
Path[Column:integer or string, Row:integer or string]

The method [,] writes data to the TableFile designated by Path.

Parameters

  • The parameter Column of data type integer or string designates the column of the cell.

  • The parameter Row of data type integer or string designates the row of the cell.

First you access the cell by its index and then you assign a value. The index statement begins and ends with a bracket. In the bracket you first enter the column and then the row of the cell to be accessed. You can use the system index or a user-defined index, which you assigned.

The data type of the return value is the same as the data type of the table cell. Note that Plant Simulation reads the contents of the cell, but does not remove it!

The data type of the new value has to be the same as the data type of the table cell.

Example:

 
MyTableFile[2,3] := 12.34 -- system index
orderList["OrderNo","Miller"] := 12.34 -- user-defined index

Plant Simulation reads the contents of the cell, but does not remove it.
