Inherit Source Code
2D Reference Help > Information Flow Objects > Method > The Method Window > Edit Ribbon Tab > Inherit Source Code

The command Inherit Source Code activates or deactivates the inheritance of the source code of a Method object which you insert into a Frame. As long as it is active, you cannot modify the source code that tells Plant Simulation what to do; the Method then uses the source code of the Method from which it is derived, i.e., its origin.
