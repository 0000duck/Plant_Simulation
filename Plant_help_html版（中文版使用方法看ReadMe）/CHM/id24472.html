Editor
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Preferences > Editor > Editor

On the tab Editor you can select the global settings that determine how Plant Simulation formats and shows SimTalk source code in all Method objects.

For individual instances of methods you can also select local settings on the Tools ribbon tab of that Method.
