goToPool
2D Reference Help > Resource Objects > Worker > Methods of the Worker > goToPool

Syntax:

 
Path.goToPool → boolean

The method goToPool sends the Worker back to the WorkerPool.

Data Type of the Value You Can Assign

The return value has the data type boolean.

If the Worker is exporting a service at the moment, he stays where he is at. In this case the method returns false.

Example:

 
.Resources.MyWorker:1.goToPool
