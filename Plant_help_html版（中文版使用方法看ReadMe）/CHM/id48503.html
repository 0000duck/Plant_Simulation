Cursor
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Attributes of Lists and Tables > Miscellaneous Attributes of Lists and Tables > Cursor

Syntax:

 
Path.Cursor

The attribute Cursor sets the cell in which the cursor is placed. Note that all methods that employ ranges, such as find or max, use the current position of the cursor. Plant Simulation ignores ranges located before the current cursor position. The method starts its task beginning at the cursor position. If a method does not return the expected values, check the cursor position.

This attribute applies to the CardFile, the StackFile, and the QueueFile.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyStackFile.Cursor := 1
MyCardFile.Cursor := MyCardFile.cursor + 1
