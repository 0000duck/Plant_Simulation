PortalAvailability
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Attributes of the MultiPortalCrane > Attributes of the MultiPortalCrane Proper > PortalAvailability

Syntax:

 
Path.PortalAvailability

The attribute PortalAvailability sets the availability of the portal.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
MyPortalcrane.PortalAvailability := 90
