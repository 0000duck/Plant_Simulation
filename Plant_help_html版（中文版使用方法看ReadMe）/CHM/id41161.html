setStationtype
2D Reference Help > Tools > TransferStation > Methods of the TransferStation > setStationtype

Syntax:Path.setStationtype

Syntax:

 
Path.setStationtype

The method setStationtype sets the type of the TransferStation: You can enter Load, Unload, Reload or Move.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyTransferStation.setStationtype("Load")
