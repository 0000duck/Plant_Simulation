Change Values in the Dialog Show Attributes and Methods
Step-by-Step Help > Getting to Know Tecnomatix Plant Simulation > Working with the Program, Basics > Changing the Settings of the Objects > Change Values in the Dialog Show Attributes and Methods

You can also access the values of the settings of the objects in the dialog Show Attributes and Methods of the respective object.

  • To open the dialog, click the object and press the F8 key.

  • Then, double-click the respective attribute.

  • Enter the new value into the dialog that opens or select a value in the drop-down list.

To toggle the value of an attribute of data type boolean from true to false and vice versa, double-click the row of the attribute in the dialog Show Attributes and Methods.
