appendList
2D Reference Help > Information Flow Objects > CardFile > Methods of the CardFile > appendList

Syntax:

 
Path.appendList(List:any)

The method appendList adds the contents of all cells of a list to the CardFile. The contents of the existing cells remains untouched.

Parameter

The parameter List of data type any designates the list you want to append.

Example:

 
var s: stack[string]
s.create
s.insert("one")
s.insert("two")
MyCardFile.appendList(s)
