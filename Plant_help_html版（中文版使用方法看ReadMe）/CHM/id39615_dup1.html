2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Graphics > Object graphics > Graphic Groups > Add > Internal

To make the graphic group only visible within its object, select this. The graphic group will then only be used for the internal decoration of the object. The external graphic groups on the other hand represent the object towards the outside.

Instead, you can also select the check box on the table on the tab itself.

Internal graphic groups are always shown inside of its object, when you also select the check box Visible; they are never visible on the outside of the object.

External graphic groups, for which you selected the check box Visible, will only be visible on the outside of the object. If you want to show them while you are editing, you can click Show External Graphics on the View ribbon tab to also make them visible internally.
