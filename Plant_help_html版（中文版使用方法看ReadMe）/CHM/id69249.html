Pen width
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Curve > Pen width

Enter the pen width of the outline around the border of the length-oriented curved objects. This can be a value between -1 and 100. The value -1 makes the outline invisible. The value 0 stands for a pen width of 1 pixel. The value 1 stands for a stroke weight of 1. This affects zooming in the Frame.

3D does not use this setting.
