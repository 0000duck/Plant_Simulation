isInteger
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > Querying the Data Type of a Variable of Data Type any > isInteger

Syntax:

 
isInteger(Argument:any) → boolean

The function isInteger returns if the argument designated by any is of data type Integer (true) or not (false).

Parameter

The parameter Argument of data type any designates the argument to be checked.

Return Value

The return value has the data type boolean.

Example:

 
param arg: any
if isInteger(arg) 
   print "Integer = ", arg
end
