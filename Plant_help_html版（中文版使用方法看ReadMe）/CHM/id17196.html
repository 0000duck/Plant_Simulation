FrontPos
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for the Location of the Mobile Objects > FrontPos

Syntax:

 
MU-path.FrontPos

The read-only attribute FrontPos returns the location of the front of the MU designated by MU-path. Remember that an MU enters a length-oriented object at position 0 and exits it after covering a certain length of the object.

Return Value

The return value has the data type length. Note that on point-oriented objects (Station, ParallelStation, etc.) the return value is not defined.

Example:

 
print @.FrontPos
