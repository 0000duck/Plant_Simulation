sensorID
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of the Material Flow Objects > Methods of Sensors of the Length-Oriented Objects > sensorID

Syntax:

 
Path.sensorID(SensorID:integer) → any

The method sensorID accesses a sensor of the object designated by Path.

The methods sensorNo(SensorID:integer) and sensorID(SensorID:integer) may reference different sensors.

Creating and manipulating sensors via SimTalk for the Transporter only works if you select the loading space Types Track or Line!

Parameter

The parameter SensorID of data type integer designates the unique Sensor-ID which Plant Simulation assigns when you create the sensor.

Return Value

The return value has the data type any.

Example:

 
MyTrack.sensorID(3).Front := true

Example:

 
MyTwoLaneTrack.A.sensorID(1).Rear := true

Example:

 
MUs.Transporter:1.sensorID(2).Rear := true
