numOfLimitedObjects
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > numOfLimitedObjects

Syntax:

 
numOfLimitedObjects → integer

The function numOfLimitedObjects counts all of the built-in objects, which are contained in the entire model. It does not count the following objects: Lists and Tables, Methods, Variables, Comments, MUs, Connectors, Interfaces, EventControllers, the Toolbar, and Folders.

Class objects are not counted as well.

You might want to know this number if you are working with a Plant Simulation Standard License which is limited to 4000 objects.

Return Value

The return value has the data type integer.

Example:

 
print numOfLimitedObjects
