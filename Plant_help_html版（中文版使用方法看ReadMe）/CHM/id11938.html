promptList1
SimTalk Reference > General SimTalk Functions > Predefined Functions > Input Functions > promptList1

Syntax:

 
promptList1(Entry:list [,Text:string, ...]) → integer

The function promptlist1 allows the user to pick an entry from the list, which is displayed in the dialog window.

The user can select a single entry and click OK. The function returns the index of the selected entry. When the user clicks Cancel, Plant Simulation returns zero.

Parameters

  • The parameter Entry of data type list designates a list of any type with one column.

  • The optional parameter(s) Text of data type string designate the text above the list box. You can enter any number of optional parameters of data type string with a maximum length of 45 characters.

Return Value

The return value has the data type integer. It is the index of the selected entry. When the user clicks Cancel, Plant Simulation returns zero.

Example:

 
promptList1(CardFile, "Select an entry from","the list below",
"additional text, text, text, text, ...")
-- the entries are contained in the object CardFile
