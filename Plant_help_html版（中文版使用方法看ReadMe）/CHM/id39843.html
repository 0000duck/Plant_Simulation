SpeedCtrl
2D Reference Help > Material Flow Objects > Conveyor > Attributes of the Conveyor > SpeedCtrl

Syntax:

 
Path.SpeedCtrl

The attribute SpeedCtrl designates a Method object, the Speed control, which the Conveyor activates as soon as it reaches its final speed after accelerating or slowing down or when it stops after decelerating. When you did not enter a Method, the attribute is VOID.

You can only set the speed control after you activate Acceleration/AccelerationEnabled.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
MyConveyor.SpeedCtrl := &mySpeedControl
