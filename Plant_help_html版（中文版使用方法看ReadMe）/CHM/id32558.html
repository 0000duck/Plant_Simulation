StatWaitTimePortion
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for Statistics of the Mobile Objects > StatWaitTimePortion

Syntax:

 
MU-path.StatWaitTimePortion

The read-only attribute StatWaitTimePortion returns the portion of the statistics collection period during which the MU designated by MU-path was waiting.

Return Value

The return value has the data type real.

Example:

 
print ParallelStation.MU(1).StatWaitTimePortion
