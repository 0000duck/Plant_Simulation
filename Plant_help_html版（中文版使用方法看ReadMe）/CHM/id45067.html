The following objects are available for each license type:

Materialflow objects: Connector, EventController, Frame, Interface, Source, Drain, Station, ParallelStation, AssemblyStation, DismantleStation, PickAndPlace robot, Store, PlaceBuffer, Buffer, Sorter, Conveyor, AngularConverter, Converter, Turntable, Turnplate, Track, TwoLaneTrack, and Cycle.

Fluid objects: Pipe, FluidSource, FluidDrain, Tank, Mixer, Portioner, DePortioner, and MaterialsTable.

Resource objects: Workplace, Footpath, WorkerPool, Worker, Exporter, Broker, ShiftCalendar, and LockoutZone.

Information Flow objects: Method, Variable, TableFile, FileInterface, and FileLink.

User Interface objects: Comment, Display, Chart, Button,Checkbox, and Drop-Down List.

Mobile objects: Part, Container, and Transporter.

Tools: Bottleneck Analyzer, Layout Optimizer, SankeyDiagram, Worker Chart.

Libraries and modeled objects: EOM library; Cross-sliding Car library, High Bay Warehouse library, Kanban Objects library, and Portal Crane/Cranes library.
