checkID
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > checkID

Syntax:

 
checkID(Expression:string) → boolean

The function checkID checks if you can use an expression as the name of an object (true) or not (false).

Parameter

The parameter Expression of data type string designates the expression. The function considers all keywords and function calls.

Return Value

The return value has the data type boolean.

Example:

 
checkID("sin")
