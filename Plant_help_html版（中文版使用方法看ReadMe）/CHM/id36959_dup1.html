GARangeAllocation
Add-Ins Reference Help > Genetic Algorithms > GARangeAllocation > GARangeAllocation

You can define allocation tasks with the object GARangeAllocation. The individual items of the chromosomes may take values from a range defined by a minimum and a maximum, the distance of which is determined by a fixed interval.
