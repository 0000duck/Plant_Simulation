Dialog Window of the FootPath
2D Reference Help > Resource Objects > FootPath > Dialog Window of the FootPath > Dialog Window of the FootPath

Double-click the icon of the FootPath, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Resources in the Toolbox. The dialog elements and menus, which all material flow objects share, are described under Dialog Items of the Objects.
