The Logical Operator NOT
SimTalk Reference > General SimTalk Functions > Operators and Expressions > The Logical Operator NOT

The logical operator NOT negates a value of data type boolean, i.e., it changes true to false and vice versa. 

failure := false
print failure         -- outputs false
print NOT failure     -- outputs true
print NOT NOT failure -- outputs false

The comparison operators (, =, =, /=, >=, >) are used for mathematical expressions. They compare two expressions and they return a boolean value. You can then use this value in turn in other expressions and thus create complex expressions.

The source code looks like this:

if (0  index)
AND (index = List.Dim) 
AND (NOT value_found OR all_processed) 
   -- complex query
end
