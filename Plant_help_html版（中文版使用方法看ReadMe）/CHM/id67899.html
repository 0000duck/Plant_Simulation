Operating System Functions
SimTalk Reference > General SimTalk Functions > Predefined Functions > Operating System Functions > Operating System Functions

Plant Simulation provides these functions for accessing functions of the operating system:

availableMemory, browseForFolder, copyFile, copyObjectsToClipboard, copyTextToClipboard, getCurrentDirectory, getEnv, getFilesOfFolder, getRegistry, getTextFromClipboard, selectFileForOpen, selectFileForSave, setCodePage, setCurrentDirectory, setEnv, shGetFolderPath, sleep, startExtProc, and system.
