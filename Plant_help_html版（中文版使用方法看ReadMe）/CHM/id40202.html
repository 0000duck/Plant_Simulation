Tab Times
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Times > Tab Times

On the tab Times you can select a distribution from the drop-down list and enter the values that the distribution requires for the Processing time, the Set-up time, the Recovery time, when the Recovery time starts, and the Cycle time.
