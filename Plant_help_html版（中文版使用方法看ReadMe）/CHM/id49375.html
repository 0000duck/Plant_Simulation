Show object labels
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > Modeling > Show object labels

To show the Labels of the objects below their icons in a Frame, select this. When you also select Show object names, it shows them below the name of the object.

On

Off

To show or hide them, you can also click the button Options > Show Object Labels on the General ribbon tab.
