_3D.generateRectangle
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating Extrusion Profiles > _3D.generateRectangle

Syntax:

 
basis.~._3D.generateRectangle(BasePoint:length[2], Width:length, Height:length)

The method _3D.generateRectangle creates a two-dimensional rectangular profile which you can use in your user-defined extrusion configuration.

  • The parameter BasePoint is an array of data type length which sets the base point of the rectangular profile.

  • The parameter Width of data type length sets its width.

  • The parameter Height of data type length sets its height.

Example:

 
var profile : any := basis.~._3D.generateRectangle([2.6,4.5],4, 2)
 --  returns a two-dimensional array containing the values defining a profile; to be used for an extrusion profile
