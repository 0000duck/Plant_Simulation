Dialog Window of the Buffer
2D Reference Help > Material Flow Objects > Buffer > Dialog Window of the Buffer > Dialog Window of the Buffer

Double-click the icon of the Buffer, which you inserted into a Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Material Flow in the Toolbox. The dialog elements and menus, which all material flow objects share, are described under Dialog Items of the Objects.
