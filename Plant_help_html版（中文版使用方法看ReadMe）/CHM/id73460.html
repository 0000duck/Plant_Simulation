Stop on Error Handlers
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Debugger Ribbon Tab > Stop on Error Handlers

The command Stop on Error Handlers stops the simulation and opens the Debugger whenever an error handling method is being executed.

You can also select Stop on Error Handler on the Debug Menu of an open Method, which only applies to that Method.
