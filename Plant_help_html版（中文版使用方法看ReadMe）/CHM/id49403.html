StatSetUpTimePortion
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for Statistics of the Mobile Objects > StatSetUpTimePortion

Syntax:

 
MU-path.StatSetUpTimePortion

The read-only attribute StatSetUpTimePortion returns the portion of the statistics collection period during which the MU designated by MU-path was set up.

Return Value

The return value has the data type real.

Example:

 
print Station.Cont.StatSetUpTimePortion
