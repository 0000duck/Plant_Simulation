_3D.ScaleAutomatically
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for the Appearance of the MUs in 3D > _3D.ScaleAutomatically

Syntax:

 
MU-path._3D.ScaleAutomatically

The attribute _3D.ScaleAutomatically activates (true) or deactivates (false) the feature auto graphics of the object designated by MU-path.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
.MUs.Part._3D.ScaleAutomatically := true
