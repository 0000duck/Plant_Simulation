sumAttr
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Indirectly Accessing Lists and Tables > sumAttr

Syntax:

 
Path.sumAttr(Range:listrange, ... AttributeName:string) → real

The method sumAttr returns the sum of the designated attributes of all objects in the designated range of the list.

The method applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Parameters

  • The parameters Range of data type listrange designate the range of the list.

  • The parameter AttributeName of data type string designates the name of the attribute.

Return Value

The return value has the data type real.

Example:

 
-- cycle through all cells in the QueueFile, sum up the processing times 
-- and return the sum as the result
MyQueueFile.Cursor := 1
print MyQueueFile.sumAttr({1}..{5},"ProcTime")
