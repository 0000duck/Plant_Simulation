Defining Properties of a Dialog Item
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Defining Properties of a Dialog Item > Defining Properties of a Dialog Item

Depending on the type of item you define, the Dialog shows one or all of these items in the dialog of the item you are defining: Name, Caption, Callback argument, X, Y, Group ID, Image ID, Width, Height, Enable or Items.
