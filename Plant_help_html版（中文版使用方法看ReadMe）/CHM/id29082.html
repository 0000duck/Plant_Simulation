openFile
2D Reference Help > Information Flow Objects > FileLink > Methods of the FileLink > openFile

Syntax:

 
Path.openFile

The method openFile opens the linked file in its originating application or which is assigned to the respective file type on the computer. You can then edit it.

If you deactivate the safety setting File > Model Settings > General > Access to the computer is permitted, the FileInterface cannot open the embedded files in the associated application with a double-click. The same applies for the method openFile.

Example:

 
frame1.MyFileLink.openFile
