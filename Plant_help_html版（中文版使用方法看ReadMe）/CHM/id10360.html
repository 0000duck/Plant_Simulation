Show Attributes and Methods
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Show Attributes and Methods

The command Show Attributes and Methods shows the attributes and methods of the selected object in the window Show Attributes and Methods.

Instead, you can also right-click the object in the Frame and select Show Attributes and Methods on the context menu or press the F8 key.
