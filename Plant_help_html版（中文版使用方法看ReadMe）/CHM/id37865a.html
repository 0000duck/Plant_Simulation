getRouteLength
2D Reference Help > Mobile Objects > Transporter > Methods of the Transporter > getRouteLength

Syntax:

 
MU-path.getRouteLength(DestinationObject:object [,TargetTable:table]) → length

The method getRouteLength returns the length of the shortest route in the direction of motion to a destination object.

The destination may either be a material flow object, which can be reached directly via Connectors, or a destination, which you entered into a sensor.

When you entered a the destination into a sensor, then it does not matter if the sensor is front or rear triggered. Plant Simulation always computes the length of the route from the side of the Transporter which is located in the front in the direction of motion, in relation to the sensor.

Parameters

  • The parameter DestinationObject of data type object designates the destination object. The method returns -1, when the Transporter did not find a route.

  • The optional parameter TargetTable of data type table designates the name of a TableFile into which the Transporter enters the objects it would pass en-route to the destination.

Return Value

The return value has the data type length.

The methodreturns -1, when Plant Simulation did not find a route.

Example:

 
print .MUs.Transporter:1.getRouteLength(track6)
