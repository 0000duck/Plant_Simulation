succLaneNo
2D Reference Help > Material Flow Objects > TwoLaneTrack > Methods of the TwoLaneTrack > succLaneNo

Syntax:

 
Path.A.succLaneNo → integer
Path.B.succLaneNo → integer
Path.A.succLaneNo(LaneNumber:integer) → integer
Path.B.succLaneNo(LaneNumber:integer) → integer

The method succLaneNo returns the number of the succeeding lane of the specified lane of the TwoLaneTrack designated by Path.

The method succLaneNo(LaneNumber:integer) returns the number of the lane of the n-th successor designated by the parameter of data type integer. Note that the objects have to be connected for this to work.

Return Value

The return value has the data type integer.

Example:

 
if MyTwoLaneTrack.A.succLaneNo = 1 
   print "succeeding lane is lane A"
end
