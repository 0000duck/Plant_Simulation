toUpper
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Strings > toUpper

Syntax:

 
toUpper(Text:string) → string

The function toUpper changes lower-case letters in a string to upper-case letters.

Parameter

The parameter Text of data type string can consist of any combination of upper-case letter, lower-case letters, and special characters. The result string then only contains upper-case letters and special characters.

Return Value

The return value has the data type string.

Example:

 
print toUpper("shania") -- returns "SHANIA"

Example:

 
print toLower("Hello World!") -- returns "HELLO WORLD!"
