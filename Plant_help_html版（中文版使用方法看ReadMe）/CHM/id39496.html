goToLine
2D Reference Help > Information Flow Objects > FileInterface > Methods of the FileInterface > goToLine

Syntax:

 
Path.goToLine(LineNumber:integer) → boolean

The method goToLine moves the cursor to the start of the line designated by the parameter. The FileInterface returns false when the file contains less lines than you enter. If the file is not open, the FileInterface opens it and leaves it open after accessing it.

Parameter

The parameter LineNumber of data type integer designates the line.

Return Value

The return value has the data type boolean.

Example:

 
MyFileInterface.goToLine(2)
