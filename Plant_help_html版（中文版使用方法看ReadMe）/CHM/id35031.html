_3D.PointCloudRotationAngle
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes for Point Clouds of the Frame > _3D.PointCloudRotationAngle

Syntax:

 
Path._3D.PointCloudRotationAngle

The attribute _3D.PointCloudRotationAngle sets the rotation angle of the point cloud in the Frame designated by Path in degrees.

3D interprets the rotation angle as a rotation around the negative z-axis. This corresponds to the rotation angles in 2D.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
.Models.MyFrame._3D.PointCloudRotationAngle := 30
