Cont
2D Reference Help > Display and User Interface Objects > Comment > Attributes of the Comment > Cont

Syntax:

 
Path.Cont

The attribute Cont sets the contents of the text box on the tab Comment. Enter a backslash to insert a line break.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyComment.Cont := "Species: Aardvark\
                 amount: 45 \
                 weight: 780kg" -- assign a multi-line comment
