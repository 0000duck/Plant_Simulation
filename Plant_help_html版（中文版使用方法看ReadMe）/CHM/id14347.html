existsIcon
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Animating All Objects > existsIcon

Syntax:

 
Path.existsIcon(IconName:string) → boolean

A 3D Only model does not provide the method existsIcon.

The method existsIcon checks if the icon designated by the parameter exists (true) or does not exist (false).

Parameter

You can enter the number of the icon as a parameter of data type integer or the name of the icon as a parameter of data type string.

Return Value

The return value has the data type boolean.

Example:

 
press.existsIcon("failed")
press.existsIcon(2)
