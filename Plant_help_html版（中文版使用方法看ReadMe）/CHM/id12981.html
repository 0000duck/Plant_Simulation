getAttributeValue
2D Reference Help > Information Flow Objects > XMLInterface > Methods of the XML Interface > getAttributeValue

Syntax:

 
Path.getAttributeValue(Parameter:any) → string

The method getAttributeValue returns the value of the attribute designated by the parameter of data type any. It applies when you randomly traverse data.

Return Value

The return value has the data type string.

Example:

 
MyXMLInterface.getAttributeName(i)+":" + MyXMLInterface.getAttributeValue(i)
