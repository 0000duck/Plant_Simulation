ShowComment
2D Reference Help > Information Flow Objects > AttributeExplorer > Attributes of the AttributeExplorer > ShowComment

Syntax:

 
Path.ShowComment

The attribute ShowComment sets if the AttributeExplorer shows the text box for the comment above the list field (true), or not (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyAttributeExplorer.ShowComment := true
