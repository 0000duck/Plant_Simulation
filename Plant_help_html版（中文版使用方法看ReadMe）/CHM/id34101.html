MutationProb, MutationDelta
Add-Ins Reference Help > Genetic Algorithms > GASequence > Attributes of the GASequence > MutationProb, MutationDelta

Syntax:

 
Path.MutationProb
Path.MutationDelta

When the attribute Mutation takes the value true, the attribute MutationProb sets the probability with which a mutation occurs within the first generation. This value changes for each new generation by the value of the attribute MutationDelta. When the attribute MutationDelta takes the value 0, the probability for a mutation to take place during the evolution cycle does not change.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
GASequence.Mutation := true
GASequence.MutationProb := 0.4
GASequence.MutationDelta := 0
