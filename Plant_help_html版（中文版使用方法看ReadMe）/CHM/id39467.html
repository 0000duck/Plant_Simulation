Methods for Columns of the TableFile
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > Methods for Columns of the TableFile

Plant Simulation provides the methods cutColumn, getAlignmentColumn, getColumnNo, getColumnWidth, getColumnYDim, getDataType, getEditorRightsColumn, getBackgroundColorColumn, getFontColorColumn, getFontSizeColumn, getFormatString, getVisibility, insertColumn, isAlignmentColumn, setAlignmentColumn, setColumnWidth, setDataType, setDataTypeDefault, setEditorRightsColumn, setBackgroundColorColumn, setFontColorColumn, setFontSizeColumn, setFormatString, and setVisibility for columns. Note that the first parameter designates the column, either the column index, the column range or the column number.

To view all of the methods and attributes of the list object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methodsof the selected Instance.
