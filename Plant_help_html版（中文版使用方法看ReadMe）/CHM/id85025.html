NumMU
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Miscellaneous read-only attributes of the Material Flow Objects > NumMU

Syntax:

 
Path.NumMU

The read-only attribute NumMu returns the number of MUs booked on the object designated by Path.

On length-oriented objects MUs may only be located on a part of the object. In this case they will only be counted if their booking point is actually located on the object.

When the object is not booked on another object either, Plant Simulation counts the MU as being part of the length-oriented object on which a part of the MU is located. This might, for example, occur, when an MU is moved from a Station onto a Conveyor.

Return Value

The return value has the data type integer.

Example:

 
print ParallelStation.NumMU
