ArgumentForOpen
2D Reference Help > Display and User Interface Objects > Dialog > Attributes of the Dialog > ArgumentForOpen

Syntax:

 
Path.ArgumentForOpen

The attribute ArgumentForOpen designates an Argument for open for the CallbackMethod method, when the user opens the dialog window of the Dialog you designed. You might, for example, use it to preallocate data in the dialog window.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyDialog.ArgumentForOpen := "Open"
