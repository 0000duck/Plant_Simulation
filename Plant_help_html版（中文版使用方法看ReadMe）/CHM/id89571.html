drawRectangle
2D Reference Help > Material Flow Objects > Frame > Methods of the Frame > drawRectangle

Syntax:

 
Path.drawRectangle(Layer:integer, X:integer, Y:integer, Width:integer, Height:integer, RGB:integer, LineWidth:integer [,Gradient:integer])

The method drawRectangle draws a rectangle on the background of the Frame designated by Path.

To draw the rectangle in the foreground, i.e., in front of the objects in the Frame, enter a negative number for the layer, designated by the parameter integer.

Parameters

The parameter of data type integer

Designates the

Layer

layer.

X

position of the rectangle on the x-axis.

Y

position of the rectangle on the y-axis.

Width

width of the rectangle.

Height

height of the rectangle.

RGB

line color of the rectangle.

LineWidth

line width of the outline of the rectangle in pixels. Enter -1 to draw a filled rectangle.

If you do not want to zoom the outline of the rectangle, enter a LineWidth of 0 instead of 1 .

Gradient

color gradient for filled rectangles:

0 stands for None

1 stands for Top To Bottom

2 stands for Bottom To Top

3 stands for Left To Right

4 stands for Right To Left

Example:

 
.Models.Frame.drawRectangle(3,30,30,80,40,133,1)
