BrokerPath
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Importer > BrokerPath

Syntax:

 
Path.failImp.BrokerPath
Path.imp.BrokerPath
Path.setupImp.BrokerPath
Path.transportImp.BrokerPath

The attribute BrokerPath designates the Broker of the object designated by Path, which the Exporter uses to provide its services.

For Broker hierarchies Plant Simulation only calls the control of this Broker, not the controls of any existing sub Brokers.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
ParallelStation.imp.BrokerPath := Broker3
MyStation.failImp.BrokerPath := myBroker2
