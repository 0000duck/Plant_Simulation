Print List
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Print List

The command Print List opens the dialog Print in which you can select settings for printing the list.

Select the printer and the paper format from the drop-down lists. Select the orientation, Landscape or Portrait, with the radio buttons and enter the number of copies into the text box.
