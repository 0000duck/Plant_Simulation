Caption
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Defining Properties of a Dialog Item > Caption

For a dialog item of Type New Static Text Box, New Button, New Group Box, New Check Box, New Radio Button, New Tab Page or New Menu/New Menu Command the Dialog shows the text box Caption.

Enter the text the Dialog shows as the caption of the dialog item in your dialog.

A caption is especially handy if you develop application object libraries in several languages and want the user to be able to switch languages. In addition you can enter special characters and blanks, which you cannot use in the name.
