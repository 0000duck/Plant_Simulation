Speed
2D Reference Help > Material Flow Objects > Conveyor > Attributes of the Conveyor > Speed

Syntax:

 
Path.SpeedCtrl

The attribute Speed sets the Speed of the Conveyor. Speed and Length form the processing time on the object. The processing time is the Length divided by the Speed. You can also enter -1 for an infinite speed.

Data Type of the Value You Can Assign

You can assign a value of data type speed.

Example:

 
MyConveyor.Speed := 0.5
