Duplicate
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Duplicate

The context menu command Duplicate copies the selected object or model and creates a new class. Changes you make to the original object are not passed on to the copy you create.

Instead of selecting Duplicate, you can hold down Ctrl and click the object you wish to duplicate with the left mouse button. Then, hold down the mouse button, drag the copy to the location of your choice, which as a rule will be another folder, in the Class Library, and release the Ctrl key and the mouse button.
