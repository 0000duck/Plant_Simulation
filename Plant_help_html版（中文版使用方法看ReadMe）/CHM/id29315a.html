Blocking
2D Reference Help > Material Flow Objects > FlowControl > Dialog Window of the FlowControl > Tab Exit Strategy > Blocking

Select if the exit strategy is blocking or non-blocking :

  • When you select Blocking , the FlowControl only moves the part on, when the designated successor can receive it.

  • When you clear the check box, the FlowControl only moves the part on, when any of its successors can receive it.

    The non-blocking strategy of the exit strategy > MU Attribute moves the part on when any of the desired successors can receive it.
