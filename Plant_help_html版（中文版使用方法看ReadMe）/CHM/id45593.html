Release all services, when one of them is failed
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Importer > Sub-tab Processing > Release all services, when one of them is failed

To make the material flow object release all processing services or set-up services respectively, when one or more of these services is failed, paused, or interrupted, select this. To only release the failed services, clear it.

This only works, when you also select Fail services in the dialog of the Exporter.
