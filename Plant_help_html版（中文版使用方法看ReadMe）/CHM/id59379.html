Open External Connections List
2D Reference Help > Material Flow Objects > Frame > The Frame Window > Context Menu of the Selected Object in the Frame > Open External Connections List

To open the list, which shows the paths of the external connections in the Frame in which you inserted the Interface object, select this command.

Column 1 shows the Connector at the exit of the Frame, column 2 the name of the succeeding object.

Instead, you can also select View > External Connections in the dialog of the Interface.
