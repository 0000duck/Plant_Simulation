Automatic stop
2D Reference Help > Material Flow Objects > Turntable > Dialog Window of the Turntable > Tab Attributes > Automatic stop

To set the current speed of the Turntable to 0 when it does not transport a part, select this. This might, for example, be the case when it is empty or when it is blocked when a part cannot leave it.

When the speed of the Turntable is 0, the energy state changes to Operational.
