CurrentValue
2D Reference Help > Information Flow Objects > Trigger > Read-Only Attributes of the Trigger > CurrentValue

Syntax:

 
Path.CurrentValue

The read-only attribute CurrentValue returns the current value of the Trigger’s designated by Path. The data type matches the Trigger’s data type.

Return Value

The return value has the data type any.

Example:

 
print MyTrigger.CurrentValue
