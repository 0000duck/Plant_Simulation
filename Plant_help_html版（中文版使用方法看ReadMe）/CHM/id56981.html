current
SimTalk Reference > General SimTalk Functions > Names > Anonymous Identifiers > current

The anonymous identifier current returns the Frame within which the Method object is located. This way you can easily enter the location of a Frame into lists and tables or enter it as a parameter to methods in other Frames.

You can also use current to distinguish between a local and a global variable with the same name.

The parameter name identifies a local variable, while current.name designates the global variable.
