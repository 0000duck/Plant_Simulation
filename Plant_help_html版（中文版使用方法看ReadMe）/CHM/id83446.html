Interval
2D Reference Help > Information Flow Objects > Generator > Dialog Window of the Generator > Tab Times > Interval

Enter the Interval, i.e., the time between two activations of the Interval control method.

When you enter 0, the program does not trigger any controls and stops the Generator.

You can also select any of the probability distributions. Then, you have to enter the parameters, which the selected distribution requires.
