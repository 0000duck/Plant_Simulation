isAlignmentRow
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Rows of the TableFile > isAlignmentRow

Syntax:

 
Path.isAlignmentRow(Row:any, Alignment:string) → boolean

The method isAlignmentRow returns if the alignment of a row of the TableFile designated by Path matches the designated value (true) or not (false).

Parameters

  • The parameter Row of data type any designates the row.

  • The parameter Alignment of data type string designates the value: Right, Left or Center.

Return Value

The return value has the data type boolean.

Example:

 
print MyTableFile.isAlignmentRow(7,"center")
