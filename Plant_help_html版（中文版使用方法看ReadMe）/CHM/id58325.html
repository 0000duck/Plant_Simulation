Tab Data Type
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Format > Tab Data Type > Tab Data Type

On the tab Data Type you can select the Data type and enter a Format string for the list. For the data types Table, List, Stack, and Queue you can select if you want to use or not use a Common format.

Plant Simulation only shows the Tab Data Type and the Tab Dimension after you selected one or several columns by clicking into the column header!
