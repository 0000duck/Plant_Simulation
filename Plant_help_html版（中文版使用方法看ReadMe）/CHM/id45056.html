goToAngle
2D Reference Help > Material Flow Objects > Turntable > Methods of the Turntable > goToAngle

Syntax:

 
Path.goToAngle(Angle:real)

The method goToAngle sets the angle to which the side of the end point of the Turntable designated by Path rotates. This only applies, when no MU is located on it and when no MU is waiting for it.

Parameter

The parameter Angle of data type real designates the angle.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
?.goToAngle(45)
