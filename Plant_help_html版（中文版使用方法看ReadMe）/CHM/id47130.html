Charge
2D Reference Help > Mobile Objects > Transporter > Dialog Window of the Transporter > Tab Battery > Charge

Enter the active charge of the battery in ampere hours (Ah). You can enter any value between 0, which means that the battery is empty, and the Capacity of the battery.

The Transporter does not automatically update the value of the battery charge during a simulation run, when the tab is displayed on screen. To update it, select View > Refresh on the View menu or press F5.
