Detach Camera
3D Reference Help > 3D Reference > View Ribbon Tab > Detach Camera

The command Detach Camera or the Esc key detach the camera, you attached to an object, from it. Plant Simulation then returns to normal view. Removing an object camera automatically resets the view.

Plant Simulation also detaches an object camera:

  • When you delete the object the camera is attached to.

  • When you change the scene.
