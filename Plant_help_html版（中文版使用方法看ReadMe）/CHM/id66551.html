_3D.renameGraphicGroup
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Graphic Groups > _3D.renameGraphicGroup

Syntax:

 
Path._3D.renameGraphicGroup(OldName:string, NewName:string)

The method _3D.renameGraphicGroup renames the specified graphic group of the object designated by Path.

The method deactivates graphic inheritance.

Parameters

  • The parameter OldName of data type string designates the current name of the graphic group.

  • The parameter NewName of data type string designates the new name of the graphic group.

Example:

 
MyStation._3D.renameGraphicGroup("MyGraphicGroup", "MyNewGroup")
