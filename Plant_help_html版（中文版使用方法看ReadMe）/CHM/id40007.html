Methods for the Order of Cells within Lists and Tables
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for the Order of Cells within Lists and Tables > Methods for the Order of Cells within Lists and Tables

Lists and tables provide the methods find, findCeil, findFloor, inOrder, setCursor, and sort for manipulating the order of cells.

To view all of the methods and attributes of the list object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.
