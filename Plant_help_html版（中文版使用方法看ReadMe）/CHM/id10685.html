setFontSizeRow
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Rows of the TableFile > setFontSizeRow

Syntax:

 
Path.setFontSizeRow(Row:any [,Row:any, ...], FontSize:integer)

The method setFontSizeRow sets the font size of row(s) of the TableFile designated by Path to the designated value.

Parameters

  • The parameters Row of data type any designate the rows whose font size you want to set.

  • The parameter FontSize of data type integer designates the font size. You can enter 1 for Small, 2 for Medium, 3 for Large or 4 for Extra Large.

Example:

 
-- sets the font size of row 2 to medium
TableFile3.setFontSizeRow(2,2)
-- sets the font size of rows 3 and 5 to small
TableFile3.setFontSizeRow(3,5,1)
