UserGraphics
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Attributes of the MultiPortalCrane > Attributes for Displaying the Crane in 3D > UserGraphics

Syntax:

 
Path.UserGraphics

The attribute UserGraphics sets if external graphics are used for displaying the MultiPortalCrane in 3D (true). To use the built-in graphics, enter false.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyPortalCrane.UserGraphics := true
