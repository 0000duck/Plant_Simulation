openObjectSelectBox
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > openObjectSelectBox

Syntax:

 
openObjectSelectBox(Filter:string, FrameOrFolder:object) → string

The function openObjectSelectBox opens the dialog Select Object.

Parameters

  • The parameter Filter of data type string designates the filter, i.e., the objects the list box below Objects shows.

  • Pass an empty string ("") to show all objects.

  • Pass an internal class name to only show objects of that type.

  • Pass ":Table" to only show lists and tables.

  • Pass ":MatCarrier" to only show material flow objects.

  • Pass ":Importer" to only show material flow objects that have importers.

    You can also enter several of these object types separated by colons.

    The internal identifiers for the classes are case-sensitive.

  • The parameter FrameOrFolder of data type object designates the Frame or the folder for which the dialog Select Object will be opened initially.

Return Value

The return value has the data type string.

Example:

 
var path := openObjectSelectBox("Place;NwAssembler", current)
-- only shows the Station and the AssemblyStation in the 'Select Object' dialog
var path2 := openObjectSelectBox(":Table", .InformationFlow)
-- shows all lists and tables in the 'Select Object' dialog
