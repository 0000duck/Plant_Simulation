succConnector
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods and Read-Only Attributes for the Location, Predecessor, and Successor of All Objects > succConnector

Syntax:

 
Path.succConnector → object
Path.succConnector([SuccessorNumber:integer]) → object

The method succConnector returns outgoing connections joined directly to the object designated by Path via the successor number. Note that adding and deleting connections may change the index number of a connection.

Parameter

The optional parameter SuccessorNumber of data type integer designates the number of the successor.

Return Value

The return value has the data type object.

The method returns VOID if the connection does not exist.

Example:

 
MyTrack.succConnector.deleteObject 
-- delete the connector to the succeeding object
