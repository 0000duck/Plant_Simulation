Station
2D Reference Help > Material Flow Objects > Station > Station

The Station has a single station for processing a part. The Station receives a part from its predecessor, processes it and moves it on to the successor.

If the types of MUs are not the same, i.e., if they do not have the same name, the Station has to set up to process this new type of MU. While a part is located on the Station, it does not receive any additional parts. A part may only enter, when the Station is available, i.e., when no other part is located on it. Plant Simulation always moves the part as a whole not continually, i.e., as soon as its front is located on the Station, the entire part is located on it.

Compare the sample models: Click the Window ribbon tab, click Start Page > Getting Started > Example Models and click Small Examples. Then, select the respective Category, the Topic, and the Example in the dialog Examples Collection and click Open Model.

As the Station is such a central object in Plant Simulation, it figures prominently in a number of topics in the Step-by-Step Help.

To add the Station to your simulation model, click Manage Class Library > Basic Objects > MaterialFlow > PickAndPlace on the Home ribbon tab.
