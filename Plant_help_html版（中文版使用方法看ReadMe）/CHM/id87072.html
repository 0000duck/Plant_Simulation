InheritValue
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Attributes and Methods of User-defined Attributes > InheritValue

Syntax:

 
Path.AttributeName.InheritValue

The attribute InheritValue sets if the value of the user-defined attribute of the object designated by Path is inherited (true) or not (false). Activating InheritValue will also automatically activate inheriting the data type.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyStation.myAttribute.InheritValue := true
