str_to_num
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Numerical Values > str_to_num

Syntax:

 
str_to_num(string) → real

The function str_to_num converts the parameter of data type string into a floating-point value of data type real.

Apart from digits the string can optionally contain leading white spaces, a sign, a decimal point and an e, E, d or D for the exponent. In case the string contains an invalid character, the function str_to_num only converts all preceding characters into a number.

Parameter

The parameter has the data type string.

Return Value

The return value has the data type real.

Example:

 
var answer: string
answer := prompt("6*7=")
if str_to_num(answer) = 42 
   print "fourtytwo"
end

Example:

 
x := str_to_num("   -7") -- minus seven

Example:

 
y := str_to_num("3.141xyz") -- 3.141

Example:

 
w := str_to_num("-1e6") -- minus one million
