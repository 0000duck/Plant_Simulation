time_to_str
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Numerical Values > time_to_str

Syntax:

 
time_to_str(time [,FormatLikeDialogs:boolean]) → string

The function time_to_str converts the time designated by the parameter of data type time into a string.

Parameters

  • The parameter Time of data type time designates the time that is going to be converted.

  • The optional parameter FormatLikeDialogs of data type boolean sets if Plant Simulation uses the same time format as in text boxes dialogs (true) or as in a TableFile (false), which shows four decimal places.

Return Value

The return value has the data type string.

Example:

 
var t: time
t := 60
Dialog.setCaption("myProcTime", time_to_str(t, true))
