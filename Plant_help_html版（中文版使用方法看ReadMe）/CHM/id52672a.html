StatWaitingPartsDelta
2D Reference Help > Material Flow Objects > AssemblyStation > Read-Only Attributes of the AssemblyStation > StatWaitingPartsDelta

Syntax:

 
Path.StatWaitingPartsDelta

The read-only attribute StatWaitingPartsDelta returns the deviation from the mean value of the time spans during which the AssemblyStation designated by Path was Waiting for MUs.

Return Value

The return value has the data type real.

Example:

 
print MyAssembly.StatWaitingPartsDelta
