makePathRelative
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > makePathRelative

Syntax:

 
makePathRelative(Path:string [,StartObject:object]) → string
makePathRelative(Path:object [,StartObject:object]) → string

The function makePathRelative transforms an absolute path to a relative path.

Parameters

  • You can enter the path as a parameter of data type string or as a parameter of data type object. Plant Simulation transforms this path into a relative path starting with the Frame in which the Method is located.

  • Optionally you can enter a second parameter of data type object. This parameter sets the object from which on Plant Simulation starts evaluating the path.

Return Value

The return value has the data type string. Plant Simulation returns an empty string (""), when the path is not valid.

Example:

 
Station.ExitCtrl := makePathRelative(Station.ExitCtrl)
