Plant Simulation Runtime License
Setting-Up and Starting > Plant Simulation License Types > Plant Simulation Runtime License

The Plant Simulation Runtime license enables you to interactively modify model parameters and to execute simulation runs within Plant Simulation models.

With a Plant Simulation Runtime license you can neither insert any of the objects built into Plant Simulation nor user-defined objects nor can you use the programming language SimTalk.

The Plant Simulation Runtime license includes the optional modules of the interface package, Gantt, and the Assembly and Shop libraries.

With the Plant Simulation Runtime license you can:

  • Interactively set the parameters of the class objects in the dialogs as well as restricted access to the built-in objects contained in the user-defined objects, for example lists and tables.

  • Execute simulation runs with 2D and 3D graphics and animate simulation models, record statistics and analyze simulation runs and save the results and the statistics tables.

  • Optimize the simulation results by modifying model parameters via file interfaces, databases or binding of ActiveX, COM, C-Programming, ODBC, Socket, ORACLE SQL interfaces.

  • Automatically optimize the simulation model with Genetic Algorithms.

  • Use the user-defined objects contained in the Assembly and Shop libraries.

  • Display the results in Gantt Charts according to resources and orders.
