Dialog Window of the Part
2D Reference Help > Mobile Objects > Part > Dialog Window of the Part > Dialog Window of the Part

Double-click the icon of an Part, which is located on one of the material flow objects in the Frame, to open its dialog window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Mobile Units in the Toolbox. Enter settings of your choice to produce the desired results. The dialog items are described under Dialog Items of the Objects.
