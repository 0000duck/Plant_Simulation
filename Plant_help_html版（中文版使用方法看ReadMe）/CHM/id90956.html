_3D.setGraphicPosition
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Detailed Access To Graphics > _3D.setGraphicPosition

Syntax:

 
Path._3D.setGraphicPosition(GraphicPath:any, Position:length[3] [,GraphicGroupName:string])

The method _3D.setGraphicPosition sets the position of the designated graphic of the object designated by Path.

The method deactivates graphic inheritance.

Parameters

  • The parameter GraphicPath of data type any designates the path of the graphic. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

  • The parameter Position is an array of data type length which designates the position of the graphic.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described under The Parameters GraphicPath and GraphicGroupName.

Example:

 
Station._3D.setGraphicPosition([1,1], [2,2,2])
ParallelStation._3D.setGraphicPosition(["Graphic1"], [2,2,2])

Example:

 
Station._3D.setGraphicPosition([1,1], [2,2,2])
ParallelStation._3D.setGraphicPosition(["Graphic1"], [2,2,2], "deco")
