Show Attributes and Methods
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > View Menu > Show Attributes and Methods

The menu command Show Attributes and Methods opens shows the attributes and the methods of the selected object in the dialog Show Attributes and Methods.

Instead, you can also right-click the object in the Frame and select Show Attributes and Methods on the context menu or press the F8 key.
