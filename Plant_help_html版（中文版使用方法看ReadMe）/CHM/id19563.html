getLane
2D Reference Help > Mobile Objects > Transporter > Methods of the Transporter > getLane

Syntax:

 
MU-path.getLane([Position:length]) → string

The method getLane returns the number of the lane on which the Transporter is located on a TwoLaneTrack.

Parameter

The optional parameter Position of data type length designates the position starting from the front of the Transporter. When you do not enter the parameter, Plant Simulation uses the position of the front end of the MU. This parameter is of importance, when the Transporter stretches across several Tracks.

Return Value

The return value has the data type string.

Example:

 
print.MUs.Transporter:1.getLane
