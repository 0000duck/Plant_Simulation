YScaleMax
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > YScaleMax

Syntax:

 
Path.YScaleMax

The attribute YScaleMax sets the upper bound of the range on the y-axis.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyChart.YScaleMax := 1.5
