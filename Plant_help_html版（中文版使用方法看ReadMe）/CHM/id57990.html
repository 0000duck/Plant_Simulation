Recovery time
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Times > Recovery time

The recovery time is the time which is required to set a station into a defined state before it can start processing the next part. The recovery time is, for example, useful to model materials handling equipment, such as a robot, which requires a certain time to insert work pieces into or remove them from processing stations.

You can use the attribute EntranceOpen to check if the entrance is currently open. You can use the attribute TimeUntilEntranceOpen to query the time at which the entrance will be open again.

Select a distribution, and enter the values, which that distribution requires, into the text box. The program shows these values along the top border of the tab. You can also enter a constant time (Const). When you select the Formula distribution, you can enter a numeric expression or the name of a Method. You can use the anonymous identifier @ to access the MU for which the recovery time applies.

The LED of a recovering station is cyan .

For some stations you can also select, when the Recovery time starts.
