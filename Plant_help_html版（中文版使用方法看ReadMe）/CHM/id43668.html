New Button
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New Button

When the user clicks a button , it calls the Callback method with the parameter you entered into the text box Callback argument. It in turn executes the actions you programmed.

  • Enter the Name of the button.

  • Enter the text the button displays (Caption).

  • Enter its X and Y position in the Dialog.

  • Enter its Width. The default value of 0 automatically sets the width of the button to the width of the OK button.

  • Select if you want to Enable it or not.

  • To insert the dialog item into your dialog, click OK.
