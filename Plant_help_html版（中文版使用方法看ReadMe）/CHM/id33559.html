_3D.deleteGraphicGroup
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Graphic Groups > _3D.deleteGraphicGroup

Syntax:

 
Path._3D.deleteGraphicGroup(Name:string)

The method _3D.deleteGraphicGroup deletes the specified graphic group from the object designated by Path.

The method deactivates graphic inheritance.

Parameter

The parameter Name of data type string designates the name of the graphic group.

Example:

 
MyStation._3D.deleteGraphicGroup("MyGraphicGroup")
