setEditorRightsColumn
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > setEditorRightsColumn

Syntax:

 
Path.setEditorRightsColumn(Column:any [,Column:any, ...], ReadOnly:boolean)

The method setEditorRightsColumn sets the editing permissions of the designated column(s) of the TableFile designated by Path to the designated value.

Parameters

  • The parameters Column of data type any designate the column(s) whose editing permissions you want to set.

  • The parameter ReadOnly of data type boolean designates read-only access (true) or read and write access (false).

Example:

 
-- sets the editor rights of column 1
Tab1.setEditorRightsColumn(1,true)
-- sets the editor rights of columns 2 and 4
Tab1.setEditorRightsColumn(2,4,true)
