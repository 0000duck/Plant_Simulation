Occupied
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Read-Only Attributes of Lists and Tables > Occupied

Syntax:

 
Path.Occupied

The read-only attribute Occupied returns if at least one cell of the list designated by Path contains an entry (true) or not (false).

The read-only attribute applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Return Value

The return value has the data type boolean.

Example:

 
print MyStackFile.Occupied
print MyCardFile.Occupied
print MyTableFile.Occupied
