BatCharge
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > BatCharge

Syntax:

 
MU-path.BatCharge

The attribute BatCharge sets the active battery Charge of the Transporter designated by MU-path. Enter a value between 0 (empty) and the capacity of the battery (BatCapacity). The unit is ampere hours (Ah).

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
-- This method controls the battery charge of a Transporter. 
if @.BatteryUsed and (@.BatCharge = @.BatReserve) 
print "Attention: Charge on reserve"
print "Driving to charge bay."
@.Destination := .building3.charge_bay
end
