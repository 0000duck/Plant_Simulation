Merge
2D Reference Help > Material Flow Objects > Frame > The Frame Window > General Ribbon Tab > Replacement Mode > Merge

The command Merge is the default setting. In this mode Plant Simulation preserves changed parameters in instances of the replaced Frame and imports them into the replacing object. Plant Simulation then deletes the replaced object in the Class Library. Note that this option only works for Frames that are identical. Compare Example 1 below.
