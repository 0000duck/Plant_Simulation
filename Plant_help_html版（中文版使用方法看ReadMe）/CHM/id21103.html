Create a Control Which is Part of the Object
Step-by-Step Help > Modeling in 2D > Modeling the Flow of Materials, Advanced > Customize the Behavior of Objects > Define Controls > Create a Control Which is Part of the Object

When you want all instances of a class to provide the same control, you typically create that control as a user-defined attribute in the class.

To create a control as a user-defined attribute of data type method of the object:

  • Enter a meaningful name into the text box, right-click in the text box and select Create Control. Plant Simulation then inserts self.Name_you_entered_for_the_control, such as self.mySelectControl.

  • Right-click in the text box and select Create Control on the context menu. Plant Simulation then inserts self.OnBuilt_in_name_of_the_control, such as self.OnSelect.

    Enter the source code of this control into the Method that opens.

    This control is a user-defined attribute of data type method. To open and modify it, do one of the following:

    • Click in the text box and press F2.

    • Shift and double-click in the text box.

    • Select the tab User-defined Attributes and double-click the name of the Method in the list pane.
