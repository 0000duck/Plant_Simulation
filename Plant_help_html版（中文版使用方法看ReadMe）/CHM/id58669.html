Visualize the Occupancy of the Store Over Time
Step-by-Step Help > Modeling in 2D > Modeling the Flow of Materials, Basics > Placing Parts into Stock and Removing Parts from It > Visualize the Occupancy of the Store Over Time

To visualize the occupancy of the Store, insert a Chart into your model and select the following settings:

  • Select input channels is the data source. Then click Table File. Enter the name of the Store, MyStore in our case, into row 1. Enter the name of the Store and the name of the method whose values you want to display into cell 1 of column 1, MyStore.numMU in our case.

  • Click the tab Display and select the category Plotter and the chart type Line for displaying the plotted values.

    You’ll notice that the icon of the Chart changes to the icon of the Plotter in the simulation model when you click Apply or OK.

  • Click the Plotter in the model with the right mouse button, select Show, start the simulation, and watch how the values develop over time.
