StatServicesWaitingTime
2D Reference Help > Resource Objects > Worker > Read-Only Attributes of the Worker > StatServicesWaitingTime

Syntax:

 
Path.StatServicesWaitingTime

The method StatServicesWaitingTime returns the time which the services of the Worker designated by Path were waiting in relation to the statistics collection period.

The value of StatServicesWaitingTime is the sum of StatServicesWaitingImpTime and StatServicesWaitingMUTime.

Return Value

The return value has the data type time.

Example:

 
print .Resources.MyWorker:2.StatServicesWaitingTime
