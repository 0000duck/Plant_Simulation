Decrypt Methods
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Options > Decrypt Methods

The command Decrypt Methods decrypts the contents of the methods in your simulation model. Enter the password, which you used to encrypt the methods, into the dialog Decrypt Methods.

To decrypt the source code of a method which you encrypted by clicking Encrypt Method on the Tools ribbon tab, click Decrypt Method in the method editor of that method.
