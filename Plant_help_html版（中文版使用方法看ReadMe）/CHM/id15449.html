Logarithmic X-axis
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Axes > Logarithmic X-axis

To activate logarithmic scaling on the x-axis, select this. This is useful if the data to be displayed is scattered across several orders of magnitude. Logarithmic representation cannot show negative values.

Plant Simulation does not show the check box Logarithmic X-axis for all Chart types.
