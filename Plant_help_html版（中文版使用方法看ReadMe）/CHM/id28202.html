insertNodes
2D Reference Help > Information Flow Objects > XMLInterface > Methods of the XML Interface > insertNodes

Syntax:

 
Path.insertNodes(NodesTable:list) → boolean

The method insertNodes inserts the nodes contained in a table into the object designated by Path. It applies when you randomly access data.

Parameter

The parameter NodesTable of data type table designates the name of the table.

Return Value

The return value has the data type boolean.

Example:

 
MyXMLInterface.insertNodes(tbl)
