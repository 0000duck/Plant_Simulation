BookPos
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for Accessing the Cargo of Container and Transporter > BookPos

Syntax:

 
MU-path.BookPos

The read-only attribute BookPos returns the position of the booking point of the MU according to the current ConveyingDirection on the object, designated by MU-path, on which it is located. You can only get the attribute, you cannot set it.

Return Value

The return value has the data type real.

The attribute only returns the position, when the active MU is located on a length-oriented object! For point-oriented objects the return value always is 0.

Example:

 
print@.BookPos
