StatExporterFailedDelta
2D Reference Help > Resource Objects > Exporter > Read-Only Attributes of the Exporter > StatExporterFailedDelta

Syntax:

 
Path.StatExporterFailedDelta

The read-only attribute StatExporterFailedDelta returns the standard deviation of the failure time of the Exporter designated by Path from the mean value.

Return Value

The return value has the data type time.

Example:

 
print MyExporter.StatExporterFailedDelta
