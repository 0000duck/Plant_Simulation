Resource statistics
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Dialog Window of the Mobile Objects > Tab Statistics > Resource statistics

To collect resource statistics data for the MU and to display the most important statistical data on the tab, select this. The values displayed are described under Resource Statistics of the MUs.

Plant Simulation does not continuously update statistical data displayed in an open window during a simulation run! Select View > Refresh or press F5.
