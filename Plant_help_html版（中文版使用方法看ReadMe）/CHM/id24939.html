OpenModal
2D Reference Help > Display and User Interface Objects > Dialog > Attributes of the Dialog > OpenModal

Syntax:

 
Path.OpenModal

The attribute OpenModal sets if the dialog window of the object designated by Path will be opened modal (true) or not modal (false). Modal means that the user cannot open any other Plant Simulation dialog windows until he closes the window of the user-defined dialog. Not modal means that the user can open other Plant Simulation dialog windows when the window of the user-defined dialog is open.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
.Models.MyPlant.MyDialog.OpenModal := true
