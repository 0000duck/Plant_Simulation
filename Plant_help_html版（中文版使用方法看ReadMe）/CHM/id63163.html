setIcon
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > setIcon

Syntax:

 
Path.setIcon(DialogItem:string, ImageID:string) → boolean

The method setIcon sets the Image ID of the dialog item designated by the parameter string to the value designated by the parameter string.

Parameters

  • The parameter DialogItem of data type string designates the dialog item,

  • The parameter ImageID of data type string designates the image ID. You can enter an image ID or an image name that is an icon of the Dialog to which the image belongs.

Return Value

The return value has the data type boolean.

Syntax:

 
MyDialog.setIcon("My_Image","Logo")
MyDialog.setIcon("My_Image","3")
