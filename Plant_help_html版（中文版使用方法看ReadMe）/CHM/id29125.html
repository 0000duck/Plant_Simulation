Define the number of processing stations of the ParallelStation in a net of coordinates.

  • Enter the number of stations on the x-axis into the text box X-dimension.

  • Enter the number of stations on the y-axis into the text box Y-dimension.

  • Activate Start processing when full if you want the ParallelStation to only start processing parts when each of the stations contains a part.

The ParallelStation accesses the individual stations using their x-coordinate and their y-coordinate.
