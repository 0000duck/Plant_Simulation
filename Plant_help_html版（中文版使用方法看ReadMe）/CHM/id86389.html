MUTarget
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Setting the Exit Strategy of the Material Flow Objects > MUTarget

Syntax:

 
Path.MUTarget

The attribute MUTarget sets the station to which the Worker carries the part for the exit strategy carry part away. The MU target is the Workplace attached to the work station, which processes the part next, or the work station itself.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
MyStation.MUTarget := MyWorkplace3
