Built-in Methods for Accessing 3D As Such
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing 3D As Such > Built-in Methods for Accessing 3D As Such

Plant Simulation provides the method _3D.closeAllWindows for accessing 3D.

To access the attributes in 3D you can enter:

basis.~._3D.NameOfTheAttribute designates the path to 3D.

To set an attribute, you will type MyObject._3D.NameOfTheAttribute := "MyString"
