Start at successor 1
2D Reference Help > Material Flow Objects > FlowControl > Dialog Window of the FlowControl > Tab Exit Strategy > Strategy > Start at successor 1

To move the MUs on to the first available successor, select Start at successor 1 from the drop-down list. The FlowControl checks the successors starting with the successor with the number 1. It moves the MU on to successor number 3, for example, if it cannot move it on to successor 1 and 2. As long as the successor with the number 1 can receive MUs, the FlowControl always moves them to it.
