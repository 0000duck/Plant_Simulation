getBackgroundColorColumn
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Columns > getBackgroundColorColumn

Syntax:

 
Path.getBackgroundColorColumn(Column:any) → integer

The method getBackgroundColorColumn returns the background color of a column of the TimeSequence designated by Path.

Parameter

The parameter Column of data type of data type any designates the column whose background color you want to get.

Return Value

The return value has the data type integer.

Example:

 
print timeSequence1.getBackgroundColorColumn(4)
