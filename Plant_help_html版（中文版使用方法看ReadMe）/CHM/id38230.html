ExitStrategyMUAttributeList
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Setting the Exit Strategy of the Material Flow Objects > ExitStrategyMUAttributeList

Syntax:

 
Path.ExitStrategyMUAttributeList

The attribute ExitStrategyMUAttributeList sets the contents of the attribute list of the material flow object designated by Path for the Exit strategy MU Attribute.

You can enter several numbers of the desired successors into the attribute list for attribute value. In the non-blocking case Plant Simulation checks which of the desired successors can receive the MU.

Data Type of the Value You Can Assign

You can assign a value of data type table.

Example:

 
MyStation.ExitStrategyMUAttributeList := MyTableFile.copy
