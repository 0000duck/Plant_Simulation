Insert a Horizontal Line
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Insert a Horizontal Line

Enter three hyphens ---, which are not followed by text, to create a solid horizontal line. You can also enter more than three hyphens. If you type in other characters than the hyphens into the line of characters, Plant Simulation interprets --- as the start of a comment. 

Text above the line.
------
Text below the line.
--- This is a comment ---
