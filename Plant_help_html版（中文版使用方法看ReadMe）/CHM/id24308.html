Attributes of the Display Panel
2D Reference Help > The Plant Simulation Program Window > Display Panel > Attributes of the Display Panel > Attributes of the Display Panel

Plant Simulation addresses the attributes of the Display Panel as sub-attributes of the attribute DisplayPanel. You cannot assign the attribute to a Variable of data type object or any.

Syntax:

 
Path.DisplayPanel.<name of the sub-attribute>

The display panel provides these sub-attributes:

Active

Width

BackgroundColor

X

BorderColor

Y

Height

A 3D Only model does not provide these attributes.
