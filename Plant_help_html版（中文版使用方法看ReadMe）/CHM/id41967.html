Primitive Empirical (Emp)
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Empirical Distributions > Primitive Empirical (Emp)

You can use the primitive empirical distribution to randomly sample discrete times.

The random numbers, which Plant Simulation generates, are located in the range between 1 and the y-dimension of the frequency table. The values on the table can be of data type real, time, or integer. The contents of a cell in row i of this table is the frequency that the random numberi occurs. The value in row 5 is the number of times the value 5 occurred.

When you need an integer random number, whose realizations is located in the range between two positive integer numbers a and b, create a column with b entries. The first (a - 1) entries are 0.

Clicking Apply in the dialog of the object automatically formats the table so that the data types of the columns match the parameters.

You can also create the primitive empirical distribution with this SimTalk function: 

z_emp(Stream:integer,Table:table,Column:integer)

The parameter Stream (integer) designates the random number stream.

Parameters

The parameter Table designates the path to a table or an attribute of data type table.

The parameter Column of data type integer designates the column containing the frequency data.
