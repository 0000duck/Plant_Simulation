New Check Box
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New Check Box

A check box displays a setting or a set of settings that do not exclude each other. The user can select or clear one or several check boxes at the same time.

  • Enter the Name of the check box.

  • Enter the Caption that describes what the check box does.

  • Enter the Callback argument, which will be passed to the Callback method as a parameter.

    It will be executed, when the user selects or clears the check box.

  • Enter its X and Y position in the Dialog.

  • Select if you want to Enable it or not.

  • To insert the dialog item into your dialog, click OK.
