_3D.GraphicsWhenEmpty
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes for Graphic Sequences of the Worker > _3D.GraphicsWhenEmpty

Syntax:

 
Path._3D.GraphicsWhenEmpty

The attribute _3D.GraphicsWhenEmpty defines a sequence of graphic groups of the Worker designated by Path. The Worker cycles through the graphic groups in the sequence which you enter here when he walks on a FootPath while he does not carry a part. When this sequence is empty, 3D uses the visible graphic groups as defined in the attribute _3D.VisibleGraphics, i.e., the default behavior.

Data Type of the Value You Can Assign

You can assign a value of data type array.

Example:

 
.Resources.MyWorker._3D.GraphicsWhenEmpty := ["Pose1", "Pose2"]
