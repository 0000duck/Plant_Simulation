DismantleMode
2D Reference Help > Material Flow Objects > DismantleStation > Attributes of the DismantleStation > DismantleMode

Syntax:

 
Path.DismantleMode

The attribute DismantleMode sets how the DismantleStation handles the MUs.

Data Type of the Value You Can Assign

You can assign a value of data type string: To create MUs, enter Create MUs, to detach them from the main MU, enter Detach MUs.

Example:

 
MyDismantleStation.DismantleMode := "Detach MUs"
