Display a Picture of a 3D Scene
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Select an Object or an Object Attribute to be Displayed > Display a Picture of a 3D Scene

To display a picture of a 3D scene on the HTML page, type in the following statement:

[>Object._3D, w, h] 
[>Object._3D, [caption,] w, h]
[>Object._3D, w, h, crx, crz]
[>Object._3D, [caption,] w, h, crx, crz]
[>Object._3D, w, h, cpx, cpy, cpz, crx, crz]
[>Object._3D, [caption,] w, h, cpx, cpy, cpz, crx, crz]

The setting [>Object._3D, w, h] creates the picture with the angles crx=45° and crz=15° and as a View All view.

The individual parameters designate the following settings:

  • The optional parameter caption designates the caption which the HtmlReport shows below the picture.

  • The parameter object designates the object whose picture you want do display.

  • The parameter w designates the width of the picture in pixels.

  • The parameter h designates the height of the picture in pixels.

  • The parameter cpx designates the x-coordinate of the camera position.

  • The parameter cpy designates the y-coordinate of the camera position.

  • The parameter cpz designates the z-coordinate of the camera position.

  • The parameter crx designates the rotation angle of the camera on the x-axis.

  • The parameter crz designates the rotation angle of the camera on the z-axis.

You might, for example, enter:

[Station._3D, 150, 100, -29.596, -31.985, 39.985, 46.983, -36.669]
