2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Failures > New > Failed

To fail this failure profile select this. To not fail this failure profile, clear the check box. The setting/attribute is observable and Plant Simulation can call a control. The row of a failed failure profile is shown in red in the List of Failure Profiles.
