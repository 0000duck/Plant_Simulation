setSubAttribute
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Managing Attributes of All Objects > setSubAttribute

Syntax:

 
Path.setSubAttribute(AttributeName:string, SubAttributeName:string, Value:any)

The method setSubAttribute sets the value of a sub-attribute of an attribute of the object designated by Path.

The method setSubAttribute sets the sub-attribute designated by the parameter string of the attribute designated by the parameter string to the value designated by the parameter of data type any of the object designated by Path.

Parameters

  • The parameter AttributeName of data type string designates the attribute.

  • The parameter SubAttributeName of data type string designates the sub-attribute.

  • The parameter Value of data type any designates the value of the sub-attribute.

Example:

 
MyStation.setSubAttribute("ProcTime","Type","uniform")
