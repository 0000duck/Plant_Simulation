_3D.getObject
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > _3D.getObject

Syntax:

 
Path._3D.getObject(Name:string)
Path._3D.getObject(Index:integer)

The method _3D.getObject returns the designated inner object of the object designated by Path.

The method excludes objects with a counterpart in 2D, such as stations within a Frame or MUs on a station.

Parameter

The parameter Name of data type string designates the name of the object. Or

The parameter Index of data type integer designates the n-th inner object.

Example:

 
var o := Machine._3D.getObject("Arm")
