profiler
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > profiler

Syntax:

 
profiler(Activate:boolean)

The function profiler activates (true) or deactivates (false) the Profiler. While it is active, it continuously collects information pertaining to the frequency of calls and the runtime of methods.

Parameter

The parameter Activate of data type boolean designates the action to be taken.

Example:

 
profiler(true) -- activate Profiler
