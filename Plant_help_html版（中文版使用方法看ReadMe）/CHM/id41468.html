StatStoreStoppedTime
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for Statistics of the Mobile Objects > StatStoreStoppedTime

Syntax:

 
MU-path.StatStoreStoppedTime

The read-only attribute StatStoreStoppedTime returns the time during which the MU designated by MU-path was located on a material flow object of type Storage that was stopped by a LockoutZone.

Return Value

The return value has the data type real.

Example:

 
print .MUs.Transporter.StatStoreStoppedTime
