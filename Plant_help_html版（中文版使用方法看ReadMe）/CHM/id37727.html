ShowColumnIndex
2D Reference Help > Information Flow Objects > TimeSequence > Attributes of the TimeSequence > ShowColumnIndex

Syntax:

 
Path.ShowColumnIndex

The attribute ShowColumnIndex sets if the TimeSequence shows the column index (true) or hides it (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
timeSequence2.ShowColumnIndex := false
