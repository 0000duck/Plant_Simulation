New Edit Text Box
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Inserting a Dialog Item into Your Dialog > New Edit Text Box

The user can enter text into the edit text box or edit text it already contains.

  • Enter the Name of the text box.

  • Enter the Callback argument, which will be passed to the Callback method as a parameter.

    It will be executed, when the user changes the contents of the Text Box and selects another dialog item afterwards, clicks in another text box, or clicks OK, Apply or Cancel.

  • Select what kind of characters the user can enter into the text box.

    • Any Character: The user can enter any character, which includes special characters, lower and upper case letters and numbers.

    • Letters and Digits: The user can enter lower and upper case letters, including special characters, such as umlauts, and letters of any kind of language, underscores, and blank spaces. The user can also enter the digits 0 to 9.

    • Letters: The user can enter lower and upper case letters, including special characters, such as umlauts, and letters of any kind of language, and blank spaces.

    • Decimal Numbers: The user can enter 0123456789

    • Signed Decimal Numbers: The user can enter decimal numbers with a negative or positive sign: -+0123456789

    • Hexadecimal Numbers: The user can enter ADbf09, for example.

    • Octal Numbers: The user can enter 01234567, but not 18 for example.

    • Binary Numbers: The user can enter 0 or 1

    • Floating Point Numbers: The user can enter 12.3E-45, for example.

    • Time: The user can enter time values.

    • Date with Time: The user can enter a date containing time values.

    • Date: The user can enter a date.

    • Positive Real Number: The user can enter a positive real number.

  • Enter its X and Y position in the Dialog.

  • Enter its Width.

  • Select if you want to Enable it or not.

  • To show a password that the user enters into the text box with superscripted lower case Xes , instead of as clear text , select this .

  • To insert the dialog item into your dialog, click OK.
