getUnit
SimTalk Reference > General SimTalk Functions > Predefined Functions > Output Functions > getUnit

Syntax:

 
getUnit(Unit:any) → string

The function getUnit returns the unit, which you selected for the data types length, weight, time, speed, money, and acceleration.

Parameter

The parameter Unit of data type any designates the unit.

Return Value

The return value has the data type string.

Example:

 
var l: length
l := 100 -- meters
print l, getUnit(l)     -- if length is kilometers, output: 0.1 km
print l*l, getUnit(l*l) -- outputs 10000 m²
