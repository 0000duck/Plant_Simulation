pe, [X,Y]
2D Reference Help > Material Flow Objects > Store > Methods of the Store > pe, [X,Y]

Syntax:

 
Path.pe(X:integer,Y:integer)
Path[X:integer,Y:integer]

The method pe accesses the designated storage place in the Store designated by Path. Instead of pe you can also use [X:integer, Y:integer].

To access the MU that is located on the designated storage place, append .Cont.

The method Cont returns the MU at the location called.

Parameters

  • The parameter X of data type integer designates the X-dimension of the storage place.

  • The parameter Y of data type integer designates the Y-dimension.

Example:

 
@.move(MyStore.pe(2,3))
MyStore[2,3].Cont.move(Station)
