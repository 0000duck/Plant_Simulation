setCheckBox
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > setCheckBox

Syntax:

 
Path.setCheckBox(DialogItem:string, Active:boolean) → boolean

The method setCheckBox sets if the New Check Box or New Radio Button designated by the parameter of data type string will be selected (true) or cleared (false).

Parameters

  • The parameter DialogItem of data type string designates the dialog item.

  • The parameter Active of data type boolean sets if it will be active or not.

Return Value

The return value has the data type boolean.

Example:

 
MyDialog.setCheckBox("My_CheckBox",true)
