AutomaticRouting
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > AutomaticRouting

Syntax:

 
MU-path.AutomaticRouting

The attribute AutomaticRouting activates (true) or deactivates (false) Automatic routing of the Transporter designated by MU-path.

For tugger trains automatic routing only works, when you enter the Destination into the tractor.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
.MUs.MyTransporter.AutomaticRouting:=true
