create
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Instantiating the TableFile > create

Syntax:

 
Path.create

The method create creates a data structure without contents.

Example:

 
var OrderList : table[string,real]
OrderList.create
OrderList[1,1] := "Cans"
OrderList[2,1] := 3000.0

The method create only applies to a local variable.
