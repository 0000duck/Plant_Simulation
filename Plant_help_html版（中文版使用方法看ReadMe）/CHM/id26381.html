Date
SimTalk Reference > General SimTalk Functions > Data Types > Date

A variable of data type date designates a date between 1.1.190 and 31.12.9999.

When assigning a value to a variable or an attribute of this data type, you have to convert the date with the function str_to_date.

Before you assign a value to the local variable, it has an initial value. It is 0 for all numerical data types, false for the data type boolean, empty string ( "" ) for the data type string, and void for the data types Object and List/Table. The initial value for the data types date and DateTime depends on the Plant Simulation version in which you initially created your simulation model. Plant Simulation version in which you initially created your simulation model. For models, which you create in Plant Simulation 14.0 or higher, the start value is 1.1.1900.
