getRefPoint
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > Methods for Animating All Objects > getRefPoint

Syntax:

 
Path.getRefPoint(byRef X:integer, byRef Y:integer)

A 3D Only model does not provide the method getRefPoint.

The method getRefPoint returns the coordinates of the reference point of the object designated by Path. It then assigns them to the local variables, whose names you enter.

Local Variables

  • The local variable X designates the x-coordinate of the reference point.

  • The local variable Y designates its y-coordinate.

    Example:

    		 
    MyStation.getRefPoint(x,y)
