Booking point height
2D Reference Help > Mobile Objects > Transporter > Dialog Window of the Transporter > Tab Attributes > Booking point height

Enter the relative height of the booking point of the Transporter. The height is only important if you want to use the z-dimension of the part in 3D. You can enter a relative value between 0 and 1. Plant Simulation shows the absolute height of the booking point in parentheses.

When you assign a value to the booking point in a Method object, Plant Simulation interprets this number as meters. Only when outputting it with the method print, it converts the value according to the settings you selected under File > Model Settings/Preferences > Units > Length.
