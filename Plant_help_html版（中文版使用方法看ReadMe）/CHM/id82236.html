Animation offset
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab MU Animation > Animation offset

For the length-oriented objects you can enter an animation offset.

At times you might not want the parts to be placed flush onto the surface of the length-oriented object but with an offset to this surface. To do so:

  • Click the toggle button so that it looks like this .

  • Enter the offset of the animation path to the surface of the object along the X direction into the text box or click the spin buttons to set a positive or a negative offset.

  • Enter the offset of the animation path to the surface of the object along the Y direction.

  • Enter the offset of the animation path to the surface of the object along the Z direction.

Compare this example:

Offset X = 0, Y = 0, Z = 0

Offset X = 0.5, Y = 0.5, Z = 1
