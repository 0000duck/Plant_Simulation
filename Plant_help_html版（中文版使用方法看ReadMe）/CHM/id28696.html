RotateMovables
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Curved Objects > RotateMovables

Syntax:

 
Path.RotateMovables

The attribute RotateMovables sets if the MU turns in the direction of motion when it turns in a curve (true) on the curved object designated by Path. When you type in false, the parts keep their previous direction/rotation.

We advise to not deactivate Rotate movables as this setting does not affect the behavior in 3D.

Rotate MUs active/true

Rotate MUs not active/false

A rotation triggered by the setting Rotate movables (RotateMovables) on a length-oriented object only has an effect as long as the part is located on this object. When leaving the object the part again assumes the rotation it had before. When entering a new object, the part considers the rotation of this new object when Rotate MUs is active.

When Plant Simulation shows the length-oriented object in curve mode, Plant Simulation evaluates the setting Rotate MUs (RotateMovables).

When Plant Simulation shows the object as an icon , Plant Simulation evaluates the setting Same as the Object (RotateMUs).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
if MyConveyor.CurveActive 
   MyConveyor.RotateMovables := true
else
   MyConveyor.RotateMUs := true
end
