Built-in Methods and Attributes for Scheduling and Playing Animations
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods and Attributes for Scheduling and Playing Animations > Built-in Methods and Attributes for Scheduling and Playing Animations

Plant Simulation provides a set of functions for scheduling animations for the animation types self animation and camera animation:

The three animation types share a common set of functions of data. They also provide a number of functions specific to a certain type. All of the functions can handle saved animations though.

Plant Simulation provides these attributes and methods for saved animations in 3D:

_3D.SelfAnimations.deleteAllAnimationBlocks, _3D.SelfAnimations.deleteNextAnimationBlock, _3D.TypeOfAnimations.pause, _3D.TypeOfAnimations.play, _3D.TypeOfAnimations.reset, _3D.SelfAnimations.Animation.play, _3D.SelfAnimations.playRotation, _3D.SelfAnimations.playTranslation, _3D.TypeOfAnimations.scheduleRotation, _3D.SelfAnimations.scheduleTranslation, _3D.SelfAnimations.startNextAnimationBlock, _3D.TypeOfAnimations.EstimatedBlockAnimationTime, and _3D.TypeOfAnimations.EstimatedTotalAnimationTime.

The animation types self animations and camera animations have saved and temporary animation paths.
