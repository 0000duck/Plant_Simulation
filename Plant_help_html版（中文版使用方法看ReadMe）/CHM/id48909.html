getEditorRightsCell
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for the Format of Lists and Tables > getEditorRightsCell

Syntax:

 
Path.getEditorRightsCell(Column:any, Row:any)

The method getEditorRightsCell returns the editor rights of a cell of the StackFile, QueueFile, CardFile, TableFile, and TimeSequence designated by Path.

Parameters

  • The parameter Column of data type any designates the column of the cell, whose editor rights you want to get.

  • The parameter Row of data type any designates the row of the cell, whose editor rights you want to get.

Example:

 
MyTableFile.getEditorRightsCell(2,5)
