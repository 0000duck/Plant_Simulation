Manipulate an Object Precisely
Step-by-Step Help > Modeling in 3D > Working with Objects > Manipulate an Object Precisely

You can modify the objects in the scene of your simulation model manually with the mouse.

When the object is selected, you can press Esc to undo any modifications. Press Esc repeatedly until you return to the object in its unselected state.

For some purposes this will not be precise enough. Then, you can enter exact values into the text boxes in the dialog 3D Properties > Tab Transformation to fine tune the position, the rotation and the scale of the selected object:

To precisely manipulate an object:

  • Select the object you would like to edit.

  • Click on the Home ribbon tab and click the tab Transformation.

  • Click Apply to apply your changes once you’re satisfied with the results.
