HookGraphicsFile
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Attributes of the MultiPortalCrane > Attributes for Displaying the Crane in 3D > HookGraphicsFile

Syntax:

 
Path.HookGraphicsFile

The attribute HookGraphicsFile sets the name of the graphics file to be used for displaying the hook of the MultiPortalCrane designated by Path in 3D.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyPortalCrane.HookGraphicsFile := "MyHookGraphic.JT"
