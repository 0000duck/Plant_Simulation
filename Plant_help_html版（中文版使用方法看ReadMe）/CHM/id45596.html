Tab 3D
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Dialog Window of the MultiPortalCrane > Tab 3D > Tab 3D

On the tab 3D you can set to use your own graphics files for displaying all portals, trolleys, and hooks of the MultiPortalCrane respectively in 3D. You can specify a graphics file for the portals, the trolley, and for the hook. When you specify the graphics file for the portal, the MultiPortalCrane uses this portal for all newly created portals. When you use an external graphics file, the MultiPortalCrane scales it according to the masses of the portal. The MultiPortalCrane also uses the scaling factor, which it calculated for the portal, for the trolley and for the hook.

For the three graphics files Plant Simulation creates a file link each in the folder GraphicFiles. This embeds the respective graphics file into the model. This way you can pass on your model together with the graphics files. If you select other graphics files in the dialog, Plant Simulation updates the file links.

When you change the graphics files, these changes are not reflected in the model as the model uses the embedded graphics files. For this reason we recommend to name the new graphics files differently

The tab 3D provides these dialog items:

Use external graphics

Trolley animation offset X

Portal graphics file

Trolley animation offset Z

Trolley graphics file

Hook graphics file
