If you have not completed Section 1 of the tutorial, you can open the respective tutorial model from the Plant Simulation installation folder > Plant Simulation > Tutorial, compare the example below.

We will start this section by creating another Frame within our folder TestingConveyor with a Source, a Drain and several Stations. We will also be using the Frame named Test, which we created in Section 1 of the tutorial.

  • Create a new Frame is described in section 1.

  • Click the folder TestingConveyor in the Class Library with the right mouse button and select New > Frame.

  • Rename the new Frame to ProcessingConveyor.

  • Double-click the icon of the Frame named ProcessingConveyor.

    We will now insert the objects which we need into the Frame named ProcessingConveyor.

  • Insert a Source .

  • Insert a Station and rename it to PreTest.

  • Open the dialog window of the Station named PreTest.

  • On the tab Times select NegExp is the Processing time.

  • Type in 1:00 is the Processing time.

  • Insert a Drain .

    Next, we will insert the two Test stations, which we created earlier, between the station PreTest and the Drain.

  • Click the Frame Test in the folder TestingConveyor.

  • Drag it between the station PreTest and the Drain, and drop it there.

  • Rename this Frame to Test1.

  • To create a second Test Frame, hold down the Ctrl key, move the mouse over the Frame named Test1, and click the left mouse button.

  • Drag Test1 down and drop it.

  • Rename this Frame to Test2.
