Mode
2D Reference Help > Information Flow Objects > AttributeExplorer > Attributes of the AttributeExplorer > Mode

Syntax:

 
Path.Mode

The attribute Mode sets the mode that the AttributeExplorer uses.

Data Type of the Value You Can Assign

You can assign a value of data type string: Watch, Edit, or Read only.

Example:

 
MyAttributeExplorer.Mode := "Edit"
