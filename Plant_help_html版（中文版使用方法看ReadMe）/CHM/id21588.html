StatServicesFailedTime
2D Reference Help > Resource Objects > Exporter > Read-Only Attributes of the Exporter > StatServicesFailedTime

Syntax:

 
Path.StatServicesFailedTime

The read-only attribute StatServicesFailedTime returns the total time during which services of the Exporter designated by Path were failed.

The Exporter only collects failure times for the services when you select the check box Fail services.

Return Value

The return value has the data type time.

Example:

 
print MyExporter.StatServicesFailedTime
