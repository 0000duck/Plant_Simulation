applicationHome
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > applicationHome

Syntax:

 
applicationHome → string

The function applicationHome returns the folder in which Plant Simulation is installed.

Return Value

The return value has the data type string.

Example:

 
print applicationHome -- might return C:\Plant Simulation 14.1\
