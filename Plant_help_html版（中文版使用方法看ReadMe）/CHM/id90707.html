Open List of Attached Conveyors
2D Reference Help > Tools > CrossSlidingCar > Dialog Window of the CrossSlidingCar > Tab Track Attributes > Open List of Attached Conveyors

To open the list of the branch lines connected with the cross-sliding car, click this. The list shows the names of the branch lines, the side at which they connect to the cross-sliding car, the position of the sensors, and the sensor ID.

You can connect the cross-sliding car with any number entrances and exits, i.e., branch lines.

The cross-sliding car and the attached branch lines have to be located in the same Frame .
