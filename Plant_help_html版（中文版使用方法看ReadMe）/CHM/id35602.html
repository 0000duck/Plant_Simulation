DrivingHome
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > DrivingHome

Syntax:

 
MU-path.DrivingHome

The attributes DrivingHome and HasOrder change the state of the Transporter designated by MU-path. Both attributes are important for statistics collection only. As soon as DrivingHome is set to true, HasOrder changes to false and vice versa.

Plant Simulation does not automatically set the attribute DrivingHome to true when a Transporter reaches its destination, but remains in the present state! You have to explicitly set the state.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTransporter.DrivingHome
MyTransporter.DrivingHome := true
