Set Next Statement
2D Reference Help > Information Flow Objects > Method > The Method Window > Method Debugger > Menu Bar of the Method Debugger > Debug Menu > Set Next Statement

The menu command Set Next Statement, or Ctrl+F10, sets the next statement that is going to be executed. Click Yes in the dialog that opens if you want Plant Simulation to continue executing the current Method in the line in which the cursor is located. Click No if you do not want Plant Simulation to do that and return to the current position of the cursor.

Let us illustrate this with an example: Suppose a Method is caught in an endless loop. To get out of this, hold down the Shift+Ctrl+Alt keys to stop Method execution, and set the cursor outside of the loop.

Then, press Ctrl+F10 to get out of the loop and press F5 to continue executing the Method.

The feature Set Next Statement provides the same function as Ctrl+double-clicking the left mouse button.
