PortalWidth
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Attributes of the MultiPortalCrane > Attributes of the MultiPortalCrane Proper > PortalWidth

Syntax:

 
Path.PortalWidth

The attribute PortalWidth sets the width of the portal. You can also make the portal width greater than the Width of runway. In this case you have to enter an Overhang on the right hand side of the portal in the direction in which you inserted the MultiPortalCrane.

Data Type of the Value You Can Assign

You can assign a value of data type length.

Example:

 
MyPortalcrane.PortalWidth := 20
