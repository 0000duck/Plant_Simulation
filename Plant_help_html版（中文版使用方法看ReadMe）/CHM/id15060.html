BwEntranceCtrlRear
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Controls of the Material Flow Objects > BwEntranceCtrlRear

Syntax:

 
Path.BwEntranceCtrlRear

The attribute BwEntranceCtrlRear sets if the length-oriented objects Conveyor, Track, or TwoLaneTrack designated by Path activate the Backward entrance control, when the Rear of the MU enters the length-oriented objects via their exit and/or when it moves towards their entrance after entering (true) or not (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTrack.BwEntranceCtrlRear := true
MyTwoLaneTrack.B.BwEntranceCtrlRear := true
MyConveyor.BwEntranceCtrlRear := false

Example:

 
.MUs.Transporter:2.LoadBayType := "Track"
.MUs.Transporter:2.BwEntranceCtrlRear := false
