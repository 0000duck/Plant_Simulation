append
2D Reference Help > Display and User Interface Objects > HtmlReport > Methods of the HtmlReport > append

Syntax:

 
Path.append(Text:string)

The method append appends text to the content of the HtmlReport designated by Path.

Parameter

The parameter Text of data type string designates the text that you want to append.

Example:

 
MyReport.Content := "#Heading"
   for var i := 1 to numNodes 
   MyReport.append(chr(10))
   MyReport.append("Name: ")
   MyReport.append(object(i).name)
