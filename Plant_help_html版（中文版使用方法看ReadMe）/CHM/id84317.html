getAlignmentColumn
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Columns > getAlignmentColumn

Syntax:

 
Path.getAlignmentColumn(Column:any) → string

The method getAlignmentColumn returns the alignment of a column of the TimeSequence designated by Path.

Parameter

The parameter Column of data type of data type any designates the column whose alignment you want to get.

Return Value

The return value has the data type string.

Example:

 
str := timeSequence1.getAlignmentColumn(1)
