Watched entity types
Add-Ins Reference Help > Object Libraries > Kanban Objects > KanbanChart > Dialog Window of the KanbanChart > Watched entity types

To watch a certain type of part within the KanbanBuffer, double-click the row which contains the data of this part. An X in the row designates that the KanbanChart watches that part.
