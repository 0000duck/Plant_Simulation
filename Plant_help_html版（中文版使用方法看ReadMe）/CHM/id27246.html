Inherit View Options
2D Reference Help > Material Flow Objects > Frame > The Frame Window > General Ribbon Tab > Inherit View Options

The command Inherit View Options activates or deactivates inheritance of the settings of the active Frame from its class.

  • When you activate it, the current Frame inherits all settings on the Options menu from its class object.

  • When you deactivate it, the settings you select only apply to the current Frame.
