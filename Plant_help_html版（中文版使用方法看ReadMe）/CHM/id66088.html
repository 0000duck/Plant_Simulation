Activate
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Sensors of the Length-Oriented Objects > Activate

Syntax:

 
Path.Sensor.Activate

The attribute Activate sets when the Transporter triggers the sensor of the Track designated by Path.

  • Always, independent of the destination of the Transporter.

  • Only when the Transporter has the same destination as the one, which you entered into the sensor.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
MyTrack.sensorID(2).Activate := "Only when destination"
