Any
SimTalk Reference > General SimTalk Functions > Data Types > Any

A variable of data type any can take any value. You cannot change this data type after you initially assigned it to a variable. You can use the following functions to query the data type of a variable of data type any. The function expects a parameter whose data type Plant Simulation compares.

Return Value

The return value has the data type boolean.

You can query the data type of a variable of data type any with these functions:

getSimTalkTypename

isLength

isSpeed

isAcceleration

isList

isStack

isBoolean

isListRange

isString

isDate

isObject

isTable

isDatetime

isQueue

isTime

isInteger

isReal

isWeight

Example:

 
param arg: any -- declaring the method myPrint
if isLength(arg) 
   print "length = ", arg 
end
    
-- calling the method myPrint 
myPrint(1)
myPrint("any text")
myPrint(MyConveyor.length)

Example:

 
var a : any
a := MyConveyor.acceleration
print a," ",getUnit(a)
print isAcceleration(a)

For isObject, isTable, and isList this applies: The value VOID does not have a data type as local variables of data types object, table, and list can all take the value void. For this reason a local variable of data type any does not take a data type when you assign void to it. If the function isObject is called with such a variable, the function returns the result false.

Example:

 
var a: any := void
print isObject(a)   -- prints false
print a = void      -- prints true

var b : any := 123
print b = void      -- prints false

You can also use the function getSimTalkTypename to get the data type of a variable of data type any.

Return Value

The return value has the data type string.

Example:

 
param a : any
switch getSimTalkTypename(a)
case "integer" 
   print "integer argument"
case "string" 
   print "string argument"
end
