infoBox
SimTalk Reference > General SimTalk Functions > Predefined Functions > Output Functions > infoBox

Syntax:

 
infoBox(Text:string, Modal:boolean)

The function infoBox opens a message box, which shows the designated text.

Close the message box by calling the function infoBox a second time and by passing an empty string ("").

Open a modal Infobox when you would like to prevent the user from modifying a simulation model during a simulation run.

When the Debugger is opened, any open message box will be closed. Otherwise an open modal message box would prevent you from continuing your work.

When you inadvertently opened a modal message box without closing it, hold down the Shift+Ctrl+Alt keys down for 5 seconds. This closes the message box and opens a dialog you can close by clicking No.

Parameters

  • The parameter Text of data type string designates the text, which the message box shows.

  • The parameter Modal of data type boolean sets if the dialog is modal, i.e., that you cannot access any other Plant Simulation window (true) or not (false).

Example:

 
infoBox("info string",false) -- shows the "info string" 
infoBox("",false)            -- closes the info box
