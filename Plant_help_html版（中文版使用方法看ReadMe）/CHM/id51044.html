LoadBayCapacity
2D Reference Help > Tools > CrossSlidingCar > Attributes of the CrossSlidingCar > LoadBayCapacity

Syntax:

 
Path.LoadBayCapacity

The attribute LoadBayCapacity sets the length of the loading space of the Vehicle of the cross-sliding car designated by Path.

Note that the loading space is rotated by 90 degrees in regard to the direction of motion of the Vehicle. The loading space width is identical with the Length of the loading space of the Transporter.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyCrossslidingcar.LoadBayCapacity:=4
