Orientation of the MUs
2D Reference Help > The Plant Simulation Program Window > Icon Editor > General Ribbon Tab > Orientation of the MUs > Orientation of the MUs

The command Orientation of the MUs provides the subcommands Same as the Object and Inherit.

You can only change the Orientation of the MUs for the material flow objects.

We strongly advise to not switch off Same as the Object as this setting does not influence the behavior in 3D.
