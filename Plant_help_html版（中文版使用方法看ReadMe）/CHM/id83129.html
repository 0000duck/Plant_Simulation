eraseFailureEvents
2D Reference Help > Mobile Objects > Transporter > Methods of the Transporter > eraseFailureEvents

Syntax:

 
MU-path.eraseFailureEvents

The method eraseFailureEvents deletes failure events, which the methods startFailureIn, and endFailureIn entered into the event list in the EventController, for the Transporter designated by MU-path.

You can use it to terminate an existing failure and, at the same time, delete eraseFailureEvents, as these would end new failures too early.

Example:

 
@.MUs.Cont.eraseFailureEvents
