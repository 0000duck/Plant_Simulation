getDataType
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > getDataType

Syntax:

 
Path.getDataType(Column:any) → string

The method getDataType returns the data type of a column of the TableFile designated by Path.

Parameter

The parameter Column of data type any designates the column.

Return Value

The return value has the data type string.

Example:

 
print MyTableFile.getDataType(1)
