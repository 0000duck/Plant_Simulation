str_to_speed
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Physical Data Types > str_to_speed

Syntax:

 
str_to_speed(string) → speed

The function str_to_speed converts the data designated by the parameter of data type string into a number of data type speed. You can enter one of these unit into which the function converts the string: mm/s, cm/s, m/s, km/h, m/min, in/s, ft/s, yd/s, or mph,

If you do not enter a unit, Plant Simulation outputs the acceleration according to the settings you selected under File > Model Settings/Preferences > Units > Speed.

Suppose Plant Simulation outputs the speed in a dialog window. Plant Simulation outputs the speed according to the settings you selected under File > Model Settings/Preferences > Units > Speed. When evaluating the value, Plant Simulation has to reconvert it to m/s (meters per second). The function str_to_speed does this in a single step.

Parameter

The parameter Data has the data type string.

Return Value

The return value has the data type speed.

Example:

 
-- reads values in dialog window
var entry : string
entry := dialog.getValue("move forward")
print str_to_speed(entry)

Example:

 
str_to_speed("10")
-- interprets 10 according to the speed setting you selected under Preferences

Example:

 
str_to_speed("10m/ss")
-- interprets 10 as meters per square second
