Indent Text
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Indent Text

Enter a greater than sign > at the start of a line to indent a paragraph. The paragraph can span several lines. Terminate it with an empty line. To indent the paragraph more, enter more than one greater than sign >> at the start of the line. 

> This paragraph
is indented.

Non-indented body text after the indented paragraph.
