Export XML File
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Export XML File

The command Export XML File saves the list as an XML file (.xml). It opens the dialog Save As in which you select the drive, and the file type and enter a new name for the file to be saved. Note that Plant Simulation uses UTF-8 encoding. Click Save to save the file.
