dayOfWeek
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Managing Date and Time > dayOfWeek

Syntax:

 
dayOfWeek(Date:date) → integer
dayOfWeek(DateTime:dateTime) → integer

The function dayOfWeek returns the number of days that has elapsed since the last Sunday.

The return value for Monday is 1, the return value for Sunday is 0.

Parameter

The parameter Date of data type date designates the date.

Return Value

The return value has the data type integer.

Day of Week

Return Value

Monday

1

Tuesday

2

Wednesday

3

Thursday

4

Friday

5

Saturday

6

Sunday

0

Example:

 
var d: date, dt: dateTime
dt := str_to_dateTime("3.1.99 1:20:30")
d := str_to_date("1.1.99")
print dayOfWeek(dt)  -- 0 
print dayOfWeek(d)   -- 5
