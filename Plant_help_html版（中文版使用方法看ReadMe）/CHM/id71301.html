appendToContent
2D Reference Help > Display and User Interface Objects > Comment > Methods of the Comment > appendToContent

Syntax:

 
Path.appendToContent(Text:string)

The method appendToContent appends the passed text to the existing text on the Tab Comment which the object Comment shows. Plant Simulation requests memory in larger blocks from the operating system so that memory does not have to be increased with each call. This makes putting a message together from many text blocks faster.

Parameter

The parameter Text of data type string designates the text which you want to add.

Example:

 
MyComment.appendToContent("my added text")
