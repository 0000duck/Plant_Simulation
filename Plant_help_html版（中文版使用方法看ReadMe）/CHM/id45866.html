Export Object
3D Reference Help > 3D Reference > Edit Ribbon Tab > Export Scene > Export Object

The command Export Object opens the dialog Save As. You can export the object as an object file in the format 3D Files (*.s3d). s3d is a proprietary format which can store the graphic data together with the animation data. You can then exchange the graphic data and the animation data of another object in the active model with this file. To do so, click Exchange Graphics on the Edit ribbon tab.

When you select an object which does not have an origin any more, i.e. a class without origin or an animated object which you inserted into the respective inheritance level, the command Export Object applies to this object instead of the object that is displayed as the scene.

Select drive and folder from the drop-down list and enter a name for the scene you are going to export into the text box next to File name.

You can open these files again and continue modeling with them. Note that a scene encompasses all objects that are descendants of the current root of the scene.
