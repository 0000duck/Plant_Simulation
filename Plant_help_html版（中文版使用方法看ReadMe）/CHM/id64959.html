Stop on Controls
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Debugger Ribbon Tab > Stop on Controls

The command Stop on Controls stops the simulation when a control is called, for example an entrance or exit control you entered into an object. Plant Simulation shows the called method in the Method Debugger. Plant Simulation only stops when the control itself is called, not when subprograms are called.

You can also select Stop on Controls on the Debug Menu of the Method Debugger to toggle the setting Stop on Controls.

When you click the Down arrow on the button, Plant Simulation shows the command Stop on Controls in Object.
