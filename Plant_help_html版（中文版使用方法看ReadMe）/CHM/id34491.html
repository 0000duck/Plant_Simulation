Group Graphics
3D Reference Help > 3D Reference > Edit Ribbon Tab > Group Graphics

The command Group Graphics groups the selected graphics and combines them into one object. This combined graphic then contains the selected graphics.

To select several graphics, hold down Shift and click the objects. Then click Group Graphics or press Ctrl+G.
