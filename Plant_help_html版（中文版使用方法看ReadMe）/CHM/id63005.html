printList
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Accessing Lists and Tables > printList

Syntax:

 
Path.printList([string]) → boolean

The method printList prints the contents of the list designated by Path, without the icon and information about origin and class. The printout is identical with the printout you get when you click Print on the List ribbon tab.

The method applies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence.

Return Value

The return value has the data type boolean.

Example:

 
MyQueueFile.printList
MyTimeSequence.printList
