Toolbar of the Method Debugger
2D Reference Help > Information Flow Objects > Method > The Method Window > Method Debugger > Toolbar of the Method Debugger

The toolbar of the Debugger has these buttons:

To do this

Click

Menu command, Method

Open the location of the Method, i.e., the Frame in which the Method is located.

Open Location, location/ ~

Open the origin of the Method from which the active Method was derived.

Open Origin, origin

Return the Debugger to the currently executed Method.

--

Open the Watch Window.

Watch Window

Execute one line of the debugged Method at a time. Any methods being called in that line of the source code will not be loaded.

Step Over

Execute one line of the debugged Method at a time. If that line of the source code happens to call other methods, these are loaded as well.

Step Into

Step out of the current Method and move on to beginning of the next statement in the calling Method.

Step Out

Continue executing the active Method.

Continue

Stop the simulation, i.e., stop the execution of all active methods.

Terminate Simulation

Insert a breakpoint into the class or remove it from it.

Class Breakpoint

Delete all breakpoints in the Method.

Remove All Breakpoints , clearAllBreakpoints

Open the Find dialog.

Find

Search toward the beginning of the Method.

Find Previous

Search toward the end of the Method.

Find Next

Show or hide the line number at the beginning of a line.

Show Conveyor Numbers
