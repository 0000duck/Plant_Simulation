Icons Showing the Function of the Method
2D Reference Help > Information Flow Objects > Method > Icons Showing the Function of the Method

Just like other objects the object Method has a number of icons. Plant Simulation shows the Method in the Class Library with the icon named Default. If you find that the icons we defined do not meet your needs, you can also draw any number of additional icons and assign a name to each one of them. Note that Plant Simulationautomatically selects the corresponding icon when it calls the methods init, reset, and endSim.

As long as the source code of the object still contains syntax errors, the icon of the object switches to the icon named Error . This makes finding the Methodwithin a Frame very easy. As soon as you corrected the syntax error, the object returns to the original icon. Note that this mechanism does not work in the Class Libraryas the Default icon is always displayed there.

To show and to edit the icons of the Method, right-click the icon and select Edit Icons. Click the Previous Icon or Next Icon button on the Edit ribbon tab of the Icon Editor to cycle through the predefined icons.
