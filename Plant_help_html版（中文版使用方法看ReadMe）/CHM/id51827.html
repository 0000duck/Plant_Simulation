ConnectCtrl
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of All Objects > Attributes for Controls of All Objects > ConnectCtrl

Syntax:

 
Path.ConnectCtrl

The attribute ConnectCtrl designates a Method object. The program calls it, when you connect two objects with a Connector and when you delete the connection between two objects.

When you insert a Connector, Plant Simulation first calls the Connect control for the Connector before calling the Connect controls for the objects you are connecting.

When you delete a Connector, Plant Simulation first calls the Connect control for the connected objects before calling Destructor control.

Parameters

The program passes three parameters to the Method that is called:

  • The parameter ConnectionStart of data type object designates the source object at which the connection starts.

  • The parameter ConnectionEnd of data type object designates the target object where the connection ends.

  • The parameter ConnectionEstablishedDeleted of data type boolean indicates if the connection is to be established (true) or to be deleted (false).

A connect control might look like this:

Example:

 
param ConnectionStart, ConnectionEnd: object, ConnectionEstablishedDeleted: boolean
if ConnectionEstablishedDeleted
   print "New"
else
   print "Deleted"
end
   print "from:",ConnectionStart," to:",ConnectionEnd
end

Data Type of the Value You Can Assign

You can assign a value of data type method.

Example:

 
MyStation.ConnectCtrl := &myConnectCtrl
