copy
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Strings > copy

Syntax:

 
copy(String:string, Position:integer, NumberOfCharacters:integer)

The function copy copies a segment of a string and returns it.

Parameters

  • The parameter String of data type string designates the string from which Plant Simulation copies the segment.

  • The parameter Position of data type integer designates the position from which on Plant Simulation copies the segment.

  • The parameter NumberOfCharacters of data type integer designates number of characters you would like to copy.

Plant Simulation does not display an error message when integer is less than 1 or integer is greater than the length of the parameter of data type string.

Example:

 
print copy("abcdef",2,3)   -- returns "bcd"
print copy("abcdef",4,10)  -- returns "def"
print copy("abcdef",-1,4)  -- returns "ab" (-1 and 0 letter are counted)
