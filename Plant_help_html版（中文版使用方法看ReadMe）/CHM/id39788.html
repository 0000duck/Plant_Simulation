Illustrating State Graphics
Step-by-Step Help > Modeling in 3D > Working with Graphics > Illustrating State Graphics > Illustrating State Graphics

State graphics correspond to the LEDs in Plant Simulation 2D, compare States and Icons of the Material Flow Objects. Every state graphic visualizes the object state it names and has the same color as the respective LED in 2D. Plant Simulation show them whenever it displays the respective LED in 2D. Note that several state graphics can be displayed at the same time.

You can select the settings for state graphics on the Tab Graphics > States orientation.

  • Show States at the Front of the Machine

  • Show States on a Pole Above the Machine
