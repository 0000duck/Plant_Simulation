ProdStatOn
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes of All Mobile Objects > ProdStatOn

Syntax:

 
MU-path.ProdStatOn

The attribute ProdStatOn activates (true) or deactivates (false) the collection of Product statistics for the MU class designated by MU-path.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MUs.Part.ProdStatOn := true
