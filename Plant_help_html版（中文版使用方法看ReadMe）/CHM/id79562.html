Automatic stop
2D Reference Help > Material Flow Objects > Turnplate > Dialog Window of the Turnplate > Tab Attributes > Automatic stop

To set the current speed of the Turnplate to 0 when it does not transport a part, select this. This might, for example, be the case when it is empty or when it is blocked when a part cannot leave it.

When the speed of the Turnplate is 0, the energy state changes to Operational.
