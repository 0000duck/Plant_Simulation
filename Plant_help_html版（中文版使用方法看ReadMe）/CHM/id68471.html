userInterfaceLanguage
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > userInterfaceLanguage

Syntax:

 
userInterfaceLanguage → integer

The function userInterfaceLanguage returns the language of the user interface of Plant Simulation.

0 designates German, 1 English, 2 Japanese, 3 Chinese, 4 Russian, and 8 designates Hungarian

Return Value

The return value has the data type integer.

Example:

 
if userInterfaceLanguage = 0      -- German
   print "Guten Morgen"
elseif userInterfaceLanguage = 1  -- English
   print "Good morning"
elseif userInterfaceLanguage = 2  -- Japanese
   print "Ohayo gozaimasu"
elseif userInterfaceLanguage = 3  -- Chinese
   print "Zao an"
elseif userInterfaceLanguage = 4  -- Russian
   print "Dobroje utro"
elseif userInterfaceLanguage = 8  -- Hungarian
   print "Jó reggelt"
end
