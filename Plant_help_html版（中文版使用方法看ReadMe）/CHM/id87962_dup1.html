Locked
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab Graphics > Object graphics > Graphic Groups > Add > Locked

To lock the graphic group in the 3D windows which are opened for this object, select this. This then prevents you from selecting this graphic group in the 3D windows which are opened for this object. This makes, for example, sense for the graphic of a factory building which might otherwise be selected inadvertently.

Instead, you can also select the check box on the table on the tab itself.
