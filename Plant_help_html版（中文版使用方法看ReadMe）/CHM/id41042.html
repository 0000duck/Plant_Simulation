Define the Input Values in the Experiment Table
2D Reference Help > Tools > ExperimentManager > Defining the Experiments > Define the Input Values in the Experiment Table

To open the experiment table ExpTable, click Define Experiments.

In this table you can define the input values, which only depend on the number of the experiment. We call this type of parameterization static parameterization. To deactivate static parameterization, clear the check box Use input values. You can only open the experiment table ExpTable by clicking the button Define Experiments after you did set the input values.

Defining the input values and the experiment table

You can exclude individual experiments from a simulation study by typing false into the corresponding cell in the column Active. By typing in true, you can execute these experiments at a later point in time by clicking Start without having to repeat already executed experiments. The following columns show the input values which you entered into the table Input. You can enter the values for the individual experiments.
