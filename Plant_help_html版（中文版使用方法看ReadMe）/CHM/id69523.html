ShowFillLevel
2D Reference Help > Material Flow Objects > Buffer > Attributes of the Buffer > ShowFillLevel

Syntax:

 
Path.ShowFillLevel

The attribute ShowFillLevel sets if the buffer shows its fill level below its icon in the Frame (true) or not (false).

Fill Level On

Fill Level Off

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
Buffer.ShowFillLevel := true
