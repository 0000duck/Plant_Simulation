Tab Advanced
Add-Ins Reference Help > Object Libraries > Kanban Objects > KanbanBuffer > Dialog Window of the KanbanBuffer > Tab Advanced > Tab Advanced

On the tab Advanced you can extend and modify the properties of the Kanban system. You can access the tables and the Store on which the KanbanBuffer is based. In addition, you can manipulate how parts are ordered.

The tab Advanced provides these dialog items:

Calculate capacity

Open User-defined Method sortList

Show List of Open Requests

Open Workplace

Direct order mode
