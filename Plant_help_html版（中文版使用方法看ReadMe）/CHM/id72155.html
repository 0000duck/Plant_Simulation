stopMU
2D Reference Help > Material Flow Objects > Turntable > Methods of the Turntable > stopMU

Example:

 
Path.stopMU

The method stopMU stops the MU while the Turntable designated by Path is still rotating. Once the rotation is finished, the MU automatically moves again. You might use this method to stop the MU before it reaches the end of the table to prevent collisions for example. You will mostly use the method in a sensor control.

Example:

 
MyTurntable.stopMU
