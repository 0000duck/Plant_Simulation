getItems
Add-Ins Reference Help > Inter-Process Communication Interfaces > OPCClassic > Methods of the OPC Interface > getItems

Syntax:

 
Path.getItems(Table:table)

The method getItems returns the items which you defined in the Items table, and writes them into a table.

Parameter

The parameter Table of data type table designates the name of the table.

Example:

 
var MyItemsTable: table
OPCClassicInterface.getItems(MyItemsTable)
