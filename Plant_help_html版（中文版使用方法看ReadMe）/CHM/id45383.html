BottleneckAnalyzer
2D Reference Help > Tools > BottleneckAnalyzer

To add the BottleneckAnalyzer to the Class Library, click Manage Class Library > Libraries > Tools > Bottleneck Analyzer on the Home ribbon tab.

The BottleneckAnalyzer visualizes the standard statistics of the material flow objects and sorts the data into a table. It displays the statistics of the material flow objects that are located in the same Frame as the BottleneckAnalyzer or in Sub-frames of that Frame. The BottleneckAnalyzer works statically, i.e., it does not collect data during the simulation run. For this reason you can insert it even after the simulation run is finished.

If, for example, you would like to increase the throughput of a production, you will start looking for the bottleneck at the Drains. The machines causing the bottleneck will have a high working portion. When the machines causing the bottleneck work faster, the throughput of the entire system will increase.

For a Frame the BottleneckAnalyzer shows the chart of the machine, which has the greatest working portion. To continue looking for the bottleneck in a Frame, check the chart of the objects, which you inserted into the Frame.

When you click OK, the BottleneckAnalyzer opens a table, which shows all material flow objects with the statistical values, which you selected.

We modeled the BottleneckAnalyzer as an application object in a Frame. For this reason opening help with F1 and What’s This Help in the dialogs of the objects do not work. To open help for the object, select Help > Help on BottleneckAnalyzer in its dialog.

We modeled the dialogs of the objects with the object Dialog. The windows of the objects do not use the standard Siemens PLM theme, but your Windows theme.
