Window of the Table File
2D Reference Help > Information Flow Objects > TableFile > Window of the Table File

Double-click the icon of the TableFile, which you inserted into a Frame, to open its window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Information Flow in the Toolbox. You can access the functions of the list objects on the List Ribbon Tab.
