closeDialog
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Methods of All Objects > General Methods of All Objects > closeDialog

Syntax:

 
Path.closeDialog([ApplyChanges:boolean]) → boolean

The method closeDialog closes the dialog window of the object designated by Path.

Parameter

The optional parameter ApplyChanges of data type boolean determines if Plant Simulation ignores all changes you made in the dialog (false) or if it accepts and applies your changes (true).

Return Value

The return value has the data type boolean.

Example:

 
MyStore.closeDialog -- apply changes
MyTrack.closeDialog(false) -- do not apply changes
