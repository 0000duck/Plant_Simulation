decrypt
2D Reference Help > Information Flow Objects > Method > Methods of the Method > decrypt

Syntax:

 
&Method.decrypt(Key:string) → boolean

The method decrypt decrypts the Method designated by &Method you or somebody else encrypted.

Parameter

The parameter Key of data type string designates the password.

Return Value

The return value has the data type boolean.

Example:

 
&MyMethod.decrypt("zTrqYa8%e")
