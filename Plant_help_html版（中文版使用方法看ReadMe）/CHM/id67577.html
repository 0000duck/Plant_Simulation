Attributes of the Failure Profiles
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for Defining Failures of the Material Flow Objects > Attributes of the Failure Profiles > Attributes of the Failure Profiles

The static material flow objects provide the attributes Active, Availability, AvailabilityOn, Duration, Failed, Interval, Mode, MTTR, Name, Start, and Stop for defining failure profiles.

The objects address failure profile attributes as sub-attributes of Failures.

Example:

 
MyStation.Failures.MyFailureProfile1.Active := true
