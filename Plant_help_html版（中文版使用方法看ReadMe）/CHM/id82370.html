getEditorRightsColumn
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > getEditorRightsColumn

Syntax:

 
Path.getEditorRightsColumn(Column:any) → boolean

The method getEditorRightsColumn returns the editor rights of a column of the TableFile designated by Path.

Parameter

The parameter Column of data type any designates the column whose editor rights you want to get.

Return Value

The return value has the data type boolean: true for read-only access, false for read and write access.

Example:

 
MyTableFile.getEditorRightsColumn(1)
print MyTableFile.getEditorRightsColumn(1)
