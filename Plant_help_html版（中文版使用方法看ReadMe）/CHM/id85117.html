close
2D Reference Help > Display and User Interface Objects > Dialog > Methods of the Dialog > close

Syntax:

 
Path.close(OK:boolean)

The method close closes the dialog window of the Dialog you defined. When you enter false as parameter, the Dialog cancels all changes the user made in the dialog. When you enter true it accepts and applies the changes the user made.

Parameter

The parameter OK of data type boolean designates the action the dialog takes, OK or Cancel.

Example:

 
MyDialog.close(true)
