Efficiency
2D Reference Help > Resource Objects > Worker > Attributes of the Worker > Efficiency

Syntax:

 
Path.Efficiency

The attribute Efficiency sets the Efficiency of the Worker, i.e., how fast he performs the task assigned to him:

  • With an efficiency of 100 percent he consumes the exact processing time, which you entered.

  • With an efficiency of 200 percent he consumes half of the processing time, which you entered.

  • With an efficiency of 50 percent he consumes twice of the processing time, which you entered.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
.Resources.MyWorker:2.Efficiency := 90
