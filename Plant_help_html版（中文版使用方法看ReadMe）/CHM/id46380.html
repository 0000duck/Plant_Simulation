CollectData
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > CollectData

Syntax:

 
Path.CollectData

The attribute CollectData sets if a Chart of type Histogram or Plotter collects data (true ) or does not collect data (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyChart.CollectData := true
