StatStoreSetupTime

Syntax:

 
MU-path.StatStoreSetupTime

The read-only attribute StatStoreSetupTime returns the time during which a material flow object of type Storage was setting-up for the MU class designated by MU-path.

Return Value

The return value has the data type real.

Example:

 
print .MUs.Transporter.StatStoreSetupTime
