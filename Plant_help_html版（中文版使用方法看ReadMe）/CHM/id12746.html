animIcon
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > animIcon

Syntax:

 
animIcon([Activate:boolean]) → boolean

The function animIcon activates (true) or deactivates (false) Icon Animation . When you deactivate MU animation, Plant Simulation redraws the icon only when necessary, i.e., when the icon was selected or was covered by another item. Note that this does not affect the display of MUs and that the function also returns the previous setting.

Parameter

When you do not enter the optional parameter, the function returns if icon animation is activated or not.

Return Value

The return value has the data type boolean.

Example:

 
animIcon(false)
print animIcon
