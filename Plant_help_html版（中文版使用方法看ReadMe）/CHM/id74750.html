Sampler
2D Reference Help > Display and User Interface Objects > Display > Attributes of the Display > Sampler

Syntax:

 
Path.Sampler

The attribute Sampler activates Sample mode (true) or deactivates it (false).

  • Sample mode updates the display periodically, independent of changes of the input value.

  • Watch mode updates its display, whenever the input value changes.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyDisplay.Sampler := true -- activates sampling
