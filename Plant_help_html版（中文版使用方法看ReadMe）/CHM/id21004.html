Items
2D Reference Help > Display and User Interface Objects > Drop-Down List > Dialog Window of the Drop-Down List > Tab Attributes > Items

Click this to open the list into which you enter the items which the Drop-Down List displays when you click it.

If you want to use the index number to access an item in the list, you can use the attribute Value.

If you want to use the name to access an item in the list, you can use the attribute Item.
