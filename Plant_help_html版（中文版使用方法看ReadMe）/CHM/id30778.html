Using a Named-User License
Setting-Up and Starting > Licensing Plant Simulation > Using a Named-User License

A named-user license is a floating license that assigns a license to a single individual user. A named-user license utilizes the login name of the user for the local area network. The user can execute the program on any computer if his name is contained in a list of users in the ugslmd.optfile.

The SPLM License Service has to run on a server to assign and manage the licenses. You do not have to install it on the individual client computers running Plant Simulation.
