Find Help Topics Using the Toolbar Buttons
Using HTML Help > Working with Full Text Search > Find Help Topics Using the Toolbar Buttons

To find help topics, click the respective button on the toolbar of the Help Viewer:

  • Locate show where the current topic is located in the table of contents.

    If the current topic is not listed on the table of contents, this command will not work. This might, for example, happen if you clicked an index entry, which does not exactly match the title of the page. To locate the topic anyway, click the button or the button in the Topic pane and then click Locate again and return to the topic, which you wanted to locate in the first place.

  • Back show the last topic you viewed.

  • Forward show the next topic in a previously displayed sequence of topics.

  • Home show the Home page topic for the help file you are viewing.
