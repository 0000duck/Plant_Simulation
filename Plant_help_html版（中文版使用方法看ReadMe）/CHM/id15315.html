RotationPoint
2D Reference Help > Material Flow Objects > Turntable > Attributes of the Turntable > RotationPoint

Syntax:

 
Path.RotationPoint

The attribute RotationPoint (Rotation point) sets the position of the point around which the Turntable rotates. You can enter a value between 0 and the length of the Turntable. 0 designates the point at which you start inserting the Turntable.

Data Type of the Value You Can Assign

You can assign a value of data type length.

Example:

 
MyTurntable.RotationPoint := 4
