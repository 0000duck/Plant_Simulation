rollDice
2D Reference Help > Information Flow Objects > Variable > Methods of the Variable > rollDice

Syntax:

 
Path.rollDice → time

For the data type randtime the method rollDice creates a new random number and saves it to the Variable.

Return Value

The return value has the data type time.

Example:

 
for var i := 1 to 10 
   variable.rolldice -- new random number
   print variable    -- output random numbers
