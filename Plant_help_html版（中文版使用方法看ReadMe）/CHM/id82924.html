setColumnWidth
2D Reference Help > Information Flow Objects > TimeSequence > Methods of the TimeSequence > Methods for Columns > setColumnWidth

Syntax:

 
Path.setColumnWidth(Column:any, ..., ColumnWidth:integer)

The method setColumnWidth sets the width of a column of the TimeSequence designated by Path to the designated value.

Parameters

  • The parameters Column of data type any designate the columns whose width you would like to set.

  • The parameter ColumnWidth of data type integer designates the width in character widths of a non-proportional font.

Example:

 
timeSequence1.setColumnWidth("column2",10)
