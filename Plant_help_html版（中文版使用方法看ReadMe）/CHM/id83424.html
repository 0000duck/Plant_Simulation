_3D.exchangeGraphic
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for Accessing Graphics > _3D.exchangeGraphic

Syntax:

 
Path._3D.exchangeGraphic(FilePath:string[, UseNewGraphicMeasurement:boolean]) → boolean

The method _3D.exchangeGraphic exchanges the entire graphic of the selected object designated by Path with the graphic you imported from a file.

Parameter

  • The parameter FilePath of data type string sets the path and the name of the file you want to import.

  • The optional parameter UseNewGraphicMeasurement of data type boolean sets if the new graphic measurements are also to be used for the simulation measurements of the MU. When you enter true and the object is an MU and when the attribute _3D.AutoGraphicsActive of the loaded .s3d file is set to false, this attribute is set to true again and the graphic measurements are applied to the simulation. In all other cases the method does not execute anything in addition.

Return Value

The return value has the data type boolean.

Example:

 
MyFrame._3D.exchangeGraphic("C:\Program Files\Siemens\Tecnomatix Plant Simulation 14.0\3D\s3D-graphics\Lathe.s3d")
