Absolute
2D Reference Help > Information Flow Objects > Trigger > Attributes of the Trigger > Absolute

Syntax:

 
Path.Absolute

The attribute Absolute sets the Time reference of the Trigger to Absolute (true) or Relative (false). After this time interval has elapsed, the Trigger value will attain the default value. Define the default value under Values > Values > Value Table.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyTrigger.Absolute := true
