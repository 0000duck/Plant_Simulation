Window of Lists and Tables
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > Window of Lists and Tables

Double-click the icon of a list object to open its window. To change the properties of the Class of the object, double-click it in the Class Library or on the tab Information Flow in the Toolbox. You can then view and edit the data or enter new information. You can also customize the format and the data types to meet your needs.

You can access the functions of the list objects on the List Ribbon Tab.
