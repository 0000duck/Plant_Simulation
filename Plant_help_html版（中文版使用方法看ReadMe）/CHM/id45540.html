_3D.addGraphicGroup
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Graphic Groups > _3D.addGraphicGroup

Syntax:

 
Path._3D.addGraphicGroup(Name:string, Visible:boolean, Internal:boolean [, Locked:boolean])

The method _3D.addGraphicGroup adds the graphic group to the object designated by Path.

The method deactivates graphic inheritance.

Parameter

  • The parameter Name of data type string designates the name of the graphic group.

  • The parameter Visible of data type boolean sets if the graphic group will be visible (true) or not visible (false).

  • The parameter Internal of data type boolean sets if the graphic group will be internal (true) or external (false).

  • The optional parameter Locked of data type boolean sets if the graphic group will be locked (true) or not locked (false).

Example:

 
MyStation._3D.addGraphicGroup("MyGraphicGroup", true, true)
