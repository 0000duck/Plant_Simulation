ExperimentManager
2D Reference Help > Tools > ExperimentManager > ExperimentManager

The ExperimentManager supports you in executing simulation studies.

To add the ExperimentManager to the Class Library, click Manage Class Library > Libraries > Tools > Experiment Manager on the Home ribbon tab.

Your Plant Simulation license determines what you can do with ExperimentManager.

We modeled the ExperimentManager as an application object in a Frame. For this reason opening help with F1 and What’s This Help in the dialogs of the objects do not work. To open help for the object, select Help > Help on ExperimentManager in its dialog.

We modeled the dialogs of the objects with the object Dialog. The window of the ExperimentManager does not use the standard Siemens PLM theme, but your Windows theme.

Execute Experiments with the ExperimentManager shows how you can run your simulation experiments even more productively.

Compare the sample models: Click the Window ribbon tab, click Start Page > Getting Started > Example Models, and click Small Examples. Then, select the respective Category > Tools and Optimization and the Topic >Experiment Manager, Buffer Conveyor Optimization, Experimental design, and Variance Reduction in the dialog Sample Models and click Open Model.

Compare the video Using the ExperimentManager under Window > Start Page > Getting Started > Videos.
