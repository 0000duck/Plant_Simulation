_3D.createTiledPlate
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating 3D Shapes > _3D.createTiledPlate

Syntax:

 
_3D.createTiledPlate(BasePoint:length[3], FilePath:string, Orientation:integer, Size:length[2], Bifacial:boolean, TileSize:length[2] [, Thickness:length, GraphicGroupName:string]) → string

The method _3D.createTiledPlate creates a plate which shows the picture or the texture loaded from a picture tiled.

Parameters

  • The parameter BasePoint is an array of data type length which sets the base point of the picture.

  • The parameter FilePath of data type string sets the path and the name of the file you want to import.

  • The parameter Orientation of data type integer sets the sets the orientation of the picture. 1 stands for Floor, 2 stands for Front Wall, and 3 stands for Side Wall.

  • The parameter Size is an array of data type length which sets the width and the height of the picture. Enter a value less than or equal to 0 to ignore the width and/or the height.

  • The parameter Bifacial of data type boolean sets if the texture will be shown on both sides of the plate (true) or only on the top side of the plate (false).

  • The parameter TileSize is an array of data type length which sets the width and the height of a tile. Enter a value less than or equal to 0 to ignore the width and/or the height.

  • The optional parameter Thickness of data type length designates the thickness of the plate.

  • The optional parameter GraphicGroupName of data type string designates the name of the graphic group you are working with. The parameter is described underThe Parameters GraphicPath and GraphicGroupName.

Return Value

The return value has the data type integer and specifies the numerical index of the textured plate, compare _3D.getGraphicIndex.

Example:

 
_3D.createTiledPlate([2.0,2.0,1], "D:\Picture.png", 1, [4.0,2.0], true, [2.0, 2.0], 0.1)

Example:

 
_3D.createTiledPlate([2.0, 2.0, 1], "D:\Picture.png", 1, [4.0, 2.0], true, [2.0, 2.0], 0.1, "deco")

Example:

 
var index := _3D.createTiledPlate([2.0,2.0,1], "D:\Picture.png", 1, [4.0,2.0], true, [2.0, 2.0], 0.1, "deco")
_3D.setGraphicMaterial([index], makeRGBValue(255,0,0), 0, 0, 0, 0, 0, "deco")
