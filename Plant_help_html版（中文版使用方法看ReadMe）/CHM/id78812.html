Before we can connect and run this model, we have to edit the Test objects. To do this, we will edit the object Test in the Class Library. Editing the class object automatically propagates our changes to all of its instances in our simulation model.

  • Open the Frame Test in the Class Library.

  • Delete the Source, the Drain, and the EventController from the Frame named Test.

You will notice that this also deleted the Source, the Drain, and the EventController from the Frames Test1 and Test2.
