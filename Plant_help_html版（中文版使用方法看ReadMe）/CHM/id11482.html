trim
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Strings > trim

Syntax:

 
trim(Text:string) → string

The function trim returns a string from which Plant Simulation removed leading and trailing blanks, tab stops, and carriage returns.

Parameter

The parameter Text of data type string designates the string, which contains leading and trailing blanks, tab stops, and carriage returns, i.e., whitespace characters.

Return Value

The return value has the data type string.

Example:

 
print trim(" ab c ") -- returns "ab c"
