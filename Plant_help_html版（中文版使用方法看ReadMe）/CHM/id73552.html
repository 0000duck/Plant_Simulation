Camera Marks
3D Reference Help > 3D Reference > View Ribbon Tab > Camera Marks > Camera Marks

The command Camera Marks saves the active view of the scene. It opens the dialog Mark Current Camera Settings, where you enter a name for your camera mark.

To return to this camera mark, you can click the button on the View ribbon tab and select it. Plant Simulation shows the path to the scene in the field Scene Path. You can define up to 40 camera marks.

  • Enter a name for the current camera setting into the text box Name or accept the name, which 3D suggests.

  • 3D shows the path of the scene in the field Scene path. Click OK to set the name. Plant Simulation then saves this setting together with the active root object of the scene in the model file. This way you can easily return to this saved view after you changed the scene or reopened the model.

  • Then, 3D shows the name in the list, when you click the small down arrow at the bottom of the button . Plant Simulation enters the camera mark you defined last at the top of the list. Select the camera mark you would like the 3D to use there. You can define as many camera marks as you like.

  • The command Camera Marks opens the dialog Camera Marks. You can activate, rename, move up or move down, or delete the selected camera mark in the list.

  • The command Generate SimTalk Code and Copy to Clipboard generates SimTalk code for the currently defined camera parameters and for the object of interest and copies that code to the clipboard.

At present 3D provides three cameras:

  • The main camera through which you view the scene in normal view.

  • The object camera, which you attach to a specific object and which moves with this object through the model.

  • The animation camera, which moves along an animation path you defined.

The command removes any of the transformations you defined and returns the object or animation camera to the initial settings the cameras had before you attached a camera or started the animation.

You can set, save and call view points, called camera marks, which save the position and the orientation of the main camera used for displaying the scene in normal view together with the root object of the scene. The button and list Camera Marks allows you to return to these camera marks at any point in time later on.

3D, on the other hand, automatically saves the active setting of the camera together with the root object of the scene when you change the view on the root object of the scene.

You can also animate the camera. This creates the impression of you flying on the animation path of the selected object or of the root object of the scene.

The horizontal viewing angle in the 3D windows is 51.2 degrees.
