Built-in Methods for Creating 3D Shapes
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating 3D Shapes > Built-in Methods for Creating 3D Shapes

Plant Simulation provides these methods for creating 3D shapes in 3D:

_3D.createBox, _3D.createCone, _3D.createCuboid, _3D.createCylinder, _3D.createFrustum,_3D.createIndexedFaceSet, _3D.createIndexedConveyorSet, _3D.createPicture, _3D.createSphere, _3D.createText, and _3D.createTiledPlate.

You can view and access the methods for accessing graphics by clicking Auto Complete on the Edit ribbon tab of the method editor.
