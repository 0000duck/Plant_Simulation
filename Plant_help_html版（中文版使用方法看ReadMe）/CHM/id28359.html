getVisibility
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Columns of the TableFile > getVisibility

Syntax:

 
Path.getVisibility(Column:any) → boolean

The method getVisibility returns if a column of the TableFile designated by Path is visible (true) or hidden (false).

Parameter

The parameter Column of data type any designates the column.

Return Value

The return value has the data type boolean.

Example:

 
print MyTableFile.getVisibility(4)
