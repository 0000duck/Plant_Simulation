Sequence
Add-Ins Reference Help > Object Libraries > Kanban Objects > KanbanSingleProc > Attributes of the KanbanSingleProc > Sequence

Syntax:

 
Path.Sequence

The attribute Sequence sets if the KanbanStation randomly selects a new order (false) or uses a sequence list to do so (true).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyKanbanStation.Sequence := true
