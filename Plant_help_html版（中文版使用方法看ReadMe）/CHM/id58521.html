StopCtrl
2D Reference Help > Resource Objects > LockoutZone > Attributes of the LockoutZone > StopCtrl

Syntax:

 
Path.StopCtrl

The attribute StopCtrl designates a Method object, the Stop control, which the LockoutZone calls when one of the objects assigned to the LockoutZone fails. The anonymous identifier @ designates the triggering station, the anonymous identifier ? designates the LockoutZone.

Data Type of the Value You Can Assign

You can assign a value of data type method.

Example:

 
var row:integer := ReportStopping.ydim + 1
ReportStopping["First Failed Station",row] := @.name
ReportStopping["Start Stopping",row] := eventcontroller.simTime

Example:

 
MyLockoutZone.StopCtrl := &myStopCtrl
