_3D.MaterialSpecularColor
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Methods for the Appearance of the MUs in 3D > _3D.MaterialSpecularColor

Syntax:

 
MU-path._3D.MaterialSpecularColor

The attribute _3D.MaterialSpecularColor sets the specular color of the auto graphics of the mobile object designated by MU-path.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
.MUs.Part._3D.MaterialSpecularColor := makeRGBValue(0,255,0)
