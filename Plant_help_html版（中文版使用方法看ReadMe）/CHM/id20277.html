Tab Statistics
2D Reference Help > Material Flow Objects > Source > Dialog Window of the Source > Tab Statistics > Tab Statistics

In addition to the features described under the Tab Statistics, the tab Statistics provides the dialog items Creation table and Open.

To view more detailed statistics in the Statistics Report, click the object in the Frame, and press F6 (Show Statistics Report), or click Show Statistics Report on the Home ribbon tab.

You can also click the object in the Frame with the right mouse button and select Show Statistics Report on the context menu.
