GapWhenNull
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > GapWhenNull

Syntax:

 
Path.GapWhenNull

The attribute GapWhenNull sets if the Chart interrupts the line it draws for values that are not defined (true) or if it continues drawing the line (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyChart.GapWhenNull := false
