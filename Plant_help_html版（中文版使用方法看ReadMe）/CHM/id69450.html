MUs To Be Deleted
2D Reference Help > Material Flow Objects > AssemblyStation > Dialog Window of the AssemblyStation > The View Menu > MUs To Be Deleted

The menu command MUs To Be Deleted opens a list with the MUs located at present on the AssemblyStation that it deletes when the main part exits.
