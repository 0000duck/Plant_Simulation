ExitLocked
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes for the State of the Material Flow Objects > ExitLocked

Syntax:

 
Path.ExitLocked

The attribute ExitLocked locks (true) or unlocks (false) the exit of the object designated by Path, so no MU can exit. Plant Simulation enters MUs attempting to exit unsuccessfully into the Exit Blocking List of the object. When you set ExitLocked to false MUs entered into the blocking list will make another attempt to exit the object.

You can also lock or unlock the exit with the check box Exit locked.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
MyStation.ExitLocked := true
