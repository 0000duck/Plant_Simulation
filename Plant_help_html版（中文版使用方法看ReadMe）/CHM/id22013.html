_3D.generateHollowRectangle
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Creating Extrusion Profiles > _3D.generateHollowRectangle

Syntax:

 
basis.~._3D.generateHollowRectangle(BasePoint:length[2], Width:length, Height:length, BarThickness:length)

The method _3D.generateHollowRectangle creates a two-dimensional hollow rectangular profile which you can use in your user-defined extrusion configuration.

  • The parameter BasePoint is an array of data type length which sets the base point of the hollow rectangular profile.

  • The parameter Width of data type length sets its width.

  • The parameter Height of data type length sets its height.

  • The parameter BarThickness of data type length sets the thickness of the bar.

Example:

 
var profile : any := basis.~._3D.generateHollowRectangle([2.6,4.5], 4, 2, 3, 0.1)
 --  returns a two-dimensional array containing the values defining a profile; to be used for an extrusion profile
