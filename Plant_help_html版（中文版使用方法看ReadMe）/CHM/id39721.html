StatRotationEmptyTime
2D Reference Help > Material Flow Objects > Turntable > Read-Only Attributes of the TurnTable > StatRotationEmptyTime

Syntax:

 
Path.StatRotationEmptyTime

The read-only attribute StatRotationEmptyTime returns the total time during which the Turntable designated by Path was rotating Empty, i.e., no part being located on the table.

Return Value

The return value has the data type time.

Example:

 
print MyTurntable.StatRotationEmptyTime
