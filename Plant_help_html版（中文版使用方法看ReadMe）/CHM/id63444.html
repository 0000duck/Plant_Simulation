Analyzing the Quality of the Neural Network
2D Reference Help > Tools > Artificial Neural Network > Analyzing the Quality of the Neural Network

Before you make the neural network compute forecasts, you will check with which quality the artificial neural network has learned the input data. For this minimizing process the error is of no immediate importance. The error can be calculated depending on the noise. If the variations are located within a tolerance interval, which results from the variance of the output values or from a percentage, which you define, the error will be set to 0. You can enter this percentage into the dialog Configuration > Settings > Noise , which the command Settings > Extended Settings opens.

Clicking Show next to As a chart on the tab Forecast to open a chart, which shows the errors of the individual data sets.

Tolerance interval for error computing

Using Tools > Calculator, you can compute all output values for a set of input values. To compute several combinations of input values one after the other, change the input values and click Apply. The grayed-out columns contain the output values, which you cannot change.
