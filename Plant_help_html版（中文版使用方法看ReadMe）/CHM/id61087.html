getPortals
Add-Ins Reference Help > Object Libraries > Cranes > MultiPortalCrane > Methods of the MultiPortalCrane > Methods of the MultiPortalCrane Proper > getPortals

Syntax:

 
Path.getPortals(Targetlist:list)

The method getPortals returns all portal objects which were created for the crane and writes them into a list or a table. These portals are required to access the functions of each portal.

Parameter

The parameter TargetList of data type list can either be a list of data type object or a table of data type table in which the column with the number 1 has the data type object. The method automatically formats the list or the table.

Example:

 
MyPortalcrane.getPortals(MyPortallist)
MyPortal := MyPortallist[1,1]
