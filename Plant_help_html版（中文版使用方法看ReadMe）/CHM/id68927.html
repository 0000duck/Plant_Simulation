Decrypt Method
2D Reference Help > Information Flow Objects > Method > The Method Window > Tools Ribbon Tab > Decrypt Method

The command Decrypt Method decrypts an encrypted Method.

Enter the correct password you used to encrypt the Method into the text box Password. In addition, you can decrypt the entire model by selecting File > Options > Decrypt Methods in the Plant Simulation window.
