Alpha
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Attributes of the Distributions > Alpha

Syntax:

 
Attribute_path.Alpha

The attribute Alpha is a positive number.

Data Type of the Value You Can Assign

You can assign a value of data type real.

You can assign the following attributes/parameters to these distributions:

Distribution

Set of parameters

Range of values for the parameters

Gamma

Alpha, Beta, LowerBound, UpperBound

Alpha > 0, Beta > 0, Upper Bound > Lower Bound

Loglogistic Distribution

Stream, Alpha, Beta, LowerBound, UpperBound

Alpha > 0, Beta > 0, Upper Bound > Lower Bound

Paralogistic Distribution

Stream, Alpha, Theta, LowerBound, UpperBound

Alpha > 0, Theta > 0, Upper Bound > Lower Bound

Pareto Distribution

Stream, Alpha, Theta, LowerBound, UpperBound

Alpha > 0, Theta > 0, Upper Bound > Lower Bound

Weibull

Alpha, Beta, LowerBound, UpperBound

Alpha > 0, Beta > 0, Upper Bound > Lower Bound
