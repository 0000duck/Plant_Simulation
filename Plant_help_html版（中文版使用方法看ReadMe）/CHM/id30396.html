Waiting
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Read-Only Attributes of All Mobile Objects > Read-Only Attributes for the State of the Mobile Objects > Waiting

Syntax:

 
MU-path.Waiting

The read-only attribute Waiting returns if the MU designated by MU-path is waiting, i.e., if the processing time of the MU has not started yet or is already finished (true). During processing the method Waiting returns false, even if processing is interrupted by a failure or a pause.

Return Value

The return value has the data type boolean.

Example:

 
var pallet : object
pallet := Station.Cont
if pallet.Waiting 
   pallet.move(Station.succ(2))
end
