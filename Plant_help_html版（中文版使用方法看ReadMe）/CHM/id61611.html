Predefined Names
SimTalk Reference > General SimTalk Functions > Names > Predefined Names > Predefined Names

Before and after a simulation run certain tasks will have to be performed: lists have to be initialized, tables have to be analyzed, and the results of a simulation run have to be written to a file. Specific method names are assigned to the four consecutive phases of a simulation (reset), initialization (init), simulation run (autoexec), and simulation end (endSim). Plant Simulation calls each method in the respective phase. These methods correspond to the respective buttons of the EventController on the tab Controls.
