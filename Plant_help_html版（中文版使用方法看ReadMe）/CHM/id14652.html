Ungroup Group of Graphics
3D Reference Help > 3D Reference > Edit Ribbon Tab > Ungroup Group of Graphics

The command Ungroup Group of Graphics ungroups the selected group of graphics, i.e., a single graphic that contains other graphics. You can then manipulate each object individually again. Ungrouping only works for a single group!

When you group any number of graphics, Plant Simulation creates a single new graphic. When you ungroup this group, 3D replaces this group with the individual members of the group and restores the state of the individual graphics as it was before the grouping action.
