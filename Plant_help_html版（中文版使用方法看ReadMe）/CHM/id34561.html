_3D.TypeOfAnimations.createAnimationCurve
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods and Read-Only Attributes for Animations in 3D > _3D.TypeOfAnimations.createAnimationCurve

Syntax:

 
Path._3D.MUAnimations.createAnimationCurve(Name:string) → boolean
Path._3D.SelfAnimations.createAnimationCurve(Name:string) → boolean
Path._3D.CameraAnimations.createAnimationCurve(Name:string) → boolean

The method _3D.TypeOfAnimations.createAnimationCurve creates a new saved animation path of type Polycurve.

You can set the values of the animation path with the method _3D.TypeOfAnimations.setTable.

Parameter

The parameter Name of data type string designates the name of the new animation path to be created.

Return Value

The return value has the data type boolean.

It tells if the animation path was created successfully, i.e., if the name has not been assigned yet.

Example:

 
var a: boolean := self.~._3D.SelfAnimations.createAnimationCurve("curved movement")
if not a 
   print "The animation curve could not be created."
end

Example:

 
Buffer._3D.MUAnimations.Default.delete
Buffer._3D.MUAnimations.createAnimationCurve("Default")
var t : table
Buffer._3D.MUAnimations.Default.getTable(t)
t.maxyDim := 2
t[1, 1] := 0
t[2, 1] := 0
t[6, 1] := 1.5
t[1, 2] := 0
t[3, 2] := 720
t[4, 2] := 2
t[5, 2] := false
t[6, 2] := 2
Buffer._3D.MUAnimations.Default.setTable(t)
