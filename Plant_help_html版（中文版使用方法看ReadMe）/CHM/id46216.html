StateResourceMissing
2D Reference Help > Material Flow Objects > Frame > Attributes of the Frame > StateResourceMissing

Syntax:

 
Path.StateResourceMissing

The attribute StateResourceMissing sets the state of the Frame to resource missing/Waiting for an Exporter (true) or not missing (false). This state applies when the Frame is waiting for resources, i.e., for services or MUs.

Then, the Frame shows an orange dot in the LED area of the icon.

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
Frame.StateResourceMissing := true
