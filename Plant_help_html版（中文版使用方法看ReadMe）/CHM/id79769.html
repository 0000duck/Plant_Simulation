Y
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Defining Properties of a Dialog Item > Y

Enter the position on the y-axis, where the program places the dialog item in the Dialog you define. The value you enter corresponds to lines of characters of the system font, not to pixels. You can enter a value from 0 to 10 for example.

If you cannot see the dialog items you inserted, check the settings for the position first. If you entered identical values for different dialog items, the Dialog places them on top of each other, so that you can only see the first item(s) you inserted.
