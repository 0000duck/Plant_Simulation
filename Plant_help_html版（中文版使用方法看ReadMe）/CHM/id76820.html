Select the Input Values for the Experiments
2D Reference Help > Tools > ExperimentManager > Defining the Experiments > Select the Input Values for the Experiments

The ExperimentManager sets the input values in the model before it starts an experiment. The input values are not changed for the simulation runs of the experiment. The input values have to be of data type string, boolean, speed, length, acceleration, real, integer, or time. You can also use parameters of probability distributions or the contents of a table as input values.

When you hold down Shift and drag a material flow object onto the ExperimentManager, it opens a dialog offering all attributes of this object. Hold down Ctrl and select one or several of these attributes.

Selecting input values and output values with drag-and-drop

The ExperimentManager only provides attributes of the data types described above. After you have selected the attributes, the ExperimentManager opens the table Input

Entering the input values

In addition to selecting attributes, you can also type the names of the attributes into it. You can also click the button Define Input Variables to open this table.

You can enter input values into the table Input or you can delete them from it. If you want to, you can also enter any explanatory text which describes the output value in plain words. All results are then represented by this text. After closing the table Input , the ExperimentManager checks the plausibility of the input values.
