Open Frame in 2D/3D
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Open Frame in 2D/3D

The command Open Frame in 3D opens the window of the counterpart of the current object in 3D or in 2D respectively.

  • When you are working with a Frame, the command opens the window of that Frame in 3D or vice versa.

  • When you are working with an object, the command opens the dialog window of that object in 3D or vice versa.
