YDim
2D Reference Help > Material Flow Objects > ParallelStation > Attributes of the ParallelStation > YDim

Syntax:

 
Path.YDim

The attributes YDim sets or gets the number of stations on the y-axis. The capacity is the product of X-dimension times Y-dimension. The greatest allowed value is ten million.

When you decrease the dimension of the ParallelStation, make sure that no MUs are located on the stations that will be deleted by this action! When MUs are located on the object, the dimension is limited, as Plant Simulation neither deletes MUs outside of the new dimension automatically nor moves them to another station!

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
ParallelStation.XDim := 3
ParallelStation.YDim := 4 -- 12 stations
