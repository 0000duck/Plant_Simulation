Destination
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Attributes of the Material Flow Objects > Attributes of the Sensors of the Length-Oriented Objects > Destination

Syntax:

 
Path.Sensor.Destination

The attribute Destination designates the object to which the Transporter on the Track designated by Path drives.

Data Type of the Value You Can Assign

You can assign a value of data type object.

Example:

 
MyTrack.sensorID(2).Destination := Station
