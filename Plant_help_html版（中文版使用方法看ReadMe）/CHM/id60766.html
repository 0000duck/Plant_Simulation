Mode
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > Tab Data > Mode

Select the mode with which the Chart updates the displayed data. Sample mode updates the Chart periodically after the interval you enter. Watch mode updates the Chart, whenever the watchable input data changes. Plot mode updates the Chart, whenever a simulation event takes place. SimTalk: Mode.

  • Sample mode updates the Chart periodically after the interval you enter.

    Enter the time span ( Interval ) after which the Chart takes the next sample. SimTalk: NumIntervals, SampleInterval.

  • Watch mode updates the Chart, whenever the observable input data changes. The column Watchable in the dialog Show Attributes and Methods, shows the methods and attributes which Plant Simulation can observe.

  • Plot mode updates the Chart, whenever a simulation event takes place.
