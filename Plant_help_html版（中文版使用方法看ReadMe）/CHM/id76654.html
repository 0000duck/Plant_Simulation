Show Connections
3D Reference Help > 3D Reference > View Ribbon Tab > Show Connections

The command Show Connections shows or hides Connectors in the active 3D scene window.

By default, Plant Simulation does not show Connections in the 3D windows. When you insert a Connector or an Interface into a Frame, the program activates the button.
