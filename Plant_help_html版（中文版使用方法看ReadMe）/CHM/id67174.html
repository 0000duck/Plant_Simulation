Draw Textbox
2D Reference Help > Material Flow Objects > Frame > The Frame Window > Vector Graphics Ribbon Tab > Draw Textbox

The command Draw Textbox draws a textbox on the background of the Frame. Enter the text into the dialog that opens.

Enter the text into the text box Text in the dialog Graphic Settings and select the text color, the layer on which the text will be placed, and the font size of the text.
