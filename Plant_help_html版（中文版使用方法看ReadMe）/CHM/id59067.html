Define Set-up Behavior in the Class of the Processing Stations
Step-by-Step Help > Modeling in 2D > Modeling the Flow of Materials, Basics > Producing and Processing Parts with a Work Plan > Create the Processing Stations > Define Set-up Behavior in the Class of the Processing Stations

To automatically set the stations up depending on the name of a user-defined attribute of the MU:

  • Select the check box Automatic.

  • Select User-defined Attribute from the drop-down list Set-up depends on. Enter the name of the user-defined attribute which we defined in the Sequence table with which the Source produces parts, compare Produce the Parts with a Source Using a Sequence Table. We typed in PartType.
