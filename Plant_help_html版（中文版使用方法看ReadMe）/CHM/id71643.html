_3D.unlockGraphicGroup
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Graphic Groups > _3D.unlockGraphicGroup

Syntax:

 
Path._3D.unlockGraphicGroup(Name:string)

The method _3D.unlockGraphicGroup unlocks the specified graphic group of the object designated by Path.

This clears the check box Locked on the Tab Graphics.

The method deactivates graphic inheritance.

Parameter

The parameter Name of data type string designates the name of the graphic group.

Example:

 
MyStation._3D.unlockGraphicGroup("MyGraphicGroup")
