Power up early
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Energy > State Transition > Unplanned > Power up early

To power the object up early during an unplanned phase so that the machine has the correct energy state at the end of that phase, select this. When you clear the check box, powering up starts at the end of the unplanned phase.

When you select the check box, the machine powers up/down while the machine is unplanned. Therefore the powering up/down time is part of the unplanned time.
