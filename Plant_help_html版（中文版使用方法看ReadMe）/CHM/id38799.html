Show units
2D Reference Help > Information Flow Objects > Variable > Dialog Window of the Variable > Tab Display > Show units

To also show the unit of the physical size of a Variable, select this. Select the unit under File > Model Settings/Preferences > Units. To hide the units, clear the check box.

For the data type randtime the check box Show units also sets if Plant Simulation shows the distribution parameters in the Frame window or not.

When you clear Show units, you can also enter the number of decimal places.
