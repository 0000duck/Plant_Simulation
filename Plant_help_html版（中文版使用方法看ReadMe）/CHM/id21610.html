Context Menu of a Graphic Group
2D Reference Help > The Plant Simulation Program Window > Class Library > Context Menu of the Selected Object in the Class Library > Show 3D Graphic Structure > Context Menu of a Graphic Group > Context Menu of a Graphic Group

The context menu of a graphic group in the structure window of the dialog Show 3D Graphic Structure provides these commands:

Visible

Rename

Internal

Copy

Locked

Paste

Generated

Delete

When you paste a graphic into a graphic group of a 3D object which inherits its graphics, 3D asks if you want to cut graphic inheritance for this object or not. 3D only pastes if you cut graphic inheritance by clicking Yes.
