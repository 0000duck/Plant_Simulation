Show states as colors if no state graphics are present
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > File Menu > Model Settings > 3D > Show states as colors if no state graphics are present

To show the external representation of simulation objects in different colors when no state group with individual state graphics for the different states exists, select the check box. Then 3D automatically colors the visible graphics of the object red for failed, blue for pausedor yellow for waiting, when the object is in the respective state during the simulation.

When you clear the check box, 3D immediately removes all state colors.
