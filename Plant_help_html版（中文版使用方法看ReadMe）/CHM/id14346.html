getLibraryVersionFromFile
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > getLibraryVersionFromFile

Syntax:

 
Path.getLibraryVersionFromFile(FilePath:string) → string

The function getLibraryVersionFromFile returns the version of the library designated by Path.

Parameter

The parameter FilePath of data type string has to be the path to a library file (.lib).

Return Value

The return value has the data type string.

Example:

 
print getLibraryVersionFromFile("C:\Plant Simulation 14\Libraries\Tools\Experiment.lib")
