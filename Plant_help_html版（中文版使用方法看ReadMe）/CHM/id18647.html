currentEventCtl
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > currentEventCtl

Syntax:

 
currentEventCtl → object

The functioncurrentEventCtl returns the EventController that controls the current simulation run.

Return Value

The return value has the data type object.

Example:

 
if currentEventCtl /= VOID
   print "simulation running in frame", currentEventCtl.location
   print "simulation running in frame", root
end
