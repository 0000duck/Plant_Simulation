omit
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Manipulating Strings > omit

Syntax:

 
omit(SourceText:string, Position:integer, NumberOfCharacters:integer) → string

The function omit copies a string and deletes a succession of characters from it.

Parameters

  • The parameter SourceText of data type string designates the string, which you want to copy.

  • The parameter Position of data type integer designates the position within the string from which on Plant Simulation deletes characters.

  • The parameter NumberOfCharacters of data type integer designates number of characters which Plant Simulation deletes.

Return Value

The return value has the data type string.

Example:

 
print omit("abcdef",3,2) -- returns "abef"
print omit("abcdef",0,3) -- returns "cdef" (0 letter is counted)
