ActiveX Events
Add-Ins Reference Help > ActiveX > Converting Data Types Between Plant Simulation and ActiveX > ActiveX Events

If an ActiveX control triggers an event, Plant Simulation starts the method that you entered into the text box Callback method. A parameter of data type integer containing the event ID is passed to this method.

A callback method might look like this: 

param event: integer, a,b,c,d: any
if event=1 
   print MyActiveX.PercentDone
end
