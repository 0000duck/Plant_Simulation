Insert Cylinder
3D Reference Help > 3D Reference > Edit Ribbon Tab > Insert Cylinder

The command Insert Cylinder creates and inserts a cylinder into the selected Frame.

  • Select the Graphic group to which you want to add the new graphic. The list shows the graphic groups which are defined for the object.

    • Default (external): Designates the standard graphic group of the 3D objects. It initially is external and represents the object towards the outside. When you deactivate Show External Graphics, 3D hides the graphic groups designated as Default.

    • One of the user-defined graphic groups you added on the Tab Graphics. Newly created 3D Frames by default provide the graphic group Deco (internal), which are intended for graphics for decorating the interior view of the Frame.

  • Enter the Radius and the Height of the cylinder which you want to create.

    A cylinder with shell, top cover, and bottom cover looks like this.

  • To create an open cylinder with the shell only, select Create shell.

  • To create the cylinder with shell and lid, select Create top cover.

  • To create the cylinder with shell and bottom, select Create bottom cover.

  • For special effect you can just create the lid or the bottom of the cylinder.

  • To create the cylinder, click Create. 3D attaches the shape to the mouse pointer. Drag the mouse to the position at which you would like to insert the shape and click the left mouse button. To cancel inserting, you can click the right mouse button or press the Esc key.

  • To move the cylinder to the location of your choice, click it and drag the mouse.

  • To change the size of the cylinder, select it and select 3D Properties > Tab Transformation. Then change the size under Scale.

  • To set if the cylinder is an obstacle for the worker, select it and then click Edit 3D Properties > Tab Graphic Settings on the Home ribbon tab.

  • To assign a material and a color, select the cylinder and click 3D Properties > Tab Material on the Home ribbon tab.
