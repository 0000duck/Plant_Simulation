Syntax:

 
Attribute_path.Type := string>

The sub-attribute Type sets the type of the probability distribution for the addressed attribute. Attribute_path designates an attribute of data type time or a user-defined attribute of data type randtime.

Example:

 
MyStation.proctime.Type := "Uniform"

The different distributions have different attributes. You can set these:

  • With the method setParam.

  • With the method setTypeAndAttr.

  • By direct assignments to the attribute.

    Example:

    		 
    MyStation.proctime.Mu := 0.50
MyStation.proctime.Sigma := 0.1

You can assign the following attributes/parameters to the individual distributions:
