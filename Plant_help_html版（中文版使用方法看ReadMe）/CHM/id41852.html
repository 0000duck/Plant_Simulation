Help on Debugger

The menu command Help on Debugger opens the help topic pertaining to the Method Debugger.

Help on Word

The menu command Help on Word or the F1 key open help for the word over which the input cursor is located. This way you can quickly and easily get help for an instruction, for a built-in attribute or for a built-in method.
