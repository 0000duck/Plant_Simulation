RouteWeightingAttr
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > RouteWeightingAttr

Syntax:

 
MU-path.RouteWeightingAttr

The attribute RouteWeightingAttr is used for controlling the Route weighting attribute, when you activate Automatic routing. The value of the attribute RouteWeightingAttr determines the name of user-defined attributes of Tracks and TwoLaneTracks. All lengths of the Tracks and TwoLaneTracks will then be multiplied with the value you assigned to the user-defined attribute. The user-defined attributes have to have the data type real. If you assign a negative value to the user-defined attribute, this Track or TwoLaneTrack will not be used for automatic routing. All Tracks and TwoLaneTracks, which do not have a user-defined attribute with the name designated by RouteWeightingAttr use the factor 1.0.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
.MUs.Transporter.RouteWeightingAttr :="MyWeightFactor"
Track5.MyWeightFactor := 0.7 -- preferably use this main road
Track7.MyWeightFactor := 1.5 -- try to avoid this side road
Track8.MyWeightFactor := -1 -- do not use this road
