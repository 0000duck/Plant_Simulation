drawText
2D Reference Help > Material Flow Objects > Frame > Methods of the Frame > drawText

Syntax:

 
Path.drawText(Layer:integer, X:integer, Y:integer, RGB:integer, FontSize:integer, Text:string)

The method drawText draws the text, which you enter as the parameter of data type string, on the background of the Frame designated by Path.

To draw the text in the foreground, i.e., in front of the objects in the Frame, enter a negative number for the layer, designated by the parameter integer.

Parameter

The parameter

Designates the

Layer of data type integer

the layer.

X of data type integer

the position of the text on the x-axis.

Y of data type integer

the position of the text on the y-axis.

RGB of data type integer

the color of the text.

FontSize of data type integer

the size of the text in pixels.

Text of data type string

the text proper that Plant Simulation draws.

Example:

 
current.drawText(4,48,15,80,18,"Forge in plant anytown, MyTown")
