getExitList
2D Reference Help > Material Flow Objects > FlowControl > Methods of the FlowControl > getExitList

Syntax:

 
Path.getExitList(ExitList:any)

The method getExitList returns the contents of the internal exit list of the FlowControl designated by Path for the Exit strategies Percentage or Cyclic sequence, and writes it to a list.

Parameter

The parameter ExitList of data type any designates the name of the list.

Example:

 
MyFlowControl.getExitList(cardFile)
MyFlowControl.setExitList(cardFile.copy)
