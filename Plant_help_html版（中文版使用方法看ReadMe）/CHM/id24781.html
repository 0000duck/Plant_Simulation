setTypeAndAttr
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Times and Distributions > Methods of the Distributions > setTypeAndAttr

Syntax:

 
Path.Attribute.setTypeAndAttr(Type:string,SetOfParameters:string)

The method setTypeAndAttr sets the type of the distribution and assigns a complete set of parameters to the distribution.

Parameters

  • The parameter Type of data type string designates the type of the distribution.

    Distribution

    Name in Plant Simulation English/German

    Constant number

    Const/Konst

    Uniform distribution

    Uniform/Gleich

    Normal distribution

    Normal/Normal

    Lognormal distribution

    Lognorm/Lognorm

    Negative exponential distribution

    Negexp/Negexp

    Geometric distribution

    Geom/Geom

    Hypergeometric distribution

    Hypgeo/Hypgeo

    Erlang-distribution

    Erlang/Erlang

    Weibull-distribution

    Weibull/Weibull

    Triangular distribution

    Triangle/Dreieck

    Binomial distribution

    Binomial/Binomial

    Poisson-distribution

    Poisson/Poisson

    Gamma distribution

    Gamma/Gamma

    Beta distribution

    Beta/Beta

    Discrete Empirical (dEmp) distribution

    dEmp/dEmp

    Continuous Empirical (cEmp) distribution

    cEmp/cEmp

    Primitive Empirical (Emp) distribution

    Emp/Emp

    Formula

    Formula/Formel

    Type-dependent distribution

    List (Type) /Liste(Typ)

    List-dependent distribution ( ParallelStation )

    List (Place) /Liste(Platz)

    Matrix distribution for setting-up objects

    Matrix(Type) /Matrix(Typ)

  • The parameter SetOfParameters of data type string designates the set of parameters.

Return Value

The return value has the data type boolean.

Example:

 
MyStation.proctime.setTypeAndAttr("Normal", "30,10")
MyStation.proctime.setTypeAndAttr("Normal", "30,10,1,100")
MyStation.setuptime.setTypeAndAttr("Formula", "Variable1/20")
MyGenerator.duration.setTypeAndAttr("dEmp", "TableEmp")

Example:

 
var type: string := Station.ProcTime.type
var SetOfParameters: string := Station.ProcTime.getAttr
StationNew.ProcTime.setTypeAndAttr(type, SetOfParameters)

The method getAttr returns the set of parameters in the format that the method setTypeAndAttr understands.
