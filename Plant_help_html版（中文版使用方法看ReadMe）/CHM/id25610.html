openDialog
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Accessing Lists and Tables > openDialog

Syntax:

 
Path.openDialog([CallOpenControl:boolean]) → boolean

The method openDialog opens the window of the object designated by Path.

The methodapplies to the objects StackFile, QueueFile, CardFile, TableFile, and TimeSequence, as well as to sub-tables and to user-defined attribute tables.

Parameter

The optional parameter CallOpenControl of data type boolean sets if the open control will be executed (true) or not (false, which is the default.

Return Value

The return value has the data type boolean.

Example:

 
MyTableFile.openDialog
MyStackfile.openDialog(true)
