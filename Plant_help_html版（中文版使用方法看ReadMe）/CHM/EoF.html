EoF
2D Reference Help > Information Flow Objects > FileInterface > Read-Only Attributes of the FileInterface > EoF

Syntax:

 
Path.EoF

The read-only attribute EoF (end of file) returns true, if the file designated by Path is still closed, as Plant Simulation cannot process data. After you opened the file with the method open, it returns false as long as lines may still be read in with the method readLn.

Return Value

The return value has the data type boolean.

Example:

 
MyFileInterface.open("C:\temp\data.txt")
while not MyFileInterface.EoF 
   print MyFileInterface.readLn
end
MyFileInterface.close
