Methods for Accessing Lists and Tables
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Methods of Lists and Tables > Methods for Accessing Lists and Tables > Methods for Accessing Lists and Tables

Lists and tables provide the methods closeDialog, copy, delete, insertList, interSection, openDialog, openDialogBox, printList, refillDialog, and showPrintDialog for accessing them.

Read and write access depend on the object class and are described in the sub-chapters.

To view all of the methods and attributesof the list object, open the window Show Attributes and Methods. The figure below illustrates the information using the example of the object Source.

  • Select Show Attributes and Methods on the context menu of the Class Library to show the attributes and methods of the selected Class.

  • Press the F8 key or click Show Attributes and Methods on the Home ribbon tab of the Frame into which you inserted an instance to show the attributes and methods of the selected Instance.
