XGridlines
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > XGridlines

Syntax:

 
Path.XGridlines

The attribute XGridlines sets the number of grid lines that the Chart shows on the x-axis. The Chart also shows a label for each grid line on the x-axis. When you are displaying a large number of data sets, it makes sense to enter a smaller number so that the labels can be shown completely.

When the number of grid lines is greater than the number of data sets on the x-axis, the Chart only shows as many grid lines as there are data sets. When the number of grid lines is smaller than the number of data sets, the Chart reduces the number of grid lines and labels.

When you select the Category > XY-Graph, the Chart only evaluates the number, when you also enter a fixed Range X, i.e., when you did not enter an asterisk *.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
MyChart.XGridlines := 10
