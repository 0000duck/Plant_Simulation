calculateList
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Miscellaneous Methods of the TableFile > calculateList

Syntax:

 
Path.calculateList([Range:listrange])

The method calculateList recomputes formulas of the TableFile designated by Path.

Parameter

The parameter Range of data type listrange designates the range of cells for which the formulas are computed.

Example:

 
table1.calculateList := false -- deactivates calculation
table1.calculateList({2,*},{3,3}) -- computes column 2, cell 3,3
