Callback argument
2D Reference Help > Display and User Interface Objects > Dialog > Dialog Window of the Object Dialog > Tab Elements > Defining Properties of a Dialog Item > Callback argument

For a dialog item of Type New Button, New Drop-Down List Box, New Check Box, New Radio Button, New Edit Text Box, New List Box, New List View, New Tab Control, or New Menu/New Menu Command the Dialog shows the text box Callback parameter.

Enter the parameter that is passed to the callback method. The Callback method executes this parameter when the user performs the respective action. Be aware that the parameters are case-sensitive!

  • Closes the drop-down list box.

  • Selects and double-clicks an item in the list box.

  • Changes the contents of the Text Box and selects another dialog item afterwards, clicks in another text box, or clicks OK, Apply or Cancel.

  • Clicks the button.

  • Selects or clears the check box.

  • Selects the radio button.

  • Selects a row in the list view and double-clicks it.

  • Selects a tab in the tab control.

  • Selects a menu/menu command.
