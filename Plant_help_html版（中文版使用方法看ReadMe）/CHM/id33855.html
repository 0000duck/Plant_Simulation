Display a TableFile or a List Object
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Select an Object or an Object Attribute to be Displayed > Display a TableFile or a List Object

To display a TableFile or a list object (CardFile or QueueFile and StackFile) in table form on the HTML page, click on the toolbar .

Select the TableFile/List whose values you would like to show. Click OK.

Enter the caption which the HtmlReport shows below the TableFile/List.

Enter the identifiers of the columns of the TableFile/List which the HtmlReport shows on the HTML page.

Instead, you can also type in the instructions that correspond to the settings you can enter into the dialog.

[ThroughputList] -- our TableFile is called ThroughputList
[ThroughputList, "Tab. 1: Mean value of the throughput times"]
[ThroughputList, 0, 2] -- columns 0 and 2
[ThroughputList, *..4, 7] -- all columns until column 4 and column 7
[ThroughputList, 0, 2..4, 7..*]
[ThroughputList, 0, 2..4, 7, #MyColumnName] -- columns 0, 2 to 4, and column 7, and the columns named MyColumnName

Access cells of lists and tables to add their content to the HtmlReport

The syntax for accessing cells of lists and tables to add their content to the HtmlReport is similar to the syntax used in formulas: 

[MyTableFile[Column, Row]]

You can specify the Column or Row with its number or with its index (if available). The text in the column or row index has to be surrounded by quotation marks and has to be started with the number sign #: 

[MyTableFile[1, 1]]
[MyTableFile["Name", 1]]
[MyTableFile["Name", "Part type 1"]]
[MyTableFile[#Name, 1]]
[MyTableFile[#Name, #Part type 1]]

You have to protect characters within the text of the column index, that have a special meaning in the HtmlReport: 

[MyTableFile["\"Name\"", 1]
[MyTableFile["Speed\, km/h", 1]]
[MyTableFile[#Speed\, km/h, 1]]
