2D Reference Help > Display and User Interface Objects > Display > Dialog Window of the Display > Tab Display > Type

Select how the Display displays the data it records.

  • Text mode shows the value as text.

  • Bar and Pie mode map the value to a range of values. They visualize values inside the range by the height of a bar or a filled segment of the pie. An empty bar/pie corresponds to values below or equal to the lower bound. A full bar/pie represents values above or equal to the upper bound.

    Plant Simulation converts the current value to the range, whose bounds you enter and shows it averaged.
