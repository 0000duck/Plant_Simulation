DestinationListRow
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > DestinationListRow

Syntax:

 
MU-path.DestinationListRow

When you assigned true to the attribute AutomaticRouting and when the attribute Destination points to a TableFile or a CardFile respectively, the attribute DestinationListRow points to the Destination to which the Transporter drives next. This means that the Transporter next drives to the object, which is located in column 1 of the row with the number DestinationListRow. When the Transporter has reached the destination, Plant Simulation automatically increments the value of the attribute by 1. When the end of the column is reached, the attribute is reset to the value 1.

For tugger trains this applies: When the first Transporter of the tugger train in the direction of motion has reached the destination, Plant Simulation increments the value of the attribute by 1. When the tugger train drives backwards, the first Transporter in the direction of motion is the last Transporter.

Data Type of the Value You Can Assign

You can assign a value of data type integer.

Example:

 
@.DestinationListRow :=1
print@.Destination[1, @.DestinationListRow]
