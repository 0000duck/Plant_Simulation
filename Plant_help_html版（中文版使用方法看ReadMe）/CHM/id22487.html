[ ] (read the contents of a cell)
2D Reference Help > Information Flow Objects > TableFile > Methods of the TableFile > Methods for Accessing the TableFile > [ ] (read the contents of a cell)

Syntax:

 
Path[Column:integer or string, Row:integer or string] → contents of the cell

The method print[,] returns the data of the designated cell of the TableFile designated by Path.

When you use a string as the column/row index, the string is case-insensitive, for example when you print it with the method TableFile["my text", "aa bb"].

If the table contains a large number of entries, you might want to activate Fast index access.

Parameters

  • The parameter Column of data type integer or string designates the column.

  • The parameter Row of data type integer or string designates the row.

Return Value

The return value is the contents of the cell. If the cell is Empty, Plant Simulation returns a zero value. For numerical data types this is the number 0, for the data types Boolean it is false, for strings it is the empty string "", and for the data types object, table, list, stack, and queue it is VOID.

Example:

 
print MyTableFile[1,2]
print MyTableFile["Value",2]
