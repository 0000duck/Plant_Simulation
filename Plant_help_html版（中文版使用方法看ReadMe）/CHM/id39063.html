Pause
2D Reference Help > Resource Objects > ShiftCalendar > Read-Only Attributes of the ShiftCalendar > Pause

Syntax:

 
Path.Pause

The read-only attribute Pause returns, if the ShiftCalendar designated by Path is Paused (true) or not (false). The attribute is observable.

Return Value

The return value has the data type boolean.

Example:

 
print MyShiftCalendar.Pause
