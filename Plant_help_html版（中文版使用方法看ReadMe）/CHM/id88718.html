Display Statistics Values of an Object as a Table in the Report
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Select an Object or an Object Attribute to be Displayed > Display Statistics Values of an Object as a Table in the Report

To display statistics values of an object as a table on the HTML page, click on the toolbar .

Select the object for which you would like to show statistics values. Click OK.

Enter the caption which the HtmlReport shows below the statistics table on the HTML page.

Select the type of statistics table you want to show for the object. The different object types provide different table types. If you do not enter anything into the text box List columns, the HtmlReport shows all portions.

The settings above result in this source code:

[Wareneingang, "Zustandsstatistik der Quelle 'Wareneingang'", %Stati]??

The result might look like this:

To only show certain colums, enter their column index, separated by commas, into the text box List columns.

The settings above result in this source code:

[Wareneingang, "Zustandsstatistik der Quelle 'Wareneingang'", %Stati, 4,6]??

If the selected object collected statistics values, the HtmlReport shows values in a table. If the object did not collect statistics, the Report naturally cannot show any data.

When the HtmlReport lists several objects one below the other, it aggregates their values if possible, i.e., it lists their values row-by-row in a table. If the HtmlReport cannot aggregate the values, it shows the non-aggregated values and starts a new table. If you want to show instances of a class object, the HtmlReport lists the values of all instances row-by-row.

Instead of selecting the statistics table type in the dialog and entering the caption, you can also type the instructions into the tab Contents of the dialog of the HtmlReport: 

[Object path, caption, statistics table type]

You might, for example, enter:

[Receiving, "State statistics of the Source 'Receiving'", %States]
