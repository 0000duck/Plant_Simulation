pop
2D Reference Help > Information Flow Objects > QueueFile and StackFile > Methods of the QueueFile and the StackFile > pop

Syntax:

 
Path.pop

The method pop removes a cell from a list with one column. StackFile and QueueFile cut the entries according to their built-in properties.

Parameter

The data type of the return value matches the data type of the StackFile or the QueueFile.

Example:

 
value := MyStackFile.pop
value := MyQueueFile.pop
