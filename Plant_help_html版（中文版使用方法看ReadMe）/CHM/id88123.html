_3D.LockedGraphics
SimTalk Reference > 3D-related SimTalk Functions > Built-in Attributes and Read-Only Attributes for Accessing Graphic Groups > _3D.LockedGraphics

Syntax:

 
Path._3D.LockedGraphics

The attribute _3D.LockedGraphics represents the list of all locked graphic groups. This then prevents you from selecting this graphic group in the 3D windows which are opened for this object. This makes, for example, sense for the graphic of a factory building which might otherwise be selected inadvertently.

Data Type of the Value You Can Assign

You can assign a value of data type string.

Example:

 
.Models.Frame.Station1._3D.LockedGraphics := ["Group1", Group2"]
var a: string[] := .Models.Frame.Station1._3D.LockedGraphics
