Entry length
2D Reference Help > Material Flow Objects > AngularConverter > Dialog Window of the AngularConverter > Tab Attributes > Entry length

Enter the length of the first leg of the AngularConverter. It covers the distance from the entry point to the point at which the AngularConverter switches the conveying direction.

When Transfer length on the Tab Curve is active, the program adapts the graphic of the AngularConverter, when you change the entry length. When the length cannot be exactly displayed in pixels, the program adapts the length of the graphic. If you need to use the exact length, clear transfer length.
