StatisticsActive
2D Reference Help > Information Flow Objects > Variable > Attributes of the Variable > StatisticsActive

Syntax:

 
&Variable.StatisticsActive

The attribute StatisticsActive activates if the Variable collects statistics values (true) or if it does not collect them (false). Note that the Variable only collects statistics for the data types string and integer.

Data Type of the Value You Can Assign

You can assign a value of data typeboolean.

Example:

 
&MyVariable.StatisticsActive := true
