addItem
Add-Ins Reference Help > Inter-Process Communication Interfaces > OPCClassic > Methods of the OPC Interface > addItem

Syntax:

 
Path.addItem(Item:string) → boolean

The method addItem defines access to the item designated by the argument string in the PLC.

Parameter

The parameter Item of data type string designates the item in the PLC.

Return Value

The return value has the data type boolean.

Example:

 
opcinterface.addItem("Random.INT4")
