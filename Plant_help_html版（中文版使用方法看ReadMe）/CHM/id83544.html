resetProfile
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > resetProfile

Syntax:

 
resetProfile

The function resetProfile deletes all data collected by the Profiler up until you reset the model. Call this function before activating the Profiler to receive valid data for the present simulation run only.

Example:

 
resetProfile
