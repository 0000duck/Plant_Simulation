num_to_speed
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Values without Units into Physical Data Types with Units > num_to_speed

Syntax:

 
num_to_speed(real) → speed

The function num_to_speed converts the number without a unit into a value of data type speed.

Parameter

The parameter Number of data type real designates the number.

Return Value

The return value has the data type speed.

Example:

 
var a: speed
a := num_to_speed(4.1) / Conveyor2.Time
