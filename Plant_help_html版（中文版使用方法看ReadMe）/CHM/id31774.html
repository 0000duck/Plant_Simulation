StatUnplannedTime
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Read-Only Attributes for Statistics of the Material Flow Objects > StatUnplannedTime

Syntax:

 
Path.StatUnplannedTime → time

The read-only attribute StatUnplannedTime returns the total time during which the object designated by Path was not planned to work.

Return Value

The return value has the data type time.

Example:

 
print ParallelStation.StatUnplannedTime
