Add User-defined Charts to the Report
2D Reference Help > Tools > ExperimentManager > Viewing Evaluations > View Output Values of Interest > Add User-defined Charts to the Report

If you want to show charts with special evaluations for each experiment in the report, you can drag these chart objects onto the ExperimentManager. This opens a table with all charts that are registered for this purpose. Delete the charts which you do not want to show in the Report from this table. You can also open the table by selecting Tools > Charts.

Registered Chart objects

As the Report uses the labels of the charts to display them, the different charts should have different labels.

Report of an experiment study with two user-defined charts

If you would like the ExperimentManager to save the model files after each experiment, we recommend to not include charts in the report as the user-defined graphics greatly increase the size of the model file.

When you use a distributed simulation, you cannot add user-defined charts to the Report.
