BatCharging
2D Reference Help > Mobile Objects > Transporter > Attributes of the Transporter > BatCharging

Syntax:

 
MU-path.BatCharging

The attribute BatCharging sets if the battery of the Transporter designated by MU-path will be recharged (true) or not (false).

Data Type of the Value You Can Assign

You can assign a value of data type boolean.

Example:

 
.Station.Cont.BatCharging :=true -- starts charging

Failures or pauses do not affect charging.
