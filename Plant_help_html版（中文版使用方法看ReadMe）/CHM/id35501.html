Test path
2D Reference Help > The Plant Simulation Program Window > Ribbon Bar > Home Ribbon Tab > Edit 3D Properties > Tab MU Animation > Test path > Test path

The group box Test path provides settings and actions for testing the MU animation path which exists in the scene. When you start testing, 3D creates a temporary test object which moves on the selected animation path.

You can:

  • Select the Object, which represents the test object.

  • Enter the Factor with which the test object moves on the animation path.

  • Select if the test object moves in the forward direction or if it moves Backwards.

  • Play the animation with the settings you selected.

  • Pause the animation.

  • Stop the animation.

    3D does not save these settings to the model file.
