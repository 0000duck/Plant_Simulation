Use Metric Prefix
2D Reference Help > Display and User Interface Objects > Chart > Dialog Window of the Chart > The Tools Menu > Use Metric Prefix

The menu command Use Metric Prefix activates or deactivates if the Chart shows a unit prefix on the y-axis for very great or very small values, for example 100K instead of 100000.

The prefix

stands for

p

pico

n

nano

u

micro

m

milli

K

kilo

M

million

B

billion

T

trillion
