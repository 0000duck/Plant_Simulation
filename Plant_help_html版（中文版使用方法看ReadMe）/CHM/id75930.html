Font size
2D Reference Help > Information Flow Objects > Shared Properties of Lists and Tables > Window of Lists and Tables > List Ribbon Tab > Edit Format > Tab Settings > Font size

Select the font size for the text you enter into the cells of the column(s) or into the cell(s) you selected: Small, Medium/Normal, Large, or Extra Large/Huge.

The list automatically adjusts the width and the height of the cells to the font size you selected.
