Class name
Add-Ins Reference Help > ActiveX > Dialog Window of the Object ActiveX > Tab Attributes > Class name

Enter the class name of the required ActiveX control. The command Tools > ActiveX Class Overview opens a table that shows all ActiveX controls, which are installed on your computer.

You might, for example, enter excel.application, for Microsoft Excel, word.application, for Microsoft Word, or access.application for Microsoft Access.
