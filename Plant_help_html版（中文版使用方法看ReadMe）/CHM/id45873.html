Recording the Results of a Simulation Study
2D Reference Help > Tools > ExperimentManager > Recording the Results of a Simulation Study

On the tab Report you can enter a description of the project and designate folders into which you want to save the report and the simulation models.

For documenting and archiving the reports you can enter important information about the project on the tab Report. The report shows this information along the top of the page. After an experiment run the ExperimentManager opens the report showing all results. When you select the check box Save report, the ExperimentManager saves this report to the Folder whose name you enter. Compare Show the Results in a Report.

If you would like the ExperimentManager to save the model files after each experiment, we advise against including charts in the report as the user-defined graphics greatly increase the size of the model file.

If you would like to start several simulation studies one after the other with a BAT file, compare Entering Start Options, you have to prevent Plant Simulation from opening any modal dialogs which require user actions. To prevent the program from showing message boxes, select the respective check box. In addition you might not want to open reports after such a simulation run.

saveExcelAtEnd

Saves the table Results as an Excel file after the study is finished.

If you do not want to show the message asking to save the model befire executing distributed simulations, you have to set the user-defined attribute AskForSaveModel to false.

Within tables the ExperimentManager adjusts the column width to the length of the expression you enter as the column index. This way, the program always shows the entire column index. When displaying output tables with many columns it might be helpful to set a constant column width. To do so, change the value of the user-defined attribute columnWidth_standard from the default value 0 to the column width of your choice. To change the values of these attributes, click the ExperimentManager in the Frame into which you inserted it and press the F8 key.

If you would like the ExperimentManager to inform you when a time-consuming simulation study is finished, select the check box Send e-mail. Then enter your e-mail address and the name of your e-mail server.
