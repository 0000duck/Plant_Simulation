append
SimTalk Reference > General SimTalk Functions > Data Types > Array > Methods and Attributes of Arrays > append

Syntax:

 
append(Value:any)

The method append appends the value designated by the parameter of data type any to the end of a one-dimensional array that does not have a fixed size.

Parameter

The parameter Value of data type any designates the value that you want to append.

Example:

 
var a: string[]
a.append("string")
a.append("string")
print a[2]
