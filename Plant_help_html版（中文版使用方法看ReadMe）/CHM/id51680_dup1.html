Min. num. in
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Exit > Strategy > Min. num. in

To move the MUs on to the successor that received the smallest number of MUs, select Min. num. in from the drop-down list.

This only works correctly, when resource statistics of the successor or of the predecessor is active

  • When you select Blocking , the object moves the MU to the next successor.

  • When you clear Blocking , the object moves the MU to the next successor, which can receive the MU.
