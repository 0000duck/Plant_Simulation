Rework
Add-Ins Reference Help > Object Libraries > Value Stream Mapping > Value Stream Mapping Objects > Rework > Rework

The object Rework represents a reworking station within your value stream model. Products, which have been marked as defective, will be repaired here and be fed back into the value stream. Be aware that the name of the product might change during this process.

The arriving defective part will be repaired according to the required repair time. This time will be determined according to a constant time located between the minimum reworking time and the maximum reworking time.
