_3D.isGraphicGroupLocked
SimTalk Reference > 3D-related SimTalk Functions > Built-in Methods for Accessing Graphic Groups > _3D.isGraphicGroupLocked

Syntax:

 
Path._3D.isGraphicGroupLocked(Name:string) → boolean

The method _3D.isGraphicGroupLocked returns if the specified graphic group of the object designated by Path is locked (true) or not (false). This then prevents you from selecting this graphic group in the 3D windows which are opened for this object. This makes, for example, sense for the graphic of a factory building which might otherwise be selected inadvertently.

Parameter

The parameter Name of data type string designates the name of the graphic group.

Return Value

The return value has the data type boolean.

Example:

 
print MyStation._3D.isGraphicGroupLocked("MyGraphicGroup")
