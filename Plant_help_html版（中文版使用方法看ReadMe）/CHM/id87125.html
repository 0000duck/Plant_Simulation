MULength
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes of All Mobile Objects > MULength

Syntax:

 
MU-path.MULength

The attribute MULength sets the length of the MU designated by MU-path.

Only the length-oriented objects use the MULength to determine how many MUs they can hold, when the ConveyingDirection (Conveying direction) is forward or backward.

Plant Simulation interprets the length in meters. Only at output time, for example with the method print, does Plant Simulation convert the value to the unit you selected under File > Model Settings/Preferences > Units > Length.

Data Type of the Value You Can Assign

You can assign a value of data type length.

Example:

 
.MUs.Part.MULength := 2.5 -- 2.5 meters
