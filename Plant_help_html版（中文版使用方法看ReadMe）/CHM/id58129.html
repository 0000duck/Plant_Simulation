applicationVersion
SimTalk Reference > General SimTalk Functions > Predefined Functions > Miscellaneous Global Functions > applicationVersion

Syntax:

 
applicationVersion

The function applicationVersion returns the coded version number of Plant Simulation.

Return Value

The return value of data type integer is composed as follows: VVMMMPPP

  • VV is the major version number.

  • MMM is the minor version number.

  • PPP is the number of the patch level.

Example:

 
print applicationVersion -- might return 14001002, i.e. version 14.1.2
