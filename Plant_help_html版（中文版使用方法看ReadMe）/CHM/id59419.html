StatEnergyOffConsumption
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Read-Only Attributes of the Material Flow Objects > Read-Only Attributes for Energy Statistics of the Material Flow Objects > StatEnergyOffConsumption

Syntax:

 
Path.StatEnergyOffConsumption → real

The read-only attribute StatEnergyOffConsumption returns the total energy consumption during which the object designated by Path was off.

Return Value

The return value has the data type real.

Example:

 
print MyStation.StatEnergyOffConsumption
