StatDeleted
2D Reference Help > Material Flow Objects > Drain > Read-Only Attributes of the Drain > StatDeleted

Syntax:

 
Path.StatDeleted

The read-only attribute StatDeleted returns the number of MUs which the Drain designated by Path removed from the plant.

Return Value

The return value has the data type integer.

Example:

 
print MyDrain.StatDeleted
