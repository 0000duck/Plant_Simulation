BookPntHRelative
2D Reference Help > Mobile Objects > Shared Properties of the Mobile Objects > Attributes of All Mobile Objects > BookPntHRelative

Syntax:

 
MU-path.BookPntHRelative

The attribute BookPntHRelative sets the relative position of the booking point of the MU designated by MU-path in the z-dimension. You can enter a value between 0 and 1.

Data Type of the Value You Can Assign

You can assign a value of data type real.

Example:

 
.MUs.Transporter.Length := 4.5  -- meters
.MUs.Transporter.BookPnt := 0.1 -- meters

A long MU may be located on several objects at the same time!
