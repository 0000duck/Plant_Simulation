Height
2D Reference Help > Display and User Interface Objects > Chart > Attributes of the Chart > Height

Syntax:

 
Path.Height

The attribute Height sets the height of the viewing angle for 3D Chart types.

Data Type of the Value You Can Assign

You can assign a value of data type integer. You can enter a number between -50 and 50.

Example:

 
MyChart.ChartType := "3D Columns"
MyChart.Height := 45
