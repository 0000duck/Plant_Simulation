Display a Chart
2D Reference Help > Display and User Interface Objects > HtmlReport > Dialog Window of the HtmlReport > Tab Content > Select an Object or an Object Attribute to be Displayed > Display a Chart

To display the diagram picture of a Chart on the HTML page, click on the toolbar .

Select the Chart which you would like to show. Click OK.

Enter the caption which the HtmlReport shows below the Chart.

Enter the Image width with which the HtmlReport shows the picture of the object. You can also enter an asterisk (*) for either the width or the height to ensure that the image will not be distorted.

Enter the image height with which the HtmlReport shows the picture of the object. You can also enter an asterisk (*) for either the width or the height to ensure that the image will not be distorted.

Instead, you can also type in the instructions that correspond to the settings you can enter into the dialog.

[StationChart] -- our Chart is called StationChart
[StationChart, 80, 60]
[StationChart, "This is my caption"]
[StationChart, "This is my caption", 80, 60]
