timeRepresenation
SimTalk Reference > General SimTalk Functions > Predefined Functions > Functions for Converting Data Types > Functions for Converting Times and Dates > timeRepresenation

Syntax:

 
timeRepresentation

The function timeRepresentation returns the preferences you selected for the time scale under File > Model Settings/Preferences > Units > Time scale. The return value is a CardFile of data type real. The first entry contains the conversion factor for the time. Note that you can only enter the denominator. The conversion factor is the reciprocal value. The following three entries are the values taken from File > Model Settings/Preferences > Units > Time Scale. They are normally represented by 60, 60 and 24.

Example:

 
var l: list[real]
var timeDivision: real
var SecondCarriedOver, HourCarriedOver, DayCarriedOver : real
l := timeRepresentation
timeDivision := l.read(1)
secondCarriedOver := l.read(2)
hourCarriedOver := l.read(3)
dayCarriedOver := l.read(4)
