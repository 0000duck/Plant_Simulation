Release all services when one of them is failed
2D Reference Help > Material Flow Objects > Shared Properties of the Material Flow Objects > Dialog Items of the Objects > Tab Importer > Sub-tab Failure > Release all services when one of them is failed

To make the material flow object release all repair services, when one or more of the services is failed, paused, or interrupted, select All. To only release the failed services, clear it.

This only works, when you also select Fail services of the Exporter.

failImp.ReleaseAllServices
